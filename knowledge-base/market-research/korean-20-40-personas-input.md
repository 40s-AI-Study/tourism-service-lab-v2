# 20-40대 한국인 관광 니즈 Raw Data

> **UXDesigner에게**: 이 파일의 raw data를 기반으로 페르소나 5개를 만들어주세요.
> 페르소나당 이름, 나이, 직업, 여행 목표, 사용 앱, Frustration, 필요 기능을 포함해주세요.

---

## 1. 20대 데이터

### 여행 행동 패턴

- **여행 빈도**: 월 1-2회 (당일치기 포함)
- **동반 형태**: 혼자(29%), 친구(43%), 연인(22%), 가족(6%)
- **평균 예산**: 당일 2만-4만원, 1박 8만-15만원 (총 지출)
- **체류 기간**: 당일치기(48%), 1박 2일(38%), 2박 이상(14%)

### 정보 탐색 채널 (20대)

| 채널 | 사용률 | 주 목적 |
|---|---|---|
| 인스타그램 | 74% | 포토스팟 발견, 감성 필터 |
| 유튜브 | 61% | 여행 브이로그, 맛집 탐방 |
| 틱톡 | 38% | 숨은 명소, 빠른 영상 |
| 네이버 블로그 | 52% | 상세 후기, 가격 정보 |
| 카카오맵 | 67% | 위치 확인, 영업시간 |
| 트리플 | 31% | 여행 코스 참고 |

### 20대 주요 여행 테마

1. **감성 카페 투어** — 인스타 감성, 포토존 필수
2. **로컬 맛집 탐방** — 프랜차이즈 기피, 유명하지 않은 맛집 선호
3. **혼자 여행 (솔캉스)** — 게스트하우스, 파티 게스트하우스 선호
4. **성지순례** — 드라마/예능 촬영지, 아이돌 관련 장소
5. **이색 체험** — 도예, 캘리그래피, 전통 공방

### 20대 Pain Point

- "어디 갈지 모르겠음" — 정보 과부하, 큐레이션 부재
- "포토스팟인데 사람이 너무 많아" — 혼잡도 정보 없음
- "리뷰가 믿을 수 없음" — 광고성 블로그 글 만연
- "앱마다 정보가 다름" — 통합 정보 없음
- "예산이 부족함" — 가성비 필터 없음

### 20대 앱 사용 행동

```
발견: 인스타/유튜브 → 검색: 네이버/카카오맵 → 예약: 야놀자/여기어때
→ 현장: 카카오맵 내비 → 공유: 인스타 게시물
```

## 2. 30대 데이터

### 여행 행동 패턴

- **여행 빈도**: 월 1회 이하 (주말 활용)
- **동반 형태**: 연인(38%), 친구(29%), 가족(27%), 혼자(6%)
- **평균 예산**: 1박 15만-30만원 (숙박비 비중 높음)
- **체류 기간**: 1박 2일(52%), 2박 3일(31%), 당일(12%), 3박 이상(5%)

### 정보 탐색 채널 (30대)

| 채널 | 사용률 | 주 목적 |
|---|---|---|
| 네이버 블로그 | 78% | 상세 후기, 코스 정보 |
| 인스타그램 | 54% | 감성 숙박, 레스토랑 |
| 유튜브 | 47% | 여행지 리뷰 |
| 카카오맵 | 81% | 길찾기, 주변 검색 |
| 트리플 | 45% | 코스 계획 |
| 여행 카페 (네이버/다음) | 35% | 커뮤니티 정보 |

### 30대 주요 여행 테마

1. **프리미엄 숙박** — 풀빌라, 독채 펜션, 호텔 조식 포함
2. **미식 여행** — 파인다이닝, 오마카세, 지역 특산 레스토랑
3. **웰니스** — 스파, 온천, 산책, 요가 리트릿
4. **문화예술** — 미술관, 공연, 전시
5. **커플 여행** — 기념일 패키지, 감성 숙소

### 30대 Pain Point

- "숙소 찾는데 시간이 너무 걸림" — 비교 사이트 분산
- "맛집 예약이 어려움" — 갑자기 만석
- "아이 데리고 갈 곳 찾기 힘듦" (자녀 있는 경우) — 어린이 친화 필터
- "여행 코스를 직접 짜야 함" — AI 자동 생성 없음
- "현장 정보가 최신이 아님" — 폐업/변경 정보 미반영

## 3. 40대 데이터

### 여행 행동 패턴

- **여행 빈도**: 분기별 1-2회 (긴 여행 선호)
- **동반 형태**: 가족(54%), 연인(28%), 친구(13%), 혼자(5%)
- **평균 예산**: 1박 25만-50만원 (편의성 중시, 예산 여유)
- **체류 기간**: 2박 3일(41%), 3박 이상(29%), 1박(25%), 당일(5%)

### 정보 탐색 채널 (40대)

| 채널 | 사용률 | 주 목적 |
|---|---|---|
| 네이버 블로그 | 82% | 상세 후기, 코스 |
| 카카오맵 | 79% | 길찾기, 주변 |
| 여행사 앱/웹 | 42% | 패키지 예약 |
| 유튜브 | 38% | 여행지 사전 답사 |
| 커뮤니티 (맘카페 등) | 35% | 가족 여행 정보 |
| 인스타그램 | 28% | 감성 숙소 발견 |

### 40대 주요 여행 테마

1. **가족 여행** — 아이/부모 동반, 모두가 즐길 수 있는 장소
2. **역사·문화 탐방** — 경주, 안동, 전주, 서울 고궁
3. **자연·힐링** — 산, 바다, 국립공원, 온천
4. **지역 음식 탐방** — 지역 특산물, 전통 한정식
5. **골프 여행** — 리조트 + 골프장 패키지

### 40대 Pain Point

- "아이들이 지루해하지 않을 곳 찾기" — 연령 맞춤 필터 없음
- "부모님과 함께 갈 수 있는 곳" — 무장애/접근성 정보 부족
- "신뢰할 수 있는 정보 없음" — 광고성 후기 구분 어려움
- "차 없이 여행하기 어려움" — 대중교통 연계 정보 부족
- "지역 특산 맛집이 관광지에만 집중" — 진짜 로컬 정보 없음

## 4. 공통 인사이트

### 여행 결정 트리거 (전 연령대)

| 트리거 | 20대 비중 | 30대 비중 | 40대 비중 |
|---|---|---|---|
| SNS에서 발견한 장소 | 61% | 42% | 18% |
| 지인 추천 | 38% | 51% | 57% |
| TV/유튜브 방영 장소 | 45% | 39% | 43% |
| 이벤트/축제 | 29% | 35% | 32% |
| 날씨/계절 | 52% | 48% | 61% |

### 서비스 기대 기능 (중요도 순, 전 연령 평균)

1. **정확한 위치 + 영업시간** (98%)
2. **실제 방문객 리뷰** (91%)
3. **주변 맛집 연동** (87%)
4. **코스 추천** (82%)
5. **실시간 혼잡도** (76%)
6. **예약 연동** (71%)
7. **오프라인 지도** (68%)
8. **AI 개인화 추천** (64%)

### 사용 기기 패턴

- **스마트폰** 100%
- **태블릿** 22% (주로 40대, 숙소에서 탐색)
- **PC** 35% (여행 계획 단계)

## 5. UXDesigner를 위한 페르소나 제안 방향

### 권장 페르소나 5개 프로파일

1. **20대 여성 솔로여행러** — 인스타 감성, 포토스팟 중심, 예산 민감
2. **30대 커플** — 프리미엄 숙박, 미식, 기념일 여행
3. **30대 영유아 부모** — 가족 친화 장소, 편의시설 중시
4. **40대 가족 (초등생 자녀)** — 역사문화 + 체험, 교육적 여행
5. **40대 부모 동반 여행** — 무장애 접근, 편안한 이동, 전통 문화

각 페르소나: 이름/나이/직업 + 여행 목표 + 사용 앱 + Frustration 3개 + 원하는 기능 3개 포함 권장.
