# 한국 관광 트렌드 통계 (2024-2025)

## 1. 방한 외래 관광객 현황

### 연도별 방한 관광객 추이

| 연도 | 방한 외래객 (만명) | 전년대비 |
|---|---|---|
| 2019 (팬데믹 전) | 1,750 | — |
| 2020 | 252 | -85.6% |
| 2021 | 97 | -61.5% |
| 2022 | 320 | +229.9% |
| 2023 | 1,103 | +244.7% |
| 2024 (추정) | 1,620 | +46.9% |

> 출처: 한국관광공사 출입국관광통계 (2024 추정치는 월별 집계 기준 연간 환산)

### 2024년 국가별 방한 비중 (상위 5개국)

| 국가 | 비중 | 특이사항 |
|---|---|---|
| 중국 | 약 24% | 2023.08 단체관광 재개 후 회복세 |
| 일본 | 약 22% | 엔저 지속에도 꾸준한 방문 |
| 대만 | 약 6% | 한류 + 음식 관심 |
| 미국 | 약 6% | K-컬처 세계화 효과 |
| 홍콩 | 약 4% | 쇼핑+의료관광 |

### 외국인 방문 주요 지역

1. **서울** (명동, 강남, 홍대, 북촌) — 약 70% 이상 방문
2. **부산** — 해운대, 광안리, 감천문화마을
3. **제주** — 자연경관, K-드라마 촬영지
4. **경주** — 역사문화 관광
5. **전주** — 한옥마을, 한식

## 2. 국내 관광 현황

### 국내 관광 규모

- **2023년 국내 관광 총 이동 횟수**: 약 3억 2천만회 (한국문화관광연구원)
- **1인당 평균 국내 여행 횟수**: 연 5.8회
- **평균 여행 지출**: 1회당 약 15만원 (당일) / 33만원 (1박 이상)

### 계절별 패턴

- **성수기**: 7-8월 (여름휴가), 추석 연휴, 10월 단풍
- **봄 여행**: 4-5월 (벚꽃, 봄 나들이) — 20-30대 선호
- **겨울 트렌드**: 크리스마스·연말 여행, 스키, 온천

## 3. 20-40대 관광 트렌드

### 연령대별 여행 특성

#### 20대 (MZ 세대 전반)
- **여행 형태**: 솔로 여행(34%), 친구 동반(41%), 커플(22%)
- **평균 여행 예산**: 일 2만~5만원 (저비용 최적화)
- **정보 탐색 채널**: 인스타그램 릴스(56%), 유튜브(48%), 틱톡(31%)
- **선호 테마**: 감성카페·포토존, 이색체험, 로컬 맛집
- **키워드**: #혼자여행, #감성여행, #포토스팟, #로컬맛집

#### 30대 (밀레니얼 세대)
- **여행 형태**: 커플(38%), 가족(29%), 친구 소그룹(23%)
- **평균 여행 예산**: 일 5만~10만원 (품질 중시)
- **정보 탐색 채널**: 네이버 블로그(62%), 인스타그램(47%), 카카오맵
- **선호 테마**: 웰니스, 프리미엄 숙박, 미식, 문화예술
- **키워드**: #주말여행, #가심비, #힐링여행, #프리미엄펜션

#### 40대 (X세대)
- **여행 형태**: 가족(54%), 커플(28%), 친구(13%)
- **평균 여행 예산**: 일 8만~15만원 (편의성·안전 우선)
- **정보 탐색 채널**: 네이버 블로그(68%), 카카오맵, 여행사 앱
- **선호 테마**: 역사문화, 가족 체험, 자연/산, 온천
- **키워드**: #가족여행, #국내여행, #역사기행, #아이와함께

### 공통 트렌드 (20-40대 전반)

1. **짧고 잦은 여행 (short & frequent)**: 주말 1박 2일 선호 증가
2. **로컬 소비**: 프랜차이즈보다 로컬 식당·카페 선호
3. **SNS 공유 가치**: '인증샷' 가능한 장소 우선 고려
4. **실시간 정보 수요**: 혼잡도, 영업 여부 실시간 확인 니즈
5. **AI 추천 수용**: 챗GPT/AI 기반 여행 계획 수용률 증가

## 4. 주요 관광 트렌드 키워드 (2024-2025)

| 트렌드 | 설명 | Visit Korea API 연계 가능성 |
|---|---|---|
| **웰니스 투어** | 스파, 명상, 산림욕 결합 | `wellness-tourism` |
| **반려동물 동반** | 펫프렌들리 숙박·명소 급증 | `pet-friendly-travel` |
| **에코 관광** | 환경 의식 고취, 탄소발자국 최소화 | `eco-tourism`, `durunubi-trails` |
| **AI 큐레이션** | 개인화 여행 코스 자동 생성 수요 | `related-attractions`, `tourism-big-data` |
| **혼잡 회피** | 붐비지 않는 '숨은 명소' 선호 | `visitor-concentration-forecast` |
| **무장애 여행** | 시니어·장애인·영유아 접근성 | `barrier-free-travel` |
| **야간관광** | 야경·야시장·축제 중심 야간 활동 | `tourism-info-korean` |
| **고캠핑·글램핑** | 럭셔리 캠핑 시장 급성장 | `gocamping` |

## 5. 데이터 시사점 (우리 서비스 기획 관점)

1. **2024년 방한 외래객 1,620만명 추정** → 다국어 서비스 필수
2. **중국(24%)·일본(22%)·영어권(10%)** → 최소 3개 언어 필수
3. **20-40대 국내 관광 3.2억회** → 국내 사용자도 큰 시장
4. **혼잡도·실시간 정보 수요 증가** → `visitor-concentration-forecast` API 핵심 차별화 요소
5. **SNS 연동형 여행** → 포토 콘텐츠(`tourism-photos`, `photo-contest-winners`) 통합 가치 높음
