## 서비스별 상세 분석

### 1. Visit Korea 공식 (korean.visitkorea.or.kr)

**강점**
- 한국관광공사 공식 데이터 기반 — 신뢰성 최고
- 9개 언어 지원 (한/영/일/중간/중번/독/불/서/노)
- 27개 OpenAPI 개방 → 개발자 활용 가능
- 오디오 가이드(odii), 반려동물, 무장애, 웰니스 등 특수 정보 보유

**약점**
- UI/UX 구식 (2010년대 디자인 수준)
- 개인화 기능 전무
- 실시간 데이터 없음
- 앱스토어 평점 낮음 (2.8/5 수준)
- SNS 연동 없음

**타겟**: 외국인 관광객 정보 탐색 (한국 진입 전)

---

### 2. 트리플 (Triple)

**강점**
- 한국 내 최고 수준 여행 앱 UX
- 여행기(코스) 사용자 공유 커뮤니티
- 타임라인 기반 여행 계획
- 항공·숙박·액티비티 예약 통합

**약점**
- 한국어 전용 → 외국인 이용 불가
- 실시간 혼잡도 없음
- 공식 관광 데이터 부재
- 지역 깊이 있는 정보 부족

**타겟**: 국내 20-30대 여행자

---

### 3. 야놀자 (Yanolja)

**강점**
- 숙박 예약 시장점유율 1위
- 레저/액티비티 예약 강세
- 대규모 할인 프로모션
- MAU 약 700만명 (2024 기준)

**약점**
- 관광 정보 콘텐츠 빈약
- 여행 코스/계획 기능 없음
- 예약 외 기능 매우 약함
- 외국인 서비스 없음

**타겟**: 국내 전 연령 (숙박·레저 예약 중심)

---

### 4. 여기어때 (Yeogi Eottae)

**강점**
- 숙박 + 레저 통합 예약
- 실속 여행자용 할인 강세
- 모바일 UX 최적화

**약점**
- 관광 정보/콘텐츠 부족
- 여행 계획 기능 없음
- 외국인 서비스 없음

**타겟**: 국내 2040 실속 여행자

---

### 5. 마이리얼트립 (My Real Trip)

**강점**
- 로컬 가이드 투어 연결 강점
- 해외 여행 투어·액티비티 1위
- 검증된 현지인 가이드 네트워크
- 세밀한 여행 카테고리

**약점**
- 국내 여행 서비스 상대적으로 약함
- 개인 자유여행보다 투어 상품 중심
- 실시간 정보 없음
- 외국인 대상 아님 (한국어 중심)

**타겟**: 해외여행 및 국내 투어 관심 20-40대

---

### 6. Google Maps

**강점**
- 전 세계적 신뢰도 — 외국인 필수 앱
- 실시간 혼잡도(Popular Times)
- 다국어 리뷰 자동 번역
- 내비게이션 + 정보 통합

**약점**
- 한국 특화 콘텐츠 부족
- Visit Korea OpenAPI 데이터 미활용
- 여행 코스 계획 기능 없음
- 한국 로컬 맛집/소규모 장소 누락 많음

**타겟**: 외국인 관광객 (필수 의존)

---

### 7. Tripadvisor

**강점**
- 글로벌 최대 여행 리뷰 플랫폼
- 외국인 의사결정 70% 이상 관여
- 호텔·레스토랑 통합 리뷰

**약점**
- 한국 콘텐츠/리뷰 상대적으로 부족
- 한국 특화 정보 없음
- OpenAPI 연동 없음
- 실시간 정보 없음

**타겟**: 서구권 외국인 관광객

---

### 8. Klook

**강점**
- 아시아 액티비티·티켓 예약 강자
- 외국인 대상 한국 상품 풍부
- 모바일 티켓 즉시 사용

**약점**
- 예약 중심 → 여행 정보 약함
- 여행 계획/코스 기능 없음
- 한국 공식 데이터 미활용

**타겟**: 아시아 외국인 관광객 (액티비티 예약)

---

## 경쟁 지형도 요약

```
            외국인 특화
                ↑
    Visit Korea  |  Google Maps
    (정보강/UX약) |  (범용/혼잡도 有)
                |
국내 서비스 ←———+———→ 외국인 서비스
                |
    트리플      |  Klook / Tripadvisor
    야놀자/여기  |  (예약/리뷰 중심)
    어때         |
                ↓
            국내 사용자 특화
```

**공백 지점**: 외국인+국내 모두 커버하며, 실시간 데이터 + AI 개인화 + Visit Korea OpenAPI를 통합한 서비스 **없음**
