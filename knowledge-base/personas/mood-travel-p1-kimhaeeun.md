# P1 — 김하은, 26세 · KR 내국인

## 기본 정보

| 항목 | 내용 |
|---|---|
| 이름 | 김하은 |
| 나이 | 26세 |
| 거주지 | 서울 마포구 |
| 직업 | 스타트업 UX 디자이너 (입사 2년차) |
| 소득 | 월 280만원 |
| 여행 빈도 | 연 3~4회 (주로 1박 2일) |
| 주 기기 | iPhone 15 Pro, 앱 중심 |

## 심리 프로파일

- **핵심 감정**: 번아웃, 고갈, "뭔가 채워지지 않는" 공허감
- **여행 동기**: 일상 탈출 / 감정 회복 / SNS 감성 사진
- **의사결정 패턴**: 즉흥적·충동적. 검색보다 추천 선호
- **정보 채널**: 인스타그램 → 블로그 → 카카오맵 순

## 고통 포인트 (Pain Points)

1. 주말 새벽 2시에 "어디든 가고 싶다" 충동이 오지만 어디를 검색해야 할지 막막함
2. 여행 앱에서 날짜·목적지·인원을 입력하는 절차가 너무 많고 귀찮음
3. 기분에 딱 맞는 여행지를 찾는 데 30분 이상 소비하다 포기
4. 막상 가도 기대와 달라 실망하는 경험 반복

## 목표 (Goals)

- "지금 기분에 딱 맞는 곳"을 5초 안에 추천받고 싶음
- 여행 후 감정이 회복됐다는 만족감을 느끼고 싶음
- 나중에 "그때 거기 갔을 때 기분이 어땠는지" 기억하고 싶음

## 핵심 시나리오

> 금요일 밤 11:30, 퇴근 후 소파에 누워 스마트폰을 켬  
> → Mood Travel Matcher 열기  
> → 감정 "위로" 선택 (3초)  
> → 흐림·혼자·2시간 이내 자동 설정  
> → 수안보 온천 힐링 숲 매칭 (5초)  
> → 즉시 예약 링크 클릭 → 다음날 출발

## 서비스 기대 가치

- 검색 시간 30분 → 5초 단축
- "내 기분을 이해해주는 앱" 정서적 연결
- 감정 일지로 자기 이해 향상
