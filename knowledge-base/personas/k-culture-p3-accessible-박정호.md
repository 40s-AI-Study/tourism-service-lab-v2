# P3 (접근성 약자): 박정호 — 전동 휠체어 이용 K-드라마 팬

## 기본 정보

- **이름**: 박정호
- **나이**: 45세
- **직업**: 프리랜서 번역가 (재택)
- **거주지**: 경기 수원시
- **장애 유형**: 하지 마비 — 전동 휠체어 사용
- **소득**: 월 220만원
- **보조기기**: 전동 휠체어, 스마트폰 음성 접근성 사용

## 배경 (서사)

정호는 10년 전 사고로 하지를 다친 이후 여행을 거의 포기했지만, 넷플릭스로 한국 드라마에 빠지면서 직접 촬영지를 방문하고 싶은 꿈이 생겼다. 문제는 대부분의 여행 정보가 계단·경사로 정보를 제공하지 않아 현장에서 낭패를 봤던 경험이 여러 번이다. 국립박물관 방문 때 입구 경사로가 없어 그냥 돌아온 적도 있다. 아내와 함께 K-드라마 촬영지를 "배리어프리"로 방문할 수 있는 루트가 있다면 꼭 가보고 싶다.

## 목표 (Goals)

- 드라마 촬영지 중 휠체어 접근 가능 장소만 필터링
- 배리어프리 인증 경로로 성지 방문 루트 생성
- 이동 중 장애인 화장실·휴식 공간 위치 실시간 확인
- 아내와 함께 K-드라마 촬영지 방문 인증샷 남기기

## 고민 (Pain Points)

- **접근성 정보 부재**: 관광지 홈페이지에 배리어프리 정보가 없거나 오래됨
- **현장 낭패**: 접근 가능하다고 나와 있어도 실제로는 임시 공사·경사 급경사
- **경로 계획**: 휠체어 통행 가능한 인도·엘리베이터 경로를 포함한 내비게이션 부재
- **피로 관리**: 전동 휠체어 배터리·체력 고려한 동선 계획 어려움
- **커뮤니티 고립**: 같은 처지의 장애인 여행자 정보 공유 채널 부재

## 관광 패턴

- **빈도**: 분기 1회 (당일 또는 1박)
- **선호 유형**: K-드라마 촬영지, 국립·시립 박물관, 한옥 체험 (배리어프리 가능 장소)
- **예산**: 1박 2일 20만원 내외
- **동반자**: 아내 (주 보호자)
- **정보 탐색**: 장애인 여행 카페 (네이버), 배리어프리 여행 블로그, 유튜브

## 기술 친숙도

medium — 스마트폰 음성 안내 사용. 큰 글씨·고대비 UI 선호. 앱 접근성(Voice Over 호환) 필수.

## MBTI / 성향

ISFJ — 신중하고 계획적. 사전 정보 100% 확인 후 실행. 가족과의 추억을 여행 최고 가치로 여김.

## 접근성 요구사항 (Accessibility Needs)

- 텍스트 최소 16px, 고대비 색상 (WCAG AA 이상)
- VoiceOver / TalkBack 완전 호환
- 큰 터치 타깃 (44px 이상)
- 성지별 배리어프리 등급 표시 (A/B/C)
- 장애인 주차 구역, 경사로, 엘리베이터, 장애인 화장실 표시
- 배터리 소모 고려 최적 경로 (이동 거리 최소화)

## 대표 시나리오

앱 설정 → "배리어프리 모드" 활성화 → "도깨비" 촬영지 검색 → 접근성 A등급 장소만 필터링 표시 → 전동 휠체어 이동 경로(엘리베이터·경사로 포함) 자동 생성 → 중간 장애인 화장실 위치 표시 → 방문 완료 후 "배리어프리 개척자" 뱃지 획득 → 같은 장애를 가진 여행자를 위한 현장 접근성 후기 등록.

## 관련 API 시사점

- `barrier-free-travel` — 무장애 여행 정보 API (핵심)
- `tourism-info-korean` — 국문 관광지 상세 (접근성 항목 필터)
- `visitor-concentration-forecast` — 혼잡 시간 회피 → 휠체어 이동 편의
- `area-tourism-resource-demand` — 지역 관광 자원 수요 → 배리어프리 인프라 높은 지역 우선
- `durunubi-trails` — 두루누비 걷기 코스 (휠체어 통행 가능 구간 연동)
