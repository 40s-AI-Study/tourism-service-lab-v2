# P9: 아미나 사이드 (Amina Said) — 말레이시아 무슬림 한국 여행자

## 기본 정보

- **이름**: 아미나 사이드 (Amina Said)
- **나이**: 30세
- **직업**: 쿠알라룸푸르 디지털 마케팅 에이전시 크리에이티브 디렉터
- **거주지**: 쿠알라룸푸르 (말레이시아)
- **소득**: 월 6,500 링깃 (약 200만원)

## 배경 (서사)

아미나는 K-drama와 K-beauty에 빠진 말레이시아 무슬림 여행자다. 인스타그램 팔로워 4만 5천 명의 여행 인플루언서이기도 하다. 한국 여행을 항상 꿈꿔왔지만 할랄 음식 확인, 기도실(무살라) 위치 파악이 어려워 계획을 미뤄왔다. 말레이시아에서 한국 여행이 붐이지만 무슬림 친화적 정보가 한곳에 정리된 앱이 없어서 이슬람 커뮤니티 카카오톡 그룹에서 정보를 공유하고 있다.

## 목표 (Goals)

- 할랄 인증 한식당·카페 방문 (삼겹살 제외 한국 음식 체험)
- 기도 시간에 맞는 무살라(기도실) 위치 사전 확인
- K-beauty 제품 성분 확인 (돼지 유래 성분 없는 제품 필터)
- 인스타그램 팔로워를 위한 "무슬림 한국 여행 가이드" 콘텐츠 제작

## 고민 (Pain Points)

- **할랄 정보 부재**: 한국 여행 앱 어디에도 할랄 인증 레스토랑 필터가 없음
- **기도실 위치**: 서울 중심가에 무살라가 있는지, 어디에 있는지 앱으로 확인 불가
- **성분 확인**: 음식 성분표 한국어 해석 어려움 (돼지·알코올 성분 여부)
- **영어 정보 격차**: 말레이시아에서 유명한 한국 관광지도 영어 정보가 부실한 경우 많음

## 관광 패턴

- **여행 기간**: 5박 6일 (연 1~2회)
- **선호 유형**: 할랄 식당 투어, K-beauty 쇼핑, 한복 체험, 사진 콘텐츠 제작
- **예산**: 총 4,000 링깃 (약 125만원, 쇼핑 별도)
- **동반자**: 친구 2~3명 (모두 무슬림)
- **정보 탐색 채널**: 인스타그램 무슬림 한국 여행 계정, 말레이시아 이슬람 여행 앱(HalalTrip), YouTube

## 기술 친숙도

very high — 디지털 마케터로 앱·SNS에 능숙. 콘텐츠 생산 관점에서 앱 품질 평가. 영어 UI 완전 사용 가능.

## MBTI / 성향

ESFJ — 커뮤니티 중심적, 팔로워와 정보 공유 중시. 믿을 수 있는 정보를 빠르게 찾아 다른 무슬림 여행자에게 전달하는 것에 보람 느낌.

## 대표 시나리오

K-Guide Global 앱 영어 전환 → "Halal" 필터 활성화 → 명동·홍대 주변 할랄 인증 식당 목록 확인 → 인근 무살라 위치 지도 표시 → 기도 시간 알림 설정 → 여행 중 할랄 식사 + 기도 + 포토스팟을 차질 없이 완료 → "무슬림도 한국 여행 완벽히 즐길 수 있어요" 인스타 릴스 게시 → 10만 뷰 달성.
