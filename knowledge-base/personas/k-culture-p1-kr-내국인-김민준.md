# P1 (KR 내국인): 김민준 — 넷플릭스 드라마 촬영지 탐방러

## 기본 정보

- **이름**: 김민준
- **나이**: 28세
- **직업**: IT 스타트업 프론트엔드 개발자
- **거주지**: 서울 마포구
- **소득**: 월 380만원
- **디바이스**: Galaxy S24, 맥북

## 배경 (서사)

민준은 넷플릭스 한국 오리지널 시리즈 마니아다. 주말마다 드라마 촬영지를 찾아다니는 것이 취미지만, 정보가 커뮤니티 카페·인스타·유튜브에 흩어져 있어 매번 2~3시간 조사를 해야 한다. "이태원 클라쓰" 포차를 찾다가 폐업했다는 걸 현장에서야 알았던 경험이 있다. 최근 동생이 K-팝 성지순례를 같이 가자고 해 처음으로 아이돌 관련 장소에도 관심이 생겼다.

## 목표 (Goals)

- 드라마별 촬영지 리스트를 한곳에서 확인 (현재 영업 여부 포함)
- 촬영지 → 주변 맛집 → 포토스팟 이어지는 최적 동선 생성
- 직접 발굴한 숨은 촬영지를 커뮤니티에 공유하고 인정받기
- 동생과 함께 K-팝 성지도 탐방 (BTS, 뉴진스)

## 고민 (Pain Points)

- **정보 신선도**: 촬영지 폐업·이전 여부를 실시간으로 알기 어려움
- **정보 파편화**: 드라마별 촬영지 정보가 Naver 카페·인스타·유튜브에 분산
- **동선 비효율**: 여러 촬영지를 하루에 효율적으로 돌아다니는 루트 계획이 어려움
- **현장 실망**: 기대하고 갔더니 공사 중이거나 일반인 접근 불가인 경우 잦음

## 관광 패턴

- **빈도**: 월 1~2회 당일치기 또는 1박 2일
- **선호 유형**: 드라마/영화 촬영지, K-팝 성지, 숨은 로컬 맛집
- **예산**: 교통비 포함 하루 5~8만원
- **동반자**: 친구 1~2명 또는 동생
- **정보 탐색**: 나무위키 촬영지 문서 → 인스타 해시태그 → 유튜브 브이로그

## 기술 친숙도

high — 개발자 출신. API 연동, 지도 서비스에 익숙. 앱 UI에 까다로움.

## MBTI / 성향

INTJ — 사전 조사를 철저히 하지만, 현장에서 예상치 못한 발견도 즐김. 수집욕 강함.

## 대표 시나리오

앱 실행 → "이상한 변호사 우영우" 검색 → 촬영지 9곳 리스트 + 현재 운영 상태 표시 → AI가 서울 도심 반나절 루트 자동 생성 → 지하철 환승 포함 이동 경로 확인 → 당일 방문 후 "숨겨진 추가 촬영지" 직접 제보 → 커뮤니티 '드라마 헌터' 뱃지 획득.

## 관련 API 시사점

- `tourism-info-korean` — 국문 관광지 상세 정보 (운영 시간·현황)
- `area-tourism-resource-demand` — 지역별 관광 자원 수요 → 혼잡도 예측
- `visitor-concentration-forecast` — 방문 최적 시간대 추천
- `related-attractions` — 촬영지 주변 연관 명소 자동 연결
- `audio-guide` — 드라마 OST 배경 오디오 스토리텔링
