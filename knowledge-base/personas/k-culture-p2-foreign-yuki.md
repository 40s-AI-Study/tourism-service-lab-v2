# P2 (외국인): Yuki Tanaka — 방한 K-팝 성지순례 팬

## 기본 정보

- **이름**: Yuki Tanaka (田中 ゆき)
- **나이**: 24세
- **직업**: 도쿄 패션 잡지사 에디터 보조
- **거주지**: 도쿄 시부야구
- **소득**: 월 22만 엔 (약 200만원)
- **언어**: 일본어(모어), 영어(중급), 한국어(기초 읽기)

## 배경 (서사)

유키는 BTS와 뉴진스의 열혈 팬으로, 한국 여행이 이번이 두 번째다. 첫 방한 때는 패키지 투어라 HYBE 사옥만 잠깐 들렀지만, 이번엔 혼자 5박 6일로 성지순례를 계획 중이다. X(트위터) K-팝 커뮤니티에서 정보를 모으고 있지만 일본어로 된 신뢰할 만한 실시간 정보가 부족하다. 멤버들이 다닌 카페와 연습실 건물 앞에서 인증샷을 찍고 일본 팬 커뮤니티에 공유하는 것이 최고의 목표다.

## 목표 (Goals)

- 좋아하는 아이돌 관련 모든 성지 (연습실, 데뷔 장소, 자주 가는 카페) 방문
- 일본어로 안내되는 K-팝 성지 오디오 가이드
- 다른 외국인 팬과 현장에서 연결되는 소셜 기능
- 멤버별 선호 장소 → 개인화 루트 생성
- 성지순례 일정을 일본 팬 커뮤니티와 실시간 공유

## 고민 (Pain Points)

- **언어 장벽**: 한국어 안내판·메뉴 해독 불가. 앱 UI도 한국어면 이탈
- **실시간성 부재**: X/트위터 정보가 오래됐거나 폐업된 곳 많음
- **이동 동선**: 서울 지하철 복잡해 여러 성지를 효율적으로 돌기 어려움
- **팬 커뮤니티 고립**: 일본에서 교류하던 팬 친구들과 현지에서 만나기 어려움
- **아티스트별 정보 분산**: 그룹마다 성지가 달라 통합 앱 부재

## 관광 패턴

- **여행 기간**: 5박 6일 (유급 휴가 소진)
- **선호 유형**: K-팝 성지, 아이돌 콘셉트 카페, 굿즈샵, 홍대·강남 핫플
- **예산**: 항공·숙박 제외 1일 8만원
- **동반자**: 솔로 여행 (현지에서 팬 미팅 기대)
- **정보 탐색**: X(트위터) K-팝 JP 커뮤니티 → 웨이보 → 일한 여행 유튜버

## 기술 친숙도

high — Instagram, X, TikTok 헤비 유저. 앱 직관성 중시. 회원가입 단계 많으면 이탈.

## MBTI / 성향

ESFP — 즉흥적·감성적. 계획보다 현장 분위기 중시. 공유와 인정이 강한 동기.

## 대표 시나리오

앱 언어 → 일본어 선택 → "뉴진스" 검색 → 성지 12곳 지도 표시 (현재 운영 여부 실시간) → AI가 하루 루트 생성 → 각 성지 이동 시 일본어 오디오 스토리 재생 ("멤버 민지가 이 카페에서 찍은 사진이...") → 성지 방문 완료 시 '뉴진스 버니' 팬 뱃지 획득 → 달성 현황 X에 자동 공유.

## 관련 API 시사점

- `tourism-info-japanese` — 일본어 관광지 상세 정보
- `audio-guide` — 일본어 K-팝 성지 오디오 가이드
- `related-attractions` — 성지 주변 팬 방문 명소 연결
- `area-tourism-diversity` — 국제 다양성 지수 → 외국인 다수 방문 지점 안내
- `visitor-concentration-forecast` — 성지 혼잡 시간대 회피 → 독점 인증샷 타임
