# 페르소나 P1 — 이서연, 28세, 콘텐츠 크리에이터

## 기본 정보

| 항목 | 내용 |
|---|---|
| 이름 | 이서연 |
| 나이 | 28세 |
| 직업 | 인스타그램 라이프스타일 크리에이터 (팔로워 1.2만) |
| 거주지 | 서울 마포구 |
| 소득 | 월 250만원 (크리에이터 수익 + 부업) |
| 여행 빈도 | 월 2-3회 (당일 또는 1박) |

## 여행 목표

- **콘텐츠 소재** 발굴 — 인스타/릴스에 올릴 감성 있는 전통체험 장면 확보
- **진정성** 있는 로컬 체험 — 관광지화된 포토존이 아닌, 실제 장인과 함께하는 체험
- **가성비** — 3-6만원 내외, 2시간 이내, 이동 편리한 도심 공방

## 사용 앱 & 채널

```
발견: 인스타그램 릴스, 유튜브 쇼츠
검색: 네이버 블로그, 당근마켓 동네 이야기
예약: 클래스101, 네이버 예약
공유: 인스타그램, TikTok
```

## Goals & Motivations

1. 독창적인 콘텐츠 소재 찾기 — 너무 유명해지기 전의 공방/체험
2. "진짜 한국"을 표현하는 이미지 — 외국인 팔로워에게 한국 문화 소개
3. 혼자서도 편하게 예약 가능한 클래스
4. 체험 후 작품을 집에 가져갈 수 있는 것 (도자기, 천연염색 등)

## Frustrations (Pain Points)

- "공방 찾아봤는데 네이버 블로그 글이 2년 전 거라 폐업했음"
- "예약 전화해야 하는데 영업시간에 통화 못 함"
- "체험 끝나고 작품 받는 데 2주 걸린다고 함 — 여행 중 못 가져감"
- "카테고리 필터가 없어서 '한복+도자기+악기' 비교가 안 됨"
- "리뷰가 광고성인지 진짜인지 구분 어려움"

## 필요 기능

- **실시간 예약 가능 여부** 확인 (당일/주말 즉석 예약)
- **체험 카테고리 필터** (도자기, 한복, 한식, 악기, 서예, 천연염색)
- **사진 갤러리 미리보기** — 체험 공간 분위기, 결과물 사진
- **당일 픽업 가능 여부** 표시
- **인스타 공유** 기능 (체험 완료 후 태그 생성)

## 사용자 여정 터치포인트

```
인스타 릴스에서 도자기 영상 발견
→ Traditional Experience 앱 검색
→ "도예" 카테고리 필터
→ 인사동 공방 3개 비교 (별점, 사진, 가격)
→ 토요일 오후 2시 즉석 예약 (카카오페이)
→ 체험 당일 공방 도착, 장인 지도 하에 도자기 제작
→ 사진 촬영 팁 제공 (포토존 안내)
→ 작품 당일 수령 or 배송 선택
→ 앱 내 리뷰 작성 + 인스타 공유 버튼
```

## 인용구

> "이런 체험 있는 줄 알았는데 어디서 예약하는지 몰랐어요.
> 앱 하나로 카테고리 보고 바로 예약할 수 있으면 진짜 자주 올 것 같아요."
