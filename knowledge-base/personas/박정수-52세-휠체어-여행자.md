# 페르소나 P3 — 박정수, 52세, 대학 강사 (접근성 약자)

## 기본 정보

| 항목 | 내용 |
|---|---|
| 이름 | 박정수 |
| 나이 | 52세 |
| 직업 | 대학 역사학과 강사 |
| 거주지 | 서울 동작구 |
| 장애 유형 | 하지 마비 (전동휠체어 사용, 10년째) |
| 보호자 동반 | 성인 딸과 함께 (월 1-2회 문화 나들이) |
| 한국 전통문화 관심 | 매우 높음 (전공 분야) |

## 여행 목표

- **접근 가능한** 전통문화 체험 — 계단 없는 공방, 휠체어 진입 가능 동선
- **전문 지식**을 체험으로 연결 — 수업 소재 발굴
- 딸과 **함께 즐기는** 여가 — 보조 없이도 참여 가능한 체험
- **실내 착석형** 활동 (도자기 성형, 서예, 한지 공예 등)

## 사용 앱 & 채널

```
정보 탐색: 네이버 지식iN, 장애인 여행 카페, 구글 지도 (접근성 레이어)
예약: 전화 또는 이메일 선호 (온라인 폼 복잡하면 포기)
결제: 카카오페이, 체크카드
공유: 장애인 여행 블로그 (소규모 커뮤니티)
```

## Goals & Motivations

1. 방문 전 **휠체어 접근 가능 여부** 100% 확인 (현장 거절 경험 다수)
2. 체험 공간 **사진 + 동선 정보** — 입구 폭, 화장실 위치, 바닥 재질
3. **착석 상태로 진행 가능**한 체험 필터
4. 강의에 활용할 수 있는 **전통 공예 심화 정보** 포함
5. **사전 요청 시** 특별 지원 (높이 조절 작업대 등)

## Frustrations (Pain Points)

- "공방 전화하면 '계단 있어서 안 된다'고 함 — 가보지도 못하고 거절"
- "장애인 접근성 표시가 있어도 실제 가보면 달라서 믿을 수 없음"
- "온라인 예약 폼이 작고 복잡해서 딸이 대신 해줘야 함"
- "체험 종류 설명이 피상적 — 역사적 배경이나 장인 소개가 없음"
- "그룹 체험인데 혼자 다 느림 — 눈치가 보임"

## 필요 기능

- **접근성 인증 뱃지** — 휠체어 진입 가능, 경사로 있음, 무장애 화장실
- **공간 사진** — 입구 폭, 작업대 높이, 화장실 위치 (텍스트 + 이미지)
- **착석형 체험 필터** (전신 활동 체험 vs 착석형 구분)
- **대형 텍스트 모드** / 화면 접근성 지원
- **사전 요청 입력** — 예약 시 특별 지원 사항 메모 전달
- **1:1 소규모 옵션** — 그룹이 아닌 단독/2인 클래스 예약
- **체험 심화 정보** — 장인 프로필, 해당 공예 역사, 관련 문헌

## 사용자 여정 터치포인트

```
딸과 주말 계획 세우기 → 앱 "접근성" 필터 ON
→ "휠체어 접근 가능 + 착석형" 필터 결과 확인
→ 각 공방 접근성 상세 페이지 (사진 + 동선 설명)
→ 서예 공방 선택 — "작업대 높이 조절 가능, 무장애 화장실"
→ 예약 시 "전동휠체어 이용, 높이 조절 테이블 필요" 사전 메모
→ 공방 측 확인 문자 수신 ("준비 완료")
→ 체험 당일 — 장인의 서예 역사 설명 + 직접 붓글씨 작성
→ 완성 작품 액자 포함 당일 수령
→ 접근성 후기 작성 (구체적 정보 → 다른 장애인 여행자에게 도움)
```

## 인용구

> 휠체어라는 이유로요. 갈 수 있는 곳이 있다는 걸 앱이 먼저 알려줬으면 해요."
