# P3 — 이재성, 42세 · 접근성 약자 (청각장애)

## 기본 정보

| 항목 | 내용 |
|---|---|
| 이름 | 이재성 |
| 나이 | 42세 |
| 거주지 | 경기도 성남시 |
| 직업 | IT 스타트업 백엔드 개발자 |
| 장애 유형 | 청각장애 2급 (보청기 사용) |
| 여행 스타일 | 조용하고 자연 중심 / 시각적 아름다움 중시 |
| 동행 | 주로 아내와 함께 (비장애인) |

## 심리 프로파일

- **핵심 감정**: 고요함을 원함 / 소음 스트레스 없는 환경 선호
- **여행 동기**: 보청기 없이도 즐길 수 있는 자연·시각 중심 여행
- **의사결정 패턴**: 사전 정보 철저히 확인 (소음 수준, 접근성 등)
- **불안 요소**: 안내 방송·음성 중심 관광지 → 정보 누락 불안

## 고통 포인트 (Pain Points)

1. 음성 안내 중심 관광지에서 정보 접근이 어려움
2. 여행앱에서 "소음 적음", "청각장애 친화" 필터가 없음
3. "고요한 곳"을 검색해도 실제 소음 수준을 알 수 없음
4. 공연·축제 등 청각 중심 행사 추천이 자주 뜨는 문제

## 목표 (Goals)

- 청각에 의존하지 않아도 충분히 즐길 수 있는 여행지 추천받기
- 감정(고요·위로·호기심) 기반으로 나에게 맞는 장소 발견
- 텍스트·시각 중심의 접근 가능한 UI로 불편 없이 사용

## 접근성 요구사항

- 화면 내 모든 정보 텍스트 제공 (음성 의존 없음)
- "소음 낮음" · "조용한 자연" 태그 필터
- 진동·시각 피드백 (음성 알림 대체)
- 고대비 텍스트, 큰 터치 타겟

## 핵심 시나리오

> 아내와 주말 당일치기 여행 계획  
> → 감정 "고요" 선택  
> → 청각장애 친화 필터 ON  
> → 담양 죽녹원 + 보성 녹차밭 매칭  
> → "소음 낮음·시각 중심·텍스트 안내 제공" 태그 확인  
> → 출발 전 경로·시설 정보 텍스트로 모두 확인 완료

## 서비스 기대 가치

- 접근성 필터로 포용적 여행 경험 제공
- 청각 약자도 감정 기반 여행을 온전히 즐길 수 있음
- 사회적 가치: 관광 접근성 격차 해소
