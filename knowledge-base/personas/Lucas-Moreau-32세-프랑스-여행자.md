## Goals & Motivations

1. 영어로 완전히 소통 가능한 체험 클래스 (강사 or 가이드)
2. 혼자 와도 괜찮은 소규모 그룹 (6명 이하)
3. 전통 맥락 설명 포함 — "왜 이 공예가 중요한가"
4. 카드 결제 + 영어 영수증 발급 가능
5. 숙소 근처 이동 최소화

## Frustrations (Pain Points)

- "Klook에는 한복 대여밖에 없어 — 진짜 문화 체험을 원하는데"
- "한국어 사이트는 번역기 돌리면 내용이 이상해짐"

- "체험 설명에 '영어 가능 여부'가 안 적혀 있음"
- "구글맵에 없는 공방이 많아서 길 찾기 어려움"

## 필요 기능

- **5개국어 완전 지원** (EN/FR/ZH/JA/ES) — UI + 체험 설명 + 예약 폼
- **영어 진행 가능 여부** 뱃지
- **구글맵 연동** 길찾기 (숙소에서 도보/지하철 몇 분)
- **Visa/Mastercard + Apple Pay** 결제
- **문화 맥락 설명** 섹션 (한국 전통 공예의 역사 2-3줄)
- **즉석 채팅 지원** (체험 전 영어로 질문)
