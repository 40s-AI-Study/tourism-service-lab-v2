# KTO 잠재관광지 기반 비수도권 숨은 명소 발굴

## 핵심 아이디어
한국관광공사의 "강소형 잠재관광지" 선정 리스트와 빅데이터를 결합해 아직 대중에게 알려지지 않은 비수도권 명소를 먼저 발견하는 앱. "아무도 모르는 곳 먼저 가보기" 경험 제공.

## 타겟
- 대중관광지 식상한 여행 고수 20-40대
- 숨은 명소 먼저 발견하는 것을 즐기는 탐험형 여행자

## 핵심 기능
- KTO 강소형 잠재관광지 리스트 큐레이션
- 관광 다양성 지수로 "덜 알려진 정도" 점수화
- 숨은 명소 발견 시 "개척자 배지" 게임 요소
- 방문 후 리뷰로 명소 데이터 축적
- 지역별 숨은 명소 랭킹

## 활용 API
- `한국관광공사_지역별 관광 다양성`
- `한국관광공사_관광빅데이터 정보서비스`
- `한국관광공사_기초지자체 중심 관광지 정보`
- `한국관광공사_지역별 관광 수요 강도`

## 차별화
공식 잠재관광지 데이터 + 빅데이터 조합. 공모전 지역 균형 발전 가치 직접 연계.

## 기대 효과
비수도권 관광 활성화, 관광 쏠림 현상 완화

## AI 역할
- **ai_role**: `ai-enhanced` — 잠재관광지 큐레이션 및 "덜 알려진 정도" 점수화에 AI 핵심 역할
- KTO 강소형 잠재관광지 데이터 + 빅데이터 방문자 수 결합해 희소성 점수 자동 산출
- 관광 다양성 지수·수요 강도 복합 분석으로 진짜 미발굴 명소 발굴
- 사용자 발견 리뷰 데이터 학습으로 명소 데이터 지속 축적

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 구독 | 희소성 점수 상위 명소·배지 수집 | 월 2,900원 |
| 지역 관광 광고 | 비수도권 관광지 노출 | 월 10만원/지역 |
| 지자체 파트너십 | 잠재관광지 발굴 데이터 공유 | 프로젝트별 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 비수도권 여행 시장 약 4,000억원
- **SAM**: 숨은 명소 먼저 발견하고 싶은 탐험형 여행자 약 250만명
- **SOM**: 출시 2년 내 MAU 10만 목표 → 구독+광고 기준 연 약 40억원
- **근거**: KTO 잠재관광지 선정 프로그램 확대, 비수도권 관광 정책 지원 강화

## 편의성
- 희소성 점수 기반 명소 즉시 정렬 표시 (검색 없이 발견 경험)
- 개척자 배지 자동 발급으로 방문 인증 절차 불필요
- 지역별 숨은 명소 랭킹으로 여행지 미결정 시 즉시 영감 제공
- 발견한 명소 SNS 공유 원클릭 (자동 포맷 생성)
