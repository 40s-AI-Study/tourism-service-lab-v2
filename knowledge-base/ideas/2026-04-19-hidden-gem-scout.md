# KTO 잠재관광지 기반 비수도권 숨은 명소 발굴

## 핵심 아이디어
한국관광공사의 "강소형 잠재관광지" 선정 리스트와 빅데이터를 결합해 아직 대중에게 알려지지 않은 비수도권 명소를 먼저 발견하는 앱. "아무도 모르는 곳 먼저 가보기" 경험 제공.

## 타겟
- 대중관광지 식상한 여행 고수 20-40대
- 숨은 명소 먼저 발견하는 것을 즐기는 탐험형 여행자

## 핵심 기능
- KTO 강소형 잠재관광지 리스트 큐레이션
- 관광 다양성 지수로 "덜 알려진 정도" 점수화
- 숨은 명소 발견 시 "개척자 배지" 게임 요소
- 방문 후 리뷰로 명소 데이터 축적
- 지역별 숨은 명소 랭킹

## 활용 API
- `한국관광공사_지역별 관광 다양성`
- `한국관광공사_관광빅데이터 정보서비스`
- `한국관광공사_기초지자체 중심 관광지 정보`
- `한국관광공사_지역별 관광 수요 강도`

## 차별화
공식 잠재관광지 데이터 + 빅데이터 조합. 공모전 지역 균형 발전 가치 직접 연계.

## 기대 효과
비수도권 관광 활성화, 관광 쏠림 현상 완화
