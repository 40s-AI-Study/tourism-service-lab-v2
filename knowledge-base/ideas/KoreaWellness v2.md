# KoreaWellness v2 — 감정 치유 여행 큐레이터

## 갈래 A 고도화 요약

**모 아이디어**: KoreaWellness — 번아웃 직장인을 위한 웰니스 관광 큐레이션 (5축 73점)

**v2 변경 지점**:
- 감정 JTBD를 "스트레스 진단 → 여행 추천" 로직에서 **"앱을 여는 순간의 감정 포착"** 으로 재정의
- Hero Screen을 입력 폼에서 **감정 5선택 첫 화면**으로 재설계
- 5성향(조용/활달/미식/전통감성/계절사진) 웰니스 타입 매핑 추가
- 국내/외국인 이중 렌즈: K-웰니스(한방·산림욕) vs 글로벌 웰니스(스파·요가) 이중 서술
- 4계절 웰니스 콘텐츠 분기 명시

## Hero Screen 묘사

화면 중앙에 큰 질문 하나:
**"지금 가장 필요한 게 뭐예요?"**

아래 5개 카드 (아이콘 + 한 단어):
- 🌿 **고요** — 아무것도 안 하고 싶어
- 🔥 **열기** — 몸을 움직이고 싶어
- 🍵 **맛** — 뭔가 특별한 걸 먹고 싶어
- 🏯 **감성** — 한국다운 것을 느끼고 싶어
- 📷 **순간** — 계절을 담고 싶어

선택 즉시 해당 감정에 맞는 웰니스 코스 추천. 트리플·야놀자가 "어디 갈까?" 묻는다면, 이 앱은 **"지금 어떤 기분이야?"** 를 먼저 묻는다.

## 성향별 시나리오 3개

### 시나리오 1 — 조용한 혼자 여행자 (30대 직장인)
박서연(34세, 광고팀 기획자)이 월요일 아침 KoreaWellness v2를 열어 "고요"를 선택한다. 앱이 강원도 인제 합강정 산림욕장을 추천— 이번 주말 수요 밀도 하위 10%. 금요일 저녁 출발 → 토요일 산림욕 + 한방 족욕 → 일요일 조용한 귀경. 혼잡도 알림으로 "토요일 오전 9시 전 도착 권장" 메시지 수신.

### 시나리오 2 — 전통감성 선호 커플 (40대)
이민수·김혜진(42/40세) 부부가 "감성"을 선택한다. 앱이 전북 순창 전통 한옥 스파 + 한방차 체험 + 가을 단풍 포인트를 묶은 2박 코스를 제안. K-웰니스 특화 콘텐츠(한방·찜질·사찰 명상)를 묶어 "외국에서 절대 못 하는 회복 방식"으로 포지셔닝.

### 시나리오 3 — 외국인 K-웰니스 탐방객
일본인 여행객 Yui(27세)가 "한국식 힐링을 경험하고 싶다"며 앱 실행. 영문 인터페이스에서 "CALM" 선택. 앱이 "K-Wellness Signature: 황토 찜질 → 산림욕 → 한방차" 코스를 제안하며 일본인 관점 설명("일본 온천과 다른 한국 황토 치유 원리") 제공.

## 계절별 분기

| 계절 | 콘텐츠 포인트 | 추천 지역 |
|---|---|---|
| 봄 | 벚꽃·봄나물 힐링 산책 + 야생화 명상 | 지리산 둘레길, 경남 하동 |
| 여름 | 계곡 족욕·산림욕 열 해소 + 새벽 고요 | 강원 인제, 경북 봉화 |
| 가을 | 단풍 컬러테라피 + 전통 한방 보양 | 전북 순창, 충남 공주 |
| 겨울 | 황토 찜질·한증막 + 설경 명상 | 강원 평창, 충북 수안보 |

## 국내/외국인 이중 렌즈

**같은 POI: 충남 공주 마곡사 산사 명상 프로그램**

- 🇰🇷 한국인 관점: "도심 번잡함을 완전히 끊는 디지털 디톡스 — 24시간 산사 스테이, SNS 없는 하루"
- 🌍 외국인 관점: "UNESCO 세계유산 사찰에서 불교 명상 체험 — 절 식사(공양), 새벽 예불 참여. 일본 젠 문화와 다른 한국 선(禪) 스타일"

## 활용 API (한글 파일명 기준)

| 한글 API 파일명 | 서비스 내 역할 | 호출 빈도 |
|---|---|---|
| `한국관광공사_웰니스관광정보` (wellness-tourism) | 한방·스파·명상·산림욕 핵심 POI 데이터 | 코스 생성 시 |
| `한국관광공사_지역별 관광 수요 강도` (area-tourism-demand-density) | 한적한 웰니스 지역 선별 | 추천 시 실시간 |
| `한국관광공사_생태 관광 정보` (eco-tourism) | 산림욕·자연치유 생태관광지 연계 | 코스 생성 시 |
| `한국관광공사_관광빅데이터 정보서비스` (tourism-big-data) | 계절별 웰니스 트렌드 분석 | 일 1회 캐시 |
| `한국관광공사_관광지 집중률 방문자 추이 예측 정보` (visitor-concentration-forecast) | 혼잡도 알림 | 시간별 |
| `한국관광공사_두루누비 정보 서비스` (durunubi-trails) | 웰니스 도보 산책로 연계 | 코스 생성 시 |
