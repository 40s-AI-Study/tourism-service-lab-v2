# 아이디어 5: WellKorea

## 한 줄 정의
스파·온천·산림욕·명상·트레킹·글램핑을 통합해 심신 회복을 목적으로 하는 '웰니스 여행' 전문 큐레이터 앱.

## 타겟 사용자
- **1차**: 30-40대 번아웃 직장인 — 주말 힐링을 원하는 여행자
- **2차**: 외국인 방한 관광객 — K-wellness(한방·찜질방·산림욕) 체험 원하는 여행자

## 핵심 기능
1. **웰니스 테마 필터**: 스파·온천·산림욕·명상·요가·트레킹·글램핑 카테고리 선택
2. **1박 2일 웰니스 패키지 코스**: 도착일 스파 → 다음날 트레킹 → 귀경 전 한방 체험
3. **두루누비 트레일 통합**: 국내 도보여행길 코스 정보 + 난이도·소요시간 안내
4. **글램핑 조회**: 인근 글램핑장 위치·시설 정보 및 예약 연동
5. **효능별 추천**: "피로 회복" → 온천 + 숲 프로그램 / "스트레스 해소" → 명상 + 트레킹

## 활용 API
- `wellness-tourism` — 웰니스 관광 정보 (스파·온천·한방 등)
- `eco-tourism` — 생태 관광지 (자연 힐링)
- `durunubi-trails` — 전국 도보여행길 코스
- `gocamping` — 글램핑·캠핑 시설 정보
- `visitor-concentration-forecast` — 혼잡도 예측 (조용한 힐링 보장)

## 차별화 포인트
- 웰니스 카테고리들이 현재 앱마다 분산 — 하나로 통합한 서비스 없음
- "힐링 목적" 기반 추천 → 증상/목표 입력으로 코스 매칭 (차별화 UX)
- K-wellness 콘텐츠는 외국인 고관심 분야 → 다국어 확장 용이

## 공모전 강점
- 5개 API 조합으로 데이터 활용 점수 공략
- 웰니스 관광 시장 연 15% 성장 — 시장 타당성 입증 용이

## 예상 사용 시나리오
```
사용자: "몸도 마음도 지쳤어요, 주말 1박 2일" →
  거리 설정: 서울 2시간 이내 →
  Day1: 충주 수안보 온천 (혼잡도 낮은 토요일 오후 3시) →
        인근 한정식 저녁 →
        글램핑장 1박
  Day2: 월악산 둘레길 (두루누비 코스 3.2km) →
        오전 산림욕 산책 → 귀경
```

## 리스크
- 예약 연동 구현 복잡도 (1차 버전은 외부 링크 연동으로 단순화 가능)
- 글램핑 시즌성 (봄·가을 집중) → 사계절 콘텐츠 확충 필요
