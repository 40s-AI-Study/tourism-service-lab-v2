# 지역 특산 맛집 루트 자동 생성기

## 핵심 아이디어
MZ세대 여행지 선택 1순위 "맛집"을 공략. 지역별 특산 음식과 숨은 맛집을 관광 동선에 자연스럽게 녹여 "미식 투어 루트"를 자동 생성.

## 타겟
- 미식 중심 여행자 20-40대
- "맛집 없으면 안 가요" 인식의 MZ세대

## 핵심 기능
- 지역 선택 → 특산 음식 + 추천 식당 + 관광지 혼합 루트
- 식사 시간대 고려한 동선 배치 (아침/점심/저녁/야식)
- 빅데이터 기반 현지인 vs 관광객 맛집 분리 표시
- 계절별 특산물 캘린더 연동
- 먹방 루트 저장/공유 기능

## 활용 API
- `한국관광공사_국문 관광정보 서비스` (음식점 카테고리)
- `한국관광공사_관광빅데이터 정보서비스`
- `한국관광공사_관광지별 연관 관광지 정보`

## 차별화
관광지 + 식당을 동선으로 통합. 단순 맛집 앱(망고플레이트)이 아닌 "미식 여행 코스" 설계.

## 기대 효과
MZ세대 핵심 여행 동기 해소, 지역 음식 산업 활성화
