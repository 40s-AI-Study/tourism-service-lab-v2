# 외국인 긴급상황 대응 + 의료관광 통합앱

## 핵심 아이디어
한국 여행 중 아프거나 긴급상황이 생겼을 때 언어 장벽 없이 빠르게 대응 가능한 앱. 의료관광 정보 + 실시간 통역 + 긴급 연락처 통합.

## 타겟
- 한국어 모르는 외국인 관광객 전체
- 의료관광 목적 방한 외국인 (시장 규모 연간 50만명+)

## 핵심 기능
- 증상 선택 → 가까운 외국어 가능 병원 안내
- 의료관광 클리닉 예약 연동
- 응급상황 119/112 다국어 문자 전송
- 여행자보험 청구 서류 자동 번역
- 약국/처방전 다국어 안내

## 활용 API
- `한국관광공사_의료관광정보`
- `한국관광공사_영문 관광정보서비스`
- `한국관광공사_일문 관광정보서비스`
- `한국관광공사_중문 간체 관광정보서비스`

## 차별화
여행 안전/건강 특화. 공모전 심사: 실용성 + 외국인 방한 촉진 가치.

## 기대 효과
외국인 한국 여행 안전망 강화, 의료관광 시장 확대
