# 아이디어 8: FamilyKorea

## 한 줄 정의
초등생 자녀·노부모·반려동물 등 다양한 가족 구성원의 조건을 동시에 만족하는 코스를 자동 생성하는 가족여행 전문 플래너.

## 타겟 사용자
- **1차**: 40대 초등생 자녀 가족 — "아이가 좋아하면서 어른도 즐길 수 있는 곳" 탐색
- **2차**: 30대 영유아(0-5세) 부모 — 유모차 접근 가능, 수유실 있는 여행지

## 핵심 기능
1. **가족 구성원 프로필 입력**: 자녀 나이·노부모 동반 여부·반려동물 동반 여부 → 조건 매칭
2. **연령별 교육 콘텐츠 연계**: 역사 관광지에서 초등 교과 연계 설명 제공 (학부모 니즈)
3. **무장애 + 반려동물 동시 필터**: 노부모 접근성 + 펫 동반 가능 장소 교집합 검색
4. **이동 피로도 계산**: 아이 연령·거리·이동 수단 → "이 코스 소요 시간 + 쉬는 곳 3개" 제시
5. **추천 숙소 유형**: 풀빌라·독채 펜션·가족실 호텔 — 가족 수·자녀 나이 기반 추천

## 활용 API
- `barrier-free-travel` — 노부모·유모차 접근성 정보
- `pet-friendly-travel` — 반려동물 동반 가족 옵션
- `related-attractions` — 가족 여행지 연관 관광지 추천
- `visitor-concentration-forecast` — 어린이 있는 가족 위한 한산 시간 추천
- `tourism-info-korean` — 기본 관광지 정보

## 차별화 포인트
- 40대 Pain Point #1이 "아이 데리고 갈 곳 찾기"지만 가족여행 전용 앱이 없음
- 노부모 접근성 + 아이 연령 + 반려동물 동시 필터 조합 = 완전히 새로운 검색 경험
- 교육 콘텐츠 연계 → 부모 "교육적 여행" 니즈 충족 (40대 구매 의사 결정 강화)

## 공모전 강점
- 40대 가족여행 타겟은 실제 소비력이 높아 서비스 기획 타당성 명확
- 복합 API 조합(barrier-free + pet-friendly + related) → 데이터 활용 점수 공략

## 예상 사용 시나리오
```
사용자: 아빠(42세), 엄마(40세), 딸(8세), 외할머니(68세, 무릎 약함) →
  입력: 1박 2일, 전라도 →
  Day1: 전주 한옥마을 (휠체어/지팡이 접근 가능 경로 안내) →
        아이 체험: 한지 공예 →
        가족실 한옥 숙박 1박
  Day2: 순창 강천산 (유아 코스 1.5km) →
        지역 순창고추장 비빔밥 식사 → 귀경
```

## 리스크
- 맞춤 필터 복잡도 → 초기 버전은 "자녀 연령"과 "노부모 동반" 2개 조건만으로 단순화 가능
- 교육 콘텐츠 제작 비용 → 초기에는 문화재청 공개 자료 활용
