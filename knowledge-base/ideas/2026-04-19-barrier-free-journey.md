# 무장애 여행 통합 정보 앱

## 핵심 아이디어
장애인/고령자/영유아 동반 가족이 접근 가능한 관광지, 숙박, 이동 수단을 한 앱에서 통합 제공. 흩어진 공식 무장애 정보를 현대적 UX로 통합.

## 타겟
- 장애인 약 263만명 + 가족
- 65세 이상 시니어 여행자 약 970만명
- 영유아 동반 부모 (유모차 이동 필요)

## 핵심 기능
- 장애 유형별 필터 (휠체어/시각/청각/영유아)
- 접근성 상세 정보 (경사로/엘리베이터/점자블록)
- 무장애 동선 지도 (턱 없는 경로)
- 장애인 화장실 위치 실시간 안내
- 접근성 정보 사용자 업데이트 기능 (크라우드소싱)

## 활용 API
- `한국관광공사_무장애 여행 정보`
- `한국관광공사_관광지별 연관 관광지 정보`
- `한국관광공사_국문 관광정보 서비스`

## 차별화
고령화 사회 필수 서비스. 공모전 사회적 가치 심사항목 직접 연계. 현재 공식 정보 접근성 낮음.

## 기대 효과
관광 약자 여행권 보장, 사회적 가치 공모전 가점 기대
