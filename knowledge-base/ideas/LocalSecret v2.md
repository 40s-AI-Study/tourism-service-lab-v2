# LocalSecret v2 — 감정으로 발견하는 숨은 한국

## 갈래 A 고도화 요약

**모 아이디어**: LocalSecret — 혼잡 회피 + 숨은 명소 발견 앱 (5축 82점)

**v2 변경 지점**:
- 감정 JTBD 재정의: "혼잡 회피" 기능 중심 → **"나만 아는 곳을 발견하는 설렘"** 감정 중심
- Hero Screen 재설계: 혼잡도 히트맵 → **"지금 어떤 발견을 원해?"** 감정 선택 첫 화면
- 4계절 숨은 명소 콘텐츠 분기: 같은 숨은 명소의 계절별 다른 매력
- 국내/외국인 이중 렌즈: 동일 로컬 명소의 한국인/외국인 의미 차이
- 5성향별 숨은 명소 추천 로직 강화

## Hero Screen 묘사

어두운 배경에 빛나는 별들처럼 지도 위에 점들이 깜박인다.
상단에 한 줄:
**"오늘, 어떤 발견을 원해?"**

아래 4개 선택:
- ✨ **비밀** — 아직 아무도 모르는 곳
- 🌿 **고요** — 사람 없는 자연
- 🏮 **정취** — 오래된 골목과 시간
- 📸 **계절** — 지금만 볼 수 있는 풍경

선택 즉시 해당 감정에 맞는 숨은 명소 목록 표시. 트리플·야놀자는 인기 순위를 보여주지만, 이 앱은 **혼잡도 최하위 + 관광 다양성 지수 기반**으로만 추천한다.

| 구분 | 트리플 / 야놀자 / Klook | LocalSecret v2 |
|---|---|---|

| 첫 화면 | 목적지 검색 | 발견 감정 선택 |

| 기존 v1 | 혼잡도 히트맵 중심 | **감정 발견 UI + 4계절 숨은 명소 + 이중 렌즈** |

## 성향별 시나리오 3개

### 시나리오 1 — 조용한 독립 여행자 (20대)
이서준(26세, 사진작가 지망생)이 "비밀"을 선택한다. 앱이 "지금 이 순간 가장 한산한 곳 TOP 3"를 표시 — 경기도 양주 天寶山 산정 호수, 충남 부여 궁남지 새벽 산책로, 전남 담양 메타세쿼이아 샛길. 수요 밀도 하위 5%, 방문자 예측 혼잡도 8%. 아무에게도 알리지 않고 혼자 간다.

### 시나리오 2 — 전통정취 선호 커플 (30대)
박민준·이수아(34/32세)가 "정취"를 선택한다. 앱이 "오래된 골목 + 현재 방문객 12명" 장소를 추천 — 전북 군산 신흥동 일본식 가옥 골목, 경북 안동 임청각 인근 하회마을 외곽. 혼잡한 본 하회마을 대신 "하회마을 외곽 3km 토담길 — 지금 방문객 0명"을 우선 제안.

### 시나리오 3 — 외국인 "진짜 한국" 탐험가
캐나다인 Sarah(29세)가 "HIDDEN" 모드로 서울에서 앱 실행. 앱이 서울 내 관광지도에 없는 곳을 표시: 마포구 연남동 경의선 숲길 비공개 구간, 도봉구 도봉산 무명 약수터. 영문으로 "Not on any tourist map — visited by locals only"라는 태그가 붙어 있다. 한국인 현지인 관점("주민 산책로")과 외국인 관점("authentic Seoul off the beaten path")을 병기.

## 계절별 분기

**같은 숨은 명소, 다른 계절 콘텐츠 — 충남 태안 신두리 해안사구 예시**:

| 계절 | 숨은 매력 포인트 | 추천 방문 타이밍 |
|---|---|---|
| 봄 | 사구 위 해당화 군락 — 5월 한정 | 이른 아침, 방문객 없음 |
| 여름 | 모래 언덕 석양 — 해수욕객 없는 구역 | 오후 6시 이후 |
| 가을 | 갈대밭과 모래 대비 — 사진 최적 | 주중 오전 |
| 겨울 | 눈 덮인 사구 — 연 3-4일만 가능 | 첫눈 다음날 새벽 |

## 국내/외국인 이중 렌즈

**같은 POI: 인천 강화도 동막해변 인근 논길**

- 🇰🇷 한국인 관점: "어릴 때 외갓집 가던 길 느낌 — 논과 갯벌이 보이는 아무도 모르는 산책로. SNS에 올리면 인기 생길까봐 안 올리게 되는 곳"
- 🌍 외국인 관점: "Korea's countryside farmland trail — rice paddies, tidal flats, and no crowds. What rural Korea looked like 30 years ago, still preserved"

## 활용 API (한글 파일명 기준)

| 한글 API 파일명 | 서비스 내 역할 | 호출 빈도 |
|---|---|---|
| `한국관광공사_관광지 집중률 방문자 추이 예측 정보` (visitor-concentration-forecast) | 향후 30일 혼잡도 예측 — 핵심 추천 필터 | 실시간 |
| `한국관광공사_지역별 관광 다양성` (area-tourism-diversity) | 덜 알려진 지역 발굴 — 다양성 지수 낮은 곳 보완 | 일 1회 캐시 |
| `한국관광공사_지역별 관광 수요 강도` (area-tourism-demand-density) | 수요 밀도 하위 구간 선별 | 추천 시 |
| `한국관광공사_관광지별 연관 관광지 정보` (related-attractions) | 혼잡한 명소의 한적한 대체지 추천 | 추천 시 |
| `한국관광공사_관광빅데이터 정보서비스` (tourism-big-data) | "아직 덜 알려진 곳" 트렌드 감지 | 일 1회 캐시 |
