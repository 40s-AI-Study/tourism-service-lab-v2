## 한 줄 정의

관광지 QR 스캔 한 번으로 한·영·일·중 4개 언어 오디오 가이드와 사진 공모전 수상작을 동시에 즐기는 몰입형 문화재 스토리텔링 앱.

## 타겟 사용자

- **주 타겟**: 역사·문화재에 관심 있는 20-40대 내국인, 한국 문화 관광객 외국인
- **세부 세그먼트**: 혼자 여행하는 솔로 여행자, 인솔자 없이 자유 관람하는 외국인 개별 관광객(FIT), 문화재 해설사 대기 시간 기피층
- **부 타겟**: 학교 현장학습 단체, 시각장애인 등 접근성 필요 사용자

## 핵심 기능

- **QR 스캔 즉시 오디오 가이드 실행**: 관광지 안내판 QR 스캔 시 해당 문화재 오디오 가이드 즉시 재생 — 대기 없는 셀프 해설
- **4개 언어 동시 지원**: 한국어·영어·일본어·중국어간체 관광 정보 API를 통합해 언어 선택 즉시 전환 가능
- **사진 공모전 수상작 연동 갤러리**: 오디오 재생 중 해당 문화재 수상 사진을 슬라이드쇼로 함께 감상 — 청각+시각 몰입 경험
- **오프라인 다운로드**: 와이파이 환경에서 사전 다운로드 후 현장에서 데이터 없이도 완전한 가이드 이용 가능
- **스토리 챕터 구성**: 단순 설명이 아닌 '건립 배경 → 주요 사건 → 현재 의미' 챕터 형식의 내러티브 구성으로 몰입감 강화

## 활용 API

| API | 활용 목적 |
|-----|-----------|
| `audio-guide` | 문화재·관광지별 오디오 가이드 콘텐츠 — 핵심 기능의 콘텐츠 소스 |
| `photo-contest-winners` | 사진 공모전 수상작 — 오디오 재생 중 고품질 비주얼 콘텐츠 연동 |
| `tourism-info-english` | 영어 관광 정보 — 외국인 영어권 사용자 대응 |
| `tourism-info-japanese` | 일본어 관광 정보 — 일본인 관광객 대응 |
| `tourism-info-chinese-simplified` | 중국어간체 관광 정보 — 중국인 관광객 대응 |

## 차별화 포인트

- **audio-guide API 유일 활용**: 공모전 참가작 중 오디오 가이드 API를 핵심 기능으로 배치한 앱 전무 — 블루오션 포지셔닝
- **다국어 + 오디오 조합**: 텍스트 다국어 지원은 흔하지만 오디오 + 4개 언어 조합은 접근성과 몰입감을 동시에 달성
- **청각 우선 UX**: 스마트폰 화면을 보지 않아도 걸으면서 감상 가능 — 문화재 현장 특성에 최적화된 인터페이스
- **콘텐츠 자산 활용 극대화**: 한국관광공사의 오디오 가이드 + 사진 공모전 두 가지 자산을 동시 활용하는 유일한 설계

## 공모전 강점

- **audio-guide API 독점 활용**: 다른 참가팀이 간과하는 API를 핵심으로 배치 — 심사위원 신선도·차별성 가점 기대
- **외국인 관광객 접근성 극대화**: 4개 언어 오디오 지원으로 Visit Korea 2026 외국인 유치 목표와 직결
- **문화재 해설 인프라 보완**: 전국 문화재 해설사 부족 문제를 기술로 해결 — 정책적 필요성 어필 가능
- **장애인·고령자 접근성**: 오디오 중심 UI로 디지털 취약계층 포용 — 공익성·포용적 관광 심사 기준 충족

## 예상 사용 시나리오

> **"경복궁에 왔는데 해설사 투어는 마감, 안내판은 읽기 불편하다"**
>
> 외국인 관광객(일본인)이 경복궁 근정전 앞 QR 코드를 Audio Story Korea 앱으로 스캔.
>
> 앱이 일본어를 자동 감지해 "근정전 오디오 가이드 — 일본어" 즉시 실행. 오디오가 "이곳은 조선왕조의 심장부..."로 시작되는 동안 화면에는 사진 공모전 대상 수상작 슬라이드쇼가 재생됨.
> 챕터 2 "임진왜란과 복원의 역사" 구간에서 사용자가 일시정지 후 해당 챕터만 반복 청취. 관람 후 앱 내 "오늘 방문한 곳" 기록에 근정전이 자동 저장되고, 다음 방문지 창덕궁 오디오 가이드 미리 다운로드를 권유받음.
