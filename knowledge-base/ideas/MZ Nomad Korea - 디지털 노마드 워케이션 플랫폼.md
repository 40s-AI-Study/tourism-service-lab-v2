# 아이디어 20: MZ Nomad Korea

## 한 줄 정의
원격 근무자·디지털 노마드를 위해 **고속 Wi-Fi 카페·공유오피스·숙소·관광지**를 "2주+ 체류 단위"로 묶어주는 한국형 워케이션 플랫폼. 외국인 노마드 비자(2024년 시행) 대응 포함.

## 타겟 사용자
- **1차**: 국내 MZ 원격근무자 (IT·디자인·마케팅 프리랜서) — 2024 추정 130만명
- **2차**: 외국인 디지털 노마드 — 2024년 한국 "디지털 노마드 비자(F-1-D)" 시행, 방한 노마드 급증 예상

## 문제 정의 / 근거
- 한국 디지털 노마드 비자 시행 (2024년 1월) — 글로벌 노마드 시장 공략 정책
- 현재 "한국에서 2주 체류하며 일하면서 관광" 정보는 블로그에 분산
- Pain Point:
  - Wi-Fi 빠른 카페·공유오피스 정보 부족 (Nomadlist는 서울 10개 미만 등재)
  - 월 단기 임대 숙소 정보 제한적
  - "일하는 시간 + 관광 시간" 균형 잡는 루틴 앱 없음
- 제주·부산·강릉 등 워케이션 허브 지역 정책 지원 가시화

## 핵심 기능
1. **Wi-Fi 인증 POI**: 다운로드 50Mbps+ 인증 카페·공유오피스 필터 (사용자 실측 + 공식)
2. **2주 워케이션 코스**: 도시 선택 + 체류 기간 → 평일 업무 루틴 + 주말 관광 + 맛집 자동 조합
3. **공유오피스 예약**: 패스트파이브·WeWork·로컬 공유오피스 통합 예약 (Phase 2)
4. **노마드 비자 가이드**: F-1-D 비자 신청·연장·관리 통합 튜토리얼 (영·한)
5. **노마드 커뮤니티**: 도시별 현지 노마드 밋업·이벤트 (Discord/Slack 연계)

## 활용 API
- `tourism-info-korean/english` — 도시별 관광 정보
- `gocamping` — 글램핑·워케이션 숙박 조합
- `area-tourism-diversity` — 지역별 자원 다양성 (업무+관광 균형)
- `related-attractions` — 공유오피스 주변 명소
- `wellness-tourism` — 장기 체류 중 힐링 스팟

## 차별화 포인트
- 한국 워케이션 전문 앱 전무 (외국 Nomadlist·SafetyWing은 영어권 중심, 한국 정보 부족)
- 비자 가이드 통합 = 정책 대응 가점
- 지자체 워케이션 정책 (제주·부산 등)과 파트너십 = 지역 송객 효과
- B2B: 공유오피스 예약 수수료 + 지자체 정책비 수주

## 예상 사용 시나리오
```
34세 미국 개발자 (2개월 한국 체류, F-1-D 비자):
  앱: "제주 · 4주 체류 · Wi-Fi 80Mbps+ · 예산 월 200만원" 입력
  제공:
    숙소: 제주시 1개월 단기 임대 월 90만원 (해변 뷰)
    업무: 제주 패스트파이브 (월 35만원 HotDesk)
    평일 카페: Wi-Fi 100Mbps+ 3곳 + 각 포토
    주말 관광: 4주 × 2일 = 8일 코스 자동 조합
    밋업: 매주 수요일 제주 개발자 커뮤니티 참여 권장
  비자 갱신 일정 알림 푸시 (만료 30일 전)
```

## 예상 사업성 (1-10)
- 시장 규모: 7 (국내 원격근무 130만 + 외국인 노마드)
- 차별화: 9 (한국 전문 워케이션 앱 전무)
- 구현 난이도: 7 (공유오피스·숙소 파트너십 필요)
- 공모전 적합성: 9 (디지털 노마드 비자 정책 + 지방 관광 활성화)
- **총점: 32/40**

## 리스크
- 공유오피스 파트너십 — 3-5개 주요 브랜드와 초기 MVP 연계
- Wi-Fi 실측 데이터 수집 — 사용자 기여 + 공식 인증 병행
- 정책 변경 리스크 — F-1-D 비자 조건 변동 시 즉시 업데이트
