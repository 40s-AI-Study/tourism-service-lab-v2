# 반려동물 맞춤 여행 코스 생성기

## 핵심 아이디어
반려동물의 품종/크기/나이/성격을 입력하면 그에 맞는 여행 코스를 자동 생성. 대형견과 소형견, 고양이와 개의 여행 코스가 다르게 설계.

## 타겟
- 처음 반려동물과 여행 계획하는 초보 반려인
- 특수 반려동물(고양이/토끼 등) 보유 여행자

## 핵심 기능
- 반려동물 프로필 설정 (종류/크기/나이/특이사항)
- 프로필 기반 입장 가능 장소 자동 필터링
- 이동 거리/시간 고려한 무리 없는 코스 설계
- 날씨 고려 (여름 뜨거운 바닥 주의 알림)
- 반려동물 사진 스팟 추천

## 활용 API
- `한국관광공사_반려동물_동반여행_서비스`
- `한국관광공사_관광지 집중률 방문자 추이 예측 정보` (혼잡 회피)
- `한국관광공사_관광지별 연관 관광지 정보`

## 차별화
반려동물 개체 특성 고려한 개인화. 단순 목록 제공이 아닌 "우리 아이에게 맞는" 코스.

## 기대 효과
반려인 만족도 극대화, 앱 재사용률 증가
