## 상세 평가 근거

### 1위: KoreaPath AI (44점)
- **차별화 만점(10)**: AI 코스 자동 생성 서비스가 한국 시장에 전무 → 심사위원 임팩트 최대
- **API 활용도(9)**: visitor-concentration-forecast + related-attractions + central-attractions + tourism-big-data + area-tourism-demand-density — 5개 연동
- **공모전 적합성(9)**: "AI × 혼잡도 × 개인화" 3축 조합이 기획력 30점 항목에서 강점

### 2위: K-Guide Global (44점)
- **시장 규모(9)**: 방한 외래객 1,620만명 (2024) → 가장 큰 잠재 사용자
- **API 활용도(10)**: 다국어 API 8종 + audio-guide = 9개 API 활용 → 데이터 활용 20점 만점 기대
- **구현 가능성(9)**: 기존 Visit Korea 데이터 재패키징 → 구현 난이도 낮음

### 3위: AccessKorea (40점)
- **차별화(9)**: 무장애 통합 서비스 한국 시장 전무. barrier-free-travel API 단독 특화
- **공모전 적합성(9)**: 사회적 가치 심사항목 직결, 심사위원 호감도 높음. 특별상 가능성
- 시장 규모는 시니어(970만)+장애인(263만) 합산으로 실질 여행 빈도 감안

### 4위: PetKorea (39점)
- 반려동물 가구 600만, 시장 연 25% 성장. pet-friendly API 7개 카테고리 전부 활용
- 커뮤니티+바이럴 가능성 있으나 공모전 심사 차별성 포인트 상대적으로 약함

### 5위: KoreaTrend Radar (39점)
- 빅데이터 API 4종 동시 활용으로 데이터 활용 점수 강점
- 단, '일반 사용자 앱' vs '분석 대시보드' 포지셔닝이 불명확 → 기획력 감점 가능
