# 혼잡 피크 회피 방문 타이밍 추천앱

## 핵심 아이디어
관광지 방문자 집중률 예측 API를 활용해 "지금 가면 한산한 시간대"를 실시간 알림으로 제공. 사용자가 원하는 관광지를 입력하면 향후 30일간 혼잡도 예측 데이터를 기반으로 최적 방문 시간대를 추천.

## 타겟
- 20-40대 한국인, 주말 관광객
- 혼잡을 피하고 싶은 MZ 세대 (Nielsen 2024: 55%+ 선호)

## 핵심 기능
- 관광지 검색 → 혼잡도 30일 예측 캘린더 표시
- "지금 방문 적기" 푸시 알림
- 혼잡 수준별 색상 코드 (빨강/노랑/초록)
- 알림 등록 후 임계치 이하 되면 자동 알람

## 활용 API
- `한국관광공사_관광지 집중률 방문자 추이 예측 정보`
- `한국관광공사_국문 관광정보 서비스`

## 차별화
기존 Google Maps "Popular Times"는 주간 평균치만 제공. 본 서비스는 한국 관광지 특화 30일 예측 데이터 활용.

## 기대 효과
관광지 혼잡 분산 효과, 방문자 만족도 향상

## AI 역할
- **ai_role**: `ai-assist` — 혼잡도 예측 데이터 시각화 및 알림 로직에 AI 보조
- 30일 혼잡도 예측 데이터를 패턴 분석해 "베스트 방문 시간대" 자동 산출
- 사용자 이전 방문 패턴 학습으로 개인 맞춤 알림 임계치 설정
- 날씨·공휴일 데이터 결합으로 혼잡도 예측 정확도 향상

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 알림 | 복수 관광지 동시 모니터링 | 월 1,900원 |
| 관광지 파트너십 | 한산 시간대 특별 할인 쿠폰 연결 | 거래액의 3% |
| 여행사 API | 혼잡도 예측 데이터 B2B 공급 | 월 50만원~/건 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 주말 관광객 약 1,200만명/월
- **SAM**: 혼잡 회피 선호 여행자 약 660만명 (전체의 55%)
- **SOM**: 출시 2년 내 MAU 20만 목표 → 구독 기준 연 약 46억원
- **근거**: Nielsen 2024 "55% 이상 혼잡 회피 선호", Google Popular Times 한계 인식 증가

## 편의성
- 관광지 등록 1회 → 이후 알림 자동 수신
- 혼잡도 색상 코드(빨/노/초)로 한눈에 상황 파악
- 알림 수신 즉시 해당 관광지 정보·경로 바로 접근
- 위젯 지원으로 앱 실행 없이 혼잡도 확인 가능
