# 아이디어 18: Silver Journey

## 한 줄 정의
55세+ 은퇴자·준고령층을 위한 **체력·이동거리·의료 접근성 3축**을 고려한 맞춤 슬로우 여행 플래너. 하루 도보 1km 이내 원칙 + 휴식 간격 + 의료시설 거리 자동 계산.

## 타겟 사용자
- **1차**: 55-70세 은퇴 또는 은퇴 임박 국내 여행자 — 1,000만명+ (총 인구 19%)
- **2차**: 65세+ 방한 외국인 (일본·대만) — 꾸준한 실버 관광 수요

## 문제 정의 / 근거
- 한국 65세 이상 인구 약 970만명, 향후 10년간 1,300만 돌파 예상
- 은퇴자 여행 소비력 높으나(가구당 월 여행비 22만원+) **맞춤 서비스 부재**
- 기존 관광 앱은 2030 중심 UX — 글자 크기·걷는 거리·화장실·의료시설 고려 안 됨
- FamilyKorea는 "노부모 동반" 서브 필터 수준, 실버 전용 앱은 전무

## 문제 정의 / 근거 (Pain Points)
- 장시간 서 있기 어려움 → 카페·벤치 간격 10분 이내
- 낯선 화장실 찾기 어려움 → 공공 화장실 밀도 중요
- 의료 상황 불안 → 병원·약국 15분 이내 접근성 필요
- 크고 복잡한 앱 UX 어려움 → 큰 글자·버튼·음성 지원 필수

## 핵심 기능
1. **체력 레벨 기반 코스**: 상/중/하 3단계 → 하루 이동 800m / 2km / 5km 맞춤
2. **휴식 간격 자동 삽입**: 30-40분 활동 후 10분 벤치·카페 블록 자동 포함
3. **의료 접근성 필터**: 현재 위치 기준 병원·약국 15분 이내 여부 실시간 표시
4. **시니어 UX**: 글자 크기 조절 (기본 18pt+), 큰 버튼, TTS 음성 읽어주기
5. **동반자 공유**: 자녀가 원격에서 부모 여행 일정 확인·알림 받기 모드

## 활용 API
- `barrier-free-travel` — 무장애 시설 정보 (계단·경사·엘리베이터)
- `central-attractions-by-municipality` — 지자체별 편의시설 다수 명소 우선
- `visitor-concentration-forecast` — 한산한 시간대 (노인 붐비는 곳 기피)
- `tourism-info-korean` — 한글 큰 글자 UX
- `eco-tourism` — 도보 부담 적은 수목원·공원

## 차별화 포인트
- 실버 전용 국내 관광 앱 전무 (유럽·일본은 일부 존재)
- 의료 접근성 연동 + 동반자 공유 모드 = 가족·자녀의 안심도 확보
- 고령화 가속 시장 → 장기 성장성 확실 (연평균 +4% 인구 증가 세그먼트)
- 사회적 가치 = 공모전 심사 RTO 특별상 유력

## 예상 사용 시나리오
```
67세 은퇴자 부부 (부산 2박 3일):
  앱: "체력 중 / 의료 접근성 필수 / 부부 동반" 선택
  Day 1:
    09:00 해운대 벤치 산책 (400m) + 10분 카페 휴식
    10:30 미포 횟집거리 (휠체어 접근 OK 식당 3곳 마킹)
    12:00 점심 후 호텔 1시간 휴식
    14:00 동백섬 관광 (평지 600m 산책로)
    15:30 카페 귀가
  각 지점마다 가까운 병원·약국 이름 표시 (부산백병원 1.2km 등)
  자녀(서울 거주)에게 일정 공유 — 매일 저녁 "오늘 무탈하셨어요" 푸시
```

## 예상 사업성 (1-10)
- 시장 규모: 9 (65+ 인구 970만, 성장세 확실)
- 차별화: 9 (실버 전용 앱 전무)
- 구현 난이도: 6 (구현 자체 단순, 시니어 UX 테스트 필요)
- 공모전 적합성: 9 (사회적 가치 최상 + 인구 구조 정책 부합)
- **총점: 33/40**

## 리스크
- 시니어 디지털 접근성 — 인스톨·온보딩 도움말 필수
- 자녀 중심 마케팅 — 실사용자는 부모이지만 설치는 자녀가 해주는 구조
- 의료 정보 정확성 — 실시간 DB 업데이트 지자체 협업 필수
