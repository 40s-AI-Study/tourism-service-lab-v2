# 시니어 특화 여행 설계 컴패니언

## 핵심 아이디어
60대 이상 시니어의 체력/이동 편의/선호도를 고려한 여행 설계. 대중교통 환승 최소화, 무리한 걷기 없는 코스, 큰 글씨 UI.

## 타겟
- 65세 이상 시니어 여행자 (약 970만명)
- 시니어 부모와 여행 계획하는 40대 자녀

## 핵심 기능
- 이동 거리/걷기 시간 기준 코스 필터 (1km 이내 등)
- 대중교통 환승 횟수 최소화 경로
- 시니어 할인 정보 통합 (경로우대 혜택)
- 무장애 접근 가능 명소 우선 표시
- 큰 글씨/고대비 UI 모드
- 자녀가 부모 여행 일정 공유/모니터링

## 활용 API
- `한국관광공사_무장애 여행 정보`
- `한국관광공사_국문 관광정보 서비스`
- `한국관광공사_관광지 집중률 방문자 추이 예측 정보`

## 차별화
초고령화 사회 대응 서비스. 시니어 당사자 + 보호자 자녀 동시 타겟.

## 기대 효과
시니어 관광 시장 확대, 세대 간 여행 연결
