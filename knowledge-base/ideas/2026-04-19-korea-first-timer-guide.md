# 한국 첫 방문 외국인 AI 여행 가이드

## 핵심 아이디어
방한 외국인이 "한국에서 뭐해야 해?"를 해결해 주는 AI 가이드. 국적/관심사/체류 기간 입력 시 공식 관광공사 데이터 기반의 맞춤 일정 자동 생성. 다국어 지원.

## 타겟
- 방한 외래객 (2024년 약 1,620만명)
- 한국 첫 방문 20-40대 외국인
- K-문화 팬 (K-팝/K-드라마 시청자)

## 핵심 기능
- 국적 선택 → 해당 언어 자동 적용 (9개 언어)
- 관심사 선택 (K-팝/음식/전통/자연/쇼핑)
- 체류 기간별 일정 자동 생성
- 오프라인 저장 (데이터 없이 사용)
- 긴급 연락처/병원 안내 포함

## 활용 API
- `한국관광공사_영문 관광정보서비스`
- `한국관광공사_일문 관광정보서비스`
- `한국관광공사_중문 간체 관광정보서비스`
- `한국관광공사_중문 번체 관광정보서비스`
- `한국관광공사_의료관광정보`

## 차별화
Visit Korea 앱 대비 현대적 UX + AI 개인화. 공식 데이터 신뢰성 + 맞춤형 추천.

## 기대 효과
방한 외국인 재방문율 증가, 한국 관광 브랜드 경쟁력 강화

## AI 역할
- **ai_role**: `ai-only` — 외국인 맞춤 일정 생성부터 다국어 안내까지 전 과정 AI 주도
- 국적·관심사·체류기간 3가지 입력만으로 최적 일정 자동 생성 (LLM + 관광공사 API 결합)
- 9개 언어 실시간 자연어 처리로 완전 현지화된 가이드 제공
- 대화형 AI 가이드: "지금 어디야?" 질문에 실시간 맞춤 대응
- 사용자 행동 패턴 학습으로 국적별 선호 일정 모델 지속 개선

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 AI 플랜 | 개인화 일정 + 오프라인 가이드 | 1회 9,900원 |
| 예약 수수료 | 관광지·식당·체험 예약 연결 | 예약액의 6% |
| 항공사/호텔 광고 | 방한 외국인 타겟 광고 | CPM 기반 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 방한 외래객 연 약 1,620만명 (2024) × 평균 여행 앱 지출
- **SAM**: 디지털 앱으로 여행 계획하는 방한 외국인 약 800만명
- **SOM**: 출시 2년 내 다운로드 50만 목표 → 프리미엄 전환 10% 기준 연 약 50억원
- **근거**: Visit Korea 앱 UI 노후화, K-문화 팬 급증으로 방한 외국인 증가 추세

## 편의성
- 국적 선택 즉시 해당 언어 자동 전환 (설정 불필요)
- 3가지 입력으로 완성된 여행 일정 즉시 생성 (2분 내)
- 오프라인 저장으로 해외 로밍 없이도 가이드 완전 사용
- 긴급 연락처·병원 안내를 홈 화면에서 1터치로 접근 가능
