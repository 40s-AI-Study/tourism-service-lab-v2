# 한국 첫 방문 외국인 AI 여행 가이드

## 핵심 아이디어
방한 외국인이 "한국에서 뭐해야 해?"를 해결해 주는 AI 가이드. 국적/관심사/체류 기간 입력 시 공식 관광공사 데이터 기반의 맞춤 일정 자동 생성. 다국어 지원.

## 타겟
- 방한 외래객 (2024년 약 1,620만명)
- 한국 첫 방문 20-40대 외국인
- K-문화 팬 (K-팝/K-드라마 시청자)

## 핵심 기능
- 국적 선택 → 해당 언어 자동 적용 (9개 언어)
- 관심사 선택 (K-팝/음식/전통/자연/쇼핑)
- 체류 기간별 일정 자동 생성
- 오프라인 저장 (데이터 없이 사용)
- 긴급 연락처/병원 안내 포함

## 활용 API
- `한국관광공사_영문 관광정보서비스`
- `한국관광공사_일문 관광정보서비스`
- `한국관광공사_중문 간체 관광정보서비스`
- `한국관광공사_중문 번체 관광정보서비스`
- `한국관광공사_의료관광정보`

## 차별화
Visit Korea 앱 대비 현대적 UX + AI 개인화. 공식 데이터 신뢰성 + 맞춤형 추천.

## 기대 효과
방한 외국인 재방문율 증가, 한국 관광 브랜드 경쟁력 강화
