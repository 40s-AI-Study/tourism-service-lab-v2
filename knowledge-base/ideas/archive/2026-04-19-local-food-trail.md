# 지역 특산 맛집 루트 자동 생성기

## 핵심 아이디어
MZ세대 여행지 선택 1순위 "맛집"을 공략. 지역별 특산 음식과 숨은 맛집을 관광 동선에 자연스럽게 녹여 "미식 투어 루트"를 자동 생성.

## 타겟
- 미식 중심 여행자 20-40대
- "맛집 없으면 안 가요" 인식의 MZ세대

## 핵심 기능
- 지역 선택 → 특산 음식 + 추천 식당 + 관광지 혼합 루트
- 식사 시간대 고려한 동선 배치 (아침/점심/저녁/야식)
- 빅데이터 기반 현지인 vs 관광객 맛집 분리 표시
- 계절별 특산물 캘린더 연동
- 먹방 루트 저장/공유 기능

## 활용 API
- `한국관광공사_국문 관광정보 서비스` (음식점 카테고리)
- `한국관광공사_관광빅데이터 정보서비스`
- `한국관광공사_관광지별 연관 관광지 정보`

## 차별화
관광지 + 식당을 동선으로 통합. 단순 맛집 앱(망고플레이트)이 아닌 "미식 여행 코스" 설계.

## 기대 효과
MZ세대 핵심 여행 동기 해소, 지역 음식 산업 활성화

## AI 역할
- **ai_role**: `ai-only` — 미식 여행 루트 생성 전 과정 AI 주도
- 지역 특산 음식 + 관광지 + 식사 시간대를 동시에 고려한 최적 동선 자동 생성
- 빅데이터 방문자 패턴 분석으로 현지인 vs 관광객 맛집 자동 분류
- 계절·제철 데이터 실시간 반영으로 시즌 맞춤 미식 루트 생성
- 사용자 음식 선호 학습으로 개인화 루트 정확도 지속 향상

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 구독 | 맞춤 루트 무제한·저장·공유 | 월 2,900원 |
| 음식점 제휴 | 예약·쿠폰 연결 수수료 | 거래액의 5% |
| 지역 식품 광고 | 지역 특산물·식당 프로모션 | 월 10만원/업체 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 미식 여행 시장 약 8,000억원 (여행 지출 중 음식비 비중 30%)
- **SAM**: 맛집 중심 여행 의사결정 MZ세대 약 500만명
- **SOM**: 출시 2년 내 MAU 25만 목표 → 구독+수수료 기준 연 약 100억원
- **근거**: 국내 여행 동기 1위 "음식" (한국관광공사 2024), 미식 여행 수요 지속 증가

## 편의성
- 지역 선택 1회로 아침·점심·저녁·야식 포함 미식 동선 즉시 생성
- 식사 시간대 자동 배치로 이동 중 끼니 걱정 불필요
- 루트 저장→카카오맵·네이버지도 내보내기 원클릭
- 먹방 루트 SNS 공유 템플릿 자동 생성
