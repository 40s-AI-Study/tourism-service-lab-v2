# 시니어 특화 여행 설계 컴패니언

## 핵심 아이디어
60대 이상 시니어의 체력/이동 편의/선호도를 고려한 여행 설계. 대중교통 환승 최소화, 무리한 걷기 없는 코스, 큰 글씨 UI.

## 타겟
- 65세 이상 시니어 여행자 (약 970만명)
- 시니어 부모와 여행 계획하는 40대 자녀

## 핵심 기능
- 이동 거리/걷기 시간 기준 코스 필터 (1km 이내 등)
- 대중교통 환승 횟수 최소화 경로
- 시니어 할인 정보 통합 (경로우대 혜택)
- 무장애 접근 가능 명소 우선 표시
- 큰 글씨/고대비 UI 모드
- 자녀가 부모 여행 일정 공유/모니터링

## 활용 API
- `한국관광공사_무장애 여행 정보`
- `한국관광공사_국문 관광정보 서비스`
- `한국관광공사_관광지 집중률 방문자 추이 예측 정보`

## 차별화
초고령화 사회 대응 서비스. 시니어 당사자 + 보호자 자녀 동시 타겟.

## 기대 효과
시니어 관광 시장 확대, 세대 간 여행 연결

## AI 역할
- **ai_role**: `ai-assist` — 시니어 체력 기반 코스 필터링 및 대중교통 경로 최적화에 AI 보조
- 이동 거리·경사·걷기 시간 복합 분석으로 시니어 적합 코스 자동 필터링
- 대중교통 환승 최소화 경로 자동 생성 (시니어 이동 편의 최우선)
- 시니어 할인 정보 자동 매칭 (경로우대·노인 할인 혜택 통합)

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 여행 코스 예약 수수료 | 시니어 맞춤 패키지 예약 | 예약액의 7% |
| 자녀 프리미엄 구독 | 부모 일정 모니터링·알림 | 월 3,900원 |
| 시니어 보험·건강 광고 | 시니어 타겟 광고 | CPM 기반 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 65세 이상 시니어 약 970만명 × 여행 지출 시장
- **SAM**: 디지털 앱 사용 시니어 여행자 약 200만명 (스마트폰 보급률 고려)
- **SOM**: 출시 2년 내 MAU 8만 목표 → 구독+수수료 기준 연 약 35억원
- **근거**: 초고령화 사회 진입 2025년, 시니어 여행 지출 증가 추세, 자녀 세대 구독 수요

## 편의성
- 큰 글씨/고대비 UI로 시니어도 별도 설정 없이 편리하게 사용
- 걷기 거리 1km 이내 필터로 체력 부담 없는 코스 즉시 선택
- 시니어 할인 정보 자동 표시 → 별도 검색 불필요
- 자녀가 부모 일정 원격 공유·확인 (보호자 안심 기능)
