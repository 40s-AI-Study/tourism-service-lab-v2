# AI 연관 관광지 동선 자동 최적화

## 핵심 아이디어
관광지 A를 입력하면 연관 관광지 50위 데이터와 혼잡도 예측을 결합해 하루 최적 동선을 자동 생성. 이동 거리 최소화 + 혼잡 회피 + 선호 카테고리 필터를 동시에 고려.

## 타겟
- 처음 방문하는 지역 여행자
- 여행 계획을 짜기 어려워하는 직장인 20-40대

## 핵심 기능
- 대표 관광지 1개 입력 → 1일/2일/3일 코스 자동 생성
- 시간대별 혼잡도 반영한 방문 순서 정렬
- 동선별 이동 소요시간 표시
- 코스 저장/공유 기능

## 활용 API
- `한국관광공사_관광지별 연관 관광지 정보`
- `한국관광공사_관광지 집중률 방문자 추이 예측 정보`
- `한국관광공사_기초지자체 중심 관광지 정보`

## 차별화
국내 서비스 중 관광공사 공식 연관 데이터 + 혼잡도 예측 결합한 자동 코스 생성 서비스 전무.

## 기대 효과
여행 계획 시간 80% 단축, 관광지 분산 방문 유도

## AI 역할
- **ai_role**: `ai-only` — 동선 최적화 알고리즘 전체가 AI 주도
- TSP(외판원 문제) 변형 알고리즘으로 이동거리 최소화 동선 자동 산출
- 혼잡도 예측 데이터를 실시간 반영한 방문 순서 동적 재정렬
- 사용자 이동 패턴 학습으로 개인화 동선 추천 정확도 향상
- 자연어 입력("경복궁 근처 하루 코스") → 의도 파악 → 코스 생성

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 구독 | 다일정 코스·오프라인 저장 무제한 | 월 2,900원 |
| 관광지 제휴 | 동선 내 입장권·할인 쿠폰 연결 | 거래액의 5% |
| API 라이선스 | 여행사·OTA에 동선 최적화 API 제공 | 월 100만원~/ 건 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 여행 계획 앱 시장 약 5,000억원
- **SAM**: 처음 방문 지역 여행 계획 니즈 보유자 약 800만명
- **SOM**: 출시 2년 내 MAU 50만 목표 → 구독+제휴 기준 연 약 200억원
- **근거**: "여행 계획에 평균 3시간 소요" → 자동화 수요 명확

## 편의성
- 관광지 이름 1개 입력만으로 1~3일 코스 즉시 생성
- 코스 생성 소요시간 5초 이내 (사전 계산 캐시 활용)
- 코스 저장→공유 원클릭 (카카오톡/인스타 연동)
- 실시간 혼잡도 변화 시 대안 경로 자동 푸시 알림
