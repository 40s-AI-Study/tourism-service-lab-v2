# SNS/블로그 UGC 기반 실시간 숨은 명소 발굴

## 핵심 아이디어
강원관광 '가봄', 대한민국 구석구석 '다님' 기자단의 UGC와 SNS 해시태그 분석을 결합해 방금 막 발견된 숨은 명소를 실시간으로 큐레이션. "남들보다 먼저 아는" 정보 제공.

## 타겟
- SNS 기반 여행 결정 MZ세대
- 인스타그래머블 장소 먼저 발견하고 싶은 20-30대

## 핵심 기능
- KTO 공식 기자단 콘텐츠 실시간 피드
- SNS 버즈 급상승 명소 알림 (바이럴 예측)
- "이번 주 새로 발굴된 명소" 주간 리포트
- 포토 스팟 사진 비교 (시즌별)
- 빠른 예약 연동 (명소 인기 전 선점)

## 활용 API
- `한국관광공사_관광빅데이터 정보서비스`
- `한국관광공사_관광사진 정보`
- `한국관광공사_관광공모전(사진) 수상작 정보`
- `한국관광공사_지역별 관광 자원 수요`

## 차별화
공식 KTO 기자단 콘텐츠 + SNS 트렌드 결합. "먼저 아는 자가 이긴다" 심리 공략.

## 기대 효과
MZ세대 높은 참여도, 바이럴 마케팅 효과

## AI 역할
- **ai_role**: `ai-only` — UGC 분석부터 바이럴 예측까지 전 과정 AI 주도
- NLP 기반 SNS/블로그 UGC 실시간 크롤링·분류·감성 분석
- 버즈 급상승 패턴 감지 알고리즘으로 "뜨기 전 명소" 사전 포착
- 사진 AI 분석으로 인스타그래머블 점수 자동 산출
- KTO 공식 기자단 콘텐츠 + 민간 SNS 데이터 융합 모델

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 알림 | 신규 명소 조기 알림·주간 리포트 | 월 3,900원 |
| 명소 선점 예약 | 숙박·체험 조기 예약 수수료 | 예약액의 6% |
| 관광청 파트너십 | 지자체 신규 명소 홍보 채널 | 프로젝트별 계약 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 SNS 기반 여행 앱 시장 약 5,000억원
- **SAM**: "남들보다 먼저 아는" 정보 선호 MZ세대 약 350만명
- **SOM**: 출시 2년 내 MAU 20만 목표 → 구독+예약 기준 연 약 95억원
- **근거**: MZ세대 여행 결정 1위 채널 SNS, 인스타 명소 선점 심리 강화

## 편의성
- 앱 열면 오늘의 신규 발굴 명소 자동 표시 (별도 검색 불필요)
- 알림 등록 1회로 관심 지역 신규 명소 즉시 수신
- 포토 스팟 사진 비교로 방문 전 장소 분위기 즉시 파악
- 예약 버튼으로 명소 인기 전 숙박·체험 선점 가능
