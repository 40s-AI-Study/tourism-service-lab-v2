# 웰니스 관광 + 명상/요가 + 숙박 통합 플래너

## 핵심 아이디어
웰니스 관광지, 명상 센터, 요가 스튜디오, 한방 스파, 자연 치유 숙박을 하나의 여행 패키지로 묶어 설계. 글로벌 웰니스 투어리즘 $8,940억 시장의 한국 버전.

## 타겟
- 웰니스 관광 관심 외국인 (BookRetreats: 64% 리트리트 계획)
- 힐링 여행 원하는 내국인 30-40대 직장인

## 핵심 기능
- 웰니스 프로그램 유형별 검색 (명상/요가/한방/삼림욕/온천)
- 1박 2일 ~ 3박 4일 힐링 패키지 자동 구성
- 웰니스 숙박 (한옥/게스트하우스/사찰) 예약 연동
- 웰니스 여행 전후 몸/마음 상태 체크인
- 영어/일어 지원 (외국인 웰니스 관광객)

## 활용 API
- `한국관광공사_웰니스관광정보`
- `한국관광공사_생태 관광 정보`
- `한국관광공사_영문 관광정보서비스`
- `한국관광공사_일문 관광정보서비스`

## 차별화
한국 전통 웰니스(한방/사찰음식)와 현대 웰니스(요가/명상)의 융합. 글로벌 웰니스 투어리즘 트렌드 직접 공략.

## 기대 효과
고부가가치 외국인 웰니스 관광객 유치, 한국 힐링 브랜드 구축

## AI 역할
- **ai_role**: `ai-only` — 웰니스 패키지 자동 구성 전 과정 AI 주도
- 사용자 건강 상태·선호 입력 → 최적 웰니스 프로그램 조합 자동 생성
- 웰니스 관광지·명상센터·숙박 트리플 최적화 (거리+가용성+컨디션 반영)
- 다국어 자연어 처리로 외국인 웰니스 관광객 온보딩 자동화
- 여행 전후 몸/마음 체크인 데이터로 개인화 모델 지속 학습

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 패키지 판매 수수료 | 웰니스 숙박+체험 패키지 예약 | 예약액의 10% |
| 외국인 프리미엄 | 영어/일어 전용 웰니스 플랜 | 건당 5~15만원 |
| 웰니스 업체 노출 | 한방원·요가원 프리미엄 등록 | 월 30만원/업체 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 글로벌 웰니스 투어리즘 시장 $8,940억 (GWRI 2024)
- **SAM**: 한국 웰니스 관광 시장 약 2,000억원 (방한 외국인 + 내국인)
- **SOM**: 출시 2년 내 월 예약 3만건 목표 → 수수료 기준 연 약 90억원
- **근거**: 외국인 웰니스 관광객 연평균 20% 성장, K-웰니스 글로벌 관심 증가

## 편의성
- 체류 기간·선호 웰니스 유형 입력 시 패키지 즉시 자동 구성
- 숙박·체험·이동 동시 예약 (3개 사이트 방문 불필요)
- 영어/일어 완전 지원으로 외국인 단독 예약 가능
- 웰니스 체크인 기록 자동 저장 → 다음 방문 시 개인화 강화
