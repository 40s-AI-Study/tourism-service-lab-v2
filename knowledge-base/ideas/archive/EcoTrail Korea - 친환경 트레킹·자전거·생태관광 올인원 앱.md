## 한 줄 정의

두루누비 코스 + 생태관광 + 탄소발자국 계산기를 하나로 — 환경을 지키며 한국을 걷고 달리는 에코투어 플랫폼.

## 타겟 사용자

- **1차**: 20-30대 환경의식 MZ세대 (기후 위기 체감, 저탄소 여행 지향)
- **2차**: 트레킹·자전거 여행 마니아 (두루누비 코스 사용자층)
- **3차**: 한국의 자연을 탐방하려는 외국인 에코투어리스트 (유럽·북미 중심)

## 핵심 기능

- **두루누비 코스 통합 탐색**: 전국 트레킹·자전거·수상 코스를 난이도·거리·지역·테마별로 필터링, 상세 경로와 고도 프로필 제공
- **탄소발자국 계산기**: 이동 수단(도보·자전거·대중교통·자가용)별 탄소 배출량 비교, 저탄소 이동 루트 우선 추천
- **생태관광지 큐레이션**: 생태관광 인증 명소·자연 해설 프로그램·철새도래지·습지 등 자연 특화 콘텐츠 제공
- **지역 관광 다양성 기반 에코 명소 발굴**: 관광 집중도가 낮은 비주류 자연 명소를 발굴해 생태 훼손 방지·지역 분산 효과
- **에코 챌린지 & 커뮤니티**: 저탄소 여행 달성 시 배지 획득, 탄소 절감량 누적 트래킹, 코스 후기 공유

## 활용 API

| API | 활용 내용 |
|---|---|
| `eco-tourism` | 생태관광 인증 명소·자연 해설 프로그램·체험 데이터 |
| `durunubi-trails` | 전국 트레킹·자전거·수상 코스 경로·난이도·거리 데이터 |
| `area-tourism-diversity` | 지역 관광 다양성 지표로 덜 알려진 자연 명소 발굴 |
| `tourism-big-data` | 계절별 자연 관광 트렌드·인기 코스 분석 및 추천 근거 |

## 차별화 포인트

1. **durunubi-trails + eco-tourism 최초 결합**: 두 API를 연계해 트레킹 코스와 생태관광 명소를 하나의 동선으로 통합한 유일한 서비스
2. **탄소발자국 계산기 내장**: 여행 자체의 환경 영향을 수치로 보여주는 기능으로 ESG·친환경 가치 소비자 직접 공략
3. **관광 분산 알고리즘**: 인기 명소 집중으로 인한 생태 훼손 문제를 `area-tourism-diversity` 데이터로 해소
4. **글로벌 에코투어 시장 대응**: 유럽·북미 중심 에코투어리즘 성장세에 맞춘 영문 콘텐츠 확장 준비

## 공모전 강점

- **친환경 가점 항목 직결**: 지속 가능한 관광·탄소 중립 테마는 2026 Visit Korea 공모전 친환경 심사 가점 항목과 정확히 일치
- **API 조합 희소성**: `durunubi-trails` + `eco-tourism` 이중 활용은 경쟁 아이디어에서 보기 어려운 차별화 조합
- **사회·환경적 임팩트**: 탄소 절감 수치화, 관광 분산으로 지역 균형 발전 기여 — 심사위원 설득력 높음
- **글로벌 트렌드 부합**: 세계적으로 에코투어리즘 시장 연 15% 성장, Visit Korea 외국인 유치 목표와 직결

## 예상 사용 시나리오

> 환경 NGO에서 일하는 27세 이채원 씨는 친구들과 제주 에코 트레킹 2박 3일을 계획한다. EcoTrail Korea에서 "제주·도보·중급"을 선택하자 올레길 특정 구간과 생태관광 인증 곶자왈 해설 투어를 연결한 코스가 생성된다. 탄소발자국 계산기가 자가용 대신 버스+도보 조합 시 1인당 CO₂ 12kg 절감 가능하다고 알려준다. 코스 완주 후 "제로카본 챌린지" 배지를 획득하고, 코스 후기를 올리며 비주류 해안 습지 명소를 다른 사용자에게 소개한다.
