## 한 줄 정의

장애인·어르신·영유아 동반 가족 누구나 불편 없이 한국을 여행할 수 있도록 — 무장애 관광 정보와 AI 코스 추천을 한 곳에.

## 타겟 사용자

- **1차**: 시니어(65세 이상, 약 970만 명) — 계단·경사 회피, 쉬어가는 코스 필요
- **2차**: 장애인(약 263만 명) — 휠체어 접근성, 점자·음성 안내 필요
- **3차**: 영유아 동반 가족 — 유모차 이동 가능 경로, 수유실·놀이공간 정보 필요
- **4차**: 무장애 여행에 관심 있는 외국인 방문자

## 핵심 기능

- **무장애 필터 통합 검색**: 휠체어 접근 가능, 엘리베이터·경사로 유무, 장애인 화장실, 점자 안내, 청각 장애 지원 등 세부 조건 필터링
- **AI 무장애 코스 추천**: 장애 유형·이동 수단·체력 수준 입력 시 접근 가능한 명소만으로 최적 동선 자동 생성
- **지역별 무장애 명소 현황**: 시·군·구 단위 무장애 관광지 분포 지도, 접근성 등급 표시
- **오디오 가이드 연동**: 시각 장애인 및 시니어를 위한 명소별 음성 해설 자동 제공
- **사용자 접근성 리뷰**: 실제 방문자가 접근성 상태를 업데이트하는 크라우드소싱 시스템

## 활용 API

| API | 활용 내용 |
|---|---|
| `barrier-free-travel` | 무장애 관광지·숙박·편의시설 상세 접근성 정보 |
| `tourism-info-korean` | 관광지 기본 정보(위치·운영시간·연락처) 결합 |
| `audio-guide` | 시각 장애인·시니어 대상 명소별 오디오 해설 제공 |
| `central-attractions-by-municipality` | 시·군·구별 핵심 관광지 목록으로 지역 커버리지 확보 |

## 차별화 포인트

1. **무장애 특화 서비스 전무**: 현재 국내에 장애인·시니어·영유아 동반 가족을 통합 대상으로 한 관광 앱이 사실상 부재
2. **접근성 정보의 파편화 해소**: 각 기관에 흩어진 무장애 정보를 하나의 UX로 통합 제공
3. **오디오 가이드 네이티브 통합**: 별도 앱 없이 관광 정보 탐색에서 음성 해설까지 원스톱
4. **크라우드소싱 업데이트**: 공공데이터의 현장 괴리를 실사용자 리뷰로 실시간 보완

## 공모전 강점

- **사회적 가치 심사항목 직결**: 관광 취약계층(장애인·시니어·영유아 가족) 포용 — 공모전 사회적 가치 심사 항목에 정면 부합
- **명확한 타겟 규모**: 장애인 263만 + 65세 이상 970만 = 1,230만 명 이상의 직접 수혜 대상
- **정책 부합성**: 문화체육관광부 무장애 관광 활성화 정책 방향과 일치, 정부 사업 연계 가능성
- **API 조합 완성도**: `barrier-free-travel` + `audio-guide` 조합으로 접근성 정보와 콘텐츠를 동시에 충족

## 예상 사용 시나리오

> 70세 이정순 씨는 휠체어를 사용하는 남편과 함께 경주 여행을 꿈꿔왔다. AccessKorea에서 "휠체어 사용자·1박 2일·경주"를 입력하자 앱이 휠체어 접근 가능한 불국사 입구 전망대 → 무장애 인증 한식당 → 경주 역사문화관(엘리베이터 완비)을 잇는 코스를 생성한다. 각 명소마다 오디오 가이드가 자동 재생되고, 지도에는 경사로·엘리베이터 위치가 표시된다. 여행 후 남편이 직접 "불국사 입구 휠체어 경사로 상태 양호"라는 리뷰를 남겨 다음 방문자에게 도움을 준다.
