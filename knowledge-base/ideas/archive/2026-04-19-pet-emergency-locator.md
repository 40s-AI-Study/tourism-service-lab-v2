# 여행 중 반려동물 긴급 동물병원 안내

## 핵심 아이디어
여행 중 반려동물이 다치거나 아플 때 가장 가까운 24시 동물병원과 펫 용품점을 즉시 찾아주는 긴급 안내 기능. 여행 앱의 안전 기능으로 통합.

## 타겟
- 반려동물과 처음 장거리 여행하는 반려인
- 낯선 지역 여행 중 비상상황 대비 필요한 반려인

## 핵심 기능
- 현재 위치 기반 가장 가까운 동물병원 즉시 표시
- 24시 운영 여부 필터
- 진료 가능 동물 종류 표시 (개/고양이/특수동물)
- 펫 용품점/약국 위치 안내
- 응급처치 가이드 (기본 외상, 과식, 열사병)

## 활용 API
- `한국관광공사_반려동물_동반여행_서비스`
- `한국관광공사_국문 관광정보 서비스` (주변 편의시설)

## 차별화
여행 중 긴급상황 대응 = 반려인 신뢰 확보. 안전 기능으로 앱 필수 설치 이유 생성.

## 기대 효과
반려동물 여행 안전망 강화, 앱 충성도 향상

## AI 역할
- **ai_role**: `ai-assist` — 위치 기반 병원 매칭 및 응급처치 가이드 제공에 AI 보조
- GPS 현재 위치 + 24시 운영 + 진료 가능 동물 종류 조건 자동 매칭
- 증상 텍스트 입력 → 응급 여부 자동 판단 및 우선 대응 안내
- 주변 동물병원 실시간 영업 상태 확인 (운영 시간 데이터 자동 업데이트)

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 동물병원 제휴 광고 | 병원 프리미엄 노출 | 월 15만원/병원 |
| 펫 보험 연계 | 여행자 펫 보험 추천·가입 연결 | 건당 3,000원 |
| 펫 용품점 광고 | 응급 용품 구매 연결 | CPC 기반 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 반려동물 보유 가구 약 600만 × 여행 중 의료 수요
- **SAM**: 반려동물 동반 여행자 약 250만명 중 긴급 상황 대비 니즈 보유층
- **SOM**: 출시 2년 내 MAU 5만 목표 → 광고+제휴 기준 연 약 15억원
- **근거**: 반려동물 여행 중 응급 사례 증가, 안전 기능 = 앱 필수 설치 동기

## 편의성
- 긴급 상황 시 앱 홈 화면 "지금 응급" 버튼으로 즉시 가장 가까운 병원 표시
- 24시 운영 필터 기본값 설정으로 별도 조작 불필요
- 전화 연결 버튼으로 병원 즉시 연락 (검색·복사 불필요)
- 응급처치 가이드 오프라인 저장으로 데이터 없이도 즉시 접근
