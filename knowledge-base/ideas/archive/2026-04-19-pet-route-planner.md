# 반려동물 맞춤 여행 코스 생성기

## 핵심 아이디어
반려동물의 품종/크기/나이/성격을 입력하면 그에 맞는 여행 코스를 자동 생성. 대형견과 소형견, 고양이와 개의 여행 코스가 다르게 설계.

## 타겟
- 처음 반려동물과 여행 계획하는 초보 반려인
- 특수 반려동물(고양이/토끼 등) 보유 여행자

## 핵심 기능
- 반려동물 프로필 설정 (종류/크기/나이/특이사항)
- 프로필 기반 입장 가능 장소 자동 필터링
- 이동 거리/시간 고려한 무리 없는 코스 설계
- 날씨 고려 (여름 뜨거운 바닥 주의 알림)
- 반려동물 사진 스팟 추천

## 활용 API
- `한국관광공사_반려동물_동반여행_서비스`
- `한국관광공사_관광지 집중률 방문자 추이 예측 정보` (혼잡 회피)
- `한국관광공사_관광지별 연관 관광지 정보`

## 차별화
반려동물 개체 특성 고려한 개인화. 단순 목록 제공이 아닌 "우리 아이에게 맞는" 코스.

## 기대 효과
반려인 만족도 극대화, 앱 재사용률 증가

## AI 역할
- **ai_role**: `ai-only` — 반려동물 개체 특성 기반 맞춤 코스 생성 전 과정 AI 주도
- 반려동물 프로필(품종·크기·나이·특이사항) 분석 → 입장 가능 장소 자동 필터링
- 이동 거리·날씨·혼잡도·체력 소모를 종합 고려한 최적 코스 자동 생성
- 날씨 AI 예측으로 여름철 뜨거운 바닥·겨울 추위 자동 경보
- 반려동물 사진 AI 분석으로 포토 스팟 적합도 자동 점수화

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|

| 펫 숙박 예약 수수료 | 반려동물 동반 숙박 예약 | 예약액의 7% |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 반려동물 보유 가구 약 600만 (KB경영연구소 2024) × 여행 지출
- **SAM**: 반려동물 동반 여행 의향 반려인 약 250만명
- **SOM**: 출시 2년 내 MAU 12만 목표 → 구독+수수료 기준 연 약 65억원
- **근거**: 반려동물 동반 여행 연간 25%+ 성장, 개체 맞춤 서비스 공백

## 편의성
- 반려동물 프로필 1회 등록 후 이후 모든 추천 자동 개인화
- 코스 생성 시 입장 불가 장소 자동 제외 (사전 확인 불필요)
- 날씨 경보 자동 알림 → 대체 실내 코스 즉시 제안
- 포토 스팟 위치 지도에서 바로 확인 → 이동 경로 자동 추가
