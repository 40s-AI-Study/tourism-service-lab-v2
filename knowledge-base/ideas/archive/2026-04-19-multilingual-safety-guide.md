# 외국인 긴급상황 대응 + 의료관광 통합앱

## 핵심 아이디어
한국 여행 중 아프거나 긴급상황이 생겼을 때 언어 장벽 없이 빠르게 대응 가능한 앱. 의료관광 정보 + 실시간 통역 + 긴급 연락처 통합.

## 타겟
- 한국어 모르는 외국인 관광객 전체
- 의료관광 목적 방한 외국인 (시장 규모 연간 50만명+)

## 핵심 기능
- 증상 선택 → 가까운 외국어 가능 병원 안내
- 의료관광 클리닉 예약 연동
- 응급상황 119/112 다국어 문자 전송
- 여행자보험 청구 서류 자동 번역
- 약국/처방전 다국어 안내

## 활용 API
- `한국관광공사_의료관광정보`
- `한국관광공사_영문 관광정보서비스`
- `한국관광공사_일문 관광정보서비스`
- `한국관광공사_중문 간체 관광정보서비스`

## 차별화
여행 안전/건강 특화. 공모전 심사: 실용성 + 외국인 방한 촉진 가치.

## 기대 효과
외국인 한국 여행 안전망 강화, 의료관광 시장 확대

## AI 역할
- **ai_role**: `ai-assist` — 증상 분류 및 다국어 안내 생성에 AI 보조
- 증상 선택 입력 → 적합 진료과·병원 자동 매칭 (규칙 기반 + AI 보조)
- 여행자보험 청구 서류 자동 번역 (OCR + NLP 결합)
- 긴급 메시지 다국어 자동 생성 (119/112 신고용 표준 문장)

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 의료관광 예약 수수료 | 클리닉·병원 예약 연결 | 예약액의 8% |
| 여행자보험 제휴 | 보험 상품 추천·가입 연결 | 건당 5,000원 |
| 병원 프리미엄 리스팅 | 외국어 가능 병원 노출 | 월 30만원/병원 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 방한 외래객 연 1,620만명 × 여행 중 의료 수요 약 5%
- **SAM**: 의료관광 목적 방한 외국인 연 약 50만명 + 일반 의료 니즈 방한객
- **SOM**: 출시 2년 내 월 예약 5,000건 목표 → 수수료 기준 연 약 20억원
- **근거**: 외국인 한국 여행 불안 요소 1위 "언어 장벽 시 의료", 의료관광 시장 성장

## 편의성
- 증상 아이콘 선택만으로 1분 내 가까운 외국어 가능 병원 안내
- 긴급 상황 시 홈 화면 SOS 버튼 → 119 자동 문자 발송
- 서류 사진 촬영 즉시 자동 번역 (수동 입력 불필요)
- 오프라인 모드에서도 긴급 연락처·응급처치 가이드 접근 가능
