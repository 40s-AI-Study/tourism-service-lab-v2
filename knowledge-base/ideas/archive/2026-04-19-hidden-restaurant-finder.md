# 빅데이터 기반 진짜 숨은 맛집 발굴기

## 핵심 아이디어
블로그 협찬 맛집과 진짜 로컬 맛집을 빅데이터로 구분. SNS 버즈량 대비 실제 방문자 수 비율로 "찐 로컬 맛집" 지수를 산출하여 추천.

## 타겟
- 관광객 맛집 말고 현지인이 가는 곳 원하는 20-40대
- 블로그 협찬 의심하는 정보 피로도 높은 MZ세대

## 핵심 기능
- 지역별 "찐 로컬 지수" 맛집 랭킹
- 협찬 의심 필터 (블로그 방문 수 vs 실방문 비율)
- 오래된 가게(30년+) 전통 맛집 별도 카테고리
- 외국인 잘 모르는 한국인 단골집 발굴
- 관광빅데이터 방문자 추이로 검증

## 활용 API
- `한국관광공사_관광빅데이터 정보서비스`
- `한국관광공사_지역별 관광 수요 강도`
- `한국관광공사_국문 관광정보 서비스`

## 차별화
정보 피로도 시대의 신뢰 기반 큐레이션. 빅데이터로 "찐 맛집" 검증.

## 기대 효과
정보 신뢰도 차별화, 재방문율 향상

## AI 역할
- **ai_role**: `ai-enhanced` — 찐 로컬 맛집 지수 산출 및 협찬 필터링에 AI 핵심 역할
- SNS 버즈량 vs 실제 방문자 수 비율 분석으로 "찐 로컬 지수" 자동 산출
- 블로그 텍스트 패턴 분석으로 협찬 게시물 자동 감지 및 제외
- 오래된 가게 영업 기간·후기 패턴 결합으로 전통 맛집 자동 발굴

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 구독 | 찐 로컬 지수 상위 맛집·필터 무제한 | 월 2,900원 |
| 음식점 제휴 | 방문 예약·웨이팅 연결 | 거래액의 4% |
| 데이터 B2B | 지역 맛집 신뢰도 데이터 판매 | 프로젝트별 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 맛집 앱 시장 약 3,000억원 (망고플레이트·식신 등)
- **SAM**: 정보 신뢰도 중시 MZ세대 약 400만명
- **SOM**: 출시 2년 내 MAU 15만 목표 → 구독+수수료 기준 연 약 55억원
- **근거**: 블로그 협찬 피로도 증가로 신뢰 기반 큐레이션 수요 급증

## 편의성
- 지역 선택만으로 찐 로컬 지수 상위 맛집 즉시 표시
- 협찬 필터 토글 한 번으로 광고 게시물 완전 제외
- 오래된 가게·전통 맛집 별도 카테고리로 빠른 접근
- 현재 위치 기반 도보 거리 내 찐 맛집 즉시 안내
