# 휠체어 접근 가능 실시간 관광 동선 지도

## 핵심 아이디어
휠체어 사용자가 실제로 이동 가능한 경로만 표시하는 특화 지도. 계단/급경사 구간을 자동 우회하고 무장애 동선만 안내. 실시간 사용자 업데이트로 정확도 향상.

## 타겟
- 휠체어 사용 장애인 및 보호자
- 시니어 전동 휠체어 사용자

## 핵심 기능
- 출발지 → 목적지 휠체어 가능 경로 안내
- 경사도/턱 높이 정보 표시
- 장애인 전용 주차장 위치 안내
- 무장애 숙박/식당 연동
- 사용자 경로 평가 및 위험 구간 신고

## 활용 API
- `한국관광공사_무장애 여행 정보`
- `한국관광공사_두루누비 정보 서비스` (보행 경로)
- `한국관광공사_국문 관광정보 서비스`

## 차별화
Google Maps도 없는 한국 관광지 특화 휠체어 경로. 크라우드소싱으로 정확도 지속 개선.

## 기대 효과
장애인 여행 자립성 향상, 복지 관광 서비스 선도

## AI 역할
- **ai_role**: `ai-only` — 휠체어 경로 생성부터 크라우드소싱 데이터 처리까지 AI 주도
- 무장애 여행 정보 + 두루누비 보행 경로 결합으로 휠체어 전용 최적 경로 자동 생성
- 사용자 신고 데이터(위험 구간·계단) AI 검증 및 지도 자동 업데이트
- 경사도·턱 높이 데이터 기반 난이도 점수 자동 산출
- 사진 AI 분석으로 접근성 개선 구간 자동 식별

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 무장애 숙박 예약 수수료 | 접근성 인증 숙박 예약 연결 | 예약액의 7% |
| 지자체 정부 사업 | 무장애 관광 데이터 구축 용역 | 사업별 계약 |
| 기업 CSR 광고 | 장애인 관광 지원 기업 광고 | 월 기반 계약 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 장애인 약 263만명 + 가족 포함 약 800만명 관련 여행 시장
- **SAM**: 관광 의향 있는 장애인·시니어 여행자 약 150만명
- **SOM**: 출시 2년 내 MAU 5만 목표 → 예약+정부사업 기준 연 약 30억원
- **근거**: 장애인 관광 인프라 법제화 강화, 복지 관광 서비스 공모전 우수 평가 기대

## 편의성
- 목적지 입력만으로 휠체어 가능 경로 즉시 안내 (별도 필터 불필요)
- 위험 구간 신고 1터치 → 즉시 지도 반영 (커뮤니티 자동 공유)
- 장애인 화장실·주차장 위치 경로 위에 자동 표시
- 오프라인 경로 저장으로 이동 중 데이터 없이도 안내 지속
