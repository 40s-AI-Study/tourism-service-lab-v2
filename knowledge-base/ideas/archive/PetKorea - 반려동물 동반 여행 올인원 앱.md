## 한 줄 정의

반려동물과 함께하는 국내 여행의 모든 것 — 관광지·숙박·음식점·레포츠를 한 앱에서, AI 코스 추천까지.

## 타겟 사용자

- **1차**: 20-40대 반려동물 가구 (한국 내 약 600만 가구)
- **2차**: 반려동물 동반 외국인 여행자 (장기 체류 외국인, 디지털 노마드)
- **3차**: 반려동물 友好 여행에 관심 있는 시니어 가구

## 핵심 기능

- **Pet-Friendly 통합 검색**: 반려동물 동반 가능 관광지·숙박·음식점·레포츠·문화시설·쇼핑·서비스 7개 카테고리 전체 필터링 및 지도 표시
- **AI 여행 코스 생성**: 반려동물 종류(개/고양이/소형/대형 등)·이동 수단·여행 일수 입력 시 최적 코스 자동 생성, 혼잡 예측 기반 시간대 추천
- **혼잡도 실시간 알림**: 방문 예정지 혼잡 예보를 사전 제공해 반려동물 스트레스 최소화 루트 안내
- **펫 동반 커뮤니티**: 실사용자 후기·사진 공유, 반려동물 입장 정책 변경 제보, 별점 시스템
- **주변 연관 명소 추천**: 현재 위치·선택 명소 기준 반려동물 동반 가능 인근 명소 즉시 탐색

## 활용 API

| API | 활용 내용 |
|---|---|
| `pet-friendly-travel` | 반려동물 동반 가능 장소 7개 카테고리 전체 데이터 |
| `tourism-info-korean` | 국내 관광지 상세 정보(주소·운영시간·연락처) |
| `related-attractions` | 선택 명소 기준 인근 연관 관광지 추천 |
| `visitor-concentration-forecast` | 방문지별 혼잡도 예측 데이터, 최적 방문 시간 안내 |

## 차별화 포인트

1. **단독 API 전 카테고리 활용**: `pet-friendly-travel` API의 관광지·숙박·음식점·레포츠·문화시설·쇼핑·서비스 7개 카테고리를 모두 통합한 유일한 서비스
2. **반려동물 맞춤 혼잡 예측**: 단순 관광이 아닌 반려동물 동반 특성(소음·밀집 회피)을 고려한 혼잡도 기반 코스 설계
3. **커뮤니티 기반 실시간 정책 업데이트**: 공공데이터 갱신 주기의 한계를 커뮤니티 제보로 보완
4. **성장하는 시장**: 국내 반려동물 동반 여행 시장 연 25% 성장세, 관련 공공 인프라 수요 급증

## 공모전 강점

- **API 활용 다양성**: 4개 API를 유기적으로 결합, `pet-friendly-travel` 7개 전 카테고리 활용으로 심사 기준 충족
- **명확한 시장 규모**: 600만 반려동물 가구라는 구체적 수요층, 연 25% 성장하는 블루오션 시장
- **사회적 효과**: 반려동물 동반 여행 인프라 부족 문제 해소, 지역 관광 활성화 기여
- **확장 가능성**: 외국인 여행자 대상 다국어 확장, 반려동물 보험·용품 커머스 연계

## 예상 사용 시나리오

> 서울에 사는 30대 직장인 김지수 씨는 주말에 반려견 '콩이'(소형견)와 강원도로 1박 2일 여행을 계획한다. PetKorea 앱에서 반려견 정보를 입력하고 강원도·1박 2일을 선택하자, AI가 소형견 입장 가능한 카페 → 반려동물 동반 펜션 → 레포츠 파크 코스를 생성한다. 혼잡도 예측 기능이 "토요일 오후 2시 해당 카페 혼잡 예상, 오전 11시 방문 추천"을 알려주고, 커뮤니티에서 최근 방문자의 실사진 후기도 확인한다. 여행 후 콩이와 찍은 사진과 함께 후기를 올려 다른 반려인들과 정보를 나눈다.
