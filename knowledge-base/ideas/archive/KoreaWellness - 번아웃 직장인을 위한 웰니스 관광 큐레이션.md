## 한 줄 정의

스트레스 프로파일 기반 AI가 명상·스파·산림욕·한방 체험을 큐레이션하는 번아웃 직장인 맞춤 웰니스 여행 앱.

## 타겟 사용자

- **1차**: 30-40대 직장인 (한국갤럽 2024: 87%가 번아웃 경험)
- **2차**: 웰니스 트렌드에 민감한 MZ세대 (명상·요가·디지털 디톡스 관심층)
- **3차**: 건강 회복을 목적으로 한국을 찾는 외국인 의료·웰니스 관광객

## 핵심 기능

- **스트레스 프로파일 진단**: 번아웃 유형(신체 피로형·정서 소진형·인지 과부하형 등) 자가 진단 후 맞춤 웰니스 카테고리 추천
- **웰니스 코스 AI 큐레이션**: 명상·스파·산림욕·한방 체험·힐링 숙박을 조합한 개인화 1박~3박 코스 자동 생성
- **수요 밀도 기반 조용한 여행지 추천**: 관광 수요 밀도가 낮은 한적한 지역 웰니스 명소 우선 추천으로 진정한 힐링 환경 제공
- **생태 웰니스 연계**: 산림욕·자연 치유 코스와 생태관광지를 결합한 그린 웰니스 패키지
- **빅데이터 기반 트렌드 리포트**: 계절별·지역별 웰니스 관광 인기 명소·테마 실시간 업데이트

## 활용 API

| API | 활용 내용 |
|---|---|
| `wellness-tourism` | 웰니스 관광지·프로그램(명상·스파·한방·산림욕 등) 핵심 데이터 |
| `area-tourism-demand-density` | 지역별 관광 수요 밀도 분석으로 한적한 웰니스 지역 선별 |
| `tourism-big-data` | 계절·트렌드 기반 웰니스 명소 인기도 분석 및 추천 근거 제공 |
| `eco-tourism` | 산림욕·자연치유 관련 생태관광지 데이터와 웰니스 코스 연계 |

## 차별화 포인트

1. **wellness-tourism API 독점 활용**: 해당 API를 핵심으로 활용하는 경쟁 서비스 부재, 웰니스 특화 포지셔닝 명확
2. **번아웃 진단 + 여행 추천 결합**: 단순 관광지 나열이 아닌 심리·신체 상태 진단에서 여행까지 연결하는 독창적 UX
3. **조용한 여행지 알고리즘**: 수요 밀도 데이터를 역활용해 붐비지 않는 진정한 힐링 장소를 발굴
4. **한국 고유 웰니스 콘텐츠**: 한방 체험·찜질방·산사 템플스테이 등 K-웰니스 고유 콘텐츠로 외국인 관광객 차별화 어필

## 공모전 강점

- **트렌드 적합성**: 포스트 코로나 웰니스 관광 급성장, MZ세대 '조용한 사치' 트렌드와 정확히 교차
- **API 희소성**: `wellness-tourism` API를 메인으로 구성한 서비스 전무 — 심사위원 눈에 띄는 포지셔닝
- **데이터 기반 신뢰성**: `tourism-big-data`로 추천 근거를 수치화, 단순 주관 큐레이션과 차별화
- **내수·외수 동시 공략**: 번아웃 국내 직장인 + K-웰니스 외국인 관광객 두 시장 동시 겨냥

## 예상 사용 시나리오

> 광고 기획사에 다니는 35세 박서연 씨는 최근 극심한 번아웃을 느끼고 있다. KoreaWellness 앱에서 3분 자가진단을 하자 "정서 소진형 번아웃 — 자연 속 정적 활동 추천" 결과가 나온다. AI가 강원도 인제군의 산림욕 치유 프로그램 → 한적한 한옥 스파 숙박 → 한방 체험 클래스를 잇는 2박 3일 코스를 제안한다. 관광 수요 밀도 지표로 해당 지역이 "이번 주말 혼잡도 낮음"으로 표시되어 진짜 쉬어갈 수 있다는 확신을 갖고 예약한다.
