# 붐비는 명소 대신 나만의 유사 명소 추천

## 핵심 아이디어
"경복궁 대신 덜 붐비는 전통 궁궐" 같이, 인기 관광지가 혼잡할 때 비슷한 분위기의 대안 명소를 자동 추천. 관광 다양성 지수 데이터로 과잉 밀집 지역 분산 유도.

## 타겟
- SNS 인플루언서/사진작가 여행자
- 대중관광지 회피 성향 MZ세대

## 핵심 기능
- 현재 혼잡 관광지 입력 → 유사 분위기 저혼잡 대안 3곳 추천
- 관광 다양성 낮은 지역 우선 노출
- 분위기 태그 (전통/자연/도시/감성) 기반 매칭
- "덜 알려진 명소" 배지 표시

## 활용 API
- `한국관광공사_지역별 관광 다양성`
- `한국관광공사_관광지 집중률 방문자 추이 예측 정보`
- `한국관광공사_관광지별 연관 관광지 정보`

## 차별화
관광 쏠림 현상 해소 + 비수도권 명소 자연스럽게 노출. 공모전 지역 균형 발전 가치와 연계.

## 기대 효과
수도권 집중 완화, 지방 관광지 활성화

## AI 역할
- **ai_role**: `ai-enhanced` — 대안 명소 유사도 매칭 및 혼잡도 판단에 AI 보조
- 관광지 분위기 태그 임베딩 기반 유사 명소 코사인 유사도 계산
- 혼잡도 예측 API + 관광 다양성 지수 조합으로 "지금 가기 좋은 대안" 동적 선정
- 사용자 행동 데이터로 유사도 모델 지속 개선

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 지역 관광 광고 | 비수도권 관광지 노출 프로모션 | 월 15만원/지역 |
| 예약 수수료 | 대안 명소 숙박·체험 예약 연결 | 예약액의 5% |
| 지자체 파트너십 | 지역 관광청 공동 캠페인 | 프로젝트별 계약 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 여행 앱 시장 약 1조원
- **SAM**: 대중관광지 회피 성향 MZ세대 약 300만명
- **SOM**: 출시 2년 내 MAU 15만 목표 → 광고+수수료 연 약 50억원
- **근거**: 관광 쏠림 현상 심화로 대안 명소 수요 증가세

## 편의성
- 혼잡 명소 이름 검색 즉시 대안 3곳 자동 표시
- 분위기 태그 한 번 설정으로 지속 개인화 추천
- 대안 명소 → 바로 예약 또는 동선 추가 원클릭
- 오프라인 저장으로 현장에서 데이터 없이도 대안 정보 접근 가능
