# 감정 상태 기반 맞춤 여행지 추천

## 핵심 아이디어
"오늘 어떤 기분인가요?"를 앱 첫 화면으로 설정. 번아웃/우울/흥분/설렘 등 감정 태그를 선택하면 해당 감정에 최적화된 여행지와 체험을 추천. 감정 → 성향 → 명소 추천 플로우.

## 타겟
- 번아웃 경험 MZ세대 (부킹닷컴 2025: 여행 = 자기회복)
- 즉흥적 주말 여행을 원하는 20-30대

## 핵심 기능
- 감정 선택 UI (아이콘/이모지 기반, 직관적)
- 감정별 추천 여행지 카테고리 매핑
  - 번아웃 → 웰니스 관광지, 자연형 명소
  - 에너지 넘침 → 레포츠, 아웃도어 체험
  - 외로움 → 감성 카페 밀집 골목, 사람 구경 명소
- 감정 여행 일기 기록 기능
- 감정 회복 성공률 후기 공유

## 활용 API
- `한국관광공사_웰니스관광정보`
- `한국관광공사_국문 관광정보 서비스`
- `한국관광공사_생태 관광 정보`

## 차별화
Calm/Headspace의 감정 온보딩 UX를 여행 앱에 적용. 국내 최초 감정 기반 여행지 매칭 서비스.

## 기대 효과
MZ세대 여행 앱 재방문율 상승, 정신건강 웰니스 여행 시장 공략
