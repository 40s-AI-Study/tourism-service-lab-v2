# 감정 상태 기반 맞춤 여행지 추천

## 핵심 아이디어
"오늘 어떤 기분인가요?"를 앱 첫 화면으로 설정. 번아웃/우울/흥분/설렘 등 감정 태그를 선택하면 해당 감정에 최적화된 여행지와 체험을 추천. 감정 → 성향 → 명소 추천 플로우.

## 타겟
- 번아웃 경험 MZ세대 (부킹닷컴 2025: 여행 = 자기회복)
- 즉흥적 주말 여행을 원하는 20-30대

## 핵심 기능
- 감정 선택 UI (아이콘/이모지 기반, 직관적)
- 감정별 추천 여행지 카테고리 매핑
  - 번아웃 → 웰니스 관광지, 자연형 명소
  - 에너지 넘침 → 레포츠, 아웃도어 체험
  - 외로움 → 감성 카페 밀집 골목, 사람 구경 명소
- 감정 여행 일기 기록 기능
- 감정 회복 성공률 후기 공유

## 활용 API
- `한국관광공사_웰니스관광정보`
- `한국관광공사_국문 관광정보 서비스`
- `한국관광공사_생태 관광 정보`

## 차별화
Calm/Headspace의 감정 온보딩 UX를 여행 앱에 적용. 국내 최초 감정 기반 여행지 매칭 서비스.

## 기대 효과
MZ세대 여행 앱 재방문율 상승, 정신건강 웰니스 여행 시장 공략

## AI 역할
- **ai_role**: `ai-only` — 감정 입력 분석부터 여행지 매칭까지 전 과정 AI 주도
- 감정 태그 → 여행지 성격 벡터 변환 후 유사도 기반 매칭 (임베딩 모델 활용)
- 사용자 히스토리 학습으로 감정-여행지 개인화 모델 강화
- 자연어 감정 입력("요즘 너무 지쳐서…") → 감정 분류 → 추천 자동화

## 비즈니스 모델
| 수익원 | 내용 | 예상 단가 |
|--------|------|-----------|
| 프리미엄 구독 | 감정 여행 일기·분석 리포트 제공 | 월 3,900원 |
| 제휴 예약 수수료 | 추천 숙박·체험 예약 연결 | 예약액의 5~8% |
| 웰니스 브랜드 광고 | Calm·명상앱 등 타겟 광고 | CPM 기반 |

## 사업성 검증 (TAM/SAM/SOM)
- **TAM**: 국내 MZ세대 여행 앱 시장 약 1.2조원
- **SAM**: 번아웃 경험 직장인 + 힐링 여행 선호층 약 650만명 → 시장 규모 약 2,600억원
- **SOM**: 출시 2년 내 MAU 30만 목표 → 구독 기준 연 약 140억원 목표
- **근거**: 부킹닷컴 2025 "여행=자기회복" 트렌드, 정신건강 앱 시장 연 15% 성장

## 편의성
- 앱 첫 화면 = 감정 선택 (3초 내 온보딩 완료)
- 감정 선택 후 즉시 추천 결과 표시 (API 캐시 활용, 응답 <1초)
- 감정 여행 일기 자동 생성 → 별도 기록 불필요
- 오프라인 저장으로 여행 중 데이터 없이도 추천 목록 접근 가능
