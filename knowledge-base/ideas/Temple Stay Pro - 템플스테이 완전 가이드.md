# 아이디어 12: Temple Stay Pro

## 한 줄 정의
137개 등록 사찰의 템플스테이 프로그램을 한눈에 비교·예약하고, 방문 중 각 건물의 역사·의미를 **다국어 오디오 해설**로 들려주는 템플스테이 전문 앱.

## 타겟 사용자
- **1차**: 번아웃 20-30대 여성 (명상·힐링 수요) — 연 템플스테이 참가자 15만명 중 70%가 여성
- **2차**: 외국인 K-문화 체험자 (일·영·중 어권) — 방한 외국인 템플스테이 수요 지속 성장

## 문제 정의 / 근거
- 현재 템플스테이 정보는 **한국불교문화사업단 공식 사이트** 위주로 UX 구식, 앱 부재
- 사찰별 특징(명상 중심/문화체험/휴식/수행)이 불명확해 **선택 피로**
- 방문 후 각 전각·불상의 의미를 알기 어려워 **단순 관광으로 끝나는** 경험
- Visit Korea의 `audio-guide` + `barrier-free-travel` + 종교/문화 POI 조합은 현재 없음

## 핵심 기능
1. **프로그램 비교 매칭**: "주말 / 2박 / 명상 중심 / 영어 가능" 필터 → 조건 맞는 사찰 Top 5 자동 추출
2. **사찰별 오디오 가이드**: 방문 시 GPS 기반으로 각 전각·불상 해설 자동 재생 (4개 언어)
3. **명상 타이머·워크북**: 일정 내 정해진 명상 세션에 맞춘 가이드 타이머 + 성찰 노트
4. **무장애 필터**: 계단 많은 사찰 제외 + 휠체어 가능 사찰 표시
5. **체류 후 연계**: 사찰 주변 지역 관광지·채식 맛집 자동 추천

## 활용 API
- `tourism-info-korean/english/chinese/japanese` — 사찰 기본 정보 다국어
- `audio-guide` — 전각·문화재 해설
- `barrier-free-travel` — 무장애 접근 정보
- `related-attractions` — 사찰 주변 명소 연계
- `eco-tourism` — 산사 주변 자연 트레일

## 차별화 포인트
- 전용 템플스테이 앱 **국내 전무** — 공식 사이트만 존재
- 오디오 가이드 + 명상 가이드 결합은 해외에도 사례 적음
- K-컬처 붐 타고 외국인 수요 증가 — 다국어 대응으로 해외 유입 확장
- 사찰과 B2B 수수료 모델 (예약 성사 시)

## 예상 사용 시나리오
```
30세 마케터 여성 (번아웃, 주말 1박 원함):
  앱: "서울 근교 / 주말 1박 / 명상 중심" → 영평사 / 미타사 / 봉선사 제안
  선택 후 예약 완료 → D-Day 오디오 가이드 pre-download
방문 중:
  대웅전 앞 GPS 진입 → "이 건물은 1742년 중건, 조선 후기 건축 특징..." 해설 자동 재생
  저녁 명상 세션 → 앱 타이머 + 가이드 음성 + 성찰 노트 타이핑
귀가 후:
  다음 템플스테이 추천 + 친구 공유 링크
```

## 예상 사업성 (1-10)
- 시장 규모: 6 (연 15만명 참가, 잠재 30만+)
- 차별화: 9 (전용 앱 전무)
- 구현 난이도: 7 (사찰 파트너십 + 음성 콘텐츠 제작 필요)
- 공모전 적합성: 8 (K-문화 정책 부합 + 외국인 관광 활성화 + 접근성)
- **총점: 30/40**

## 리스크
- 사찰 측 보수적 성향 — 초기 5-10개 사찰과 파트너십 구축 필수
- 음성 콘텐츠 제작비 — 1차로 한국어 완성 후 다국어 확장
- 명상 콘텐츠 저작권 — 사찰 공식 승인 필수
