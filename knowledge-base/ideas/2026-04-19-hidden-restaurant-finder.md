# 빅데이터 기반 진짜 숨은 맛집 발굴기

## 핵심 아이디어
블로그 협찬 맛집과 진짜 로컬 맛집을 빅데이터로 구분. SNS 버즈량 대비 실제 방문자 수 비율로 "찐 로컬 맛집" 지수를 산출하여 추천.

## 타겟
- 관광객 맛집 말고 현지인이 가는 곳 원하는 20-40대
- 블로그 협찬 의심하는 정보 피로도 높은 MZ세대

## 핵심 기능
- 지역별 "찐 로컬 지수" 맛집 랭킹
- 협찬 의심 필터 (블로그 방문 수 vs 실방문 비율)
- 오래된 가게(30년+) 전통 맛집 별도 카테고리
- 외국인 잘 모르는 한국인 단골집 발굴
- 관광빅데이터 방문자 추이로 검증

## 활용 API
- `한국관광공사_관광빅데이터 정보서비스`
- `한국관광공사_지역별 관광 수요 강도`
- `한국관광공사_국문 관광정보 서비스`

## 차별화
정보 피로도 시대의 신뢰 기반 큐레이션. 빅데이터로 "찐 맛집" 검증.

## 기대 효과
정보 신뢰도 차별화, 재방문율 향상
