# 웰니스 관광 + 명상/요가 + 숙박 통합 플래너

## 핵심 아이디어
웰니스 관광지, 명상 센터, 요가 스튜디오, 한방 스파, 자연 치유 숙박을 하나의 여행 패키지로 묶어 설계. 글로벌 웰니스 투어리즘 $8,940억 시장의 한국 버전.

## 타겟
- 웰니스 관광 관심 외국인 (BookRetreats: 64% 리트리트 계획)
- 힐링 여행 원하는 내국인 30-40대 직장인

## 핵심 기능
- 웰니스 프로그램 유형별 검색 (명상/요가/한방/삼림욕/온천)
- 1박 2일 ~ 3박 4일 힐링 패키지 자동 구성
- 웰니스 숙박 (한옥/게스트하우스/사찰) 예약 연동
- 웰니스 여행 전후 몸/마음 상태 체크인
- 영어/일어 지원 (외국인 웰니스 관광객)

## 활용 API
- `한국관광공사_웰니스관광정보`
- `한국관광공사_생태 관광 정보`
- `한국관광공사_영문 관광정보서비스`
- `한국관광공사_일문 관광정보서비스`

## 차별화
한국 전통 웰니스(한방/사찰음식)와 현대 웰니스(요가/명상)의 융합. 글로벌 웰니스 투어리즘 트렌드 직접 공략.

## 기대 효과
고부가가치 외국인 웰니스 관광객 유치, 한국 힐링 브랜드 구축
