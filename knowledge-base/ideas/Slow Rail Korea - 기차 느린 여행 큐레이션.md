# 아이디어 14: Slow Rail Korea

## 한 줄 정의
KTX·SRT·무궁화호·관광열차를 **"서두르지 않는 여행"** 관점에서 재조합해, 각 역 주변 도보 가능 명소·카페·맛집과 연결한 철도 슬로우 여행 전문 앱.

## 타겟 사용자
- **1차**: 차 없는 20-30대 솔로/커플 (서울 거주, 당일 또는 1박 슬로우 트립)
- **2차**: 50-60대 은퇴자 — 체력 부담 적은 느린 여행 선호

## 문제 정의 / 근거
- KTX 이용객 2024년 연 9,000만명, 무궁화호·ITX 포함하면 철도 관광 시장 거대
- 그러나 "역을 거점으로 한 도보 관광" 큐레이션 서비스 부재
- 기존 여행 앱은 자차 중심 — 대중교통 여행자에게 맞춤 코스 제공 못함
- 간이역 주변 숨은 명소·카페 정보가 블로그에 분산

## 핵심 기능
1. **역 기반 코스 생성**: 출발·도착역 입력 → "도보 15분 이내 명소·카페" 자동 묶음
2. **관광열차 통합**: V-train(중부내륙), S-train(남도해양), DMZ 평화열차 등 특화 상품 직접 연결
3. **시간표 기반 일정**: 기차 시간표 가져와 관광 시간을 분 단위로 정밀 계획
4. **무궁화호 슬로우 모드**: 일부러 완행 기차만 골라 "여유 중심" 코스 옵션 제공
5. **간이역 발굴**: 승하차 거의 없는 간이역 주변 숨은 명소 큐레이션 (ex: 화본역 카페, 구둔역 감성)

## 활용 API
- `tourism-info-korean/english` — 역 주변 명소 정보
- `durunubi-trails` — 역 연결 도보여행길
- `eco-tourism` — 철도 연계 생태관광지
- `central-attractions-by-municipality` — 지자체 대표 명소
- `related-attractions` — 역별 연관 POI

## 차별화 포인트
- 철도 특화 관광 앱 국내 전무 (KORAIL 공식 앱은 예매 중심)
- "도보 15분" 반경 제약 = 대중교통 여행자 최적 UX
- 간이역 큐레이션 = 숨은 콘텐츠 자산화
- 환경 친화적 이미지 — ESG 관점 공모전 가점

## 예상 사용 시나리오
```
28세 커플 (서울 → 강릉 KTX 당일):
  앱 실행 → "서울 → 강릉 · 당일치기" 선택
  코스:
    09:00 강릉역 도착
    09:15 도보 10분 중앙시장 아침
    10:30 택시 10분 안목해변 카페거리
    12:00 경포호 도보 산책
    14:00 강릉역 복귀, KTX 복귀
  각 구간 이동 시간 + 기차 시간표 자동 정밀 계산
```

## 예상 사업성 (1-10)
- 시장 규모: 7 (KTX 이용객 연 9천만 + 무궁화호 포함)
- 차별화: 8 (철도 특화 앱 전무)
- 구현 난이도: 6 (KORAIL 시간표 API 외부 연동 필요)
- 공모전 적합성: 7 (환경·지역 활성화 정책 부합)
- **총점: 28/40**

## 리스크
- KORAIL 시간표 API 제약 — 비공식 수집 필요 or 공식 제휴
- 간이역 정보 최신성 — 지역 주민 UGC 병행 전략
- 경쟁: 네이버지도·카카오맵 대중교통 — 큐레이션 품질로 차별화
