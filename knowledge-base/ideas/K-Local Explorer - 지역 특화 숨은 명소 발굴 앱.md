## 한 줄 정의

지역 중심 관광지 데이터와 사진 공모전 수상작을 결합해 MZ세대가 SNS에 공유하고 싶은 '나만 아는 숨은 명소'를 발굴·추천하는 로컬 탐방 앱.

## 타겟 사용자

- **주 타겟**: 20-30대 로컬 탐방 선호 MZ세대, SNS 공유 문화 친화층
- **세부 세그먼트**: 인스타·릴스용 감성 사진 명소 탐방자, 지방 소도시 여행 선호 여행자, '오버투어리즘' 기피층
- **부 타겟**: 사진 취미 동호회, 지역 관광 콘텐츠 크리에이터

## 핵심 기능

- **숨은 명소 발굴 엔진**: 지자체별 관광지 데이터와 혼잡도 예측을 교차해 '덜 알려졌지만 가치 있는' 비수기 명소 추천
- **사진 공모전 수상작 갤러리**: 한국관광공사 사진 공모전 수상작을 명소별로 연결 — 방문 전 실제 비주얼 미리보기 제공
- **관련 명소 연계 탐방 코스**: 하나의 숨은 명소 클릭 시 주변 연관 명소 자동 연결 — 반나절·하루 코스 자동 생성
- **혼잡도 기반 방문 타이밍 추천**: 방문자 집중 예보로 "이 명소, 이번 주 화요일 오전이 최적" 안내
- **SNS 공유 최적화**: 수상작 사진 + 위치 정보 + 감성 문구를 조합한 인스타그램·릴스용 카드뉴스 자동 생성

## 활용 API

| API | 활용 목적 |
|-----|-----------|
| `central-attractions-by-municipality` | 지자체별 핵심 관광지 데이터 — 지역 특화 명소 목록 기반 |
| `related-attractions` | 명소 간 연관 관계 — 주변 숨은 명소 연계 코스 생성 |
| `photo-contest-winners` | 사진 공모전 수상작 — 명소별 고품질 비주얼 콘텐츠 연동 |
| `visitor-concentration-forecast` | 혼잡도 예측 — 비수기·비혼잡 방문 타이밍 추천 |

## 차별화 포인트

- **비주얼 퀄리티 보장**: 일반 UGC(사용자 생성 콘텐츠)가 아닌 공모전 수상 전문 사진으로 앱 신뢰도·심미성 극대화
- **혼잡도 역발상 전략**: 핫플을 추천하는 게 아니라 혼잡하지 않은 숨은 명소를 추천 — 오버투어리즘 해소에 기여
- **지역 균형 발굴**: 지자체별 관광지 API를 기반으로 수도권·제주 쏠림 없이 전국 고른 명소 발굴
- **SNS 연동 바이럴 구조**: 앱 내 사진 카드 공유 → 자연적 바이럴 마케팅 효과 — 사용자 획득 비용 절감

## 공모전 강점

- **지역 특화 가점(2점) 직접 공략**: 지자체별 관광지 API 중심 설계 — 지역 관광 활성화 심사 기준 정조준
- **RTO 특별상 연계 가능성**: 지역관광공사(RTO) 연계 콘텐츠 구조로 특별상 수상 전략적 포지셔닝
- **사진 공모전 수상작 활용**: 한국관광공사 자체 자산을 앱 핵심 콘텐츠로 활용 — 심사위원 친화도 높음
- **MZ세대 SNS 문화 반영**: 트렌드 부합성과 확산력을 동시에 어필 가능한 기획

## 예상 사용 시나리오

> **"SNS에서 다들 가는 곳 말고, 진짜 나만 아는 곳 찾고 싶다"**
>
> 사용자가 앱 실행 후 '경상북도 · 주말 · 혼자' 조건 설정.
>
> K-Local Explorer가 혼잡도 데이터 기반 "이번 주말 경주·안동 혼잡 예상 → 대신 청송 주왕산 달기약수터 주변 추천"을 제안.
> 해당 명소 화면에는 사진 공모전 대상 수상작 3장이 표시되고, 연관 명소로 '주산지 왕버들 연못'이 연결된 반나절 코스가 자동 생성됨.
> "화요일 오전 9시 방문 시 혼잡도 최저" 타이밍 안내와 함께 인스타 공유용 감성 카드("경북의 숨은 보석, 청송에서 찾다")가 자동 생성됨.
