# 아이디어 19: K-Baseball Fan Trip

## 한 줄 정의
프로야구 10개 구단 원정 경기를 기준점으로, **경기 + 지역 맛집 + 야경 + 숙박**을 하나의 "팬 여행 패키지"로 묶어주는 KBO 팬 전용 앱. 일·영·중 다국어로 외국인 팬 유입도 포함.

## 타겟 사용자
- **1차**: KBO 팬 국내 20-50대 — 연 관중 800만명+ (2024)
- **2차**: 일본·대만 야구 팬 방한 (김하성·류현진 현역 시절 이후 한국야구 관심 증가)

## 문제 정의 / 근거
- KBO 2024년 관중 1,088만명 돌파 (역대 최다), 원정 팬 비중 30%+
- "주말 원정 여행자"는 경기 + 여행을 묶어 계획하려 하나, 이를 도와주는 앱 부재
- 기존 야구 앱(네이버 스포츠·KBO 공식)은 기록·예매만, 여행 큐레이션 無
- 외국인 야구팬 방한 트렌드 확대 — "일본 다음은 한국야구" 해외 콘텐츠 증가

## 핵심 기능
1. **경기 일정 기반 여행 생성**: "5/17 수원 vs 롯데" 선택 → 수원 도착·경기장 도보·원정팀 숙소·돌아오는 루트 자동 생성
2. **구장별 현지 맛집**: 10개 구장 반경 1km 팬 추천 맛집 DB + 경기 전후 혼잡도
3. **원정 응원가 오디오**: 각 구단 공식 응원가 + 구장 방문 퀘스트 카드
4. **외국인 모드**: 4개 언어 지원 + 티켓 구매 가이드 + 치맥 문화 설명
5. **스탬프 투어**: 10개 구장 전 방문 인증 앱 스탬프 (연간 챌린지)

## 활용 API
- `tourism-info-korean/english/chinese/japanese` — 구장별 관광 정보
- `related-attractions` — 구장 반경 명소·맛집
- `visitor-concentration-forecast` — 경기 전/후 혼잡도
- `central-attractions-by-municipality` — 원정 지역 필수 방문지
- `photo-contest-winners` — 구장·야경 포토스팟

## 차별화 포인트
- KBO 팬 전용 여행 앱 **국내외 전무** (미국 MLB도 비슷한 사례 드묾)
- 10개 구장 스탬프 투어 = 장기 리텐션 유도
- 외국인 야구팬 유입은 정책 부합 + 지역 활성화
- B2B: 구단·지역 상권 연계 수수료

## 예상 사용 시나리오
```
32세 두산 팬 (주말 수원 원정):
  앱: "5/17 토 두산 vs 수원 · 서울 출발" 입력
  자동 코스:
    11:00 수서 출발 → GTX + 환승
    12:30 수원역 도착 + 통닭거리 점심
    14:00 수원 화성 구경 (1시간)
    16:00 KT 위즈파크 도착 (입장 1시간 전)
    18:30 경기 (응원가 플레이리스트 자동 재생)
    22:00 경기 후 교동 야맥 장소 추천
    23:30 수원역 복귀 → 서울 귀가
  일본팬 모드: 영어 티켓 구매 가이드 + 치맥 문화 소개
```

## 예상 사업성 (1-10)
- 시장 규모: 7 (KBO 팬 연 1천만 + 해외 유입)
- 차별화: 9 (야구팬 전용 여행 앱 전무)
- 구현 난이도: 7 (KBO 일정 API 외부 연동, 구단 파트너십)
- 공모전 적합성: 8 (K-스포츠 관광 트렌드 + 외국인 유입)
- **총점: 31/40**
