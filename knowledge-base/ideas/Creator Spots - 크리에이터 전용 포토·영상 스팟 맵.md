# 아이디어 17: Creator Spots

## 한 줄 정의
인스타·유튜브·틱톡 크리에이터를 위해 **최적 촬영 시각·구도·장비 세팅**까지 알려주는 포토·영상 전용 맵. 시간대별 빛 방향 + 혼잡도 + 허가 여부 통합.

## 타겟 사용자
- **1차**: 여행 콘텐츠 크리에이터 국내외 (인스타 팔로워 1만+) — 방한 크리에이터 수요 급성장
- **2차**: 일반 인생사진 선호 20대 — "인생샷 찍는 곳" 니즈 상존

## 문제 정의 / 근거
- K-컬처 붐으로 해외 크리에이터 방한 급증 — 2024 추정 월 2만+ 인플루언서급
- 기존 인생샷 앱은 "장소"만 알려줌 — 정작 중요한 **"몇 시에 어느 방향에서"** 는 누락
- 크리에이터들은 "포토그래퍼 가이드"에 시간당 30만원+ 지출 — 앱으로 대체 시 가치 큼
- `photo-contest-winners` API 수상작 데이터가 있지만 활용 서비스 없음

## 핵심 기능
1. **시간대별 빛 방향 계산**: 태양 고도 API + 각 POI의 촬영 방향 → "경복궁 근정전: 오전 9:30 정면 빛, 오후 3시 역광 실루엣"
2. **수상작 재현 가이드**: `photo-contest-winners` 수상 사진의 정확한 촬영 지점·구도·날씨 복원
3. **장비 프리셋**: 아이폰 / DSLR / 드론(허가 필요 구역 표시)별 권장 세팅
4. **혼잡도 × 촬영 시간**: 피크 시간 피해 한적한 시각 제안
5. **촬영 허가 정보**: 상업 촬영·드론 금지 구역 표시 + 신청 링크

## 활용 API
- `photo-contest-winners` — 수상작 장소·구도 DB
- `visitor-concentration-forecast` — 혼잡 회피
- `tourism-info-korean/english` — 장소 기본
- `central-attractions-by-municipality` — 지자체 포토 허가 정보
- `area-tourism-demand-density` — 한적한 시간대 분석

## 차별화 포인트
- 빛 방향 + 구도 + 허가 정보 통합 앱 글로벌 전무
- 수상작 재현 기능 = 관광공사 공모전 데이터 재활용의 모범 케이스 (공모전 심사 매력)
- 크리에이터 중심 니치 + 일반 인스타그래머 확장 가능
- B2B: 카메라 브랜드·촬영장비 광고, 지자체 포토 관광 패키지

## 예상 사용 시나리오
```
Emma Chen (23세 대만 크리에이터) 전주 방문:
  앱 자동 언어 감지 (中文繁體) → "전주 · 오전 · 인스타 포트레이트" 선택
  코스 제안:
    08:30 한옥마을 골목 (동향 빛, 역광 실루엣 10분)
    09:30 경기전 입구 (정면 빛, 정적 구도 20분)
    10:30 태조로 은행나무길 (나뭇잎 사이 빛, 필름 톤 권장)
    11:30 광장 포차 (역동 스트리트 촬영)
  각 지점 별 권장 렌즈 + 노출 세팅 + 수상작 구도 썸네일 동반
  Dcard 공유 → 대만 팬덤 바이럴
```

## 예상 사업성 (1-10)
- 시장 규모: 6 (크리에이터 니치 + 일반 확장)
- 차별화: 9 (빛·구도·허가 통합 앱 전무)
- 구현 난이도: 7 (태양고도 계산 + 수상작 메타 재구축)
- 공모전 적합성: 9 (photo-contest-winners API 핵심 재활용)
- **총점: 31/40**

## 리스크
- 상업 촬영 허가 규정 — 지자체별 상이, 정보 업데이트 부담
- 수상작 저작권 — 썸네일·구도 공유는 공정 이용 범위 확인 필요
- 경쟁: 인스타 해시태그 — 수상작 + 공식 구도 가이드로 차별화
