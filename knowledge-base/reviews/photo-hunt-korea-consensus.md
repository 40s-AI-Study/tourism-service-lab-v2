## 합의 결론

- **추천도**: 상
- **핵심 강점**:
  - `tourism-photos` + `photo-contest-winners` — Round 1 미활용 2개 포토 API 선점, 공모전 데이터 활용(20점) 고득점 유력
  - "공식 사진 DB × 혼잡도 예측 × AI 코스" 3중 결합 — 네이버지도·카카오맵이 미제공하는 포토 특화 가치
  - 다크 모드 + 핑크-퍼플 그라디언트 비주얼 시스템이 MZ 인스타그래머 타깃과 완벽 정렬 — 공모전 시연 비주얼 임팩트 팀 내 최강
  - 김지원(20대 솔로) 28/30, Emma Chen(외국인 백패커) 28/30 고득점 PASS — 한국인·외국인 핵심 타겟 PMF 동시 검증
  - SNS 바이럴 루프(앱 자체가 콘텐츠 소재) — 자발적 확산 경로 실증
- **핵심 약점**:
  - 수익 모델 공백 — 현 기획 문서에 수익화 방안 미명시, 공모전 사업성 배점 감점 리스크
  - 프로토타입이 랜딩페이지 수준 — 핵심 기능 화면(지도 뷰, 골든타임 캘린더, 코스 생성)이 심사위원 직접 체험 불가
  - KTO `tourism-photos` DB 업데이트 주기 불명확 → 장기 데이터 품질 리스크
  - 30-40대 타깃 약함 — 핵심 고객이 20대에 편중되어 시장 규모 상한선 제약
- **공모전 제출 전 필수 조치**:
  1. **수익 모델 명시 필수**: 프리미엄 코스 판매 월 3,900원 + 로컬 포토투어 업체 CPA 제휴
  2. 핵심 화면 3개(지도+스팟목록, 골든타임 예측 뷰, 포토 투어 코스 생성) 프로토타입 추가
  3. 30대 밀레니얼 대상 "웰니스+포토", "미식+포토" 테마 코스로 연령층 확장
- **사용자 선택 시 고려사항**:
  - 포토 앱 특성상 공모전 데모 시 사진 DB 시각적 설득력이 높음
  - UGC 레이어 Phase 2 추가 시 커뮤니티 락인 + DB 보완 동시 해결 가능
  - 인스타그래머·크리에이터 인플루언서 협업으로 오가닉 바이럴 조기 구축 가능

## 소수 의견 (의견 충돌 기록)

- **기획님 9점 최고점**: 기획님이 팀 내 유일하게 9점 부여 — Pain Point 해결력과 포토 API 선점 전략에 특히 높은 평가. 타 평가자 8점과 1점 차이는 수익 모델 미비 관점 차이에서 기인.
- **공통 약점 합의**: 수익 모델 미명시는 4명 전원 공통 지적. 공모전 제출 전 반드시 기획서에 반영 필요.
- **분석님 추가 지적**: KTO DB 품질 종속성 및 30-40대 커버리지 미흡 — 웰니스·미식 테마 코스 추가로 연령층 확장 시 시장 규모 제약 해소 가능.
