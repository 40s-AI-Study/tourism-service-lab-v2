# [Round 3 합의 리뷰] 치유여정 Korea — 1위 선정

> CEO 최종 선정 | 통합 점수 최상위 | Sim 리텐션 1위

## 선정 이유 요약

3개 기존 아이디어(KoreaWellness + Audio Story Korea + LocalSecret)를 융합하여 **"소리와 침묵 사이에서 회복하는 감성 여행"**이라는 기존 어떤 관광 앱에도 없는 고유 포지셔닝 확보. 첫 화면이 "어떻게 회복할까?"를 묻는 것 자체가 트리플·야놀자·Klook과 근본적으로 다르다.

## CEO 자체 검증 (4항목)

### 1. 유사도 검증 — 기존 20개 중 어느 것과 유사?

- **갈래 B 융합**: KoreaWellness(5축 73점, 6축 34→v2 50) + Audio Story Korea(6축 37→v2 53) + LocalSecret(5축 82점, 6축 34→v2 52) 3개 융합
- 개별로는 각각 중위권이었으나 **3가지를 "치유 여정"이라는 하나의 서사로 묶으면서** 감정 JTBD 10점 + UI/UX 감정 차별화 10점 달성
- 융합 근거: 한적한 숨은 명소(LocalSecret) + K-웰니스 코스(KoreaWellness) + 오디오 스토리(Audio Story) = 걸으며 듣고 쉬는 완결형 여정

### 2. UI/UX 차별성 — Hero Screen이 경쟁사와 어떻게 다른가?

**트리플·야놀자: "어디 갈까?" → 치유여정: "어떻게 회복할까?"**

3가지 치유 모드(🎧 들으며 / 🌿 쉬며 / ✨ 발견하며)를 첫 화면에서 선택. 관광 앱이 아닌 **감성 회복 앱**으로 포지셔닝. 어둑한 사찰 길 영상 배경은 기존 앱의 밝은 검색창 UI와 완전히 다른 분위기.

### 3. 리텐션 이유 — 사용자가 왜 앱을 다시 여나?

- **감정적 재사용 트리거**: "지쳤을 때" → 앱을 연다 (스트레스·번아웃은 반복되는 감정)
- **계절별 다른 치유 여정**: 같은 코스도 봄·여름·가을·겨울 다른 콘텐츠 → 재방문 이유
- **오디오 스토리 수집 욕구**: "발견한 이야기 3개 기록됨" → 수집형 리텐션
- Sim 리텐션 평가: **50개 시뮬 중 감정 공명도 1위** (시뮬님 평가)

### 4. 자산 재활용 vs 신규 관점

| 재활용 (기존 자산) | 신규 관점 (Round 3) |
|---|---|
| KoreaWellness 웰니스 코스 데이터 | "어떻게 회복할까?" 감정 첫 화면 |
| Audio Story Korea 오디오 가이드 API | 위치 감지 자동 스토리 탐지 |
| LocalSecret 혼잡도 필터 | 3치유 모드(들으며/쉬며/발견하며) |
| 기존 8개 API 카탈로그 | 계절별 치유 여정 콘셉트 분기 |
| 기존 페르소나·시뮬 자산 | 국내/외국인 동일 코스 다른 서사 |
