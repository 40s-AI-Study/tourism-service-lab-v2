# [Round 3 합의 리뷰] LocalSecret v2 — 2위 선정

> CEO 최종 선정 | 통합 134점(5축 82 + 6축 52) | BA 원본 6축 공동 1위(48점)

## 선정 이유 요약

기존 "혼잡 회피" 기능 중심에서 **"나만의 장소를 발견하는 설렘"**으로 감정 재정의. 숨은 명소 발굴력 10점 만점 — 혼잡도 예측 + 다양성 지수 역활용이 핵심 차별점. 오버투어리즘 해소라는 사회적 가치까지 포함해 공모전 심사에서 강점.

## CEO 자체 검증 (4항목)

### 1. 유사도 검증

- **갈래 A 고도화**: LocalSecret(5축 82점, BA 6축 48점 공동 1위) 기반
- v1 → v2 변경: 혼잡도 히트맵 UI → "어떤 발견을 원해?" 감정 선택 첫 화면
- K-Local Explorer(6축 44점)와 유사하나 LocalSecret v2는 **발견 감정 + 혼잡도 예측 역활용**으로 차별화
- 기존 자산 활용률: ~70% (혼잡도 알고리즘 + API 그대로, UI/감정 재설계)

### 2. UI/UX 차별성

**트리플·야놀자: 인기 순위 표시 → LocalSecret v2: "오늘, 어떤 발견을 원해?"**

어두운 배경에 빛나는 별처럼 지도 위 점들이 깜박이는 Hero Screen. 비밀/고요/정취/계절 4가지 발견 감정 선택. 인기도가 아닌 **혼잡도 최하위 + 다양성 지수 기반**으로만 추천 — 기존 앱의 정반대 로직.

### 3. 리텐션 이유

- **발견 중독**: "아직 아무도 모르는 곳"을 찾는 탐험 욕구는 반복 사용 트리거
- **계절별 새 발견**: 같은 숨은 명소도 계절마다 다른 매력 → 4번 재방문
- **독점 심리**: "내가 먼저 발견했다"는 감정 → SNS 공유 자제 + 앱 내 기록 → 충성도
- Sim 리텐션: 전 10개 PASS, 특히 조용한 여행자 페르소나 리텐션 최상위

### 4. 자산 재활용 vs 신규 관점

| 재활용 (기존 자산) | 신규 관점 (Round 3) |
|---|---|
| 혼잡도 예측 캘린더 | "발견의 설렘" 감정 JTBD 재정의 |
| 빅데이터 급상승 명소 감지 | 4발견 감정 선택 Hero Screen |
| 5개 API 연계 | 계절별 숨은 명소 다른 매력 콘텐츠 |
| 기존 페르소나·시뮬 자산 | 한국인/외국인 동일 장소 다른 의미 서술 |

### 가장 유사한 기존 앱과 개선점

| 기존 앱 | 유사점 | LocalSecret v2 개선 |
|---|---|---|
| **트리플** | 여행지 탐색·추천 | 인기도·별점 기반 → **혼잡도 최하위 + 다양성 지수 기반** 정반대 추천 |

| **네이버 블로그 검색** | 숨은 맛집·명소 | 텍스트+후기 의존 → **발견 감정 선택 첫 화면 + 30일 혼잡도 예측** |

### 사용자가 앱을 다시 여는 3가지 상황

1. **"나만 알던 곳" SNS 바이럴됐을 때** — "새 비밀 장소 찾기" 위해 즉시 앱 실행
2. **여행 3일 전 주말 계획** — "이번 주말 혼잡도 최하위 TOP 3 + 방문 타이밍 알림"
3. **계절 바뀌는 시점** — "봄 신두리 해안사구 → 겨울 눈 덮인 버전 최적 타이밍 알림"

### 2024-2026 트렌드 연결 (3개)

1. **솔로 여행 + 즉흥 여행 증가**: 아무도 없는 나만의 공간 수요 직결
2. **숏폼 SNS → 오프라인 전환율 급증**: "아직 SNS에 없는 곳" → 희귀성 욕구 자극
3. **로컬 체험 > 관광지 사진**: 현지인만 아는 길, 관광지도에 없는 곳
