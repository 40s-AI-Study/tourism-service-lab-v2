# [Round 3 합의 리뷰] Audio Story Korea v2 — 3위 선정

> CEO 최종 선정 | 통합 131점(5축 78 + 6축 53) | 4계절 콘텐츠화 10점 만점

## 선정 이유 요약

기존 "QR 스캔 오디오 가이드"에서 **"걸으면서 이야기가 따라오는 탐험"**으로 감정 재정의. 4계절 콘텐츠화 만점(10점) — 같은 경복궁이 봄·여름·가을·겨울 완전히 다른 스토리. 문화유산 + 오디오 스토리텔링은 다른 4개 아이디어와 겹치지 않는 **유일한 포지셔닝**. 공모전 심사 "데이터 활용" 항목에서 오디오 가이드 API + 수상사진 API + 다국어 API 7개 조합 = 최다 API 활용.

## CEO 자체 검증 (4항목)

### 1. 유사도 검증

- **갈래 A 고도화**: Audio Story Korea(기존 PASS, 5축 79점, 6축 37점) 기반
- v1 → v2 변경: QR 스캔 → 위치 감지 자동 탐지, 단일 스토리 → 5성향별 테마 + 계절별 분기 + 이중 렌즈
- 치유여정 Korea와 오디오 요소 공유하나, **문화유산 탐험 vs 감성 회복**으로 목적이 다름
- 기존 자산 활용률: ~60% (오디오 API 그대로, UX/계절/성향 전면 재설계)

### 2. UI/UX 차별성

**트리플·야놀자: 목적지 검색 → Audio Story v2: "지금 내 주변 이야기 탐색 중..."**

지도에 유명 관광지는 작은 핀, 숨은 문화유산은 빛나는 별 아이콘. 걷다가 5m 앞 문화유산에 접근하면 자동 진동 + "탭해서 이야기 듣기". **검색하는 게 아니라 걸으면서 이야기가 따라온다** — 세렌디피티 UX.

### 3. 리텐션 이유

- **스토리 수집 욕구**: "오늘 발견한 이야기 3개 — 지도에 기록됨" → 수집형 리텐션
- **계절별 재방문**: 같은 장소 4계절 완전히 다른 스토리 → 연 4회 이상 재사용
- **문화적 깊이**: 짧은 리뷰가 아닌 역사 스토리 → 교육적 만족감 → 장기 충성도
- Sim 리텐션: 전통감성 페르소나 리텐션 최상위, 외국인 문화 탐험가 강력 반응

### 4. 자산 재활용 vs 신규 관점

| 재활용 (기존 자산) | 신규 관점 (Round 3) |
|---|---|
| 오디오 가이드 API | 위치 감지 자동 탐지 (QR 제거) |
| 수상사진 API | 5성향별 오디오 테마 분기 |
| 다국어 관광정보 API 5개 | 4계절 같은 장소 다른 스토리 |
| 기존 합의 리뷰 자산 | 숨은 문화유산 별 아이콘 강조 |
| | 이중 렌즈: 한국인/외국인 앵글 스토리 분리 |

## 벤치마크 레퍼런스 (Board 추가 요구사항 반영)

### 가장 유사한 기존 앱과 개선점

| 기존 앱 | 유사점 | Audio Story v2 개선 |
|---|---|---|
| **Rick Steves Audio Europe** | 관광지 오디오 가이드 | 유명 명소 중심 → **비유명 문화유산 별 아이콘** + 위치 자동 감지 |
| **네이버 지도/카카오맵** | 장소 정보·길찾기 | 텍스트 중심 → **근접 감지 자동 오디오** + 계절별 스토리 분기 |
| **문화재청 공식 앱** | 문화재 QR 오디오 | QR 스캔 필요 → **걷다가 자동 진동+탐지**, 5언어 계절 분기 |

### 사용자가 앱을 다시 여는 3가지 상황

1. **새 도시 방문 전날 밤** — "내일 경주 골목에 어떤 숨은 이야기가?" 미리 탐색하며 기대감
2. **계절 바뀌었을 때** — "봄 경복궁 → 가을 '사신 맞이 외교 이야기'" 같은 장소 재방문
3. **이어폰 끼고 혼자 걸을 때** — 동네 산책 중 반경 200m 숨은 문화유산 자동 감지 → 일상 속 탐험

### 2024-2026 트렌드 연결 (3개)

1. **로컬 체험 > 관광지 사진**: "아무도 모르는 비석 하나" 발견 → 스토리 공유 문화 대응
2. **외국인 FIT 역대급**: 외국인 앵글 스토리 별도 제공 → 가이드 없이 혼자 다니는 FIT 핵심 니즈
3. **숏폼 SNS → 오프라인 전환율**: 오디오 듣고 수상사진 각도로 촬영 → 오프라인 전환 허브
