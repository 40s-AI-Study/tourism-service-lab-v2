## 합의 결론

- **추천도**: 최상 (공모전 주력 출품작 확정 권고)
- **핵심 강점**:
  - "광고 없는 실소비 데이터 맛집" USP가 기획력(30점) + 데이터 활용(20점) 양 축을 동시에 관통
  - 8개 API 활용 — 출품작 중 최다 수준, Round 1 미활용 `area-tourism-resource-demand` 독창 활용
  - 9/10 PASS — 내국인(김지원 25/30)·외국인(David Park 26/30, 타나카 하루카 26/30) 동시 고득점으로 글로벌 PMF 검증
  - 7명 전원 9/10 만장일치 — 팀 역사상 최고 합의 수준
  - 전주(한식)·부산(해산물)·전남(남도음식) 지역 특화로 지역 가점 확보 가능
  - SNS 바이럴 루프(인스타 저장 312건, X 리트윗 800건) 자발적 확산 검증
- **핵심 약점**:
  - 유일한 FAIL(아미나, 무슬림 12/30) — 할랄 필터를 기능 목록에 나열했으나 실제 API에 할랄 인증 데이터 미구축. 기대-현실 괴리로 부정 바이럴 리스크
  - 히트맵 지역 단위→개별 식당 전환 데이터 공백 — `tourism-big-data` API가 지역·업종 집계만 제공, 개별 상호명 미포함
  - 현금 위주 소규모 식당 소비 데이터 과소 반영 가능성
  - 수익화 모델 중 "광고 없음" USP와 B2B 데이터 판매의 이미지 충돌 리스크
- **공모전 제출 전 필수 조치**:
  1. 할랄 필터를 "향후 확장 예정"으로 한정, 앱스토어 기능 목록에서 제거
  2. 히트맵→식당 전환 UI를 `tourism-info` API 음식점 정보로 보완
  3. `관광지 집중률 방문자 추이 예측` API 추가 검토 — 9개 API 달성 시 배점 상승
- **사용자 선택 시 고려사항**:
  - 공모전 심사 기준(데이터 활용·기획력·지역 기여) 전 항목에서 최고 경쟁력
  - 공모전 수상 후 실제 서비스 사업화 전환 가능성 높음 (B2C 구독 + B2B 관광청 대시보드)
  - 음식 크리에이터·유튜버 협업 채널로 오가닉 바이럴 조기 구축 가능

## 소수 의견 (의견 충돌 기록)

- **만장일치 9/10**: 7명 전원 동일 점수로 의견 충돌 없음. 팀 전체가 TasteData Korea의 공모전 경쟁력과 서비스 완성도에 높은 확신을 표명.
- **공통 약점 합의**: 할랄 필터 미구현과 히트맵→식당 전환 공백은 7명 중 5명(디자인·분석·엔지니어·시뮬·전략)이 공통 지적. 이 두 항목은 공모전 제출 전 반드시 스코프 조정 필요.
- **엔지니어님 추가 제안**: 방문자 추이 예측 API 추가 시 9개 API 달성 → 데이터 활용 배점 추가 상승 가능. 전략적으로 검토 가치 있음.
