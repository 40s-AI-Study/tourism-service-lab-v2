# SIM-36: PetKorea × 김영자 (한국 시니어)

## 시나리오

2026년 10월, 김영자(67세, 은퇴교사, 부천)는 딸 가족과 함께 2박3일 가을 단풍 여행을 계획하고 있다. 영자 씨 본인은 반려묘 '복순이'를 키우지만 장거리 여행에는 복순이를 데려가지 않는다. 대신 딸 부부가 기르는 소형견 '도토리'(비숑 프리제, 5kg)가 동행한다. 영자 씨의 무릎 관절염으로 인해 평탄한 보행로, 벤치와 쉼터가 충분한 장소가 필수 조건이다.

딸이 PetKorea 앱을 설치해 여행 계획을 주도하고, 영자 씨는 옆에서 화면을 함께 본다. 영자 씨가 직접 앱을 조작하려 했으나 글씨가 작고 버튼 간격이 좁아 어려움을 느낀다. 딸이 대신 조작하며 조건을 입력한다. 목적지는 충남 공주·부여 지역 단풍 코스, 조건은 소형견 동반 가능 + 시니어 접근성 우수 + 관절염 배려 + 2박3일.

AI 코스 생성 결과로 공주 공산성(평탄 구간 선택 가능) → 반려견 동반 가능 한정식 레스토랑 → 부여 궁남지(벤치 다수) 코스가 제안된다. 영자 씨가 궁남지 사진을 보고 "여기 좋겠다"라고 직접 말하는 장면에서 앱의 가치가 드러난다.

## 사용자 여정

### 1단계: 진입
- 딸이 앱 설치 및 조작 주도 (영자 씨는 보조 사용자)
- 도토리 프로필 등록 (비숑 프리제, 5kg, 예방접종 완료)
- 가족 구성: 성인 3명 + 시니어 1명(이동 제한) 체크
- AI 코스 생성 → 조건 입력: 소형견/자가용/2박3일/충남/시니어 접근성 우수

### 2단계: 행동
- `pet-friendly-travel` API (카테고리: 관광지) → 반려견 동반 가능 + 접근성 우수 필터 동시 적용, 공산성·궁남지 포함 결과 수신
- `tourism-info-korean` API → 공산성 내 평탄 구간 정보, 벤치 위치, 화장실 거리 확인
- `pet-friendly-travel` API (카테고리: 음식점) → 소형견 동반 가능 한정식 레스토랑 검색, 영자 씨 선호 한식 필터
- `visitor-concentration-forecast` API → 10월 단풍 성수기 혼잡도 예측: 주말 "매우 혼잡" → 주중 출발 권장
- `related-attractions` API → 숙박지 반경 내 야외 평탄 산책로 추가 추천
- 딸이 영자 씨에게 각 장소 사진 보여주며 의견 수렴 → 궁남지·공산성 코스 최종 확정

### 3단계: 결과
- 2박3일 공주·부여 여행 완료. 영자 씨 무릎 무리 없이 완주
- 궁남지 연꽃 단풍 배경 도토리 사진 촬영, 딸 SNS 업로드
- 불만족: 앱 글씨 크기가 작아 영자 씨가 직접 사용 불가. 시니어 접근성 데이터가 "우수/보통/미흡" 3단계로만 표시되어 구체적 경사도 정보 부족
- 만족: 딸이 대신 사용하면서도 어르신 동반 조건 처리 결과의 정확도에 만족. 혼잡도 예측으로 주중 출발 결정이 탁월

## 개선 제안

1. **시니어 모드 (큰 글씨 UI)**: 설정에서 텍스트 크기 확대 + 버튼 간격 확장 옵션 제공, 또는 OS 접근성 설정 연동으로 시니어 직접 사용 가능하게 개선
2. **접근성 세부 정보 강화**: "접근성 우수" 배지 클릭 시 경사도(%), 보행로 포장 여부, 최장 도보 거리, 벤치 간격 등 구체적 수치 제공
3. **동반자 공유 플래닝**: 가족 구성원이 같은 여행 계획을 공유·편집할 수 있는 기능으로 딸-어머니 협업 여행 계획 경험 개선
