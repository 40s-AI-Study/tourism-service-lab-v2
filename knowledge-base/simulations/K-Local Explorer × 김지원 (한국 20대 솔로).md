# SIM-51: K-Local Explorer × 김지원 (한국 20대 솔로)

## 시나리오

김지원(25세, UX 디자이너)은 매주 주말 서울 합정·성수 일대를 탐방하며 인스타그램에 "아는 사람만 아는 감성 카페" 시리즈를 연재한다. 팔로워 8,200명을 보유한 마이크로 인플루언서로, 이미 유명해진 카페는 올리지 않는 것이 원칙. 이번 주말 홍대 라인에서 아직 미디어에 노출되지 않은 감성 공간을 찾고 싶다. 친구에게 추천받은 K-Local Explorer를 처음 설치해본다.

---

## 사용자 여정

### 1단계: 진입

앱을 설치하고 온보딩 화면에서 관심사 태그를 선택한다: `#감성카페`, `#인테리어`, `#포토스팟`, `#성수합정`. 앱이 현재 위치(합정역 근처)를 감지하고 "합정·마포 지역 숨은 명소 7곳 발견" 배너를 보여준다. 사진 공모전 수상작 갤러리가 메인 화면에 배치되어 있고, 첫 번째 이미지가 시선을 잡는다 — 낡은 벽돌 건물 2층의 창가 좌석, 필름 카메라 느낌의 색감.

`central-attractions-by-municipality` API가 마포구 데이터를 호출하고, `visitor-concentration-forecast` API가 현재 혼잡도 하(下)인 장소를 필터링해 상위 노출한다.

### 2단계: 행동

지원은 갤러리 탭을 열어 수상작 사진을 훑는다. "이 사진 어디야?" 싶은 장소를 탭하면 해당 명소 상세 페이지로 이동한다. `photo-contest-winners` API로 연결된 수상작 메타데이터에는 촬영 장소, 촬영 시간대, 추천 앵글 정보가 포함되어 있다.

"합정 로컬 반나절 코스" 탐방 코스를 선택하면 `related-attractions` API가 도보 15분 이내 연관 명소 4곳을 순서대로 제안한다. 혼잡도 예측 기능이 "이번 주 토요일 오전 10시가 방문 최적 시간"임을 알려준다. 지원은 코스를 저장하고 방문 알림을 설정한다.

토요일 방문 후 앱 내 SNS 공유 기능을 사용한다. 자동 생성된 카드뉴스 템플릿(장소명, 주소, 추천 시간대, 수상작 사진 포함)을 약간 수정해 인스타그램에 올린다. 게시물에 앱 태그를 포함시켜 팔로워들에게 앱을 자연스럽게 소개한다.

### 3단계: 결과

인스타그램 게시물이 저장 수 340건, 댓글 67건을 기록하며 평소보다 3배 높은 인게이지먼트를 달성한다. 팔로워 12명이 댓글에 "앱 이름이 뭐예요?"를 묻는다. 지원은 앱 바이럴의 자발적 앰배서더가 되었고, 다음 주 탐방 코스도 앱으로 계획한다. 혼잡도 역발상 기능 덕분에 주말임에도 줄 서지 않고 방문에 성공했다는 후기를 스토리에 추가로 올린다.

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 숨은 명소 발굴 + 수상작 비주얼 + 혼잡도 최적 타이밍이 인플루언서 니즈와 정확히 맞아떨어짐. SNS 공유 카드뉴스 자동생성은 핵심 워크플로우를 단축시킴 |
| 사용 편의 | **8** | 위치 기반 자동 감지, 갤러리 우선 UX가 디자이너 감성에 맞음. 온보딩 태그 선택이 직관적. 다만 코스 저장 후 알림 설정 단계가 2단계 더 있어 약간 번거로움 |
| 구현 타당성 | **8** | 4개 API 모두 실제 활용 가능. 카드뉴스 자동생성은 템플릿 기반이면 충분히 구현 가능. 혼잡도 예측의 정확도가 관건 |

**종합 점수: 25/30**

## Pass/Fail 판정

> **✅ PASS**

유용성 9점, 종합 25점으로 기준(유용성 6점 이상, 종합 18점 이상)을 넉넉히 충족한다. K-Local Explorer의 핵심 타겟인 MZ 인플루언서 페르소나에서 앱의 모든 핵심 기능(숨은 명소 발굴, 수상작 갤러리, 혼잡도 최적화, SNS 공유)이 실제 가치를 만들어냈다. 바이럴 구조가 자연스럽게 작동하는 이상적인 케이스.

## 개선 제안

1. **크리에이터 전용 모드 신설**: 팔로워 수 연동(인스타그램 API)으로 마이크로 인플루언서를 자동 인식하고, 수상작 사진의 RAW 파일 다운로드 권한이나 고해상도 버전을 제공하는 "크리에이터 패스" 기능을 추가하면 충성 사용자 집단을 형성할 수 있음.
2. **방문 후 검증 배지**: 사용자가 실제 방문 후 인증샷을 업로드하면 "발굴자 배지"를 부여하고 명소 페이지에 첫 번째 방문자로 등록하는 기능. 희소성과 성취감을 동시에 충족시켜 재방문율과 게시율을 높임.
