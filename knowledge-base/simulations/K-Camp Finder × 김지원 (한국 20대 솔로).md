# SIM-91: K-Camp Finder × 김지원 (한국 20대 솔로)

## 시나리오

김지원(25세, UX 디자이너)은 인스타그램에서 글램핑 감성 사진을 보고 혼자 떠나는 주말 여행을 계획 중이다. 혼자 가는 만큼 안전하고 아늑한 소규모 캠핑장을 원하고, 포토스팟이 될 만한 감성적인 장소를 찾는다. 바쁜 직장 생활로 예약부터 준비물 체크까지 한 번에 해결하고 싶다.

K-Camp Finder 앱을 발견하고, "혼자 / 글램핑 / 이번 주말 / 경기도" 조건으로 검색한다. AI가 혼잡도 낮은 감성 글램핑 TOP 5를 제시하고, 각 장소의 SNS 인기 포토스팟 정보와 인근 카페·드라이브 코스를 연계해 보여준다.

---

## 사용자 여정

### 1단계: 진입

- 인스타그램 릴스에서 K-Camp Finder 광고 노출 ("혼캠 감성 명소 AI 추천")
- 앱 다운로드 후 소셜 로그인 (카카오)
- 온보딩: 캠핑 경험 수준 선택 → "입문/글램핑 선호" 선택
- 메인 화면에서 "지금 바로 추천받기" 버튼 탭

### 2단계: 행동

- 조건 입력: 인원 1명, 날짜 이번 주 토-일, 지역 경기도, 숙박 형태 글램핑
- AI 추천 결과 수신: TOP 5 글램핑장 카드 (사진, 혼잡도 레이더, 평점, 가격대)
- 2번 후보 "별빛 글램핑 파주" 선택: 일몰 뷰 포토스팟, 인근 헤이리 예술마을 연계 코스 확인
- 체크리스트 자동 생성: 1인 글램핑 기준 준비물 (침낭 불필요, 취사도구 제공 등)
- D-3 날씨 알림 수신 확인 후 예약 진행 (gocamping API 연동 예약 페이지)

### 3단계: 결과

- 예약 완료 후 앱 내 일정 저장
- D-3 날씨 푸시 알림 수신: "토요일 맑음 18°C, 일요일 오전 흐림 예상 → 우비 챙기세요"
- 방문 후 리뷰 작성: 포토스팟 위치 핀 추가 기여

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 1인 글램핑 특화 필터, 포토스팟 정보, 혼잡도 확인이 솔로 여행자 핵심 니즈와 정확히 일치 |
| 사용 편의 | **8** | 조건 입력→추천→예약 원스톱 플로우, 체크리스트 자동화로 초보자도 부담 없음 |
| 구현 타당성 | **8** | gocamping API + visitor-concentration-forecast + related-attractions 조합으로 실현 가능, SNS 포토스팟 데이터 추가 수집 필요 |

**종합 점수: 25/30**

---

## Pass/Fail 판정

> **✅ PASS**

솔로 글램핑 여행자의 핵심 니즈(안전한 소규모 장소, 감성 포토스팟, 편리한 예약)를 모두 충족한다. 유용성 9점, 종합 25점으로 Pass 기준(18점 이상, 유용성 6점 이상)을 명확히 상회한다. 1인 여행자 안전 정보(리뷰 기반 치안 평점 등) 보강 시 완성도 더욱 향상 가능.

## 개선 제안

1. 솔로 여행자 안전 지표 추가: 캠핑장별 "혼자 방문 리뷰 비율", 치안 평점, 관리인 상주 여부를 별도 필터로 제공
2. 포토스팟 커뮤니티 기능: 방문자가 직접 감성 사진 포토스팟 핀을 남기고 공유하는 UGC 기능 추가로 콘텐츠 자생적 확장
