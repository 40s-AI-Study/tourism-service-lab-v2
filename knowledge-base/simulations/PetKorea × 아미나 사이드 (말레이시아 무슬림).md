# SIM-39: PetKorea × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

2026년 8월, 아미나 사이드(30세, 쿠알라룸푸르 크리에이티브 디렉터)는 5박6일 한국 여행 중이다. 이슬람 신자로 할랄 음식이 최우선 조건이며 반려동물은 동반하지 않는다. 무슬림에게 개는 종교적으로 접촉 시 정결 의식이 필요한 동물이기 때문에 반려견 동반 카페나 식당 방문은 아미나에게 오히려 회피 대상이다.

서울 명동 호텔에서 인스타그램을 보다가 한국 여행 앱 리뷰에서 PetKorea를 발견한다. 앱 소개 문구에서 "여행 코스 추천"이라는 키워드에 관심을 가지고 설치를 시도한다. 그러나 앱을 열자마자 반려동물 프로필 등록 권유와 함께 강아지·고양이 이미지가 가득한 화면이 나온다. 할랄 음식 필터나 무슬림 여행자 배려 기능이 없음을 즉시 파악하고, 자신에게 맞지 않는 앱이라 판단해 5분 내에 삭제한다.

---

## 사용자 여정

### 1단계: 진입
- 인스타그램 여행 리뷰 계정에서 앱 링크 클릭 후 설치
- 온보딩 화면: 반려동물 등록 권유 → 건너뜀
- 메인 화면: 반려동물 중심 콘텐츠 및 이미지 확인
- 할랄 음식 필터·무슬림 여행 카테고리 검색 → 없음 확인

### 2단계: 행동
- `pet-friendly-travel` API 기반 음식점 검색 → 반려동물 동반 가능 식당 목록만 제공, 할랄 인증 필터 없음
- `tourism-info-korean` API 일반 관광지 검색 시도 → 반려동물 맥락 없이도 일부 결과 나오지만 할랄 관련 정보 전무
- 앱의 핵심 UX(반려동물 동반 최적화)가 아미나의 니즈와 완전히 불일치함을 인지
- 앱 삭제 결정, Halal Trip 앱으로 대체

### 3단계: 결과
- PetKorea 앱 5분 내 삭제
- 반려동물 동반 니즈 없음 + 무슬림 여행자 배려 기능 부재의 이중 미스매치
- 아미나의 한국 여행은 Halal Trip + 네이버 지도 조합으로 진행, PetKorea와 무관하게 성공적으로 마무리
- 앱에 대한 부정적 인식: "반려동물 없는 여행자에게는 아무 쓸모 없는 앱"

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **1** | 반려동물 동반 니즈 없음 + 할랄 기능 부재로 실질 가치 없음 |
| 사용 편의 | **3** | 영어 지원 여부와 무관하게 앱 구조 자체가 이 페르소나와 불일치 |
| 구현 타당성 | **6** | API 인프라는 존재. 할랄 필터는 별도 데이터 소스 필요 |

**종합 점수: 10/30**

## Pass/Fail 판정

> **❌ FAIL**

반려동물 동반 니즈 부재 + 할랄 기능 전무로 유용성 1점, 종합 10점이 기준에 심각하게 미달한다. 아미나 사이드는 PetKorea의 타겟 외 사용자이며 앱 정체성과 근본적으로 불일치한다.

## 개선 제안

1. **앱스토어 타겟 명확화**: "반려동물 동반 국내 여행 전용"을 앱 설명 첫 줄에 명시해 비타겟 사용자(할랄 여행자, 비펫 외국인)의 잘못된 설치 방지
2. **무슬림 여행자 배려는 별도 앱 범위**: PetKorea 앱 내 할랄 기능 추가는 앱 정체성과 맞지 않음. 대신 K-Travel 범용 앱과의 제휴나 API 공유로 간접 지원 검토
3. **온보딩 조기 미스매치 감지**: 반려동물 프로필 등록을 건너뛰는 사용자에게 "이 앱은 반려동물 동반 여행 전용입니다. 계속하시겠습니까?" 안내를 추가해 이탈을 자연스럽게 유도하고 리뷰 점수 피해 최소화
