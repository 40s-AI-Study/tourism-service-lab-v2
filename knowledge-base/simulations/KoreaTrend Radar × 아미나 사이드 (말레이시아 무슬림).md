# SIM-49: KoreaTrend Radar × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

2026년 4월 17일 목요일 오후 9시, 쿠알라룸푸르. 인플루언서 아미나 사이드(30)는 팔로워 18만 명의 무슬림 여행 크리에이터다. 다음 달 한국 방문을 앞두고 "이번 주 뜨는 한국 명소"를 콘텐츠로 만들기 위해 자료를 수집 중이다. 팔로워들은 할랄 식당과 기도실 정보를 특히 중요하게 여긴다. KoreaTrend Radar가 "빅데이터 기반 실시간 핫플"이라는 소문을 듣고 영어로 실행했다.

홈 레이더에서 서울 히트맵을 보니 이번 주 이태원과 동대문 일대가 강하게 빛나고 있다. 이태원은 무슬림 친화적 지역으로도 알려져 있어 관심이 간다. 이태원 상세 화면을 열어보니 트렌드 지표(이동통신 +61%, 카드 +44%)는 풍부하지만 할랄 식당·기도실 정보는 없다. 아미나는 "트렌드는 알겠는데 무슬림이 편하게 갈 수 있는 곳인지 모르겠다"며 아쉬워한다.

그래도 이태원의 트렌드 급등 데이터를 콘텐츠에 활용하기로 하고 — "빅데이터가 이태원을 이번 주 서울 1위로 꼽았습니다"라는 스크린샷을 인스타그램 스토리에 올린다. 할랄 정보는 별도 Halal Trip 앱에서 보완한다.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 (영어 UI)
- 홈 레이더에서 이태원·동대문 핫스팟 확인
- 할랄 필터 탐색 → 부재 확인

### 2단계: 행동
- 이태원 상세 화면: 트렌드 지표 확인 (이동통신 +61%, 카드 +44%)
- 할랄 식당·기도실 정보 탐색 → 없음
- 동대문 트렌드 데이터 비교 확인
- 트렌드 데이터 스크린샷 저장 → 인스타그램 스토리 업로드
- Halal Trip 앱 병행 실행으로 이태원 할랄 정보 보완

### 3단계: 결과
- 인스타그램 스토리 "빅데이터 서울 1위 = 이태원" 게시 → 좋아요 2,300건
- KoreaTrend Radar는 콘텐츠 신뢰도 높이는 데이터 소스로 활용
- 할랄 정보 이중 앱 사용이 번거로움으로 남음

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **6** | 트렌드 데이터가 콘텐츠 크리에이터 역할에서 "빅데이터 근거" 제공 측면으로 유용. 그러나 무슬림 여행자 핵심 니즈(할랄, 기도실) 정보 완전 부재 |
| 사용 편의 | **6** | 영어 UI 수용 가능. 인스타그램 공유 기능 편리. 할랄 필터 없어 이 페르소나의 여행 결정에 단독 활용 불가 |
| 구현 타당성 | **6** | 트렌드 데이터 구현은 충분. 할랄 정보는 외부 DB(Korea Halal Authority 등) 연동 필요 — 추가 파트너십 과제 |

## 개선 제안

1. **무슬림 친화 필터**: 할랄 레스토랑·기도실·무슬림 친화 숙소 필터를 트렌드 데이터와 결합해 "이번 주 뜨는 무슬림 친화 명소" 뷰 제공. Korea Halal Authority DB 연동
2. **다국어 확장 (말레이어/아랍어)**: 동남아·중동 무슬림 관광객 대상 말레이어 UI 추가 및 할랄 관련 용어 현지화로 무슬림 관광 시장 공략 강화
