# SIM-54: K-Local Explorer × Emma Chen (외국인 20대 백패커)

## 시나리오

Emma Chen(23세, 대만 출신 여행 사진작가)은 인스타그램 팔로워 45,000명을 보유한 마이크로-매크로 경계의 크리에이터다. 서울 3일을 마치고 지방 도시로 이동하려 한다. "서울 말고, 외국인 관광객이 잘 모르는 한국 지방의 포토스팟"이 그녀의 다음 콘텐츠 테마. 경주·전주·통영 중 어디로 갈지 결정하지 못한 상태에서 K-Local Explorer를 발견한다. 앱이 영어를 지원한다는 것을 확인하고 설치한다.

---

## 사용자 여정

### 1단계: 진입

앱을 영어 모드로 설정하고 관심사 태그를 `#photospots`, `#hiddenplaces`, `#countryside`, `#architecture`로 선택한다. 앱이 "Regions Less Traveled" 섹션에서 전국 지방 도시 숨은 포토스팟을 지도 뷰로 보여준다. `photo-contest-winners` API 기반의 수상작 갤러리가 영어 캡션과 함께 로드된다. Emma는 통영의 수상작 사진 — 새벽 안개 속 굴 양식장 — 을 보고 즉시 결정한다: 통영.

### 2단계: 행동

통영 명소 페이지에서 `central-attractions-by-municipality` API가 통영시 핵심 명소를 불러오고, 혼잡도 낮은 장소를 "Hidden Gems" 태그로 분류해 상단 배치한다. Emma는 달동네 감성의 동피랑 벽화마을 대신 아직 인스타에 거의 노출되지 않은 "통영 서피랑 전망대"를 발견한다.

`related-attractions` API가 "통영 포토 코스 1일"을 제안한다: 새벽 굴 양식장 → 오전 서피랑 전망대 → 점심 중앙시장 로컬 식당 → 오후 케이블카 너머 한려수도 전망. 수상작 사진에서 각 장소의 촬영 시간대와 최적 앵글 정보가 영어로 제공된다. `visitor-concentration-forecast` API가 새벽 5시 30분 굴 양식장 방문을 권장한다.

Emma는 코스를 저장하고 앱 내 카드뉴스 자동 생성 기능으로 영문 포스팅 초안을 만든다. 수상작 사진의 고해상도 버전 사용 권한을 요청하는 팝업이 뜨지만, 크리에이터 권한 신청 절차가 복잡해 일단 건너뛴다.

### 3단계: 결과

통영 방문 후 올린 릴스가 조회수 230,000회를 기록하며 역대 최고 성과를 냈다. 캡션에 "Found via K-Local Explorer" 태그가 포함되었고, 수천 명이 앱을 검색한다. Emma는 앱이 "서울 편향 없이 지방을 균형 있게 커버한다"는 점을 팔로워들에게 강조한다. 단, 수상작 사진 저작권 문제로 고해상도 파일을 사용하지 못한 것이 아쉽고, 일부 영문 번역이 어색하게 느껴졌다.

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 지방 숨은 포토스팟 발굴이라는 니즈를 정확히 충족. 수상작 갤러리가 의사결정(통영 선택)까지 주도. 촬영 최적 시간대 정보는 사진작가에게 극히 유용 |
| 사용 편의 | **7** | 영어 지원이 진입 장벽 제거. 지도 뷰 + 갤러리 뷰 전환이 직관적. 수상작 고해상도 접근 절차가 복잡하고 영문 번역 품질이 일부 불균일 |
| 구현 타당성 | **8** | 영어 다국어 지원은 추가 개발 비용 있지만 필수 투자. 저작권 처리 체계 구축이 핵심 과제. API 4종 연동은 기술적으로 타당 |

**종합 점수: 24/30**

유용성 9점, 종합 24점으로 기준을 충족한다. 외국인 사진작가 크리에이터라는 페르소나에서 앱의 비서울 지방 커버리지와 수상작 갤러리가 결정적 가치를 만들어냈다. 바이럴 확산 효과까지 시뮬레이션에서 확인된 강력한 케이스.

## 개선 제안

1. **크리에이터 고해상도 라이선스 원클릭 신청**: 수상작 사진의 고해상도 사용 권한을 SNS 크리에이터에게 간소화된 절차(인스타그램 계정 연동 + 약관 동의)로 즉시 부여. 콘텐츠 품질 향상과 앱 홍보 효과를 동시에 달성.
2. **다국어 번역 품질 관리 파이프라인**: AI 번역 후 현지 원어민 검수 체계를 구축하거나, 번역 품질 낮은 항목에 사용자 피드백 버튼을 추가해 지속적 개선. 영어 외 일본어·중국어·태국어 등 아시아 주요 언어 지원으로 외국인 타겟을 확대.
