# SIM-163: KoreaFestival Live × 최동훈 (한국 40대 가족)

## 시나리오

여름방학을 앞두고 동훈은 아이들에게 의미 있는 체험을 시켜주고 싶어 "교육적인 축제"를 검색한다. 맘카페에서 정보를 얻으려 했지만 광고성 글이 섞여 있어 신뢰하기 어렵다. KoreaFestival Live를 설치해 "테마: 전통문화·체험" + "가족 여행" 필터를 적용하자, 안동국제탈춤페스티벌과 전주한지문화축제가 상단에 뜬다. 역사 교사인 동훈에게는 두 축제 모두 수업과 연계할 수 있는 교육 콘텐츠라 눈이 번쩍 뜨인다. 특히 "아이 연령 맞춤 체험 프로그램" 정보와 혼잡도 예측이 함께 제공되어, 처음으로 광고 없는 신뢰할 수 있는 가족 축제 정보를 얻었다는 만족감을 느낀다.

---

## 사용자 여정

### 1단계: 진입
- 네이버에서 "여름방학 가족 축제 추천" 검색 → KoreaFestival Live 기사 링크 발견 → 앱 설치
- "가족" + "체험" + "여름" 필터 동시 적용 → 맞춤 축제 목록 10개 내외로 정제
- 안동국제탈춤페스티벌 클릭, 상세 페이지에서 "초등생 체험 프로그램" 별도 섹션 확인

### 2단계: 행동
- 8세·11세 자녀 체험 가능 프로그램 목록 확인 (탈 만들기, 탈춤 배워보기 등)
- 방문자 집중률 예측 그래프에서 토요일 오후 최고 혼잡 → 금요일 오전 출발로 조정
- "2박 3일 가족 코스" 생성 → 탈춤 체험 → 하회마을 → 로컬 안동 찜닭 식당 → 병산서원 순서로 자동 배치
- 지역 다양성 지수 ★★★★ 확인, 축제 외 역사 유적지까지 연계돼 교육 여행으로 완결성 높음
- 노부모 동반 여부 확인란에 체크 → 무장애 접근 가능 경로 유무 표시 (안동 하회마을 일부 비포장 경고)

### 3단계: 결과
- 코스 생성 후 가족 단톡에 공유, 배우자와 최종 일정 확정
- 현장에서 앱 추천 체험 프로그램 사전 접수로 대기 없이 진행
- 동훈이 2학기 역사 수업 자료로 현장 사진 활용 → 앱 사용 경험을 동료 교사에게 추천

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **8** | 체험 필터 + 아이 연령 맞춤 프로그램 정보가 핵심 페인포인트 해소. 역사 교사로서 교육적 축제 큐레이션은 차별화 포인트 |
| 사용 편의 | **7** | 중간 기술 친숙도를 가진 동훈에게 필터 조합이 다소 복잡할 수 있음. 그러나 신뢰할 수 있는 정보 품질이 사용 지속성 보장 |
| 구현 타당성 | **7** | 아이 연령별 체험 프로그램 데이터 수집이 핵심 — 지자체 협력 없이는 데이터 완전성 확보 어려움 |

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live는 최동훈의 "광고성 정보 불신" + "아이 맞춤 체험 필터 없음" 페인포인트를 공식 API 데이터 기반 축제 정보와 체험 프로그램 필터로 해소한다. 노부모 무장애 접근 경고 기능도 현실적 니즈를 반영한다. 다만 앱의 UI 복잡도를 중간 기술 친숙도에 맞게 단순화해야 지속 사용으로 이어진다.

## 개선 제안

1. "가족 맞춤 설정" 단계를 온보딩에 포함: 자녀 나이 입력 시 전체 앱에서 해당 연령 체험 프로그램을 자동 필터링하여 매번 설정 없이 맞춤 정보 제공
2. 축제 현장 무장애 접근성 정보를 단순 경고가 아닌 "대안 경로 제안" 형태로 고도화 — 노부모 동반 가족 여행자의 불안감을 해소하고 이 앱을 "신뢰의 앱"으로 포지셔닝
