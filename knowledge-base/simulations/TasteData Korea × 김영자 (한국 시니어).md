# SIM-196: TasteData Korea × 김영자 (한국 시니어)

## 시나리오

경주 여행을 앞두고 딸이 영자의 스마트폰에 TasteData Korea를 설치해준다. "엄마, 이 앱에서 경주 제대로 된 밥집 찾아봐요. 광고 없는 데이터 맛집이래요"라고 설명해준다. 영자는 여행에서 좋은 음식이 빠지면 안 된다는 생각이 강하고, 특히 "그 지역 토박이들이 가는 식당"에 대한 로망이 있다. 딸이 앱을 조작해주면서 경주 현지인 소비 밀집 지역과 대표 음식 정보를 함께 보여준다. KoreaDemand Atlas보다 직관적인 음식 중심 콘텐츠가 영자에게 더 친숙하게 다가온다.

---

## 사용자 여정

### 1단계: 진입
- 딸이 대리 설치 및 설명 — "맛집 찾는 앱"이라는 단순 설명으로 영자가 목적 즉시 이해
- 음식 중심 UI가 수요 지수 중심 KoreaDemand Atlas보다 개념적으로 접근하기 쉬움
- 딸이 경주 검색 → 지역 대표 음식 목록(황남빵·쌈밥·경주빵) 화면을 함께 봄

### 2단계: 행동
- 딸이 경주 현지인 소비 지역 탭: 황리단길 밀집 확인 → 영자 "거기 유명한 데 아니야?"
- 지역 특산 음식 사진 갤러리: 황남빵 사진 보고 영자 "이거 먹어봐야지" → 방문 동기 자연 형성
- 딸이 음식 코스 생성: 점심 쌈밥 → 황남빵 → 저녁 한정식 코스 저장
- 영자 본인이 사진 갤러리 손가락으로 스크롤 → 음식 사진 탐색은 직접 수행 가능한 수준
- 글씨 크기 작아 텍스트 정보는 읽기 어려움, 사진 위주로 탐색

### 3단계: 결과
- 황리단길 방문 → "역시 현지 사람들이 가는 곳은 달라" 만족
- 황남빵 구매 → 딸에게도, 사위에게도 선물용으로 추가 구매
- 영자의 앱 직접 사용은 제한적이지만, 딸과 함께 화면 보며 여행 계획 세우는 과정 자체를 즐김

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **6** | 음식 중심 콘텐츠가 시니어에게 개념적으로 접근 가능. 딸 매개 경험이지만 실제 맛집 발굴 가치는 전달됨 |
| 사용 편의 | **4** | 사진 갤러리는 직접 탐색 가능하나 텍스트·코스 생성 기능은 딸 없이 불가. KoreaDemand Atlas보다는 진입 장벽 낮음 |
| 구현 타당성 | **8** | 음식 사진 중심 UI는 시니어 친화적으로 개선 여지 있으며 현재 구현도 기술적으로 타당 |

**종합 점수: 18/30**

---

## Pass/Fail 판정

> **✅ PASS**

종합 점수 18점으로 최소 기준을 충족한다. TasteData Korea는 KoreaDemand Atlas보다 음식이라는 구체적이고 친숙한 주제를 다루어 시니어 사용자에게 더 접근하기 쉽다. 딸 매개 사용이지만 영자가 사진 갤러리를 직접 탐색하는 참여가 가능했다는 점이 차이를 만든다. 글씨 크기와 UI 단순화가 개선되면 시니어 직접 사용 가능성이 높아진다.

---

## 개선 제안

1. 시니어 모드(큰 글씨 + 사진 위주 카드 UI) 옵션 추가 — 텍스트 최소화, 음식 사진 크게, 핵심 정보(위치·영업시간)만 굵게 표시
2. "가족과 함께 보기" 공유 기능 — 자녀가 부모 폰에 코스를 원격 저장해줄 수 있는 기능으로 시니어의 딸 의존 구조를 앱 내 기능으로 흡수
