# SIM-09: K-Guide Global × Emma Chen (외국인 20대 백패커)

## 시나리오

Emma Chen(23세, 대만 여행 크리에이터)은 전주 한옥마을 방문 전날 숙소에서 K-Guide Global을 설치한다. 대만 여행 커뮤니티(Dcard)에서 "한국 여행 앱 중 중국어 번체 지원 최고"라는 후기를 보고 다운로드했다.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 언어 선택 화면: **"繁體中文(번체 중국어)"** 즉시 선택
- 전체 UI가 번체 중국어로 전환 → 구글 번역기 없이 바로 탐색 시작
- 소요 시간: 약 20초

### 2단계: 행동
- "全州(전주)" 검색 → 한옥마을 상세 정보 확인
- `visitor-concentration-forecast` 데이터: 다음날 오전 9시 혼잡도 22%(한산) 확인 → 방문 계획 확정
- 포토스팟 루트 탐색: 한옥마을 내 인기 촬영 포인트 목록 + 각 장소 고화질 공식 사진 확인 (`photo-contest-winners` 연동)
- 오디오 가이드 실행: 한옥마을 역사·문화 배경을 **번체 중국어로 청취** → 팔로워에게 설명할 내용 파악
- 관련 장소 추천: `related-attractions`로 전주 전통주 박물관·비빔밥 체험 발견 → 당일 코스 추가

### 3단계: 결과
- 오전 9시 한옥마을 도착 → 인파 없이 사진 촬영 성공
- 오디오 가이드 내용을 바탕으로 한 설명 자막을 릴스에 추가 → 팔로워 반응 폭발
- Dcard에 "K-Guide Global 번체 지원 완벽, 무조건 추천" 후기 게시
- 인스타 팔로워 2천 명 추가 유입

## 개선 제안

1. **SNS 콘텐츠 생성 도구 추가**: 포토스팟 + 오디오 가이드 핵심 문구를 자동으로 번체 중국어 캡션으로 생성 → 크리에이터 워크플로우 완성
2. **크리에이터 공유 인센티브**: Emma 같은 인플루언서가 앱 링크를 SNS에 공유할 경우 배지/할인 혜택 → 커뮤니티 바이럴 가속
3. **오프라인 캐시**: 로밍 요금 절감을 위한 지역별 오디오 가이드 사전 다운로드 기능
