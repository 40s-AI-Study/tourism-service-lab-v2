# SIM-104: EuroKorea Gateway × Emma Chen (외국인 20대 백패커)

## 시나리오

Emma는 대만 출신으로 번체 중국어와 영어를 사용한다. 전주 한옥마을 방문 전날 여행 정보 앱을 검색하다 EuroKorea Gateway를 발견한다. 앱 설명에 "multilingual Korea tourism"이라는 문구가 있어 기대하고 설치하지만, 막상 언어 선택 화면에는 러시아어·스페인어·프랑스어·독일어만 있고 중국어(번체/간체)와 영어가 없다. Emma가 원하는 번체 중국어 콘텐츠, 혼잡도 정보, 포토스팟 큐레이션이 전혀 없다. 유럽어 4개에 특화된 앱이라는 것을 깨닫고 이탈한다.

---

## 사용자 여정

### 1단계: 진입
- App Store 검색 "Korea travel multilingual" 결과에서 발견
- "다국어 지원"이라는 설명에 번체 중국어 지원을 기대하고 설치
- 언어 선택 화면: 러·스·프·독 — 중국어 없음 확인

### 2단계: 행동
- 영어 UI 없어 스페인어 또는 프랑스어로 탐색 시도 (Emma는 영어가 유창하나 이들 언어는 모름)
- 비주얼 갤러리의 사진 품질은 좋음 — 전주 한옥마을 사진 다수 확인
- 사진 탭에서 흡인력 느끼지만 중국어 캡션·설명 없어 정보 접근 불가
- 혼잡도 예측 기능 없음 — 사진 촬영 최적 시간 파악 불가

### 3단계: 결과
- 사진 10장 정도 둘러보다 이탈. PTT 커뮤니티로 정보 수집 복귀
- "예쁜 사진은 있는데 내가 쓸 수 있는 정보가 없다" 평가

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 비주얼 갤러리만 소소하게 유용. 번체 중국어 미지원으로 정보 접근 불가 |
| 사용 편의 | **3** | 사진 중심 UI는 직관적이나 언어 장벽으로 실질 탐색 불가 |
| 구현 타당성 | **7** | 유럽어 타깃 서비스로는 잘 구현되나 이 페르소나에는 부적합 |

**종합 점수: 12/30**

## Pass/Fail 판정

> **❌ FAIL**

Emma Chen은 번체 중국어 사용자로 EuroKorea Gateway의 4개 유럽어 지원 범위에 포함되지 않는다. 앱의 핵심 가치인 모국어 관광 정보 제공이 Emma에게는 전혀 작동하지 않는다. 혼잡도 정보, 포토스팟 큐레이션 등 Emma의 핵심 니즈도 미충족. 종합 12점, FAIL.

## 개선 제안

1. 앱스토어 설명에 지원 언어(러·스·프·독)를 명시해 중국어·일본어 사용자의 불필요한 설치를 방지할 것
2. 장기적으로 서비스 확장 시 번체/간체 중국어를 추가하면 대만·홍콩 여행 크리에이터 시장 진입 가능
