# SIM-96: K-Camp Finder × 김영자 (한국 시니어)

## 시나리오

김영자(67세, 은퇴 교사)는 사찰 탐방과 고궁 관람을 즐기는 문화 여행자다. 등산·트레킹은 가끔 즐기지만 야외 취침에는 관심이 없다. 여행 시 편안한 침대와 화장실이 필수이며, 스마트폰은 카카오톡과 유튜브 정도만 사용한다. 손자의 권유로 가끔 새 앱을 시도하지만 복잡한 UI에서 금방 포기한다. 캠핑·글램핑 니즈는 없다.

---

## 사용자 여정

### 1단계: 진입

- 손자가 "할머니, 이 앱으로 예쁜 데 찾아봐요" 하며 설치해줌
- 앱 실행 후 캠핑·텐트 이미지 위주 UI 확인
- "텐트에서 자는 건 싫은데…" 반응

### 2단계: 행동

- 글램핑 섹션 진입: 화장실 포함, 침대 제공 여부 확인
- 가격 확인 후 "여관이랑 비슷한데 왜 밖에서 자야 해?" 의문
- 사찰·고궁 연계 코스 검색 시도: 관련 필터 없음
- 앱 내 폰트 크기·버튼 크기 접근성 이슈 발생 가능
- 별도 앱(네이버 지도, 문화포털) 이용으로 전환

### 3단계: 결과

- K-Camp Finder 미사용 결정
- 기존 여행 패턴(한국관광공사 앱 + 네이버 지도) 유지

---

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 캠핑·글램핑 니즈 자체가 없고, 사찰·고궁 연계 기능 부재로 핵심 여행 목적과 무관 |
| 사용 편의 | **3** | 시니어 접근성(큰 폰트, 단순 UI) 미최적화, 캠핑 중심 UX가 생소 |
| 구현 타당성 | **4** | 시니어 특화 모드 기술적으로 가능하나 서비스 방향성과 불일치 |

**종합 점수: 9/30**

---

## Pass/Fail 판정

> **❌ FAIL**

시니어 문화 여행자는 K-Camp Finder의 타겟 범위 밖이다. 캠핑 숙박 자체에 대한 거부감이 있고, 앱 UI 복잡도와 접근성 문제가 추가 장벽으로 작용한다. 유용성 2점, 종합 9점으로 Pass 기준 크게 미달.

## 개선 제안

1. 시니어 타겟 제외 전략 유지: 핵심 타겟 외 확장보다 20-40대 집중이 서비스 완성도 측면에서 유리
2. 장기적 시니어 글램핑 진입: "텐트 없는 글램핑 = 프리미엄 자연 숙박"으로 재프레이밍한 시니어 특화 콘텐츠로 미래 시장 선점 가능성 탐색
