# SIM-32: PetKorea × 박민준 & 이수연 (한국 30대 커플)

## 시나리오

2026년 5월 첫째 주 금요일 저녁, 박민준(33세, IT기획)과 이수연(31세, 마케터)은 성수동 집에서 다음 주말 1박2일 여행을 계획 중이다. 두 달 전부터 함께 키우기 시작한 말티즈 '버터'가 첫 여행 데뷔를 할 예정이라 숙박부터 식당까지 소형견 동반 가능 여부가 최우선이다. 수연이 PetKorea 앱을 켜고 민준은 옆에서 함께 검색한다.

목적지는 강원도 글램핑으로 이미 방향은 잡혀 있다. 하지만 글램핑장 반려동물 정책, 근처 반려견 동반 레스토랑, 다음 날 산책 코스까지 한 번에 확인하고 싶다. 수연은 프리미엄 숙박에 거부감이 없고(1박 15만원대 예산), 민준은 식당의 위생·분위기를 중시한다. 두 사람 모두 인스타그램 업로드를 염두에 두고 있어 감성 사진이 나오는 공간을 선호한다.

AI 여행 코스 생성 기능에 "소형견/차량/1박2일/강원도/글램핑 선호" 조건을 입력하자 3가지 코스 옵션이 제시된다. 두 사람은 두 번째 코스(홍천 글램핑→소양강 산책로→반려견 동반 브런치 카페)를 선택하고 숙박·식당 예약 링크까지 확인한다.

---

## 사용자 여정

### 1단계: 진입
- 이수연이 기존 설치된 앱에서 버터 프로필(말티즈, 4kg, 예방접종 완료) 등록
- 메인 화면 "AI 코스 만들기" 버튼 탭
- 조건 입력: 반려동물 종류(소형견), 이동수단(자가용), 여행 일수(1박2일), 지역(강원도), 선호 스타일(글램핑·미식)

### 2단계: 행동
- `pet-friendly-travel` API (카테고리: 숙박) → 소형견 허용 글램핑장 필터링, 버터 체중 4kg 기준 규정 자동 매칭
- AI 코스 생성 엔진이 3개 코스 제안: 각 코스마다 숙박·음식점·관광지 묶음으로 구성
- `visitor-concentration-forecast` API → 주말 홍천 글램핑 단지 혼잡도 예측 ("보통" 표시, 예약 권장 알림)
- `related-attractions` API → 글램핑장 반경 10km 반려견 산책로·포토존 3곳 추천
- `tourism-info-korean` API → 소양강 스카이워크 반려견 동반 가능 여부 확인 (리드줄 필수, 케이지 불필요)
- 코스 2 선택 후 글램핑장 외부 예약 링크 및 브런치 카페 전화번호 저장
- 혼잡도 알림: 출발 당일 오전 8시 "현재 혼잡도 여유 — 출발 추천" 푸시 알림 예약

### 3단계: 결과
- 1박2일 여행 완료. 글램핑장에서 버터 첫 캠핑 경험, 인스타 릴스 업로드 후 2,400 좋아요
- 소양강 산책로에서 반려견 동반 가능 여부를 앱에서 사전 확인해 현장 혼선 없음
- 불만족: 글램핑장 예약이 앱 내 인앱 결제가 아닌 외부 링크로 연결되어 UX 단절 발생
- 만족: AI 코스가 소형견 체중 기준 정책을 자동 매칭해 준 것이 가장 유용했다는 후기 작성

## 개선 제안

1. **인앱 예약 통합**: 글램핑·숙박 예약을 앱 내에서 완결할 수 있도록 제휴 파트너 결제 연동 구현 (UX 단절 해소)
2. **반려동물 규정 자동 적합도 배지**: 숙박·음식점 결과 카드에 "버터 입장 가능 ✓" 식의 개인화 배지를 즉시 표시해 신뢰도 강화
3. **커플 공유 플래닝**: 두 사용자가 같은 코스 플랜을 실시간으로 함께 편집·투표할 수 있는 공유 기능 추가
