# SIM-07: K-Guide Global × 박민준 & 이수연 (한국 30대 커플)

## 시나리오

박민준(33세)이 1주년 여행 준비 중 K-Guide Global을 검색으로 발견한다. "공식 데이터 기반"이라는 설명에 신뢰감을 느끼고 설치. 이수연은 "프리미엄 숙소·미식" 위주 코스를 기대하며 함께 탐색한다.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 한국어 선택
- "강원도 1박 2일 커플 여행" 검색

### 2단계: 행동
- 공식 Visit Korea 관광지 목록은 충실하게 제공
- 혼잡도 데이터(🟢🟡🔴) 확인 가능
- 그러나 앱의 커플/로맨틱 테마 필터 부재 → 풀빌라·오마카세 추천 없음
- 웰니스(스파/온천) 필터 없음
- 외국인 관광객 대상 "베스트 스팟" 위주라 감성 숙소 큐레이션 전무
- 수연: "트리플이나 야놀자보다 정보가 적네" → 실망

### 3단계: 결과
- 혼잡도 타이밍 정보는 참고하되, 숙소·맛집 예약은 다른 앱에서 해결
- 앱을 보조 도구로만 활용
- 메인 여행 기획 도구로는 채택 실패

---

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **3** | 커플 여행 핵심 니즈(프리미엄 숙소, 미식, 로맨틱 테마)가 전무 |
| 사용 편의 | **5** | 한국어 UI는 사용 가능하나 콘텐츠 구성이 한국인 30대 커플과 무관 |
| 구현 타당성 | **9** | 앱 자체 기능 문제 아님, 순수 타겟 미스매치 |

**종합 점수: 17/30**

---

## Pass/Fail 판정

> **❌ FAIL**

P2 커플의 핵심 니즈(프리미엄 숙소 큐레이션, 미식 코스, 예약 연동)는 K-Guide Global의 설계 범위 밖이다. 서비스가 외국인 FIT 대상으로 특화되어 있어 한국인 커플 여행자에게는 적합하지 않다.

---

## 개선 제안

1. **타겟 집중**: 외국인 사용자에게만 서비스 집중 — 한국인 사용자 이탈은 타겟 설계의 성공 지표로 해석
2. **혼잡도 API 공개**: 혼잡도 데이터만 별도 API/위젯으로 외부 앱(트리플, 야놀자)에 제공하면 B2B 수익화 가능
3. **현재 서비스 범위 유지**: 커플/로맨틱 기능 추가는 서비스 정체성 희석 → 외국인 특화 전략 유지 권장
