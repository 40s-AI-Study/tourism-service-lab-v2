# SIM-141: TourismCareer Korea × 김지원 (한국 20대 솔로)

## 시나리오

지원은 프리랜서 그래픽 디자이너지만 수입이 불안정해 관광 스타트업 인하우스 디자이너로의 전직을 조용히 고민 중이다. 어느 날 인스타그램 릴스에서 "관광 산업 취업 앱" 광고를 보고 흥미가 생겼다. 평소 여행을 자주 다니는 만큼 관광 업계가 낯설지 않고, "좋아하는 일을 직업으로" 라는 막연한 바람도 있었다. 앱을 설치해 '관광 스타트업 × 디자인' 포지션을 검색하다가, 채용 기업이 위치한 강릉 지역의 관광 수요 데이터까지 한눈에 보이는 기능에 예상치 못한 흥미를 느꼈다.

---

## 사용자 여정

### 1단계: 진입
- 인스타그램 광고 → 앱스토어 4.2점 리뷰 확인 후 설치
- 온보딩에서 "관심 직무: 디자인/마케팅", "관심 지역: 수도권 + 강원" 설정
- 메인 화면에서 '관광 스타트업 UI/UX 디자이너' 공고 3건 즉시 노출 — 첫인상 긍정적

### 2단계: 행동
- 강릉 소재 여행 콘텐츠 스타트업 채용공고 클릭 → 기업 페이지 진입
- `area-tourism-demand-density` 데이터: "강릉시 관광 수요 상위 8%, 체류형 관광 성장세 2년 연속 1위" — "이 지역이 이렇게 뜨고 있었구나" 반응
- 현장 탐방 코스 생성 버튼 클릭 → 강릉 중앙시장·안목해변 카페거리·정동진 코스 자동 생성
- 탐방 코스가 평소 즐기던 당일치기 여행 스타일과 일치 — 별도 동기 없이 다음 주말 방문 예약 확정

### 3단계: 결과
- 강릉 현장 탐방 실행, 앱 내 '탐방 일지' 기능으로 방문 기록 저장
- 탐방 후 채용 지원서 초안 작성 시작 (포트폴리오에 현장 사진 활용 계획)
- 즉각적인 취업 전환 결정은 아니지만, 앱을 북마크하고 정기 사용 의향 생김

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **7** | 관광 취업 정보 + 지역 데이터 결합은 신선하나, 디자이너 직군 공고 수가 제한적 |
| 사용 편의 | **8** | 인스타 유입 → 직관적 온보딩 → 공고 탐색까지 흐름 자연스러움 |
| 구현 타당성 | **8** | tourism-jobs API 기반 공고 노출은 기술적으로 명확, 현장 탐방 연계도 realistic |

## Pass/Fail 판정

> **✅ PASS**

지원은 핵심 타깃(20대, 관광 업계 관심)과 완전히 겹치지는 않지만 전직 고민 맥락에서 앱의 핵심 가치를 체험했다. 관광 빅데이터가 단순 구직 앱 이상의 흥미를 유발했고, 현장 탐방 기능이 그녀의 여행 습관과 자연스럽게 연결되어 실사용으로 이어졌다. 디자인 직군 공고 수 확대가 필요하지만 기본 사용자 여정은 유효하다.

## 개선 제안

1. 관광 분야 크로스-직군 공고 확대: 디자인·개발·마케팅 등 비전통 직무를 관광 스타트업과 연결하는 별도 카테고리 신설
2. 탐방 일지 → 지원서 연동: 현장 탐방 기록이 자동으로 지원 포트폴리오 소재로 전환되는 기능 추가
