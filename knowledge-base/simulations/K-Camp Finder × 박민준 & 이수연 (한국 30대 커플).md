# SIM-92: K-Camp Finder × 박민준 & 이수연 (한국 30대 커플)

## 시나리오

박민준(33세, 마케터)과 이수연(31세, 간호사)은 기념일 여행을 계획 중이다. 평소 펜션이나 호텔을 이용했지만 이번에는 색다른 경험을 원해 "뷰 맛집" 글램핑을 검토 중이다. 예산은 1박 15-20만원대, 일몰·별빛·오션뷰 등 인증샷 가능한 프리미엄 감성을 원한다. 이수연은 취사보다 다양한 먹거리 체험을 선호한다.

K-Camp Finder에서 "커플 / 글램핑 / 2인 / 프리미엄" 조건으로 검색, 뷰와 혼잡도 기준 TOP 5 결과 중 강릉 해변 글램핑을 선택하고 주변 카페거리·맛집 연계 코스를 확인한다.

---

## 사용자 여정

### 1단계: 진입

- 이수연이 유튜브 커플 브이로그에서 K-Camp Finder 언급 발견
- 박민준이 앱 설치, 카카오 로그인
- 온보딩: 동반 유형 "커플", 숙박 선호 "글램핑/프리미엄" 설정
- 메인 화면 "뷰 맛집 특집" 배너 탭

### 2단계: 행동

- 조건 입력: 인원 2명, 날짜 3주 후 금-토, 지역 강원도, 예산 15-20만원
- AI 추천: 해변 뷰 글램핑 TOP 5 (혼잡도 낮음 우선 정렬)
- "주문진 오션돔 글램핑" 선택: 오션뷰 360도 돔 텐트, 혼잡도 현재 낮음 표시
- 연계 코스 확인: 강릉 안목 카페거리(차로 15분), 경포대 일출 스팟, 주문진 수산시장 맛집
- 커플 패키지 확인: 와인+꽃 장식 옵션 추가
- 날씨 알림 설정 후 gocamping API 연동 예약 완료

### 3단계: 결과

- 예약 확정 메시지 + 연계 코스 일정표 PDF 저장
- D-3 알림: "금요일 맑음 22°C, 일몰 시간 19:47, 별 관측 가능성 높음"
- 방문 후 커플 리뷰 작성, 포토스팟 핀 공유

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 뷰 기준 필터, 혼잡도 확인, 커플 특화 연계 코스가 기념일 여행 니즈를 정확히 해결 |
| 사용 편의 | **8** | 조건 입력 직관적, 연계 코스 자동 생성이 플래닝 부담 제거 |
| 구현 타당성 | **8** | gocamping + visitor-concentration-forecast + related-attractions 조합으로 대부분 구현 가능, 커플 패키지 옵션은 캠핑장 개별 연동 필요 |

**종합 점수: 25/30**

---

## Pass/Fail 판정

> **✅ PASS**

커플 글램핑 여행의 핵심 니즈(뷰, 혼잡도, 낭만 연계 코스)를 통합 해결한다. 유용성 9점, 종합 25점으로 Pass 기준을 상회한다. 프리미엄 숙박 대안으로서의 포지셔닝이 명확하며, 기념일 특화 옵션 강화 시 전환율 제고 가능.

## 개선 제안

1. 기념일/이벤트 특화 필터 추가: 생일·기념일 날짜 입력 시 꽃장식·케이크·와인 패키지 제공 캠핑장 우선 노출
2. 커플 인증샷 코스 큐레이션: AI가 일출·일몰 시간대와 포토스팟 위치를 결합한 "황금 타임 인증샷 루트" 자동 생성
