# SIM-124: K-BeautyMed Tour × Emma Chen (외국인 20대 백패커)

## 시나리오

에마는 대만 K-뷰티 커뮤니티 Dcard에서 "서울 올리브영 투어 앱 추천" 스레드를 보다가 K-BeautyMed Tour를 발견했다. 번체 중국어를 지원한다는 점이 즉각적인 매력 포인트였다. 그녀는 이번 서울 방문에서 명동과 압구정 올리브영 투어를 계획 중이고, 여행 크리에이터로서 강남 뷰티 포토스팟도 찍고 싶다. 피부 레이저 시술은 예산 초과지만, 피부과 상담 정도는 한 번 받아볼 의향이 있다. 앱의 번체 중국어 UI, 포토스팟 큐레이션, 혼잡도 기능이 자신의 여행 스타일과 잘 맞았다.

---

## 사용자 여정

### 1단계: 진입
- Dcard 커뮤니티 추천 게시글을 통해 앱 발견 → 번체 중국어 지원 확인 후 즉시 설치
- 첫 화면: 언어 자동 감지로 번체 중국어 설정 → 자연스러운 첫 경험
- "올리브영 투어 + 포토스팟 + 피부과 상담"이라는 조합이 자신의 여행 계획과 정확히 일치

### 2단계: 행동
- `tourism-photos` API 기반 강남 뷰티 포토스팟 목록 탐색 → 압구정 명품 거리, 성수동 팝업스토어 등 인스타용 장소 발견
- `visitor-concentration-forecast` API 활용: 토요일 오후 2시 명동 올리브영 혼잡 예측 확인 → 오전 10시로 방문 시간 조정
- 피부과 큐레이션 탐색 → 예산 내 피부 상담 패키지 발견, 번체 중국어 안내문 확인
- `area-tourism-demand-density` API: 강남·홍대·성수 뷰티 핫플 소비 강도 지도 → 이동 동선 최적화에 활용
- 시술 후 D+0~1 실내 쇼핑 코스 기능은 현재 시술 계획이 없어 건너뜀

### 3단계: 결과
- 포토스팟 코스와 혼잡도 기능을 실제로 활용하여 여행 만족도 상승. 인스타그램에 "K-BeautyMed Tour 추천 루트로 다녀왔어요" 게시 → 팔로워 반응 호조
- 피부과 상담은 예약까지는 이르지 못했으나 다음 방문 시 활용 의향 있음

| 구현 타당성 | **8** | 대만 방한 관광객에게 최적화된 언어 지원과 뷰티 여정 큐레이션은 실현 가능하고 설득력 있음 |

## Pass/Fail 판정

> **✅ PASS**

에마는 K-BeautyMed Tour의 핵심 타겟 페르소나 중 하나다. 번체 중국어 UI, 뷰티 포토스팟 큐레이션, 혼잡도 기반 방문 타이밍 최적화가 여행 크리에이터의 실용적 니즈를 충족한다. 시술을 받지 않더라도 K-뷰티 쇼핑+사진 여정만으로도 충분한 앱 가치를 발견했으며, 향후 재방문 시 시술 기능 전환 가능성도 높다.

## 개선 제안

1. 시술 계획이 없는 "K-뷰티 쇼핑 전용" 사용자를 위한 빠른 진입 경로(온보딩 분기) 추가 — "시술 예약" vs "뷰티 쇼핑 투어만" 선택 화면
2. 여행 크리에이터를 위한 SNS 공유 기능 강화 — 앱 내 추천 코스를 인스타 스토리 형식으로 바로 내보내기 가능하도록 개선
