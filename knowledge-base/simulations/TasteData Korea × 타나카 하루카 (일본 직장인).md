# SIM-200: TasteData Korea × 타나카 하루카 (일본 직장인)

## 시나리오

하루카는 첫 한국 여행에서 한식 체험을 가장 기대하는 목적 중 하나로 꼽고 있다. 일본에서 K-drama를 보며 삼겹살·치맥·떡볶이를 동경해왔고, "진짜 현지 맛집"에 대한 기대가 크다. 출발 전날 일본어 한국 여행 유튜브 채널에서 TasteData Korea 소개 영상을 보고 설치한다. 앱을 열면 일본어 UI가 있는지 확인하는데, 있다는 것을 확인하고 안도한다. 서울 홍대·신촌 지역을 검색하면 현지인 소비 강도 히트맵과 함께 일본어로 음식 정보가 표시된다.

---

## 사용자 여정

### 1단계: 진입
- 일본어 유튜브 채널 추천 → 설치 → 일본어 UI 전환 성공 → 즉각적 안도감
- 서울 검색 → 현지인 소비 강도 히트맵 + 일본어 지역 음식 정보 화면 탐색
- "현지인이 가는 곳"이라는 개념이 일본어로 직관적으로 전달됨 ("地元の人が実際に行くお店")

### 2단계: 행동
- 홍대 현지인 소비 밀집 지역 탭: 치킨·삼겹살·떡볶이 소비 강도 높음 — 하루카가 원하던 음식들
- 일본어 음식 설명: 삼겹살 조리법·먹는 방법·주문 팁 제공 → "現地での注文方法" 섹션 유용
- 음식 코스 생성: 저녁 삼겹살 → 야식 치맥 → 다음날 아침 순대국밥 코스 저장
- 음식 포토 큐레이션: 삼겹살 인증샷 구도 포토스팟 연동 → "X에 올릴 사진 찍을 곳도 알려줘!" 반응
- 한국어 메뉴판 대응 팁(일본어 사전 포함) → 식당 주문 불안감 해소

### 3단계: 결과
- 홍대 삼겹살집 방문 → 일본어 주문 팁 덕분에 점원에게 한국어로 기본 주문 성공
- 치맥 골목에서 일본어 앱 보며 메뉴 결정 → 친구들과 함께 만족스러운 한식 체험
- X(트위터)에 "TasteData Korea로 찾은 홍대 현지 삼겹살 골목" 게시 → 리트윗 800건
- 하루카 최종 평가: "한국 여행에서 밥 걱정 없애준 앱. 일본어 있어서 진짜 좋았음"

---

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 한식 체험이 방한 목적 1순위인 일본 여행자에게 현지인 소비 데이터 기반 맛집 + 일본어 주문 팁이 완벽하게 맞아떨어짐 |
| 사용 편의 | **9** | 일본어 UI 지원으로 언어 장벽 완전 해소. 음식 포토 큐레이션이 SNS 공유 문화와 자연스럽게 연결. 즉흥형 여행자에게 맞는 직관적 탐색 |
| 구현 타당성 | **8** | `tourism-info-japanese` API + 소비 데이터 + 포토 API 조합 실현 가능. 일본 여행자 대상 PMF 가장 높은 시뮬레이션 |

**종합 점수: 26/30**

## Pass/Fail 판정

> **✅ PASS**

TasteData Korea는 타나카 하루카에게 20개 시뮬레이션 중 공동 최고 점수(David Park과 동점 26점)를 받는다. 방한 외국인 1위 국적인 일본 여행자의 한식 체험 니즈, 언어 장벽 해소 니즈, SNS 공유 니즈를 동시에 충족하는 완성도 높은 사용 경험이 확인됐다. 일본어 특화가 이 앱의 가장 강력한 차별점 중 하나임을 이 시뮬레이션이 증명한다.

## 개선 제안

1. 일본어 특화 "한식 입문 가이드" 섹션 추가 — 삼겹살 굽는 법, 쌈 싸는 법, 소주 에티켓 등 일본인이 궁금해하는 한식 문화 팁을 일본어로 제공해 첫 방문 여행자 불안감 완전 해소
2. X(트위터) 직접 공유 최적화 — 일본 여행자의 주 SNS인 X에 최적화된 공유 포맷(16:9 음식 사진 + 위치 + 일본어 캡션 자동 생성)으로 바이럴 루프 강화
