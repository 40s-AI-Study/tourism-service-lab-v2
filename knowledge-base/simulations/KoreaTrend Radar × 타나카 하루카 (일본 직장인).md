# SIM-48: KoreaTrend Radar × 타나카 하루카 (일본 직장인)

## 시나리오

2026년 4월 17일 금요일 오후 7시, 도쿄. 마케터 타나카 하루카(27)는 6월 한국 여행을 준비 중이다. K-드라마 팬인 그녀의 최우선 목표는 좋아하는 드라마 촬영지를 방문하는 것이지만, 동시에 "지금 한국에서 진짜 뜨는 곳"도 가보고 싶다. X(트위터)에서 KoreaTrend Radar를 발견하고 일본에서 미리 설치해 탐색 중이다.

앱을 실행하면 언어 설정에서 일본어 옵션이 없다. 영어로 전환해 사용한다 — 마케터로서 영어는 문제없다. 홈 레이더에서 서울 전역 히트맵을 보며 성수동과 을지로가 꾸준히 상위권임을 확인한다. "K-drama filming location" 관련 카테고리나 태그를 찾지만 존재하지 않는다. 트렌드 데이터로 "지금 뜨는 장소"는 찾을 수 있지만, 그것이 드라마 성지인지 여부는 앱이 알려주지 않는다.

그래도 성수동 트렌드 급등 데이터(이동통신 +53%, 카드 +47%)를 참고해 성수동을 여행 일정에 포함시키기로 결정한다. 드라마 성지 정보는 별도 한류 정보 사이트에서 보완한다.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 언어 설정 확인 (일본어 없음 → 영어 선택)
- 홈 레이더에서 서울 히트맵 탐색
- K-drama 성지 관련 카테고리·태그 탐색 → 부재 확인

### 2단계: 행동
- 성수동·을지로·홍대 트렌드 지표 비교
- 성수동 상세 화면: 이동통신 +53%, 카드 +47%, 주간 트렌드 지속 상승 확인
- 관광 다양성 지수(카페/쇼핑/문화) 확인
- 트렌드 데이터와 한류 정보 사이트 병행 탐색으로 성수동 드라마 촬영지 여부 확인
- 여행 일정에 성수동 1일 배정

### 3단계: 결과
- 성수동, 을지로, 북촌 3개 지역 여행 일정 초안 완성
- KoreaTrend Radar는 "어디가 지금 뜨는지" 참고용으로 활용
- 드라마 성지 정보는 별도 앱/사이트에서 보완하는 이중 구조

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **6** | 트렌드 지역 파악에 유용하나, 핵심 니즈인 K-drama 성지 정보 연계 없어 단독 활용 한계. 여행 계획 보조 도구로는 기능함 |
| 사용 편의 | **6** | 영어 UI로 대체 가능하나 일본어 미지원이 아쉬움. 영문 UI 완성도는 수용 가능 수준. K-콘텐츠 관련 카테고리 부재가 이 페르소나의 주요 마찰점 |

## 개선 제안

1. **일본어 UI 추가**: 일본은 한국 관광 최대 인바운드 시장 중 하나. 일본어 현지화를 우선순위로 적용하고, 주요 지표 설명과 장소명 번역 완성도 확보
2. **K-콘텐츠 태그 연동**: 드라마·영화·아이돌 관련 성지 태그를 트렌드 데이터와 연계해 "지금 뜨는 K-drama 성지 TOP 5" 같은 한류 특화 뷰 제공
