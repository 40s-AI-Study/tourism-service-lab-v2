# SIM-R3-017: K-Camp Finder v2 × 아미나 사이드

## 시나리오

아미나 사이드(30세)는 쿠알라룸푸르 기반 라이프스타일 인플루언서다. 인스타그램 팔로워 45만 명, 유튜브 구독자 12만 명. 콘텐츠 테마는 "무슬림 친화적 K-문화 여행"이다. 할랄 식당, 기도 시설, 무슬림 여행자 팁을 영어와 말레이어로 공유한다. 그녀의 팔로워 대부분은 한국에 관심 있는 동남아 무슬림 여성들이다.

이번 한국 방문은 콘텐츠 제작 목적의 10일 일정이다. 서울 도심 콘텐츠는 이미 많이 만들었다. 이번엔 "한국 자연 속 무슬림 캠핑 체험"이라는 새로운 카테고리를 개척하고 싶다. 하지만 캠핑장에서 할랄 음식을 어떻게 해결할지, 기도 시간에 방향을 어떻게 찾을지, 혼성 캠핑 구역이라면 히잡 착용 상태로 편안할지 — 이런 질문들에 답을 줄 정보 소스가 없었다.

호텔 로비에서 유튜브를 보다가 K-Camp Finder v2 영어 리뷰 영상을 본다. 댓글에 "Does it have halal filter?" 라는 질문이 달려 있고, 앱 공식 계정이 "Yes, we have Muslim-friendly camping filter including halal food nearby and qibla direction"라고 답변한 것을 발견한다. 아미나가 즉시 앱을 다운로드한다.

## 사용자 여정

### 1단계: 진입
- 유튜브 댓글에서 공식 할랄 필터 답변 확인 → 즉시 다운로드 (1분)
- 영어 UI 설정. 풀스크린 봄 벚꽃 캠핑 영상 — "이 장면 촬영하면 팔로워들 좋아하겠다"
- 🌿숲멍 선택 후 "Muslim-friendly" 필터 활성화

### 2단계: 행동
- **할랄 음식 필터**: 캠핑장 반경 5km 내 할랄 인증 식당 또는 채식 옵션 정보 확인. 2곳이 표시되고 운영시간·메뉴가 영어로 제공됨
- **키블라 방향 기능**: 캠핑장 상세 페이지에서 "Qibla Direction" 탭 탭. GPS 기반으로 메카 방향과 각도 표시. "이 기능 때문에라도 이 앱 써야 한다"
- **기도 시간 알림**: 해당 지역 기준 파즈르·주흐르·아스르·마그리브·이샤 기도 시간 알림 설정. 캠핑 중에도 기도를 놓치지 않을 수 있음
- **혼성 구역 여부 확인**: 캠핑장 시설 정보에 "Mixed gender zone / Family zone / Women-only area" 구분 표시. 아미나는 가족 구역이 있는 캠핑장 선택
- **콘텐츠 촬영 동선 계획**: 포토스팟 태그 + 봄 벚꽃 명소 배지 조합. 벚꽃 만개 예측 날짜와 캠핑 일정 매칭
- **예약 완료**: 영어 UI로 온라인 결제 완료. 캠핑장 측에 "무슬림 여성 방문 예정, 기도 공간 문의" 메시지 전송 기능 활용

### 3단계: 결과
- 경기도 가평 벚꽃 캠핑장 1박. 히잡 착용 상태로 벚꽃 배경 사진 촬영
- 키블라 기능으로 텐트 안에서 기도 완료. "앱이 없었으면 방향 찾느라 한참 걸렸을 것"
- 할랄 식당에서 주문한 도시락을 캠핑장으로 배달 받아 저녁 식사
- 촬영 영상 편집 후 유튜브 게시: "Muslim Camping in Korea 🌸 Halal Food + Qibla Finder in the Mountains"
- 조회수 48만 회. 말레이시아·인도네시아·중동 무슬림 여행자들 폭발적 반응
- 앱 영어 댓글로 언급 급증 → 앱 다운로드 동남아 지역 스파이크

## 평가

|---|---|---|
| 유용성 | **10** | 할랄 필터 + 키블라 방향 + 기도 시간 알림의 3중 조합이 무슬림 캠핑 여행의 실질적 장벽을 모두 해소. 시장에 대안 없음 |
| 사용 편의 | **8** | 영어 UI 완성도 높음. Muslim-friendly 필터가 직관적으로 활성화됨. 다만 캠핑장 측 무슬림 응대 역량은 앱 외부 변수 |

## Pass/Fail 판정

> **✅ PASS**

무슬림 여행자가 한국 캠핑에서 경험하는 모든 실질적 장벽(할랄 식사, 기도 방향, 기도 시간, 혼성 구역 불편함)을 앱 하나로 해소한다. 이 카테고리는 현재 어떤 한국 캠핑 플랫폼도 공략하지 않은 블루오션이다. 아미나 같은 인플루언서가 자연 바이럴을 생성하면 동남아 무슬림 여행자 대규모 유입으로 이어지는 성장 엔진이 된다.

## 리텐션 평가: 이 페르소나가 앱을 다시 열까?

**재사용 확률**: 🔥매우 높음

**감정적 이유**:
> "무슬림 여행자로서 새로운 나라에서 캠핑한다는 건 보통 엄청난 사전 조사가 필요한 일이야. 근데 이 앱은 그 조사를 다 해줬어. 한국 올 때마다 쓸 거야, 그리고 팔로워들한테도 매번 알려줄 거야."

**재사용 트리거**:
- 한국 재방문 시 첫 번째로 여는 캠핑 앱
- 팔로워들이 "한국 캠핑 어떻게 해요?" DM 올 때마다 앱 링크 공유
- 새로운 할랄 캠핑 콘텐츠 시리즈 기획 시 앱으로 리서치

**이탈 리스크**:
- 할랄 식당 DB가 오래되거나 부정확할 경우 현장에서 당황 → 신뢰 하락
- 캠핑장이 실제로 무슬림 응대에 준비되지 않았을 경우 격차로 인한 실망

## 개선 제안

1. **무슬림 여행자 리뷰 전용 섹션**: 캠핑장 리뷰 중 무슬림 방문자 후기만 모아서 보는 필터. "기도 공간 어땠어요?", "할랄 음식 해결은?", "혼성 구역 분리는?" 같은 무슬림 특화 평가 항목으로 구성. 실사용 경험 기반 신뢰 구축
2. **캠핑장 무슬림 친화 인증 프로그램**: 앱이 캠핑장과 협력하여 기도 공간 제공, 할랄 식사 옵션, 무슬림 응대 교육 이수 시 "Muslim-Friendly Certified" 배지 부여. 공급자 측 준비도를 높이면서 앱의 할랄 필터 신뢰도를 제도적으로 보강
