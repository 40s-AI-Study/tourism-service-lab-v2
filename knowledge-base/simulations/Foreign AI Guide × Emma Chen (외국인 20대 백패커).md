# SIM: Foreign AI Guide × Emma Chen (외국인 20대 백패커)

## 시나리오

Emma Chen(23세, 대만 여행 크리에이터 · 팔로워 4.2만)이 서울 홍대→이태원→경복궁 3일 일정 첫날 아침, 호스텔 체크인 직후 앱을 켠다. X(구 트위터) 대만 여행 그룹에서 "번체 중국어 완벽 지원, AI가 다음 갈 곳 추천해줌"이라는 후기를 보고 이미 설치해 둔 상태다. 오늘의 목적: 홍대 포토스팟 찾기 + 현지인 맛집 발견 + 릴스 소재 수집.

## 사용자 여정

### 1단계: 진입 (앱 최초 실행)
- 앱 실행 → 언어 선택: **繁體中文** 탭 → 전체 UI 즉시 전환 (20초)
- 위치 권한 허용 → "홍익대 근처" 자동 감지
- AI 웰컴 메시지: "안녕하세요 Emma, 홍대 인근 관광지를 안내해드릴게요 🗺️"
- 오늘의 혼잡도 미리보기 배너: 홍대 오전 9시 **집중률 18%** (한산) 표시

### 2단계: 탐색 (핵심 기능 사용)
- AI 채팅창에 타이핑: "📸 홍대에서 사진 찍기 좋은 곳 알려줘"
- AI 응답: 홍대 거리 벽화골목 / 경의선 숲길 / 홍대 앞 광장 3개 카드 제시
  - 각 카드: 공식 관광사진(`관광사진정보 API`), 혼잡도, 관련 오디오 가이드 버튼
- 경의선 숲길 선택 → `locationBasedList2(EN)` + `storyLocationBasedList` 연동
- 오디오 가이드 실행: **번체 중국어** 음성으로 경의선 역사 설명 청취 (3분)
- 연관 관광지: `관광지별 연관 관광지` API → 합정 메세나폴리스, 망원한강공원 카드 노출

### 3단계: AI 일정 최적화
- 채팅: "오늘 홍대→이태원 이동할 건데 최적 순서 알려줘"
- AI가 집중률 예측 데이터 기반으로 **시간대별 추천 루트** 생성:
  - 09:00 경의선 숲길 (집중률 18%)
  - 11:00 홍대 벽화골목 (11시 이후 30% → 오전 방문 권장)
  - 13:00 이태원 전통시장 점심 (평일 점심 43%)
  - 15:00 이태원 이슬람 거리 (15시 이후 완화)
- "일정 저장" → 앱 내 여행 플랜으로 등록

### 4단계: 실시간 활용
- 경의선 숲길 현장 도착 → GPS 기반 오디오 가이드 자동 시작
- AR 포토스팟 마커: 최적 촬영 각도 오버레이 (인스타 비율 9:16 프레임 제안)
- 릴스 촬영 완료 후 앱 내 공유: 위치 태그 + 번체 중국어 캡션 자동 생성

### 5단계: 결과
- 오전 루트 완벽 소화 → 혼잡 없이 고품질 사진 10장 확보
- 오디오 가이드 핵심 문구 기반 릴스 캡션 → 팔로워 반응 200% 증가
- Dcard 여행 게시판: "홍대 여행 앱 최고! Foreign AI Guide 번체 완벽" 후기 게시
- 인스타 팔로워 3,200명 신규 유입
