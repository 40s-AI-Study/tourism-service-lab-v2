# SIM-140: SilverTrail Korea × 타나카 하루카 (일본 직장인)

## 시나리오

하루카는 일본 한국 여행 X(트위터) 계정에서 "韓国の歴史スポット、オーディオガイドが日本語で聴けるアプリ"라는 게시글을 보고 SilverTrail Korea를 발견했다. K-drama 《도깨비》 전주 촬영지를 방문할 계획이었고, 현장 역사 설명을 일본어로 들을 수 있다는 점이 마음에 들었다. 하지만 앱을 실제로 설치해 보니 일본어 UI와 오디오 가이드는 제공되지 않았다(한국어·영어만 지원). 또한 K-drama 성지 정보나 K-pop 관련 콘텐츠가 없었고, 시니어 중심 트레일과 웰니스 기능이 주를 이뤄 자신의 여행 스타일과 거리가 멀었다.

---

## 사용자 여정

### 1단계: 진입
- X(트위터) "일본어 오디오 가이드" 추천 게시글 기대로 설치 → 일본어 UI 없음 즉시 발견
- 영어로 전환 후 탐색 계속 (영어 중급 수준으로 가능)

### 2단계: 행동
- 전주 한옥마을 탐색: `central-attractions-by-municipality` 기반 명소 목록 — 영어 정보는 있으나 K-drama 촬영지 특정 위치 정보 없음
- `audio-guide` 영어 해설 확인: 내용은 있으나 일본어 불가 — 게시글 설명과 다름. 속은 느낌
- K-drama 성지 검색(《도깨비》 촬영지) → 기능 없음
- 인증샷 포토스팟 목록 탐색 → 시니어 트레일 중심으로 구성, SNS 인증샷 뷰포인트 정보 없음
- 시니어 UI 속도감과 큰 글씨 — 27세 K-팝 팬에게 세대 부조화 느낌

### 3단계: 결과
- 앱 삭제. 일본어 한국 여행 유튜브 채널과 네이버 재팬으로 복귀

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 일본어 미지원, K-drama 성지 없음, 포토스팟 없음 — 핵심 여행 목적 3가지 모두 미충족 |
| 사용 편의 | **4** | 영어 전환 후 기본 탐색 가능하나 시니어 UI가 젊은 사용자에게 이질적. X(트위터) 게시글과 실제 기능 불일치로 신뢰 훼손 |
| 구현 타당성 | **7** | 앱 자체 구현은 타당하나 일본어 지원 없이 일본 방문객에게 배포되는 건 마케팅 미스매치 |

## Pass/Fail 판정

> **❌ FAIL**

종합 13점으로 FAIL. 하루카에게 SilverTrail Korea는 일본어 미지원, K-drama/K-pop 콘텐츠 부재, SNS 포토스팟 없음이라는 세 가지 결정적 결함으로 실사용이 불가능했다. X(트위터) 추천 게시글의 오해(일본어 지원이라 착각)가 오히려 앱에 대한 부정적 인상을 남겼다.

## 개선 제안

1. 일본어 오디오 가이드 지원 추가 — 일본인 시니어 관광객(연 200만 명 이상 방한)은 SilverTrail Korea의 실질적 외국인 타겟이 될 수 있으며, 일본어 지원이 전제되어야 함
2. 앱스토어 및 소셜 미디어 정보를 실제 지원 언어·기능에 맞게 정확히 기재 — 기대-현실 불일치는 가장 빠른 이탈 원인이므로, 일본어 지원이 없다면 일본어 홍보 채널에서 노출 차단 필요
