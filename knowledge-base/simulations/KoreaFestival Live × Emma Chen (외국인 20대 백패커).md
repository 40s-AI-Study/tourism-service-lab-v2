# SIM-164: KoreaFestival Live × Emma Chen (외국인 20대 백패커)

## 시나리오

에마는 한국 여행 7일차, 서울 일정을 마치고 지방 도시로 이동할 계획이다. 인스타그램 대만 팔로워들이 "한국 전통 축제 꼭 가봐"라고 추천하지만, 어떤 축제가 언제 열리는지 영어 정보가 없어 막막했다. 숙소 로비에서 영국인 백패커가 KoreaFestival Live 앱을 쓰는 걸 보고 관심을 가진다. 앱이 영어를 지원하고 지도 뷰에서 현재 위치 기준 가까운 축제를 보여준다는 걸 알게 된다. 강릉단오제를 발견하고, 사진 작가로서 "새벽 4시가 혼잡도 가장 낮다"는 데이터를 활용해 이른 아침 황금빛 분위기의 단독 포토스팟을 확보할 계획을 세운다.

---

## 사용자 여정

### 1단계: 진입
- 숙소에서 영어로 앱 소개 듣고 앱스토어 검색 → 설치 후 언어 자동 감지로 영어 UI 제공
- 지도 뷰에서 "현재 위치 근처 축제" 레이더 활성화 → 강릉 단오제 마커 발견
- tourism-info-english API 기반 영어 축제 상세 정보 확인 — 유네스코 무형문화유산 등재 배경 설명 포함

### 2단계: 행동
- 축제 혼잡도 시간대 그래프 확인 → 오전 6~8시 혼잡도 12%, 오후 2~5시 87% 대비
- "포토그래퍼 팁" 태그(앱 내 커뮤니티 큐레이션) — "새벽 단오굿 의식 사진 촬영 최적" 정보 발견
- 관련 관광지 코스에서 강릉 커피거리 + 경포대 + 오죽헌 연계 당일 코스 자동 생성
- 교통 안내: KTX 청량리→강릉 시간표 링크 제공 (앱 내 외부 링크)
- 사진 공유 기능으로 촬영 후 인스타그램 직접 연동

### 3단계: 결과
- 새벽 6시 도착, 인파 없는 단오제 현장에서 의식 사진 20컷 확보
- 팔로워 5만 명 계정에 "대만에서는 몰랐던 한국 전통 축제" 포스팅 → 빠른 확산
- 앱을 대만 여행자 커뮤니티 PTT에 소개하는 자발적 홍보자로 전환

| 항목 | 점수 (1-10) | 근거 |

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live는 에마가 가장 불편해하는 "한국어만 지원하는 여행 앱" 문제를 영어 전용 인터페이스로 해결하며, 혼잡도 데이터를 포토그래퍼의 촬영 타이밍 도구로 활용할 수 있다는 차별화된 가치를 제공한다. 여행 크리에이터 특성상 SNS 바이럴 효과까지 기대할 수 있어 마케팅 측면에서도 핵심 타겟이다.

## 개선 제안

1. 사진 크리에이터를 위한 "포토 타임 추천" 기능 강화: 축제별 황금 시간대(골든아워 + 혼잡도 최저 교차점)를 별도 뱃지로 표시하여 에마 같은 여행 크리에이터를 핵심 사용자로 락인
2. 다국어 확장으로 번체 중국어(Traditional Chinese) UI 추가 — 대만, 홍콩 여행자 대상으로 에마 같은 인플루언서를 통한 자연 확산 경로 확보 가능
