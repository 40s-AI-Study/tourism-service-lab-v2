# SIM-159: PhotoHunt Korea × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

아미나는 팔로워 4만 5천 명의 여행 인플루언서로, 한국 여행 중 팔로워들에게 공유할 "무슬림도 즐길 수 있는 한국 포토스팟" 콘텐츠 제작이 핵심 목표다. 인스타그램에서 한국 여행 관련 앱을 탐색하던 중 PhotoHunt Korea의 "공식 인증 포토스팟" 광고를 봤다. 디지털 마케터답게 앱의 콘텐츠 생산 지원 기능에 주목했고, 영어 UI 지원 여부를 확인 후 설치했다.

---

## 사용자 여정

### 1단계: 진입
- 인스타그램 광고 → 앱스토어 영문 설명 확인 → 영어 지원 확인 후 설치
- 앱 실행: 영문 UI 전환 성공
- 온보딩: "Street Photography", "Traditional Culture", "Nature" 선택

### 2단계: 행동
- 서울 검색 → 경복궁·창덕궁·북촌 포토스팟 지도 표시
- `photo-contest-winners` 수상작 확인: 히잡 착용자도 문화적으로 어색하지 않은 전통 의상 체험 스팟 몇 곳 포함 — 긍정적
- `visitor-concentration-forecast`: 경복궁 수요일 오전 9시 방문자 22% — 팔로워 콘텐츠용 깨끗한 배경 확보 가능 시간 확인
- SNS 공유 기능: 한국어 해시태그 + 영어 해시태그 자동 생성 — 말레이어 해시태그 없음 아쉬움
- 할랄 식당 연계 정보 없음 — 포토스팟에서 이동 후 식사 장소 별도 검색 필요
- 무살라(기도실) 인근 여부 정보 없음 — 기도 시간 관련 불편 해소 불가

### 3단계: 결과
- 포토스팟 자체는 완벽하게 활용 — 경복궁 오전 사진 콘텐츠 퀄리티 높음
- 한복 체험 스팟에서 히잡 착용 한복 사진 → 말레이시아 팔로워 반응 폭발적
- 앱 유지 및 경주·전주 방문에도 활용
- 인스타 스토리에 앱 소개 ("Great for finding photo spots in Korea!") — 말레이시아·인도네시아 팔로워에게 노출

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **8** | 포토스팟 큐레이션 + 혼잡도 예측 + SNS 연동이 인플루언서 니즈 대부분 충족 |
| 사용 편의 | **7** | 영문 UI 완전 지원, 말레이어 해시태그 미지원은 아쉽지만 영어로 대체 가능 |
| 구현 타당성 | **8** | 핵심 기능 구현 타당, 무슬림 특화 기능(할랄 연계·기도실 안내)은 추가 과제 |

**종합 점수: 23/30**

---

## Pass/Fail 판정

> **✅ PASS**

아미나는 PhotoHunt Korea의 강력한 타깃 사용자다. 인플루언서로서의 콘텐츠 생산 니즈가 앱의 공식 포토스팟 큐레이션 + SNS 공유 기능과 정확히 맞아떨어졌다. 무슬림 특화 기능(할랄 연계, 기도실 정보)이 없다는 한계가 있지만, 포토 앱 본연의 기능만으로도 높은 만족도를 얻었다. 동남아 인플루언서를 통한 자발적 바이럴이 기대되는 사례다.

## 개선 제안

1. 다국어 해시태그 자동 생성 확장: 영어·한국어 외에 말레이어·인도네시아어·아랍어 해시태그 자동 생성으로 동남아·중동 인플루언서의 SNS 공유 편의성 극대화
2. 무슬림 친화 포토 투어 패키지: 할랄 식당·무살라 근처 포토스팟을 묶은 "Muslim-Friendly Photo Tour" 코스 제공으로 한국 방문 무슬림 여행자(연 100만 명 이상) 특화 공략
