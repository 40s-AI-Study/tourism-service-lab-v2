# SIM-184: KoreaDemand Atlas × Emma Chen (외국인 20대 백패커)

## 시나리오

에마는 대만 여행 인플루언서 커뮤니티 PTT에서 누군가 "한국 관광 데이터 앱인데 영어 지원됨, 숨은 명소 찾기 좋음"이라는 글을 보고 KoreaDemand Atlas를 설치한다. 영어 UI로 전환하니 전국 히트맵이 표시되고, 서울은 이미 붉게 과포화 상태이지만 전주·강릉·통영 일대에 "Rising" 뱃지가 붙어 있다. 크리에이터로서 "아직 덜 알려진 곳"이라는 데이터 근거는 콘텐츠 차별화에 직접 도움이 된다. 통영 인근 소도시를 클릭하니 영어로 관광 정보가 나오고, 방문자 예측 그래프도 영어로 제공된다.

---

## 사용자 여정

### 1단계: 진입
- PTT 커뮤니티 추천 글 → 앱스토어 설치 (해외 앱 거부감 없는 디지털 네이티브)
- 영어 언어 전환 즉시 확인 → 히트맵 + "Rising Area" 뱃지 UI 탐색 시작
- 번체 중국어 미지원 확인 — 영어는 가능하므로 계속 사용

### 2단계: 행동
- 통영 Rising 지역 탭: 관광 잠재력 점수 64점, 국제 다양성 지수 낮음(=외국인 방문 드문 진짜 로컬)
- "국제 다양성 낮음 = 아직 외국인 발견 안 됨" 해석 → 크리에이터 관점에서 최고 가치
- 방문자 집중률 예측: 내일 오전 저점 확인 → 포토 타임 계획
- tourism-info-english로 통영 핵심 명소 정보 열람 — 영어 콘텐츠 품질 양호
- 코스 생성 후 저장 → 인스타 스토리에 히트맵 스크린샷 공유 ("데이터가 찾아준 통영")

### 3단계: 결과
- 통영 방문, 외국인 거의 없는 로컬 분위기 촬영 성공 → 팔로워 반응 역대급
- 앱 자체를 콘텐츠 소재로 활용 ("이 앱이 알려준 곳") → 바이럴 발생
- 번체 중국어 미지원, 중국어 오디오 가이드 없음은 아쉬움으로 남음

## Pass/Fail 판정

> **✅ PASS**

에마는 KoreaDemand Atlas의 이상적 사용자 중 하나다. 데이터 기반 숨은 명소 발굴 기능이 콘텐츠 크리에이터의 차별화 니즈와 정확히 맞아떨어지며, 실제 방문 성공과 바이럴 전파로 이어지는 완결된 가치 사슬이 작동했다. 유용성 8점으로 기준을 상회한다.
