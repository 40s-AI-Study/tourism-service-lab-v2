# SIM-R3-014: K-Camp Finder v2 × 김지원

## 시나리오

김지원(25세)은 홍대 인근 디자인 스튜디오에 다니는 그래픽 디자이너다. 평일엔 클라이언트 브리프와 마감에 치이고, 주말엔 그 피로를 혼자만의 감성 나들이로 푼다. 인스타그램 피드는 그녀의 취향 아카이브이자 명함이다. 팔로워 3,200명, 미적 감각 있는 일상 사진들로 가득 차 있다.

혼잡한 카페나 유명 포토스팟은 싫다. 사람이 몰리면 무조건 피한다. "남들 다 가는 곳"은 그녀에게 스트레스다. 요즘 솔로 캠핑에 관심이 생겼는데, 텐트는 있고 장비도 기본은 갖췄다. 문제는 어디를 가야 "나만의 감성 사진"을 건질 수 있는지 모른다는 것이다. 캠핏이나 야놀자 캠핑은 써봤지만 죄다 예약 목록만 보여줘서 재미없었다.

유튜브 쇼츠에서 K-Camp Finder v2 소개 영상이 자동 재생된다. 앱 첫 화면에 "지금 어떤 하늘 아래서 쉬고 싶어?"라는 카피와 계절 영상이 교차되는 15초짜리 클립. "이거 UI 예쁘다" 직관적으로 느낀다. 디자이너 눈으로 봐도 잘 만든 앱이다. ⭐별멍을 탭하는 자신의 모습이 이미 상상된다.

## 사용자 여정

### 1단계: 진입
- 유튜브 쇼츠 → 앱스토어 → 첫 실행 (2분)
- 풀스크린 여름 계곡 해먹 영상과 가을 단풍 영상이 계절 순으로 전환됨. "이 영상 퀄리티 좋다" 심리적 신뢰 형성
- ⭐별멍 선택. 광해 지수 낮은 캠핑장 필터 자동 활성화

### 2단계: 행동
- **혼잡도 예측 필터 적극 활용**: "이번 주말 혼잡도 낮음" 필터를 첫 번째로 켬. 유명한 곳은 오히려 걸러냄. "사람 없는 곳이 포인트"
- **포토스팟 태그 탐색**: 각 캠핑장의 사용자 업로드 사진 슬라이드를 꼼꼼히 봄. 구도가 식상한 곳은 넘김. 강원도 홍천의 한 소규모 캠핑장에서 "아직 덜 알려진 각도"를 발견
- **4계절 감성 뱃지 확인**: 해당 캠핑장이 "가을 단풍 명소 ★★★★☆" 뱃지를 보유. 현재 시즌과 맞아떨어짐
- **예산 필터**: 당일 + 1박 기준 4만원 이하 필터. 솔로 캠핑 사이트비가 2만 2천 원으로 확인됨
- **실시간 혼잡도 알림 설정**: "이번 주 토요일 해당 캠핑장 혼잡도 변동 시 알림" 설정. 예상보다 예약이 몰리면 대체지 추천 받을 수 있음
- **앱 내 저장 기능**: 관심 캠핑장 3곳 북마크 후 비교 뷰로 나란히 확인

### 3단계: 결과
- 토요일 오전 11시 홍천 캠핑장 도착. 예상대로 텐트 5동 이하, 조용함
- 오후 골든아워에 단풍 + 텐트 실루엣 역광 사진 촬영. "완벽하다"
- 저녁 불멍 중 별이 뜨기 시작함. 새벽 2시까지 은하수 시도했지만 구름에 막힘 (날씨 변수는 앱 한계)
- 귀가 후 사진 편집 → 인스타 게시 → "어디야?" 댓글 40개
- 앱 리뷰: "혼잡도 필터가 킬러 기능. 이제 캠핏 안 쓸 것 같음"
- 불만: 앱이 알려준 포토스팟 정보가 몇 개는 이미 인스타에 많이 올라온 곳이었음. "진짜 숨은 곳" 발굴 능력은 아직 개선 여지

## 리텐션 평가: 이 페르소나가 앱을 다시 열까?

**재사용 확률**: 🔥매우 높음

**감정적 이유**:
> "혼잡도 낮은 곳만 골라준다는 게 처음엔 반신반의했는데, 진짜로 사람이 없었다. 내 인스타 피드가 남들이랑 달라 보이려면 이 앱이 필요해."

**재사용 트리거**:
- 매 주말 전날 저녁, "이번 주는 어디 가지?" 하며 앱 오픈
- 혼잡도 변동 알림 수신 시 즉시 앱 진입
- 계절이 바뀔 때 새로운 감성 배경화면과 추천 캠핑장 확인하러 오픈

**이탈 리스크**:
- 포토스팟 정보가 인스타에 이미 포화된 유명 장소 위주로만 구성될 경우 이탈
- 혼잡도 예측이 자주 빗나가면 신뢰 하락 후 이탈

## 개선 제안

1. **"언노운 스팟" 전용 탭**: 인스타그램 태그 수 기반으로 "아직 100게시물 미만인 캠핑 포토스팟"만 모아주는 섹션. 감성파 솔로 여행자가 가장 갈증을 느끼는 "아무도 모르는 나만의 장소" 수요를 직접 공략
2. **혼잡도 예측 정확도 피드백 루프**: 방문 후 "실제 혼잡도 어땠나요?" 간단 피드백(1-5단계)을 수집. 사용자 피드백으로 예측 모델을 지속 개선하고, 정확도가 높아질수록 앱 신뢰도 상승 → 리텐션 강화
