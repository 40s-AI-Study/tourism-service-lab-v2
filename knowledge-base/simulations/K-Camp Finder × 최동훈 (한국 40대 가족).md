# SIM-93: K-Camp Finder × 최동훈 (한국 40대 가족)

## 시나리오

최동훈(44세, 초등학교 교사)은 여름방학을 맞아 초등학교 2학년 딸, 4학년 아들과 함께 2박 3일 가족 캠핑을 계획 중이다. 아이들이 직접 체험할 수 있는 프로그램(곤충 탐구, 물놀이, 별자리 관측 등)이 있는 캠핑장을 원하고, 안전 시설(구명조끼 구비, 의무실 접근, 펜스 설치 등)이 중요하다. 본인은 캠핑 중급자지만 아이들 짐까지 챙기는 것이 버거워 준비물 자동화 기능에 관심 있다.

K-Camp Finder에서 "가족 / 4인 / 체험 프로그램 / 초등생 안전시설" 조건으로 검색한다.

---

## 사용자 여정

### 1단계: 진입

- 학교 학부모 단톡방에서 K-Camp Finder 추천 링크 수신
- 앱 설치 후 네이버 로그인
- 온보딩: 동반 유형 "가족", 자녀 연령 "초등 저학년·고학년" 입력
- 메인 화면 "가족 캠핑 특집 — 체험 프로그램 있는 캠핑장 TOP 10" 배너 클릭

### 2단계: 행동

- 조건 입력: 인원 4명(성인 2, 초등생 2), 날짜 7월 말 2박 3일, 지역 충청·전라 권역, 체험 프로그램 필터 ON, 안전시설(구명조끼·CCTV·펜스) 필터 ON
- AI 추천 TOP 5 수신: 각 캠핑장의 체험 프로그램 목록, 안전시설 체크리스트, 가족 리뷰 평점 표시
- "청양 알프스마을 오토캠핑장" 선택: 곤충 체험, 천문 관측, 수영장(구명조끼 제공), CCTV 전 구역 표시 확인
- 연계 코스 확인: 청양 칠갑산 자연휴양림(도보 10분), 청양 고추&구기자 엑스포 방문
- 가족 캠핑 체크리스트 자동 생성: 4인 기준, 아이 연령 맞춤 (모기 기피제, 구명조끼 사이즈, 어린이 안전 바닥재 등 포함)
- D-3 날씨 알림 설정 후 gocamping API 예약 진행

### 3단계: 결과

- 예약 완료, 체크리스트 PDF 저장 후 프린트
- D-3 알림: "7월 27일 맑음 32°C, 28일 오후 소나기 예상 → 우비·방수 텐트 커버 챙기세요"
- 캠핑 후 아이들과 함께 리뷰 작성, 체험 프로그램 별점 5점

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 자녀 연령별 안전시설 필터 + 체험 프로그램 필터 조합이 가족 캠핑의 가장 복잡한 의사결정 문제를 해결 |
| 사용 편의 | **8** | 자녀 연령 입력만으로 맞춤 체크리스트 자동 생성, 필터 항목이 직관적 |
| 구현 타당성 | **7** | gocamping API에 체험 프로그램·안전시설 데이터 포함 여부 불확실, 추가 크롤링 또는 캠핑장 직접 데이터 수집 필요 |

**종합 점수: 24/30**

## Pass/Fail 판정

> **✅ PASS**

가족 캠핑 계획의 핵심 장벽(안전 확인, 체험 프로그램 탐색, 준비물 관리)을 원스톱으로 해결한다. 유용성 9점, 종합 24점으로 Pass 기준 충족. 안전시설 데이터의 정확성과 최신성이 서비스 신뢰도를 결정하는 핵심 변수.

## 개선 제안

1. 안전시설 데이터 검증 체계 구축: gocamping API 외 캠핑장 직접 제보·인증 배지 시스템 도입으로 안전시설 정보 신뢰도 제고
2. 가족 캠핑 공동 플래닝 기능: 배우자와 앱 내 실시간 체크리스트 공유 및 역할 분담 기능 추가로 준비 과정 협업화
