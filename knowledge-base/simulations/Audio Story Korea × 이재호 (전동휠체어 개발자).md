# SIM-67: Audio Story Korea × 이재호 (전동휠체어 개발자)

## 시나리오

이재호(38세, IT 개발자)는 전동휠체어를 사용하는 지체장애인으로, 무장애 동선이 확보된 문화재를 주로 방문한다. 덕수궁을 방문한 날, 무장애 경사로를 따라 이동하며 QR 코드를 스캔했다. 한 손으로 휠체어를 조작하며 다른 손으로 스마트폰을 들고 있어야 하는 상황이므로, 화면을 적게 터치해도 되는 오디오 우선 UX를 높게 평가했다. 이어폰을 착용하고 이동하면서 오디오를 주 콘텐츠로 소비했다.

---

## 사용자 여정

### 1단계: 진입
- 덕수궁 정문(대한문) 옆 무장애 경사로 입구 QR 스캔
- 한국어 선택 → 오디오 자동 시작
- 이어폰 착용 후 스마트폰을 휠체어 거치대에 거치

### 2단계: 행동
- 챕터 1 "덕수궁의 슬픔 — 대한제국 마지막 황제의 공간" 청취하며 경사로 동선 이동
- 음성 명령으로 챕터 이동 시도 (음성 제어 기능 여부 확인 필요)
- 현재 음성 제어 미지원 → 화면 터치로 챕터 전환, 다소 불편
- 무장애 동선과 오디오 챕터 위치가 불일치 구간 발생 → 중정당 챕터를 계단 앞에서 청취해야 하는 상황
- 공모전 수상작 갤러리는 정지 상태에서 감상, 이동 중 갤러리 탐색은 어려움
- 전체 챕터 중 무장애 동선 내 2개 챕터 완주

### 3단계: 결과
- 오디오 우선 UX 자체는 높게 평가 ("이동 중 들을 수 있어서 좋았다")
- 무장애 동선과 챕터 위치 불일치 문제를 개발자 시각으로 구체적 피드백 제출
- 음성 제어 기능 요청 피드백 앱 내 제출
- 재사용 의향: 음성 제어 업데이트 후 긍정적

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **7** | 오디오 우선 UX가 이동 중 손 자유 제약을 부분적으로 해소, 무장애 동선 내 청취 가능 |
| 사용 편의 | **5** | 음성 제어 미지원으로 한 손 조작 시 챕터 전환 불편, 무장애 동선-챕터 위치 불일치 |
| 구현 타당성 | **7** | 음성 제어 및 무장애 동선 연동은 추가 개발 필요하나 기술적으로 실현 가능 |

## Pass/Fail 판정

> **✅ PASS**

유용성 7점, 종합 19점으로 기준 충족. 오디오 우선 UX가 전동휠체어 사용자의 핵심 니즈(이동 중 핸즈프리 정보 소비)와 잘 맞지만, 음성 제어 미지원과 무장애 동선 불일치가 완성도를 낮추는 요인이다.

## 개선 제안

1. **음성 제어 지원**: "다음 챕터", "일시정지", "반복" 등 기본 음성 명령 지원으로 이동 중 핸즈프리 조작 가능하게 하여 전동휠체어·시각장애 사용자 접근성 대폭 향상
2. **무장애 동선 연동 챕터 순서**: 문화재의 무장애 접근 가능 구역 데이터를 기반으로 해당 동선에 맞춰 챕터 순서를 자동 재배열하는 "접근성 모드" 추가
