# SIM-171: KoreaWork & Wander × 김지원 (한국 20대 솔로)

## 시나리오

지원은 프리랜서 그래픽 디자이너로 재택근무 특성상 서울 카페에서 작업하는 날이 많다. 어느 날 유튜브에서 "강릉 카페에서 일하며 여행하기" 브이로그를 보고 워케이션에 관심이 생긴다. 예산이 빠듯해 호텔보다는 감성적이면서도 와이파이 빵빵한 글램핑이나 독채를 원한다. KoreaWork & Wander 앱을 설치해 "강릉, 1주일, 2인 이하, 와이파이·전기 콘센트 필수" 조건으로 검색한다. 그런데 막상 앱 주 타겟이 "디지털 노마드 외국인"과 "한 달 살기 원격근무자"임을 느끼게 되는 인터페이스를 마주한다. 혼자 일주일 워케이션을 가는 20대 프리랜서 니즈와는 스케일이 다소 맞지 않는다는 인상을 받는다.

---

## 사용자 여정

### 1단계: 진입
- 유튜브 브이로그 → 앱스토어 검색 "워케이션 앱" → KoreaWork & Wander 설치
- 홈 화면에서 언어 선택(FR/DE/ES/EN) 프롬프트 → 한국어 옵션 없어 당황 (한국인 사용자 고려 미흡)
- "한국어" 검색 후 영어 UI로 우회 진입

### 2단계: 행동
- "강릉" 지역 검색 → gocamping API 기반 글램핑·캠핑장 목록 표시
- 와이파이·전기 콘센트 필터 적용 → 후보 5곳으로 좁혀짐
- area-tourism-diversity 지수 확인: 강릉 ★★★★★ — 카페·바다·산악이 모두 있는 최고 다양성 지역
- 주변 관광 자동 추천: 강릉 커피거리, 경포대, 안목해변 → 오후 관광 코스로 활용 가능
- 단, "1주일 패키지" 중심 UI라 당일~3일 단기 워케이션 검색 경로가 불명확

### 3단계: 결과
- 글램핑 1곳 최종 선정 후 외부 예약 사이트로 이동해 별도 예약 (앱 내 예약 불가)
- 워케이션 기간 업무 효율 만족스러웠으나, 앱 사용은 검색 단계에서만 활용
- "좋은 정보긴 한데 외국인용 앱 같다"는 인상으로 이후 재사용 가능성 낮음

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **6** | gocamping × 와이파이 필터 + 지역 다양성 지수 조합은 워케이션 장소 선택에 실질적으로 유용. 그러나 1주일 이하 단기 사용자에게는 과잉 기능 |
| 사용 편의 | **4** | 한국인 사용자를 위한 한국어 UI 부재가 결정적 진입 장벽. 주 타겟이 외국인임이 UI에서 명확히 드러나 한국 20대에게 소외감 유발 |
| 구현 타당성 | **7** | gocamping API + area-tourism-diversity 조합 기술 구현 자체는 타당. 그러나 한국 사용자 온보딩 설계 부재가 실사용 전환 저해 |

**종합 점수: 17/30**

## Pass/Fail 판정

> **⚠️ CONDITIONAL PASS**

KoreaWork & Wander는 김지원에게 워케이션 장소 탐색이라는 측면에서 일정 유용성을 제공하나, 한국어 UI 부재와 단기·소규모 워케이션 사용자 고려 미흡이 결합되어 전환율이 낮다. 외국인 타겟 특화 전략 자체는 유효하지만, 한국 내 원격근무자 수요를 함께 잡으려면 한국어 지원과 단기 사용 플로우 별도 설계가 필요하다.

## 개선 제안

1. 한국어 UI 및 한국 사용자 온보딩 추가: 언어 선택 화면에 한국어를 포함하고, "국내 원격근무자" 전용 진입 경로(1~7일 단기 워케이션)를 외국인 장기 체류와 분리하여 설계
2. 단기 워케이션(1~3일) 전용 필터 및 "오늘 바로 출발" 즉흥 워케이션 기능 추가 — 지원 같은 즉흥형 프리랜서의 사용 패턴에 맞는 단순화된 검색 플로우 제공
