# SIM-120: MediTrip Korea × 타나카 하루카 (일본 직장인)

## 시나리오

하루카는 3박 4일 첫 한국 여행을 준비하다가 일본 SNS에서 "한국 미용성형 앱"으로 MediTrip Korea를 접한다. 일본에서 한국 미용의료 관광은 매우 인기 있는 트렌드다 — 라식·피부 레이저·치아 미백을 한국에서 받는 일본 여성들이 많다. 하루카도 "이번 여행에 피부 트리트먼트 한번 받아볼까" 하는 마음이 있다. 앱을 실행하면 일본어 UI를 선택할 수 있어 바로 탐색에 들어간다. 피부과 클리닉 검색에서 강남 클리닉들이 일본어 설명과 함께 나열되며, 가격도 일본의 절반 이하임을 확인한다.

---

## 사용자 여정

### 1단계: 진입
- 일본 X(트위터) 한국 미용의료 관련 계정에서 앱 추천 → 설치
- 언어 선택: 일본어 선택 → `tourism-info-japanese` 기반 일본어 UI 로드
- 홈 화면: 피부과·성형·치과 카테고리 — 하루카의 관심과 직결

### 2단계: 행동
- "피부과" 필터: 강남 레이저 트리트먼트 클리닉 8곳 — 일본어 설명, 가격, 후기 확인
- 1곳 선택 → 온라인 예약 폼: 일본어로 작성 가능, 여권 번호·방문일 입력 후 예약 완료
- "시술 후 관광 코스" 탭: 강남 → 청담 → 압구정 코스 확인 (일본어)
- `tourism-photos` 기반 사진들: 예쁜 카페·거리 사진 — 인스타 콘텐츠로 활용 예정
- K-드라마 성지 정보 없음 — 전주·경주 방문 계획에는 이 앱 미활용 결정

### 3단계: 결과
- 피부 레이저 시술 예약 완료 — 앱 핵심 기능 100% 사용
- 강남 당일 코스 저장. 시술 후 코스 그대로 실행 예정
- K-드라마 성지 탐방 부분은 별도 일본어 블로그 참고
- X에 "MediTrip Koreaで韓国の皮膚科予約できた！日本語対応◎ 강남 관광コースも便利" 포스팅

## Pass/Fail 판정

> **✅ PASS**

타나카 하루카는 MediTrip Korea의 핵심 타깃 사용자 중 하나다. 일본어 UI, 피부과 클리닉 검색·예약, 시술 후 관광 코스 연동이 모두 그녀의 실제 여행 니즈와 정확히 맞아떨어진다. 인플루언서 성향의 하루카가 SNS에 긍정 후기를 올리면 일본 미용의료 관광 시장 전체에 바이럴 효과가 기대된다. 종합 25점, 강력한 PASS.

## 개선 제안

1. K-드라마 성지 정보를 관광 코스 탭에 추가하면 하루카처럼 미용의료 + 한류 관광을 조합하는 일본 여행자의 앱 체류 시간과 만족도를 크게 높일 수 있음
2. 일본어 SNS 공유 버튼(X·Instagram 직접 연동)을 추가하면 하루카 같은 인플루언서 사용자의 자발적 바이럴을 적극 유도할 수 있음
