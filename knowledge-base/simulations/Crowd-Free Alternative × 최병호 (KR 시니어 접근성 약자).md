# SIM-CFA-03: Crowd-Free Alternative × 최병호 (KR 시니어 접근성 약자)

## 시나리오

최병호(64세, 은퇴 공무원, 수원 거주)는 무릎 관절 수술 후 6개월째 회복 중이다. 오랜만에 가족들과 창경궁을 방문하려 하지만, 붐비는 곳에서 밀려 넘어질까봐 걱정이 크다. 아들이 "아버지, 이 앱으로 언제 가면 사람 없는지 볼 수 있어요"라며 스마트폰에 설치해 준다.

---

## 사용자 여정

### 1단계: 진입 (아들이 설치 → 부친이 사용)
- 아들이 Crowd-Free Alternative 설치 및 초기 설정 완료
- 글씨 크기 자동 시스템 설정(iOS 큰 글씨 모드) 감지 → 앱 UI 자동 확대 적용
- 아들이 기본 사용법 2분 설명: "아버지, 여기 검색하고 초록색이면 안전해요"
- 최병호: "어, 이게 그렇게 어렵지 않네"

### 2단계: 혼잡도 확인
- 아들과 함께 "창경궁" 검색 → 혼잡도 타임라인 표시
- 이번 주 토요일: 🔴 혼잡도 69% (주말 가족 관광 피크)
- 이번 주 수요일 오전 10시: 🟢 혼잡도 14%
- 최병호: "수요일은 어때? 나는 평일이 더 편한데"
- 수요일 상세 보기: "방문자 예측 3,200명 (주말의 19%). 입장 대기 5분 이내 예상"
- 최병호: "이게 얼마나 정확한 거야?" → 아들: "KTO 빅데이터 기반이래요, 꽤 믿을만해요"

### 3단계: 대안 탐색 (비혼잡 시간대 선택으로 대체)
- 창경궁 자체가 비교적 덜 붐비는 대안지 → 대안 탐색보다 최적 시간 선택에 집중
- 앱 추가 정보: "창경궁 경사도 정보 — 주요 관람 동선 평균 경사 3° 이하, 휠체어·보행 보조기 접근 가능"
- 화장실 위치 + 쉼터(벤치) 위치 지도 표시
- 최병호: "화장실 위치도 알려주네, 이거 좋다"

### 4단계: 결정 & 계획
- 수요일 오전 10시로 방문 확정 → "이 날짜로 저장"
- 앱 알림 예약: "수요일 오전 8시 30분에 '창경궁 오늘 혼잡도 알림' 보내드릴까요?" → 허용
- 아들이 일정 저장 + 아내 카카오톡 공유: "수요일 창경궁 오전 10시. 아버지 무릎 때문에 사람 적을 때 가요"
- 이동 경로: 수원 → 종로3가역 (지하철) → 창경궁 정문 (도보 5분)

### 5단계: 현장 + 피드백
- 수요일 오전 8시 30분 앱 알림: "창경궁 오늘 혼잡도 현재 11% — 예측(14%)보다 더 여유롭습니다 ✅"
- 실제 방문: 관광객 수십 명 수준, 벤치에서 충분히 쉬어가며 이동
- 점심까지 여유롭게 관람 완료 → "최근 3년 만에 제일 편한 여행"
- 방문 후 피드백: 아들이 대신 슬라이더 "여유로움" 선택 → +50 포인트
- 최병호: "다음에 또 이걸로 알아보고 가자"

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 혼잡 회피 = 안전 문제와 직결되는 접근성 약자에게 핵심 가치 제공. 경사도·쉼터 정보가 보너스 |
| 사용 편의 | **6** | 아들의 도움 없이 혼자 사용하기엔 여전히 어렵다. 큰 글씨 모드 자동 적용은 좋지만 온보딩 단계 수를 줄여야 함 |
| 구현 타당성 | **8** | 혼잡도 예측은 API로 구현 가능. 경사도·쉼터 데이터는 무장애 여행 API(barrier-free-travel) 추가 연동 필요 |

**종합 점수: 23/30**

## Pass/Fail 판정

> **✅ PASS (조건부)**

최병호는 혼잡 회피 니즈가 안전과 직결되는 고가치 타겟이다. 서비스의 핵심 기능(혼잡도 예측 + 최적 시간 제안)이 이 페르소나에게도 명확한 가치를 제공한다. 그러나 시니어가 **혼자** 사용하기엔 UI 복잡도가 장벽이 되며, 보호자 동반 사용을 전제로 하는 상황이다. 접근성 강화 없이는 이 세그먼트를 완전히 잡기 어렵다.

## 개선 제안

1. **시니어 모드 명시적 추가**: 큰 글씨·높은 대비·3단계 이내 조작 달성을 목표로 하는 시니어 UI 프리셋 (AccessKorea와의 연계 또는 유사 설계 참고)
2. **경사도·접근성 정보 통합**: `barrier-free-travel` API 연동으로 "휠체어·보행 보조기 접근 가능" 뱃지 + 쉼터·화장실 지도 표시 → 시니어·장애인 모두 커버하는 강력한 차별화 포인트
3. **가족 공유 플로우 공식화**: "가족과 함께 계획하기" 기능으로 보호자가 설정 후 링크 공유 → 시니어는 알림만 받는 단순 경험으로 진입 장벽 낮춤
