# SIM-144: TourismCareer Korea × Emma Chen (외국인 20대 백패커)

## 시나리오

에마는 대만에서 프리랜서 사진작가로 활동하며 한국 방문 시 팔로워들에게 숨은 명소를 소개하고 있다. 이번 한국 방문에서 장기 체류나 워킹홀리데이 가능성을 탐색하던 중, 대만 K-culture 커뮤니티 Dcard에서 "한국 관광 관련 외국인 인턴십 정보를 찾을 수 있는 앱"이라는 후기를 보고 TourismCareer Korea를 설치했다. 관광 콘텐츠 크리에이터로서 한국관광공사나 지역 관광 스타트업 인턴십에 관심이 있었다.

---

## 사용자 여정

### 1단계: 진입
- Dcard 후기 → 앱스토어 영문 설명 확인 → 설치
- 앱 실행 시 언어 선택 화면 없음 — 한국어 UI 기본 노출 (즉각적 장벽)
- 구글 번역 앱을 켜서 병행 사용 시작

### 2단계: 행동
- "인턴십" 탭 검색 → 한국어 공고만 노출, 영문 공고 0건
- 관광공사 인턴십 공고 발견 → 지원 자격 조건 "국내 대학 재학생 이상" 명시 → 외국인 지원 불가 확인
- 지역 데이터 분석 페이지 진입 → 한국어 차트·설명으로 내용 파악 어려움
- 현장 탐방 코스 기능만 번역 도움으로 일부 활용 → 전주 탐방 코스 생성 성공, 이 부분만 유용

### 3단계: 결과
- 취업·인턴십 기능은 실질적으로 사용 불가
- 현장 탐방 코스 기능 한 가지만 활용하고 앱 유지하나 주요 앱으로 쓰지 않음
- 인스타그램에 앱 언급 없음

| 항목 | 점수 (1-10) | 근거 |

| 사용 편의 | **2** | 다국어 미지원으로 구글 번역 병행 필수, UX 심각하게 저하 |
| 구현 타당성 | **6** | 외국인 공고 추가 시 개선 가능하나 현재 구조는 내국인 전용 |

## Pass/Fail 판정

> **❌ FAIL**

에마는 관광 크리에이터로서 잠재적 타깃이지만, 외국인 인턴십 공고 부재와 한국어 전용 UI가 이중 장벽으로 작용했다. 유용성 3점은 현장 탐방 코스 기능의 소폭 활용에서 비롯된 것이고, 핵심 취업 기능은 완전히 배제됐다. 외국인 크리에이터·유학생을 타깃에 포함하려면 영문 UI와 외국인 가능 공고 필터가 선행되어야 한다.

## 개선 제안

1. 다국어 UI 지원 (최소 영어): 외국인 사용자 온보딩과 공고 탐색을 영어로 완전 지원하여 잠재 사용층 확대
2. 외국인 가능 공고 필터: 워킹홀리데이·외국인 인턴십 지원 가능 공고를 별도 카테고리로 분류, 해외 관광 크리에이터·유학생 유입 전략 구축
