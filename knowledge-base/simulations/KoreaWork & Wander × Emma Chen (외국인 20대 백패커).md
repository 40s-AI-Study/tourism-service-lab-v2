# SIM-174: KoreaWork & Wander × Emma Chen (외국인 20대 백패커)

## 시나리오

에마는 대만으로 돌아가기 전 한국에서 2주를 더 보낼 수 있게 됐다. 사진 편집 작업이 밀려있어 관광만 하기는 어렵고, 와이파이 좋은 곳에서 일하면서 틈틈이 사진 찍는 "워킹 홀리데이" 스타일을 원한다. 대만 여행 크리에이터 커뮤니티 Dcard에서 "한국 워케이션 앱" 추천 글을 보고 KoreaWork & Wander를 설치한다. 영어 UI가 즉시 작동하고, 전주와 담양이 "워케이션 추천 지역"으로 큐레이션되어 있는 것을 보고 흥미를 느낀다. 담양 글램핑에서 대나무숲 사진을 찍으면서 편집 작업도 하는 계획이 구체화된다.

---

## 사용자 여정

### 1단계: 진입
- Dcard 추천 → 앱 설치 → 영어 UI 자동 감지
- 홈에서 "Recommended Workation Regions" 카드 목록: 제주, 강릉, 전주, 담양 큐레이션
- "담양" 선택 → area-tourism-diversity 지수 ★★★ + 대나무숲·한옥 카페 관광 다양성 정보

### 2단계: 행동
- gocamping API: 담양 글램핑장 목록 중 "와이파이 ✅, 전기 콘센트 ✅" 필터 적용 → 2곳 후보
- 업무 환경 태그 확인: "파워콘센트, 조용한 환경, 고속 인터넷" 모두 충족하는 대나무숲 뷰 글램핑 선택
- area-tourism-resource-demand 분석: 담양 평일 수요 낮음 → 사진 촬영 인파 없는 시간 확인 가능
- 업무 후 관광 추천: 오전 편집 작업 → 오후 죽녹원 포토스팟 → 저녁 담양 국수거리
- 숙소 예약은 외부 링크로 이동 (앱 내 예약 미지원)

### 3단계: 결과
- 담양 글램핑에서 5박, 오전 업무 + 오후 사진 촬영 루틴 완성
- 대나무숲 사진 인스타 게시 → 팔로워 반응 폭발, "담양 워케이션" 콘텐츠 시리즈 제작
- 앱 사용 만족도 높음, 단 예약 연동 부재로 외부 이탈이 불편하다는 리뷰 남김

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **8** | 워케이션 특화 글램핑 필터 + 지역 다양성 지수 조합이 에마의 "일하면서 사진 찍는" 니즈와 잘 맞음. 비주류 지역(담양) 큐레이션이 차별화 포인트 |
| 사용 편의 | **7** | 영어 UI 즉시 작동, 지역 큐레이션 직관적. 프리랜서 크리에이터 패턴과 앱 구조가 잘 호환됨. 예약 연동 부재만 해소되면 완성도 높음 |
| 구현 타당성 | **8** | tourism-info-english + gocamping API 조합은 기술적으로 성숙. 담양 같은 비주류 지역 글램핑 데이터 커버리지 확보가 핵심 변수 |

## Pass/Fail 판정

> **✅ PASS**

KoreaWork & Wander는 에마 같은 프리랜서 여행 크리에이터에게 높은 적합성을 보인다. 영어 지원 + 워케이션 특화 글램핑 필터 + 비주류 지역 큐레이션(담양, 전주)이 결합되어 에마가 원하는 "일하면서 감성적으로 여행하는" 경험을 앱이 직접 지원한다. 예약 연동 추가 시 전환 완결성이 크게 향상된다.

---

## 개선 제안

1. "크리에이터 워케이션 모드" 추가: 사진·영상 제작을 업무로 하는 프리랜서를 위해 촬영 포토스팟 + 업무 환경(조명, 전망, 조용함) 태그를 결합한 특화 필터 제공 — 에마 같은 여행 크리에이터가 핵심 바이럴 전파자가 될 수 있음
2. 앱 내 예약 연동(에어비앤비·Klook 글램핑 예약)을 직접 지원하여 탐색→예약 전환을 앱 안에서 완결 — 에마처럼 여러 앱 전환에 민감한 크리에이터의 이탈 방지
