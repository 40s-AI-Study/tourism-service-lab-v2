# SIM: Foreign AI Guide × 김지원 (한국 20대 솔로)

## 시나리오

김지원(22세, 서울 대학교 언어학과)은 일본인 교환학생 친구 타나카 하루카와 함께 경주 1박 2일 여행을 기획했다. 서울 사람이라 경주를 잘 모르고, 일본어를 못 하는 지원은 하루카와 소통이 불편하다. 친구 추천으로 Foreign AI Guide를 설치했다: "나는 한국어, 하루카는 일본어로 쓰면 AI가 알아서 연결해준대."

---

## 사용자 여정

### 1단계: 진입 (내국인 온보딩)
- 앱 실행 → 언어 선택: **한국어** 선택
- 하루카의 핸드폰에는 同앱 **日本語** 버전 설치
- 그룹 여행 기능: "동행자 초대" → QR 코드로 연결 → 같은 여행 플랜 공유
- 지원 화면: 한국어 UI / 하루카 화면: 일본어 UI → 동일 여행 데이터 실시간 동기화

### 2단계: 경주 탐색
- 지원이 채팅 입력: "경주 불국사 내일 오전에 가려는데 혼잡해?"
- AI 응답 (한국어): 내일 오전 9시 집중률 **31%** (보통), 10시 이후 62% → "9시 오픈과 동시 방문 추천"
- 하루카 화면에는 동일 내용이 **일본어**로 표시: "明日午前9時の混雑率は31%です"
- 불국사 오디오 가이드: 지원은 한국어 버전, 하루카는 일본어 버전 동시 청취 (이어폰)

### 3단계: 한국어 ↔ 일본어 AI 통역 기능
- 지원이 채팅에 한국어로 질문: "하루카한테 오늘 저녁 황리단길 가자고 전해줘"
- AI가 자동 번역 → 하루카 앱에 일본어로 메시지 전달: "今日の夜、ファンリダンギルに行こう！"
- 하루카가 일본어로 답장: "いいね！でも辛い食べ物は苦手なんだけど..." → 지원 앱에 한국어로: "좋아! 근데 나 매운 거 잘 못 먹어..."
- 식당 검색: `searchKeyword2(KOR/JPN)` → 비교적 맵지 않은 황리단길 음식점 필터링

### 4단계: 현장 가이드
- 불국사 도착 → GPS 기반 오디오 가이드 자동 실행
- 다석가탑 앞: 지원이 앱으로 역사 설명 들으며 하루카에게 설명할 내용 파악
- 하루카는 동시에 일본어 오디오로 청취 → 언어 장벽 없이 동일 체험
- 연관 관광지: 석굴암, 첨성대, 교촌마을 추천 카드 → 지원과 하루카가 함께 다음 코스 투표 기능

### 5단계: 결과
- 9시 방문으로 불국사 인파 없이 완벽 관람
- 하루카: "한국 여행 앱인데 일본어도 완벽해서 놀랐어" → 일본 여행 커뮤니티에 후기
- 지원: "외국인 친구랑 여행할 때 완전 필수앱" → 인스타 스토리 공유
- 동반 여행자 경험 NPS 9.4/10

| 구현 타당성 | **8** | 다국어 API 실존, 그룹 기능은 추가 개발 필요 (서버-side 플랜 동기화) |

## Pass/Fail 판정

> **✅ PASS (고가치 내국인 세그먼트)**

내국인이 Foreign AI Guide를 사용하는 "동반 여행 시나리오"는 강력한 확산 채널이다. 외국인 친구·교환학생·비즈니스 파트너와 함께하는 내국인이 앱을 적극 추천하는 구조로, 한국인 유저 기반이 외국인 유입을 견인한다. 이 시나리오가 성립하면 MAU 중 내국인 비중 30-40% 확보 가능.

## 개선 제안

1. **그룹 여행 모드 강화**: 동행자 간 실시간 채팅·투표·일정 합의 기능 → 내국인+외국인 동반 여행 특화
2. **언어 자동 감지**: 사용자가 입력하는 언어를 자동 감지해 UI 전환 → 언어 설정 불필요
3. **내국인 UX 강화**: 다국어 기능 외 국문 여행 정보 품질도 경쟁 앱 수준 이상 유지
4. **교환학생 커뮤니티 연계**: 대학 교환학생 프로그램 파트너십으로 초기 외국인 유저 확보
