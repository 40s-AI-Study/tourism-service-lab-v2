# SIM-161: KoreaFestival Live × 김지원 (한국 20대 솔로)

## 시나리오

금요일 밤 11시, 지원은 인스타그램 릴스에서 "진주남강유등축제" 영상을 본다. 형형색색 등불이 강 위를 수놓는 장면에 바로 "이거다" 싶어 주말 당일치기를 결심한다. 평소 유명한 곳에 갔다가 사람 때문에 실망한 경험이 많아, 이번엔 출발 전에 혼잡도를 꼭 확인하고 싶다. KoreaFestival Live 앱을 검색해서 설치하고, 축제 페이지에서 "토요일 오후 7시 방문자 집중률 92%"를 확인한다. 대신 일요일 오전 11시는 48%로 한산하다는 데이터를 보고 일정을 조정한다. 예산 3만원 안에서 축제 + 주변 맛집 코스까지 자동으로 생성되는 기능에 "이런 앱이 필요했어"라고 감탄한다.

---

## 사용자 여정

### 1단계: 진입
- 인스타그램 릴스 → 앱스토어 검색 "한국 축제 앱" → KoreaFestival Live 설치 (약 3분 소요)
- 첫 화면에서 "이번 주 인기 축제" 카드가 뜨며 진주남강유등축제가 상단 노출
- 달력 뷰로 전환해 주말 축제 전체 목록 확인, 지도 뷰로 경남 지역 클러스터 파악

### 2단계: 행동
- 진주남강유등축제 상세 페이지 진입 → 혼잡도 그래프에서 토요일 저녁 92% vs. 일요일 오전 48% 비교
- "최적 방문일 추천" 버튼 탭 → 앱이 "일요일 오전 10시~12시 방문 추천" 자동 제안
- "당일 3시간 코스 생성" 탭 → 유등 전시 관람 → 진주 비빔밥 로컬 식당 → 촉석루 포토스팟 자동 코스 생성
- 지역 다양성 지수 ★★★★ 확인 — "축제 외에도 볼거리 다양" 안내로 체류 결정 확정

### 3단계: 결과
- 일요일 오전 10시 출발, 혼잡도 낮은 시간대에 여유롭게 포토스팟 확보
- 앱 추천 식당에서 진주비빔밥 1만 2천원, 총 지출 2만 8천원으로 예산 내 완료
- 인스타그램에 사진 게시 후 "앱 추천대로 했더니 사람 없었음" 스토리 공유 → 전환 완료

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 혼잡도 예측으로 핵심 페인포인트(현장에서야 알게 되는 혼잡) 직접 해결. 당일 코스 자동 생성은 예산 맞춤 여행에 완벽 부합 |
| 사용 편의 | **8** | 앱스토어 검색부터 코스 생성까지 10분 내 완료. 달력·지도 이중 뷰는 직관적. 신규 앱 거부감 없는 디지털 네이티브에게 최적 |
| 구현 타당성 | **8** | visitor-concentration-forecast + related-attractions API 조합은 기술적으로 구현 가능. 축제 공식 데이터와 빅데이터 연계가 핵심 |

**종합 점수: 25/30**

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live는 김지원의 가장 큰 불만인 "현장 가서야 혼잡도를 안다"는 문제를 visitor-concentration-forecast API로 정확히 해소한다. 혼잡도 데이터 기반 최적 방문 시간 추천과 예산 내 당일 코스 자동 생성이 결합되어 이 페르소나에게 매우 높은 전환 가치를 제공한다. 솔로 여행자의 즉흥적 결정 패턴과도 잘 맞는다.

## 개선 제안

1. 인스타그램 공유 버튼을 앱 내 코스 완료 화면에 직접 삽입하여 "공유 전환" 루프를 강화할 것 — 지원 같은 크리에이터형 사용자의 바이럴 효과를 극대화
2. "예산 설정 → 코스 자동 필터" 기능 추가: 2만원/3만원/5만원 버튼으로 축제 주변 맛집·카페 추천 범위를 자동 조정하면 가성비 여행자 정착률 향상
