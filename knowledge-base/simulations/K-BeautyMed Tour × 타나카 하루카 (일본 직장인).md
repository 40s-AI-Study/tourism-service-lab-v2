# SIM-130: K-BeautyMed Tour × 타나카 하루카 (일본 직장인)

## 시나리오

하루카는 X(트위터) 일본 한국 여행 계정(@koreatabi_jp)에서 "韓国の美容医療ツアーアプリ、これが最強"라는 게시글을 보고 K-BeautyMed Tour를 발견했다. 일본어를 지원한다는 점이 결정적이었다. 3박 4일 서울 여행에서 명동 쇼핑과 함께 피부 레이저나 피부과 상담을 받아보고 싶었다. 일본에서 한국 피부과가 저렴하고 효과적이라는 소문을 들어왔고, 이번이 첫 방한이라 일본어 안내가 있다는 점이 불안감을 크게 줄여줬다. 앱의 시술 후 D+1 실내 쇼핑 코스가 일정 계획에 그대로 적용 가능했다.

---

## 사용자 여정

### 1단계: 진입
- X(트위터) 일본어 한국 여행 계정 추천 → 일본어 UI 확인 후 즉시 설치
- 첫 화면 일본어 자동 전환 — 첫 방한 여행자의 불안감 즉시 해소
- "피부과 + 쇼핑 + 포토스팟" 조합이 자신의 여행 계획과 정확히 일치

### 2단계: 행동
- `tourism-info-japanese` API 기반 피부과 클리닉 목록 일본어 안내 탐색 → 강남 3개 클리닉 비교
- 피부 상담 패키지 예약: 일본어 안내문 → 예약 절차 이해 용이. 다음날 오전 예약 완료
- `visitor-concentration-forecast` 활용: 시술 당일 오후 명동 혼잡도 확인 → 오후 3시 방문 계획 수립
- 시술 후 D+1 코스 탐색: 올리브영·압구정 쇼핑 → 붓기 상태에서 가벼운 실내 코스로 적합
- `tourism-photos` 기반 포토스팟 목록 → 한복 렌탈샵 위치 확인, X 게시용 인증샷 계획 수립

### 3단계: 결과
- 피부과 상담 예약 완료 및 방문. 시술 후 코스를 앱 추천대로 진행. X에 "K-BeautyMed Tour使ってみた！韓国美容旅行完璧でした" 게시 → 리트윗 350건, 앱 언급 파급 효과 발생

## Pass/Fail 판정

> **✅ PASS**

타나카 하루카는 K-BeautyMed Tour의 이상적인 코어 타겟 페르소나다. 일본어 지원, 피부과 시술 관심, K-뷰티 쇼핑 동기, SNS 공유 문화가 앱의 모든 핵심 기능과 완벽하게 정렬된다. 시술 예약부터 시술 후 코스 활용, SNS 바이럴까지 전체 사용자 여정이 성공적으로 완결되었으며 자연스러운 앱 홍보 효과도 발생했다.
