# SIM-101: EuroKorea Gateway × 김지원 (한국 20대 솔로)

## 시나리오

지원은 인스타그램에서 우연히 EuroKorea Gateway 광고를 접한다. 피드에 올라온 프랑스어권 여행자의 한국 감성 사진들이 눈길을 끌었지만, 앱을 열자마자 UI가 러시아어·스페인어·프랑스어·독일어 중 하나를 선택하라는 화면이 뜬다. 한국어 옵션이 없다. 앱의 타깃이 유럽어권 외국인 관광객임을 파악한 지원은 "아, 이건 나한테 맞는 앱이 아니구나"라고 판단하고 30초 만에 이탈한다. 예산 맞춤 코스, 혼잡도 회피, 국내 감성 포토스팟 큐레이션이 필요한 지원에게 EuroKorea Gateway는 처음부터 잘못된 접점이다.

---

## 사용자 여정

### 1단계: 진입
- 인스타그램 광고 또는 앱스토어 노출로 접근
- 앱 설치 후 실행 — 언어 선택 화면에서 한국어가 없음을 인지
- "외국인용 앱인가?" 의문 → 한국어 콘텐츠·혼잡도 기능 없음 확인

### 2단계: 행동
- 4개 유럽어 중 영어에 가장 가까운 스페인어로 시도해보지만 UI가 완전 스페인어라 탐색 불가
- 앱스토어 리뷰를 다시 확인하니 "For European tourists visiting Korea" 설명 확인
- 지원이 원하는 국내 당일 코스, 혼잡도, 포토스팟 큐레이션 기능 전혀 없음

### 3단계: 결과
- 앱 삭제. 카카오맵과 인스타그램으로 복귀
- 유럽어권 외국인 관광객 전용 서비스로 한국인 사용자는 실질적 가치 없음

---

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **1** | 한국인 내국인 여행자에게 전혀 유용하지 않음. 타깃 미스매치 |
| 사용 편의 | **3** | UI 자체는 비주얼 갤러리 위주라 직관적이나 한국어 없어 탐색 불가 |
| 구현 타당성 | **7** | 서비스 자체는 잘 구현될 수 있으나 이 페르소나에겐 무관 |

**종합 점수: 11/30**

---

## Pass/Fail 판정

> **❌ FAIL**

EuroKorea Gateway는 유럽어권 외국인 관광객 전용 서비스로, 한국인 내국인 사용자(김지원)에게는 언어 장벽부터 시작해 콘텐츠·기능 모든 면에서 가치를 제공하지 못한다. 타깃 페르소나 자체가 서비스 대상과 완전히 불일치한다. 종합 점수 11점, 유용성 1점으로 FAIL 기준 충족.

---

## 개선 제안

1. 앱 설명 및 스토어 페이지에 "유럽어권 외국인 전용" 표기를 명확히 하여 한국인 사용자의 불필요한 이탈을 방지할 것
2. 장기적으로 한국인 사용자가 유럽어권 여행자를 위한 가이드 역할을 하는 "현지인 팁 기여" 기능을 추가하면 내국인 참여를 유도할 수 있음
