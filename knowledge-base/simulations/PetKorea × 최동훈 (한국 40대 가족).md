# SIM-33: PetKorea × 최동훈 (한국 40대 가족)

## 시나리오

2026년 7월 말, 여름방학을 맞아 최동훈(44세, 중학교 교사)은 가족 여행을 계획한다. 구성원은 아내, 초등학생 자녀 2명(8세·10세), 70대 부모님, 그리고 4살 된 골든 리트리버 '황금이'다. 2박3일 캠핑이 목표지만 조건이 복잡하다. 부모님이 함께하므로 시니어 접근성이 필요하고, 아이들은 물놀이를 원하며, 황금이는 대형견이라 허용 숙박지가 제한적이다.

동훈은 평소 여행 앱을 잘 쓰지 않지만 아내가 PetKorea를 추천해 설치한다. 앱에서 황금이 프로필(골든 리트리버, 28kg)을 등록하자 대형견 허용 캠핑장만 필터링된 결과가 뜬다. 예산은 3박 총 40만원 이내(숙박 기준), 이동은 SUV 자가용이다. 부모님 동반임을 체크하자 "접근성 우수" 필터가 활성화된다.

AI 코스 생성기에 조건(대형견/자가용/2박3일/가족+시니어/충청도 선호)을 입력하자 충남 태안 캠핑 코스가 1순위로 제안된다. 황금이가 뛸 수 있는 해변, 아이들을 위한 수상 레포츠, 노부모가 쉴 수 있는 그늘 쉼터까지 묶인 하루 일정표가 나온다.

---

## 사용자 여정

### 1단계: 진입
- 아내 권유로 앱 설치, 황금이 프로필 등록 (골든 리트리버, 28kg, 중성화 완료)
- 가족 구성 설정: 성인 4명 + 어린이 2명 + 시니어 2명 체크
- 메인 화면 "AI 코스 만들기" → 조건 입력 시작

### 2단계: 행동
- `pet-friendly-travel` API (카테고리: 캠핑/야영장) → 대형견(25kg 이상) 허용 필터 적용, 시니어 접근성 우수 필터 추가
- AI 코스 생성: 태안 해변 국립공원 인근 캠핑장 (황금이 반려견 전용 구역 보유) 1순위 추천
- `pet-friendly-travel` API (카테고리: 레포츠) → 아이 동반 수상 레포츠 + 반려견 동반 해변 접근 가능 여부 확인
- `visitor-concentration-forecast` API → 여름 성수기 태안 혼잡도 예측: 7월 마지막 주 금요일 "매우 혼잡" → 목요일 출발 권장 알림 수신
- `tourism-info-korean` API → 캠핑장 시설 정보(화장실 거리, 전기 사용 가능, 그늘막 대여) 확인
- `related-attractions` API → 캠핑장 반경 15km 반려견 동반 가능 수산시장·해산물 식당 추천
- 목요일 출발로 일정 조정, 캠핑장 예약 외부 링크 저장

### 3단계: 결과
- 2박3일 태안 가족 캠핑 완료. 황금이 첫 해변 경험, 아이들 수상 레포츠 만족
- 목요일 출발로 혼잡 피해 캠핑장 여유 있게 이용
- 부모님은 그늘 쉼터에서 황금이와 함께 휴식, 접근성 우수 정보 정확했다는 긍정 평가
- 불만족: 대형견 캠핑장 선택지가 3곳으로 적었고, 시니어 접근성 세부 정보(경사도·보행로 포장 여부)가 부족
- 만족: 성수기 혼잡도 사전 예측으로 목요일 출발 결정이 탁월한 선택이 됨

## 개선 제안

1. **대형견 등록 캠핑장 확대**: 파트너십을 통해 대형견(20kg 이상) 허용 캠핑장 DB를 현재 대비 3배 이상 확보, 선택지 다양화
2. **시니어 접근성 세부 데이터**: 경사도, 보행로 포장 여부, 화장실 거리 등 시니어 특화 접근성 정보를 숙박·캠핑 상세 페이지에 추가
3. **가족 구성원별 일정 분기**: 아이/시니어/반려동물 각각의 활동을 병렬로 보여주는 "분기형 일정표" UI로 복잡한 가족 여행 계획 지원
