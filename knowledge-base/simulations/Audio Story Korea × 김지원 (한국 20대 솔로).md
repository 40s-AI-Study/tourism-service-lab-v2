# SIM-61: Audio Story Korea × 김지원 (한국 20대 솔로)

## 시나리오

김지원(25세, UX 디자이너)은 주말을 맞아 혼자 경복궁을 방문했다. 역사 공부보다는 인스타그램에 올릴 감성 사진을 찍는 것이 주목적이다. 근정전 앞 QR 코드를 우연히 발견하고 스캔했다. 처음에는 오디오 가이드에 별 흥미가 없었지만, 앱 실행 후 화면에 사진 공모전 수상작 갤러리가 슬라이드쇼로 펼쳐지자 멈춰서 스크롤하기 시작했다. "이 사진 구도 진짜 예쁘다"며 동일한 앵글을 찾아 이동하고, 자연스럽게 오디오 해설이 배경으로 흘러나오는 상황을 경험했다.

---

## 사용자 여정

### 1단계: 진입
- 근정전 입구 QR 스캔 → Audio Story Korea 앱 실행
- 첫 화면에서 오디오 가이드 시작 버튼보다 사진 갤러리 썸네일에 먼저 눈이 감
- "공모전 수상작 보기" 탭을 먼저 탭

### 2단계: 행동
- 수상작 사진 슬라이드쇼 감상 중 오디오가 자동 재생 시작
- 특정 사진 속 촬영 위치 태그 확인 → 실제 포토스팟으로 이동
- 오디오 해설 중 "이 기둥의 단청 문양은 조선 초기 양식" 내용에 관심 → 해당 기둥 클로즈업 촬영
- 챕터 목록에서 "단청과 문양" 챕터만 선택적으로 청취
- 사진 갤러리 3장 저장, 해당 위치에서 본인 사진 촬영

### 3단계: 결과
- 오디오 가이드 전체 청취 비율: 약 20% (선택적 챕터 청취)
- 방문 시간 +30분 연장 (포토스팟 탐색)
- 인스타그램 스토리에 수상작 갤러리 화면 캡처 공유
- 앱 재방문 의향: 다른 문화재 포토스팟 탐색 목적으로 긍정적

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **6** | 오디오 가이드 자체 활용도는 낮지만, 사진 갤러리가 방문 동선과 체류 시간을 실질적으로 늘림 |
| 사용 편의 | **8** | QR 스캔 즉시 실행, 갤러리 우선 진입 경로가 이 페르소나에게 자연스러운 흐름 제공 |
| 구현 타당성 | **8** | photo-contest-winners API + 위치 태그 기능은 구현 가능하며 포토스팟 탐색 UX와 연동 용이 |

**종합 점수: 22/30**

## Pass/Fail 판정

> **✅ PASS**

유용성 6점으로 기준선 충족, 종합 22점으로 18점 이상 달성. 핵심 오디오 가이드 소비는 낮지만 photo-contest-winners API 연동 갤러리가 이 페르소나의 주요 가치 동인으로 작동한다. 감성·포토스팟 중심 20대 솔로 여행자에게도 앱이 유의미한 경험을 제공함을 확인.

---

## 개선 제안

1. **포토스팟 모드 별도 진입 경로 제공**: 첫 화면에서 "오디오 가이드"와 "포토 갤러리" 탭을 병렬로 배치하여 이 페르소나가 원하는 경로로 즉시 진입할 수 있게 설계
2. **사진과 오디오 연동 강화**: 갤러리 사진 탭 시 해당 장면의 오디오 클립(30초 내외)이 자동 재생되는 "사진별 오디오 해설" 기능 추가로 자연스러운 오디오 소비 유도
