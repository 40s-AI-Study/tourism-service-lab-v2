# SIM-151: PhotoHunt Korea × 김지원 (한국 20대 솔로)

## 시나리오

지원은 금요일 밤 인스타그램에서 "강릉 숨은 카페" 릴스를 보고 다음날 당일치기를 결심했다. 평소 포토스팟을 찾을 때마다 SNS 정보가 광고성이거나 사람이 너무 많아 실망하는 일이 반복됐다. 친구가 "PhotoHunt Korea 진짜 좋아, 혼잡도까지 알려줘"라고 카카오톡으로 링크를 공유하자 바로 설치했다. 그래픽 디자이너답게 앱의 비주얼 퀄리티도 함께 평가하며 탐색을 시작했다.

---

## 사용자 여정

### 1단계: 진입
- 카카오톡 친구 공유 링크 → 앱스토어 4.6점 확인 → 즉시 설치
- 온보딩: "감성 카페", "해변·자연", "야경" 관심 카테고리 선택 — 30초 완료
- 메인 화면: 강릉 공식 포토 스팟 지도 + 오늘의 추천 스팟 카드 — "UI 예쁘다" 첫 반응

### 2단계: 행동
- 강릉 검색 → `tourism-photos` + `photo-contest-winners` 기반 공식 인증 스팟 12곳 지도 표시
- 안목해변 카페 거리 선택 → `visitor-concentration-forecast`: "내일 오전 9-10시 방문자 15%, 오후 1시 72%" — 오전 일찍 출발 결정
- 포토 투어 코스 생성: 안목해변 → 강릉 중앙시장 → 오죽헌 3코스 자동 생성, 이동 시간 포함
- 각 스팟 공모전 수상작 사진과 "이 각도, 이 시간대에 찍으면 비슷하게 나와요" 촬영 팁 확인
- SNS 공유 기능: 방문 후 앱 내에서 위치·해시태그 자동 생성 → 인스타그램 바로 연동

### 3단계: 결과
- 다음날 오전 9시 안목해변 도착, 한산한 타이밍에 원하는 사진 촬영 성공
- 인스타그램 게시물 좋아요 평소 대비 2.3배 달성
- 앱 즐겨찾기 추가, 친구 3명에게 추가 공유
- 일주일 후 제주 당일치기 계획에 동일하게 활용

## Pass/Fail 판정

> **✅ PASS**

김지원은 PhotoHunt Korea의 핵심 타깃 페르소나와 완벽히 일치한다. 인스타그램 활용, 포토스팟 탐색, 혼잡도 회피, SNS 공유라는 그녀의 모든 여행 패턴이 앱의 기능과 1:1 매핑됐다. 실사용 → 성공 경험 → 자발적 공유로 이어지는 바이럴 루프가 작동하는 이상적인 사례다.

## 개선 제안

1. 인스타그램 게시물 성과 트래킹 연동: 앱 내에서 공유한 게시물의 좋아요·도달 수를 확인할 수 있는 간단한 성과 대시보드 추가 (크리에이터 유저 락인 강화)
2. 친구와 포토 투어 코스 공유 기능: 생성한 코스를 카카오톡·인스타 스토리로 공유하여 그룹 여행 계획 도구로 확장
