# SIM-04: KoreaPath AI × Emma Chen (외국인 20대 백패커)

## 시나리오

Emma Chen(23세, 대만 여행 크리에이터)은 전주 한옥마을 여행을 계획 중이다. 인스타에서 한국 여행 정보를 찾다가 KoreaPath AI 광고를 보고 설치한다. 한국어를 거의 못해 구글 번역기에 의존하는 상황.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 언어 설정 화면 없음, 기본 언어 한국어
- 구글 번역기로 앱 전체를 번역하며 조건 입력 시도: **"전주, 당일, 혼자, 포토스팟, 1일"**
- 입력 소요 시간: 약 4분 (번역 반복으로 지연)

### 2단계: 행동
- 코스 생성은 성공: 한옥마을 포토스팟 루트 + 전통 음식 코스
- 혼잡도 데이터는 볼 수 있으나 설명 텍스트가 전부 한국어 → 배지 색상(🟢/🟡/🔴)으로 겨우 파악
- 각 장소 설명: 한국어 전용 → 구글 번역기 재실행 필요
- 오디오 가이드: 없음 (KoreaPath AI 미지원)
- 코스 공유 기능: 링크 복사 → 대만 Dcard에 게시 (부분 성공)

### 3단계: 결과
- 혼잡도 배지 덕분에 오전 9시에 도착해 한적한 사진 촬영 성공
- 그러나 각 장소의 문화적 맥락(배경, 역사)을 이해하지 못한 채 사진만 찍고 이동
- 번역기 의존으로 앱 UX 만족도 낮음
- 대만 친구에게 "혼잡도는 좋은데 한국어 전용이라 불편해"라고 리뷰

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **5** | 혼잡도 데이터와 코스 생성은 가치 있으나, 언어 장벽으로 절반만 활용 |
| 사용 편의 | **3** | 한국어 전용 UI는 외국인에게 심각한 장벽. 번역기 병행 사용이 필수 |
| 구현 타당성 | **8** | i18n(다국어화)만 추가하면 즉시 해결 가능한 기술적 문제 |

## Pass/Fail 판정

> **❌ FAIL**

P4 Emma는 KoreaPath AI의 핵심 기능(혼잡도, 코스 생성)이 필요한 사용자지만, 한국어 전용 UI가 결정적 장벽이다. 다국어 지원 없이는 외국인 FIT 여행자에게 실질적 서비스 불가.

## 개선 제안

1. **다국어 UI 지원 (필수)**: 영어·중국어(번체/간체)·일본어 최소 4개 언어 → P4 즉시 PASS 전환 가능
2. **언어 자동 감지**: 기기 언어 설정 기반 자동 전환 (초기 언어 선택 화면 추가)
3. **오디오 가이드 연동**: 각 장소에서 중국어 번체 음성 해설 제공 시 Emma의 콘텐츠 품질도 향상
4. **SNS 친화 기능**: 포토스팟 최적 시간대 + 촬영 팁을 다국어로 제공하면 크리에이터 유입 증가
