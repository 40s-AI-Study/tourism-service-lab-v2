# SIM-149: TourismCareer Korea × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

아미나는 쿠알라룸푸르의 디지털 마케팅 크리에이티브 디렉터로, 한국 여행 인플루언서이기도 하다. 여행 중 한국 관광 업계의 무슬림 친화 현황이 궁금해졌고, 혹시 무슬림 여행 마케팅 관련 한국 기업과 협업 기회가 있는지 탐색하고 싶었다. 한국어 무슬림 여행 커뮤니티 카카오톡 그룹에서 누군가 TourismCareer Korea를 공유했고, "관광 마케팅 직군 공고가 있다"는 말에 설치했다.

---

## 사용자 여정

### 1단계: 진입
- 카카오톡 그룹 링크 → 앱스토어 설치
- 앱 실행: 한국어 전용 UI — 영어 능숙하나 한국어 앱 탐색에 구글 번역 병행
- 온보딩에서 "관심 직무: 마케팅" 선택 (번역 도움 없이 단어 인식 가능)

### 2단계: 행동
- 마케팅 직군 공고 필터 → 10건 노출 (관광 마케터, SNS 운영, 콘텐츠 기획)
- 외국인 지원 가능 여부 확인 시도 → 지원 자격에 "내국인 우선" 또는 "국내 거주자" 명시
- "무슬림 여행 전문" 또는 "할랄 관광" 관련 공고 검색 → 0건
- 지역 관광 수요 데이터 탐색 → 무슬림 관광객 관련 데이터 없음 확인
- 앱 내에서 자신의 인플루언서 역량을 활용할 파트너십 기회를 찾으려 했으나 해당 기능 없음

### 3단계: 결과
- 취업 목적 달성 불가 (외국인 지원 불가, 무슬림 관광 특화 공고 없음)
- 앱 삭제, "한국 관광 업계가 무슬림 시장에 아직 준비 안 됐다"는 인상 강화
- 팔로워에게 앱 소개 없음

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 외국인 지원 불가, 무슬림 특화 공고 전무로 핵심 니즈 완전 미충족 |
| 사용 편의 | **3** | 한국어 전용 UI, 마케팅 단어 정도만 인식 가능, 실질적 탐색 어려움 |
| 구현 타당성 | **5** | 앱 구현은 타당하나 무슬림 관광 시장 특화 기능은 설계 단계에서 고려 안 됨 |

**종합 점수: 10/30**

---

## Pass/Fail 판정

> **❌ FAIL**

아미나는 이 앱의 완전한 비타깃 사용자다. 외국 거주자에 대한 구조적 장벽(국내 거주자 우선 채용)과 무슬림 관광 특화 기능 부재가 겹쳐 실질적 가치가 없었다. 무슬림 친화 관광 시장은 한국 관광 업계의 성장 분야임에도 채용 공고와 지역 데이터 모두 이를 반영하지 않는다는 점은 앱을 넘어 한국 관광 산업의 과제를 드러낸다.

---

## 개선 제안

1. 무슬림 관광 특화 직군 공고 수집: 할랄 인증 여행사, 무슬림 친화 호텔, 동남아 관광 마케팅 분야 공고를 별도 카테고리로 신설
2. 해외 인플루언서 협업 파트너십 섹션: 취업 외에 관광 콘텐츠 크리에이터와 한국 관광 기업 간 협업 매칭 기능 추가, 아미나 같은 인플루언서 유입 경로 창출
