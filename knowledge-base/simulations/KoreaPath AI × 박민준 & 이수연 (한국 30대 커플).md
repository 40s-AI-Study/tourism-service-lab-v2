# SIM-02: KoreaPath AI × 박민준 & 이수연 (한국 30대 커플)

## 시나리오

교제 1주년이 3주 앞으로 다가왔다. 이수연(31세, 마케터)은 퇴근 후 소파에 앉아 KoreaPath AI를 열고 "강원도 1박 2일, 커플, 로맨틱+웰니스, 예산 30만원"을 입력한다. 평소엔 박민준이 숙소·맛집·동선을 각각 검색하며 주말 저녁 2-3시간을 소비했지만, 이번엔 그 시간을 아끼고 싶다.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 조건 입력: **"강원도, 1박 2일, 커플 2인, 로맨틱+스파, 예산 30만원"**
- 입력 소요 시간: 약 50초

### 2단계: 행동
- AI가 `related-attractions` + `wellness-tourism` + `visitor-concentration-forecast` 조합
- 생성된 코스:
  ```
  [1일차]
  14:00 평창 오대산 자작나무숲 (포토스팟, 혼잡도 낮음)
  17:00 펜션 체크인 (독채풀빌라 추천 3곳, 예약 가능 여부 표시)
  19:00 오마카세 레스토랑 (혼잡도 낮은 시간대 19:00 추천)

  [2일차]
  09:00 스파/온천 체험 (웰니스 API 연동 시설)
  12:00 현지 로컬 브런치
  14:00 귀가
  ```
- 예상 지출 내역: 숙박 18만원 / 식비 7만원 / 체험 3만원 / 교통 2만원 = 총 30만원
- 민준이 "숙소 예약하기" 버튼 터치 → 외부 예약 사이트 링크로 연결

### 3단계: 결과
- 코스 기획 소요 시간: 약 3분 (기존 2-3시간 대비 획기적 단축)
- 1주년 여행 완료, 두 사람 모두 만족
- 민준: "왜 이걸 이제야 알았지" 발언 → 앱 5점 리뷰 등록

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 코스 기획 부담 해소가 핵심 Pain Point. 예산 분배 자동화도 강점 |
| 사용 편의 | **9** | 50초 입력 → 완성 코스는 P2의 "완성된 플랜" 니즈에 완벽 부합 |
| 구현 타당성 | **7** | 숙소·식당 예약 직접 연동은 추가 파트너십 필요, MVP에선 외부 링크로 충분 |

**종합 점수: 25/30**

## Pass/Fail 판정

> **✅ PASS**

P2 커플의 가장 큰 Pain Point인 "코스 기획 부담(주말 저녁 2-3시간 소모)"을 3분으로 단축시킨다. 예산 관리·웰니스 연동·혼잡도 타이밍 모두 커버. 30대 직장인 커플에게 강력한 가치 제안.

---

## 개선 제안

1. **예약 원스톱 연동**: 외부 링크가 아닌 앱 내 예약(숙소·식당)으로 전환 시 컨버전율 대폭 향상 예상
2. **기념일 모드**: "100일/1주년/생일" 선택 시 해당 테마 특별 코스 + 꽃배달·케이크 픽업 연계
3. **폐업 데이터 동기화**: 추천 맛집 최근 3개월 이내 영업 여부 실시간 확인 (P2 Pain Point 직결)
