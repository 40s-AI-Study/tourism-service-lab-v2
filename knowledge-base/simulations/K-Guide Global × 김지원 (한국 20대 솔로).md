# SIM-06: K-Guide Global × 김지원 (한국 20대 솔로)

## 시나리오

김지원(25세)이 강릉 당일치기를 검색하다가 K-Guide Global 앱을 우연히 발견한다. 앱 설명에 "공식 Visit Korea 데이터 + 혼잡도" 문구를 보고 설치한다. 한국인이지만 혼잡도 정보에 매력을 느낌.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 언어 선택 화면: "한국어" 선택
- 메인 화면이 외국인 FIT 여행자 중심으로 설계되어 있음 (영어·한자 혼재, 외국인 관광객 동선 중심)
- "강릉" 검색 → 결과 나옴

### 2단계: 행동
- 강릉 주요 관광지 목록 확인 가능 (공식 Visit Korea 데이터)
- 혼잡도 배지 확인: 안목해변 🟢 한산 표시
- 그러나 추천 코스가 "외국인 베스트 코스" 위주 → 지원이 원하는 "로컬 감성 카페" 정보 부족
- 오디오 가이드: 경복궁·불국사 등 대형 관광지만 지원, 강릉 소규모 스팟 미지원
- 앱 전체 UX가 외국인 관광객 기준으로 설계 → 한국인 사용자에게 이질감

### 3단계: 결과
- 혼잡도 정보는 유용하게 사용
- 그러나 로컬 맛집·감성카페 큐레이션은 네이버·인스타에 훨씬 미치지 못함
- "외국인 여행 앱 느낌이라 나한텐 잘 안 맞아" → 앱 삭제
- 카카오맵으로 복귀

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **4** | 혼잡도 배지는 유용하나 전체 서비스가 외국인 맞춤이라 한국인 니즈 불충족 |
| 사용 편의 | **5** | 한국어 UI는 있지만 콘텐츠 큐레이션 방향이 다름 |
| 구현 타당성 | **9** | 앱 자체는 정상 작동, 타겟 미스매치 문제 |

**종합 점수: 18/30**

---

## Pass/Fail 판정

> **❌ FAIL**

K-Guide Global은 외국인 방문객을 위한 서비스로 설계된 앱이다. P1 김지원은 타겟 사용자가 아니며, 한국인 내수 여행자의 니즈(로컬 큐레이션, 감성 콘텐츠)를 충족하지 못한다. 타겟 미스매치가 핵심 실패 원인.

---

## 개선 제안

1. **타겟 명확화**: 앱스토어 설명 및 마케팅에서 "외국인 방문객 전용" 명시 → 한국인 사용자 유입 자체를 차단해 별점 하락 방지
2. **혼잡도 위젯**: 한국인 사용자도 원하는 혼잡도 데이터를 별도 소형 위젯으로 분리 제공하면 교차 사용 가능
3. **(중장기) 한국인 모드 추가**: 내국인 여행자 전용 탭 신설 시 시장 확장 가능하나 MVP 단계에서는 불필요
