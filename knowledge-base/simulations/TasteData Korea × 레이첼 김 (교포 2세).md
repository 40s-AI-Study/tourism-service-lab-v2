# SIM-198: TasteData Korea × 레이첼 김 (교포 2세)

## 시나리오

레이첼은 전주 방문을 앞두고 부모님이 어린 시절 먹던 음식을 딸에게도 먹이고 싶어 "전주 로컬 음식" 정보를 영어로 찾다가 TasteData Korea를 발견한다. "신용카드 데이터 기반 현지인 실제 소비 맛집"이라는 설명이 단순 관광 안내가 아닌 진짜 로컬 정보를 제공한다는 신호로 읽힌다. 교포 2세로서 "부모님이 살던 전주의 진짜 음식 문화"를 딸에게 보여주고 싶은 감정적 동기가 강하다. 영어 UI로 전주를 검색하면 비빔밥·막걸리·피순대 소비 강도 지역이 표시된다.

---

## 사용자 여정

### 1단계: 진입
- 교포 커뮤니티 블로그(KoreAm) 링크 → 설치 → 영어 전환
- 전주 검색 → 현지인 소비 히트맵에서 관광 지역(한옥마을)과 현지인 실소비 지역(삼천동·서신동) 분리
- "이게 진짜 전주 사람들이 가는 곳이구나" → 감정적 공명 즉각 발생

### 2단계: 행동
- 전주 지역 특산 음식 탭: 비빔밥·콩나물국밥·막걸리·피순대 데이터 확인
- 영어 음식 설명: 각 음식의 역사적 배경 및 전주 특유의 조리법 설명 → 딸에게 설명할 내용 바로 확보
- 삼천동 막걸리 타운 탭: 현지인 고소비 지역 확인 → "부모님이 말씀하시던 그 동네 분위기겠구나"
- 음식 코스 생성: 점심 비빔밥 → 막걸리 타운 저녁 → 아침 콩나물국밥 3일 코스 저장
- 10세 딸을 위한 음식 설명이 영어로 제공 → "엄마, 비빔밥이 뭐야?"에 바로 답할 수 있음

### 3단계: 결과
- 삼천동 막걸리 타운 방문 → 관광객 없는 진짜 현지 분위기 → 레이첼 감동
- 딸에게 "할머니 할아버지가 이런 곳에서 자랐어"라고 설명하며 뿌리 교육 완성
- 앱 평가: "이 앱이 없었다면 한옥마을 비빔밥 식당에서 끝났을 것. 데이터가 진짜 전주를 보여줬다"

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 현지인 소비 데이터 기반 로컬 음식 문화 발굴이 교포 2세의 "진짜 한국" 탐색 니즈에 정확히 부합. 영어 음식 설명이 딸 교육에도 직접 활용 |
| 사용 편의 | **8** | 영어 UI 완성도 양호. 현지인/관광지 소비 분리 시각화가 직관적. 건축가 수준의 분석 사용자에게 데이터 표현 방식이 신뢰 가능 |
| 구현 타당성 | **8** | 영어 음식 문화 설명 + 소비 데이터 + 지역 특산 조합 실현 가능. 교포 여행자 타겟 추가 포지셔닝 가능 |

**종합 점수: 25/30**

## Pass/Fail 판정

> **✅ PASS**

TasteData Korea는 레이첼 김에게 단순 맛집 앱을 넘어 "데이터로 찾은 진짜 고향 음식 문화" 경험을 제공한다. 교포 2세의 감정적 여행 동기(부모님 고향 이해, 딸에게 뿌리 전달)와 앱의 현지인 소비 데이터 기반 접근이 높은 정렬을 보인다. 유용성 9점, 종합 25점으로 높은 평가를 받았다.

## 개선 제안

1. "로컬 음식 문화 스토리" 탭 추가 — 지역 음식의 역사적 맥락, 세대 간 전승 이야기를 영어로 제공해 교포 사용자의 뿌리 탐방 서사를 앱이 직접 지원
2. 음식 코스에 "어린이 동반 안내" 옵션 — 10세 아이가 먹기 좋은 순한 맛 메뉴 우선 추천, 아이에게 설명하기 좋은 음식 스토리 카드 제공
