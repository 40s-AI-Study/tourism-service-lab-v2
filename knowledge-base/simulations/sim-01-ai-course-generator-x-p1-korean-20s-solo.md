# SIM-01: KoreaPath AI × 김지원 (한국 20대 솔로)

## 시나리오

금요일 밤 10시, 김지원(25세, 프리랜서 그래픽 디자이너)은 인스타그램 릴스에서 강릉 안목해변 감성카페 영상을 보고 "내일 당일치기 가자"고 결심한다. 평소엔 네이버 블로그·카카오맵·인스타 세 개를 번갈아 보며 코스를 직접 짜야 해서 1-2시간이 걸렸지만, 이번엔 KoreaPath AI를 처음 써본다.

---

## 사용자 여정

### 1단계: 진입
- 앱 설치 후 첫 화면에서 조건 입력: **"강릉, 당일치기, 혼자, 감성카페+포토스팟, 예산 3만원"**
- 입력 소요 시간: 약 40초

### 2단계: 행동
- AI가 `visitor-concentration-forecast` 데이터 조회 → 오전 10시 안목해변 혼잡도 28%(한산) 확인
- 생성된 코스:
  ```
  10:00 안목해변 카페거리 (혼잡도 낮음, 포토스팟)
  11:30 강릉 중앙시장 로컬 먹거리 (예산 8천원)
  13:00 경포호 인생사진 스팟 (무료)
  14:30 정동진 해안 산책로
  16:00 귀가 (KTX 동해선)
  ```
- 각 장소에 혼잡도 배지(🟢한산 / 🟡보통 / 🔴혼잡)가 표시됨
- 코스를 링크로 저장 후 친구에게 공유

### 3단계: 결과
- 안목해변에서 줄 없이 사진 촬영 성공
- 예산 2만 7천원 내 여행 완료
- 인스타 릴스 업로드 → 앱 링크 첨부해 "코스 출처" 공유

|---|---|---|
| 유용성 | **9** | 혼잡도+코스 자동화가 핵심 페인포인트(정보 과부하, 혼잡 타이밍) 정확히 해소 |
| 사용 편의 | **8** | 40초 입력 → 코스 완성은 탁월. 코스 저장·공유 UX도 자연스러움 |
| 구현 타당성 | **8** | 활용 API 5종 모두 실제 제공 중. AI 코스 생성은 규칙 기반으로도 MVP 가능 |

**종합 점수: 25/30**

## Pass/Fail 판정

> **✅ PASS**

P1 김지원의 핵심 Pain Point(정보 과부하 해소, 혼잡도 사전 확인, 코스 파편화 방지)를 KoreaPath AI가 직접적으로 해결한다. 당일치기 솔로 여행자에게 가장 강력한 킬러 유즈케이스.

## 개선 제안

1. **SNS 공유 카드**: 코스를 인스타 스토리 비율로 자동 레이아웃 → "이 코스로 다녀왔어요" 바이럴 유도
2. **예산 시뮬레이터**: 입력한 예산 대비 예상 지출 항목(교통비/식비/체험비) 분할 표시
3. **실시간 재조정 알림**: 현장에서 혼잡도 급증 시 "지금 옆 골목 카페로 이동 추천" push 알림
