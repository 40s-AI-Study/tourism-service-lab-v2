# SIM-188: KoreaDemand Atlas × 레이첼 김 (교포 2세)

## 시나리오

레이첼은 10박 11일 한국 방문을 앞두고 영어로 된 전주 여행 정보를 찾다가 KoreaDemand Atlas를 발견한다. "데이터 기반 관광 지도"라는 설명이 건축가 특유의 분석적 사고방식과 맞아 설치한다. 영어 UI로 전환해 전주를 검색하면 한옥마을 지역 수요 강도 히트맵과 잠재력 점수가 표시된다. 레이첼은 단순히 어디가 유명한지보다 "어떤 지역이 실제로 살아있는 문화권인지" 데이터로 확인하고 싶어 한다. 부모님의 고향인 전주 구도심 골목 지역이 히트맵에서 낮은 수요 강도지만 '다양성 지수 상위'로 표시된 것이 흥미롭다.

---

## 사용자 여정

### 1단계: 진입
- 영어 한국 여행 블로그 링크 → 앱스토어 설치
- 영어 전환 후 전주 검색 → 히트맵 + 지역별 다양성 지수 화면 탐색
- 건축가 관점: "이 데이터가 한옥 밀집 지역과 겹치는지 확인하고 싶다"는 구체적 목적 설정

### 2단계: 행동
- 전주 한옥마을 탭: 관광 잠재력 점수 69점, 소비 다양성 지수 높음, 국제 다양성 중간
- 전주 구도심 골목 지역: 수요 강도 낮음(=덜 알려짐) + 관광 다양성 지수 높음 → 레이첼이 찾던 "살아있는 로컬 문화권"에 해당
- 영어 관광 정보 페이지 열람: 한옥마을 건축 관련 설명 있으나 건축 구조·공법 같은 전문적 내용은 없음
- 딸(10세)에게 보여줄 아동 맞춤 콘텐츠 없음 → 별도 검색 필요
- 코스 생성 기능으로 전주 3일 동선 초안 생성 → 건축 투어 중심으로 수동 편집 필요

### 3단계: 결과
- 전주 구도심 골목 방문: 예상대로 관광객 적고 진짜 로컬 분위기 — 레이첼이 원하던 경험
- "데이터가 알려준 덜 알려진 전주"라는 여행 서사 완성 → 딸에게 의미 있는 설명 가능
- 건축 심층 정보, 족보·지역사 데이터는 앱 범위 밖 — 레이첼의 핵심 관심사 일부 미충족

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **7** | 관광 다양성 지수 기반 "살아있는 로컬 문화권" 발굴이 레이첼의 교포 특수 니즈와 부분적으로 맞음. 건축 심층 정보 부재는 한계 |
| 사용 편의 | **7** | 영어 UI 완성도 양호. 건축가 수준의 분석 니즈를 데이터가 부분 지원. 코스 편집 기능 미흡 |
| 구현 타당성 | **8** | 영어 관광 정보 + 다양성 지수 통합은 실현 가능하며 해외 교포 여행자에게 차별화된 가치 제공 |

## Pass/Fail 판정

> **✅ PASS**

KoreaDemand Atlas는 레이첼이 찾는 "덜 알려진 진짜 로컬 문화권" 발굴 니즈를 관광 다양성 지수와 수요 강도 교차 분석으로 충족시킨다. 유용성 7점, 종합 22점으로 기준 충족. 건축 심층 정보와 교포 특수 콘텐츠는 이 앱의 범위 밖이지만, 여행지 선택 단계에서의 역할은 확실히 수행한다.

## 개선 제안

1. "문화 깊이 지수(Cultural Depth Score)" 파생 지표 추가 — 역사 시설 수·전통문화 자원 밀도를 결합해 교포·문화 탐방 여행자에게 적합한 지역 식별 지원
2. 영어 관광 정보 페이지에 건축·역사 카테고리 심층 콘텐츠 탭 추가 — `tourism-info-english` API 데이터를 카테고리화해 전문 관심사 여행자 대응
