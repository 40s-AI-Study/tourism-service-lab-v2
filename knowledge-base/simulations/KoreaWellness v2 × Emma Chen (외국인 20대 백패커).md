# SIM-R3-022: KoreaWellness v2 × Emma Chen

## 시나리오

Emma Chen은 대만 타이베이 출신의 23세 프리랜서 사진작가다. 인스타그램 팔로워 4만 명, 브랜드 협업으로 생계를 유지한다. 이번 한국 여행은 3주짜리 단독 백패킹 — 서울 홍대 게스트하우스에서 시작해서 어디로 갈지는 "느낌대로" 정하는 방식이다. 가장 큰 고민은 하나다: 모두가 아는 장소는 찍고 싶지 않다. 이미 인스타에 수백만 장이 올라온 경복궁 사진을 또 찍는 건 그녀의 피드에 어울리지 않는다.

숙소 침대에 누워 번체 중국어로 "韓國秘境拍照" (한국 비밀 촬영 명소)를 검색하다가 한 대만 여행 블로거의 릴스에서 KoreaWellness v2를 발견했다. 블로거가 앱을 열자마자 등장한 감정 카드 화면을 캡처해둔 게 눈에 띄었다 — "這個APP先問你心情，不是問你要去哪" (이 앱은 먼저 기분을 물어봐, 어디 갈지가 아니라). Emma는 번체 언어 설정을 확인하고 즉시 설치했다.

첫 화면. 繁體中文으로 번역된 다섯 장의 카드 중 그녀는 두 개가 끌렸다 — 📷 **순간 (季節を捕まえたい)** 과 🌿 **고요**. 잠시 고민하다 📷 순간을 눌렀다. 지금 계절은 늦봄, 벚꽃은 졌고 연두색이 막 올라오는 시기였다.

## 사용자 여정

### 1단계: 진입
- 대만 여행 블로거 릴스 → 번체 중국어 UI 확인 후 설치 결정
- 📷 순간 카드 선택 → "계절 콘텐츠 + 혼잡도 낮은 장소" 필터 자동 활성화
- 앱이 현재 계절(늦봄) 기반으로 "지금만 볼 수 있는" 장소 리스트를 먼저 보여줌

### 2단계: 행동
- 추천 리스트 상단: 경남 합천 황매산 철쭉 군락지 — 방문자 집중률 예측 API 기반으로 "이번 주 혼잡도 최저점" 배지 표시됨
- 황매산 상세 페이지에서 생태 관광 API 연동 정보 확인: 일출 시간, 최적 촬영 앵글 방향, 주차장 위치
- "K-웰니스 맥락" 탭에서 산림욕과 철쭉의 계절적 의미 번체 중국어로 읽음 — 콘텐츠 캡션에 쓸 소재로 메모
- 앱 내 저장 기능으로 황매산 + 인근 산골 찻집 2곳을 "내 여정" 보드에 저장
- 숙소에서 버스 루트 확인 후 다음날 새벽 출발 계획 수립

### 3단계: 결과
- 새벽 4시 반 출발, 일출 30분 전 황매산 정상 도착. 다른 관광객은 손에 꼽을 만큼 적었다
- 2시간 동안 혼자 철쭉밭을 촬영. 안개와 조명이 어우러진 컷 23장을 건졌다
- 귀경 후 인스타에 올린 첫 장이 4일 만에 12만 하트 — 댓글에 "어디야??" 가 수백 개
- 앱 리뷰에 繁體中文으로 남긴 한 줄: "第一次覺得旅遊APP真的懂我想要什麼。" (처음으로 여행 앱이 내가 원하는 걸 진짜 이해한다는 느낌)

## Pass/Fail 판정

> **✅ PASS**

"숨은 명소 + 최적 촬영 타이밍"이라는 사진작가 특화 니즈를 감정 카드(📷 순간) → 계절 필터 → 혼잡도 데이터로 이어지는 흐름이 자연스럽게 해결했다. 번체 중국어 UI가 대만 사용자의 신뢰를 확보하는 핵심 요소로 작동했다.

## 리텐션 평가: 이 페르소나가 앱을 다시 열까?

**재사용 확률**: 🔥매우 높음

**감정적 이유**:
> "이 앱이 없었으면 황매산은 몰랐을 거야. 내 피드를 바꿔준 앱이니까, 다음 계절에 또 뭘 보여줄지 궁금해."

**재사용 트리거**:
- 계절이 바뀔 때마다 "지금 어떤 장소가 최적인지" 확인하러 앱을 열게 됨
- 팔로워들이 "다음 한국 사진은 어디서?"라고 댓글 달 때 앱에서 다음 목적지 탐색
- 브랜드 협업 촬영지 섭외에 활용 — 차별화된 배경 필요할 때

**이탈 리스크**:
- 번체 중국어 번역이 어색하거나 일부 페이지만 번역되어 있으면 즉시 신뢰 이탈
- 추천 장소가 이미 인스타에 많이 노출된 곳이라면 "이건 나한테 안 맞는 앱"으로 판단
