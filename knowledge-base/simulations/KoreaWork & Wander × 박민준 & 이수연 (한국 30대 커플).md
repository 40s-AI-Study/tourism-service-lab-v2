# SIM-172: KoreaWork & Wander × 박민준 & 이수연 (한국 30대 커플)

## 시나리오

민준은 회사에서 주 2일 재택근무가 가능해졌고, 수연도 마케팅 에이전시에서 리모트 근무 날이 생겼다. 두 사람은 목~금 2일을 맞춰 제주에서 일하고 주말은 여행으로 이어가는 "워케이션 + 주말 여행" 계획을 세운다. KoreaWork & Wander 앱에서 제주 지역을 탐색하니, 고속 인터넷이 확인된 글램핑장과 조용한 환경의 코워킹 가능 숙소가 리스트업된다. area-tourism-resource-demand 데이터로 "제주 동부는 성수기 수요 높음 → 제주 서부 추천"이라는 실질적인 지역 선택 근거도 받는다. 민준은 "IT PM으로서 이런 데이터 기반 추천이 훨씬 신뢰간다"며 만족한다.

---

## 사용자 여정

### 1단계: 진입
- 민준이 앱스토어에서 검색 → KoreaWork & Wander 설치 (영어 UI로 시작, 한국어 미지원이 다소 불편)
- "제주, 2인, 2박(목~금), 코워킹 가능" 조건 입력
- gocamping + 숙박 데이터에서 고속 인터넷·콘센트 확인된 글램핑·독채 목록 표시

### 2단계: 행동
- area-tourism-resource-demand 분석: 제주 동부 성수기 수요 과밀 → 제주 서부(한림·애월) 추천
- area-tourism-diversity 지수 확인: 애월 일대 ★★★★ — 카페·해안 드라이브·맛집 다양
- 업무 환경 태그 필터: "파워콘센트 ✅, 고속 인터넷 ✅, 조용한 환경 ✅" 적용 → 3곳 최종 후보
- 업무 후 주변 관광 자동 추천: 오전 작업 → 오후 애월 카페 투어 → 저녁 해산물 식당
- 수연이 "주말 여행 코스"도 related-attractions 기반으로 이어서 생성

### 3단계: 결과
- 목~금 업무 완료 후 토~일 여행으로 자연스럽게 전환, 총 4일 일정 완성
- 민준: "데이터 기반 지역 추천은 기대 이상" / 수연: "영어 UI가 불편하고 숙박 예약을 앱 안에서 못 하는 게 아쉽다"
- 재사용 의향: 민준 높음, 수연 중간 — 예약 연동 강화 시 커플 단위 충성 고객화 가능

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **7** | 관광 자원 수요 분석 기반 지역 추천이 IT PM 민준에게 신뢰감 제공. 워케이션 + 주말 여행 연계 플로우가 이 커플의 생활 패턴과 잘 맞음 |
| 사용 편의 | **5** | 한국어 UI 부재가 반복적 불편 요소. 앱 내 숙박 예약 불가로 외부 이동 필요 — 수연처럼 완결된 플로우를 원하는 사용자에게 이탈 포인트 |
| 구현 타당성 | **7** | area-tourism-resource-demand × gocamping API 조합은 기술적으로 구현 가능. 예약 연동은 추가 파트너십 필요 |

## Pass/Fail 판정

> **✅ PASS**

KoreaWork & Wander는 박민준 & 이수연 커플의 "워케이션 + 주말 여행" 복합 니즈에 실질적으로 대응한다. 관광 자원 수요 데이터 기반 지역 추천이 IT PM의 신뢰를 획득하고, 업무 후 관광 연계 자동화가 이 커플의 계획 부담을 낮춘다. 한국어 UI와 앱 내 예약 연동이 추가되면 전환율이 크게 오를 것으로 예상된다.

## 개선 제안

1. 한국어 UI 필수 도입 및 "워케이션 + 주말 여행 연계 패키지" 플로우 명시: 2+2일(평일 워케이션 + 주말 관광) 조합을 단일 검색으로 설계하는 기능으로 이 커플의 여행 패턴을 앱이 완전히 지원
2. 글램핑·독채 숙박 예약을 앱 내에서 직접 완료할 수 있도록 야놀자·에어비앤비 API 연동 추가 — 수연 같은 "원스톱 예약" 선호 사용자의 이탈을 방지하는 가장 효과적인 전환율 개선책
