# SIM-44: KoreaTrend Radar × Emma Chen (외국인 20대 백패커)

## 시나리오

2026년 4월 17일 토요일 오전 10시. 대만 출신 사진작가 Emma Chen(23)은 서울 홍대 게스트하우스에서 3주째 한국을 여행 중이다. 인스타그램 팔로워 4.2만 명을 보유한 그녀의 목표는 "서울 외 지방의 아직 덜 알려진 포토스팟"을 먼저 발굴해 콘텐츠화하는 것이다. 여행 크리에이터 커뮤니티에서 KoreaTrend Radar를 추천받아 영문 UI로 실행했다.

앱을 영어로 전환하니 주요 기능은 이용 가능하다. 홈 레이더에서 'Outside Seoul' 필터를 적용하면 강릉·전주·통영 일대에 트렌드 급등 신호가 감지된다. Emma는 통영의 이동통신 유동인구 지수가 전주 대비 +89%로 급등한 것에 주목한다. 세부 화면에서 관광 다양성 지수(area-tourism-diversity)를 확인하면 통영이 자연·음식·문화 3개 카테고리에서 고른 상승세를 보인다 — 포토그래퍼에게 이상적인 복합 목적지 신호다.

그녀는 통영 트렌드 데이터 스크린샷을 찍어 인스타그램 스토리에 "Data says Tongyeong is the next big thing 📊"라고 올린다. 이틀 후 통영행 버스를 예약한다.

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 언어 설정 영어로 변경
- 'Outside Seoul' / 지방 필터 적용
- 지도에서 통영 핫스팟 클릭

### 2단계: 행동
- 통영 트렌드 지표 상세 확인: 이동통신 +89%, 카드 소비 +54%, 내비 +71%
- 관광 다양성 지수(자연/음식/문화) 카테고리별 분석 열람
- 주간 트렌드 추이 그래프로 급등 시점 파악 (5일 전부터)
- 트렌드 데이터 카드 스크린샷 저장
- 인스타그램 스토리 업로드 (앱 공유 기능 활용)
- 근처 포토스팟 데이터 추가 탐색 (남해, 거제 비교)

### 3단계: 결과
- 통영 2박 3일 일정 결정
- 앱 알림 구독: "지방 급등 알림 (서울 외)" ON
- 인스타그램 스토리 조회수 1.2만 → 팔로워 반응 긍정적
