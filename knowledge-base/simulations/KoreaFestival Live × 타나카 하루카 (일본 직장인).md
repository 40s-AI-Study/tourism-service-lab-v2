# SIM-170: KoreaFestival Live × 타나카 하루카 (일본 직장인)

## 시나리오

하루카는 친구들과 3박 4일 첫 한국 여행을 앞두고 SNS에서 정보를 모으는 중이다. X(트위터) 한일 여행 계정에서 "한국 축제 앱 KoreaFestival Live 일본어 지원 시작" 트윗을 발견하고 즉시 설치한다. 한국어를 거의 못 읽는 하루카에게 일본어 UI는 그 자체로 강력한 동기 부여가 된다. 드디어 "일본어로 한국 축제 정보를 찾을 수 있다"는 흥분감 속에 전주국제영화제 기간과 여행 일정이 겹친다는 걸 발견한다. K-drama 팬으로서 한국 영화 축제 분위기를 직접 느껴보고 싶다는 욕구가 즉시 생기며, 혼잡도 데이터로 줄 없이 야외 상영 관람이 가능한 시간대를 확인한다.

---

## 사용자 여정

### 1단계: 진입
- X(트위터) 일한 여행 계정 트윗 → 앱 설치 → 일본어 UI 자동 감지 설정
- 여행 날짜 입력 → "이 기간에 열리는 축제" 자동 매칭 기능 → 전주국제영화제 카드 팝업
- K-culture 태그 필터 클릭 → 한류 관련 이벤트 상단 큐레이션

### 2단계: 행동
- 전주국제영화제 상세 페이지 → 일본어 행사 설명 확인 (tourism-info-english 기반 일본어 번역 품질 중요)
- 야외 상영 프로그램 시간표 확인, 혼잡도 예측으로 저녁 8시 이후 관람 혼잡도 62% → 오후 3시 야외 상영 혼잡도 34%로 대체
- "축제 + 전주 투어 1일 코스" 생성: 한옥마을 포토스팟 → 카페 → 영화제 야외 상영 → 저녁 전주비빔밥 자동 연결
- 포토스팟 리스트에서 "인스타·X 공유 많은 장소" 별도 태그 확인
- 일본어로 각 장소 간단 설명 + 이동 방법 텍스트 제공

### 3단계: 결과
- 친구들과 혼잡도 낮은 오후 시간 야외 상영 관람 성공, 인파 없이 영화 분위기 사진 촬영
- X에 "韓国映画祭リアルタイム混雑データで快適に楽しめた！" 게시 → 팔로워 큰 반응
- 귀국 후 일본 여행 유튜버 채널에 앱 소개 요청받음

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live의 일본어 지원과 여행 날짜 기반 축제 자동 매칭은 타나카 하루카의 "일본어 한국 여행 정보 부재"라는 핵심 페인포인트를 직접 해소한다. K-culture 팬 일본 여행자는 한국 방문 외국인 중 최대 규모 그룹 중 하나로, 이 페르소나를 통한 바이럴 효과는 일본 시장 진입에 결정적 역할을 할 수 있다.

## 개선 제안

1. 일본어 전용 "K-drama·K-pop 성지 연계 축제 큐레이션" 콘텐츠 추가: 하루카 같은 K-culture 팬에게 "이 축제 기간에 근처 드라마 촬영지도 방문 가능" 정보를 결합하면 체류 시간 연장 및 앱 활성도 극대화
2. X(트위터) 일본어 여행 계정과의 파트너십으로 "여행 날짜 입력 시 해당 기간 축제 매칭" 기능을 홍보 — 하루카의 정보 탐색 채널인 X에서 직접 앱 유입을 만드는 가장 효율적인 마케팅 경로
