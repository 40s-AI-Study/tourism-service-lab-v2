# SIM-116: MediTrip Korea × 김영자 (한국 시니어)

## 시나리오

딸이 어머니를 위해 유용한 여행 앱을 찾다가 MediTrip Korea를 발견한다. "병원 정보도 있고 관광 코스도 있다고 해서 어머니께 좋을 것 같아서" 설치해준다. 영자가 앱을 열면 한국어 UI는 잘 되어 있다. 그러나 앱 전체가 "외국인 의료관광객"을 위한 서비스임을 콘텐츠 훑어보며 파악한다. 비자 신청 안내는 영자에게 무관하고, 외국어 병원 리스트도 필요 없다. 관광 코스 탭에서 강남·신촌 주변 힐링 코스를 보지만, 영자가 원하는 불국사·경복궁 같은 전통 사찰·고궁 코스나 무장애 경로 정보는 없다.

---

## 사용자 여정

### 1단계: 진입
- 딸이 설치해줌. "어머니, 여기 관광 코스도 있고 병원 정보도 있대요"
- 영자 혼자 실행 — 한국어 화면이라 첫 인상은 괜찮음
- 홈 화면에 "Medical Tourism Visa", "병원 찾기" 배너 — "이게 뭐지?" 의아함

### 2단계: 행동
- "병원 찾기" 탭: 외국인 환자 대응 병원 목록 — 영자에게 불필요
- "관광 코스" 탭: 강남·홍대 주변 코스 — 영자의 선호(사찰·고궁)와 불일치
- 무릎 관절염으로 계단 없는 경로가 필수인데 접근성 필터 없음
- 큰 글씨 설정 없어 글자가 작아 불편 (딸에게 전화: "글씨가 너무 작아")

### 3단계: 결과
- 딸이 원격으로 확인 후 "어머, 이건 어머니께 안 맞는 앱이에요" 인정
- 앱 삭제. 네이버 지도로 복귀

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 한국어 UI로 진입은 가능하나 콘텐츠 전체가 외국인 의료관광 맥락. 영자 니즈 미충족 |
| 사용 편의 | **4** | 한국어 지원으로 접근 가능하지만 글씨 크기·UI 복잡도가 시니어에게 부적합 |
| 구현 타당성 | **7** | 외국인 의료관광객 대상으로는 타당한 설계 |

**종합 점수: 13/30**

## Pass/Fail 판정

> **❌ FAIL**

김영자는 MediTrip Korea의 타깃인 외국인 의료관광객이 아니다. 무장애 경로, 전통 사찰·고궁 코스, 큰 글씨 UI 등 시니어 여행자의 핵심 니즈가 전혀 충족되지 않는다. 유용성 2점, 종합 13점, FAIL.

---

## 개선 제안

1. 관광 코스 섹션에 무장애 필터(`barrier-free-travel` API 연동)를 추가하면 의료관광 목적 방문자 중 고령·이동 제약 환자에게도 실질적 도움이 됨
2. 시니어 사용자를 위한 큰 글씨 모드(접근성 설정)는 의료관광 앱 성격상 고령 환자 배려 차원에서도 반드시 필요한 기능
