# SIM: Foreign AI Guide × 이재호 (전동휠체어 개발자)

## 시나리오

이재호(34세, 전동휠체어 이용 IT 개발자)는 제주 스타트업 컨퍼런스 참석 후 2박 3일 개인 여행을 계획했다. 과거 제주 여행에서 "무장애 경로라고 표시됐는데 실제로 경사가 심해서 돌아와야 했다"는 경험이 있다. 해외 컨퍼런스 참가자 중 영어권 장애인 개발자 동료 Alex(호주)와 함께 움직인다. Foreign AI Guide가 무장애 정보 + 다국어 지원을 모두 제공한다는 걸 알고 설치했다.

---

## 사용자 여정

### 1단계: 진입 (접근성 설정)
- 앱 실행 → 언어: 한국어 / Alex: English
- 온보딩: "여행 편의 설정" → **전동휠체어** 선택
- Alex 앱: "Accessibility Needs" → **Wheelchair** 선택
- 앱이 자동으로 무장애(`barrier-free`) 필터 적용: 이후 모든 추천이 무장애 기준으로 필터링

### 2단계: 무장애 관광지 탐색
- 채팅: "제주 성산일출봉 전동휠체어로 올라갈 수 있어?"
- AI 응답: `무장애 여행 정보 API` 연동 → "성산일출봉 정상 접근 불가. 하지만 일출봉 해안 산책로(1.2km)는 전동휠체어 접근 가능. 경사도 2% 이하 구간 지도 표시"
- 대안 추천: 섭지코지(전면 무장애), 한라수목원(무장애 데크 4.5km), 비자림(일부 구간)
- Alex 앱에 동일 내용 **영어**로 실시간 동기화

### 3단계: 방문 계획 최적화
- 혼잡도 확인: `tatsCnctrRateList` → 섭지코지 오전 10시 **집중률 24%** (한산)
  - 주의: 혼잡 시 휠체어 통행 방해 심함 → 비수기 시간대 선택이 특히 중요
- 경로 탐색: 숙소(서귀포) → 섭지코지 → 성읍민속마을 → 비자림 순서
  - 각 이동 구간 무장애 주차장 정보 포함
- AI: "섭지코지 무장애 주차 구역은 입구에서 50m, 전동휠체어 충전소 있음"

### 4단계: 현장 가이드 (섭지코지)
- GPS 도착 감지 → 무장애 경로 AR 오버레이: 접근 가능 구역 초록색, 위험 구역 빨간색
- 오디오 가이드: 이재호 → 한국어, Alex → 영어 동시 청취
- 섭지코지 등대 도달 → "이 경로를 무장애 경로로 커뮤니티에 공유하시겠습니까?" 알림
  - 이재호가 실시간 접근성 정보 업데이트 기여 (크라우드소싱)

### 5단계: 결과
- 3일 일정 무장애 검증 완료 → 예상치 못한 장벽 없음 (과거 경험 해소)
- Alex: "Best accessibility app I've used in Asia" → AppStore 5성 후기
- 이재호: 접근성 IT 커뮤니티(카카오 오픈채팅) 공유 → 장애인 여행자 그룹 입소문
- 크라우드소싱 데이터: 3곳 무장애 경로 정보 기여 → 앱 데이터 품질 향상

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **10** | 무장애 필터 + 다국어 지원의 조합은 접근성 약자 외국인 여행자에게 전무한 차별점 |
| 사용 편의 | **8** | 접근성 설정 온보딩 직관적. 단 AR 경로 오버레이는 화면 크기/밝기 이슈 보완 필요 |
| 구현 타당성 | **8** | 무장애 API 실존, 크라우드소싱 경로 업데이트는 추가 개발 필요 |

**종합 점수: 26/30**

## Pass/Fail 판정

> **✅ PASS (사회적 가치 + 차별화 핵심)**

무장애 다국어 여행 정보는 **글로벌 유일** 포지션이다. 장애인 여행자 커뮤니티(온·오프라인)는 충성도가 높고 입소문 속도가 빠르다. 공모전 심사 기준 "사회적 가치" 항목 최고점 획득 시나리오이자, 실제 서비스 론칭 시 정부·복지기관 B2B 파트너십 가능성이 높은 세그먼트.

## 개선 제안

1. **접근성 정보 크라우드소싱 시스템**: 사용자가 현장에서 무장애 경로 정보를 업데이트하고 커뮤니티가 검증 → 데이터 품질 지속 개선
2. **실시간 접근성 경고**: "오늘 비로 인해 섭지코지 데크 미끄러움 주의" 같은 환경 기반 경고
3. **장애 유형별 세분화**: 전동휠체어 / 수동휠체어 / 시각장애 / 청각장애별 맞춤 정보
4. **국제 장애인 여행 인증 연계**: Global Access Symbol 등 국제 무장애 인증 정보 통합
