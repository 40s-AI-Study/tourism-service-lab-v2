# SIM-146: TourismCareer Korea × 김영자 (한국 시니어)

## 시나리오

영자는 교직에서 은퇴한 67세로, 스마트폰은 카카오톡과 네이버 지도 위주로만 사용한다. TourismCareer Korea와의 접점은 거의 없다. 딸이 "엄마, 이 앱으로 해설사 자격증 공부도 된대" 라며 설치해줬다. 사찰·고궁에서 관광 해설 자원봉사를 꿈꾸던 영자는 "관광 해설사 자격증 준비용 퀴즈 기능"이 있다는 말에 잠깐 흥미를 보였다. 그러나 앱의 주요 기능인 채용 공고는 자신과 관련 없고, 화면 구성이 너무 복잡하게 느껴졌다.

---

## 사용자 여정

### 1단계: 진입
- 딸이 설치 후 "해설사 자격증 퀴즈 탭 눌러보세요"라며 직접 안내
- 앱 실행 첫 화면: 채용 공고 목록 — 영자에게 의미 없는 정보, "이게 뭐야" 반응
- 딸이 하단 탭 중 "학습" 탭 찾아서 직접 열어줌

### 2단계: 행동
- 관광 해설사 퀴즈 탭 진입: 불국사, 첨성대, 경복궁 관련 퀴즈 4문제 시도
- 글씨 크기가 작아서 안경 쓰고 화면에 바짝 붙어 읽음
- 퀴즈 3문제 맞힘 — "아, 이건 재미있네" 긍정 반응
- 공고 탭·데이터 탭은 건드리지 않음, 퀴즈 기능만 반복 사용

### 3단계: 결과
- 딸이 주 1회 앱 열어서 퀴즈 몇 문제 풀어주는 패턴으로 정착
- 채용 기능은 완전 미사용
- 퀴즈 기능만으로는 앱을 스스로 유지할 동기 부족 — 딸 없으면 사용 안 함

## Pass/Fail 판정

> **❌ FAIL**

김영자는 이 앱의 명백한 비타깃 사용자다. 취업·인턴십 플랫폼에 67세 은퇴 교사는 맞지 않는다. 관광 해설사 자격증 퀴즈 기능이 유일하게 접점이 있으나, 시니어 UX 미지원과 앱 전체 맥락과의 단절로 실질적 사용 가치가 없다. 해설사 자격증 준비 기능을 시니어 친화 독립 앱으로 분리하는 것이 아니라면, 시니어층 마케팅은 삼가야 한다.

## 개선 제안

1. 시니어 UI 모드 도입 (선택적): 취업 기능 외에 "관광 학습/자격증" 모드를 별도 진입 경로로 제공, 큰 글씨·단순 탭 구조 적용
2. 앱 포지셔닝 명확화: 타깃이 아닌 사용층(시니어, 일반 여행자) 유입을 억제하는 앱스토어 설명·키워드 정비 필요
