# SIM-160: PhotoHunt Korea × 타나카 하루카 (일본 직장인)

## 시나리오

하루카는 3박 4일 첫 한국 여행을 앞두고 인증샷 명소를 열심히 찾고 있다. 일본 X(트위터)의 한국 여행 계정 @koreatabi_jp에서 "PhotoHunt Korea — 공식 포토스팟 앱, 인증샷 각도까지 알려줌"이라는 트윗을 보고 리트윗하며 즉시 설치했다. K-pop 아이돌들이 찍은 듯한 감성 사진을 SNS에 올리는 것이 이번 여행의 핵심 목표 중 하나다.

---

## 사용자 여정

### 1단계: 진입
- X 트윗 링크 → 앱스토어 설치
- 앱 실행: 일본어 UI 없음 → 영어 UI로 전환 (영어 중급으로 사용 가능)
- 온보딩: "K-Drama Spots", "Street Photography", "Cafe & Food" 선택

### 2단계: 행동
- 서울 홍대 검색 → 공식 포토스팟 지도 표시 + 공모전 수상작 사진 확인
- "K-Drama Spots" 카테고리에서 경복궁 근처 드라마 촬영지 스팟 발견 — "ここ知ってる！(여기 알아!)" 반응
- `visitor-concentration-forecast`: 경복궁 토요일 오후 1시 집중률 81%, 일요일 오전 8시 19% — 일찍 일어나서 가기로 결정
- 각 스팟 "이 각도에서 찍으면 이렇게 나옵니다" 영문 촬영 팁 확인, 구글 번역으로 일부 보완
- SNS 공유 기능: 영어 해시태그 자동 생성 → 일본어 해시태그 없음, 직접 추가 필요
- 친구(1명 동반)와 앱 화면 공유하며 같이 탐색 — "この앱すごい！(이 앱 대단해!)"

### 3단계: 결과
- 일요일 오전 8시 경복궁 방문: 사람 없는 광화문 앞 사진 성공 — "韓国来て一番の写真！(한국 와서 최고 사진!)"
- 홍대 벽화골목 포토스팟에서도 앱 촬영 팁 적용 → 만족스러운 인증샷 달성
- X에 "@koreatabi_jp この앱が最高でした！(이 앱 최고였어요!)" 태그 게시 → 200 리트윗
- 앱 일본 앱스토어 리뷰 5점: "日本語がないのが残念だけど英語でも十分使える (일본어 없는 게 아쉽지만 영어로도 충분히 사용 가능)"

## Pass/Fail 판정

> **✅ PASS**

타나카 하루카는 PhotoHunt Korea의 핵심 타깃(20대, 인스타·SNS 공유 중심 여행자)과 완전히 일치한다. 일본어 UI 부재라는 한계에도 불구하고 영어로 대체 사용하여 높은 만족도를 얻었다. X 트위터를 통한 자발적 추천과 앱스토어 5점 리뷰가 일본 사용자 바이럴로 연결될 잠재력이 크다. 일본어 UI 추가 시 일본 시장에서 강력한 성장 동력이 될 수 있다.
