# SIM-43: KoreaTrend Radar × 최동훈 (한국 40대 가족)

## 시나리오

2026년 4월 17일 금요일 오후 9시. 경기도 분당에 거주하는 최동훈(44, 중학교 교사)은 다음 달 가족 여행지를 고민 중이다. 초등학생 두 자녀(9세, 11세)와 함께하는 여행이라 안전성과 교육적 요소가 중요하다. 아내가 "요즘 트렌드 앱이 있다더라"며 KoreaTrend Radar를 추천했고, 호기심에 실행해봤다.

홈 화면의 실시간 급등 레이더와 화려한 히트맵이 눈에 들어온다. 흥미롭긴 하지만 "지금 당장 뜨는 곳"보다 "아이들과 가기 좋은 검증된 곳"을 원하는 최동훈에게는 트렌드 데이터보다 안전 정보·편의시설·교육 콘텐츠가 우선이다. '가족여행' 필터를 찾아보지만 카테고리는 음식·카페·자연·관광지로만 나뉘어 있어 가족 특화 필터가 없다. 트렌드 히트맵의 지표(이동통신 유동인구 지수, 카드 소비 지수 등)가 직관적으로 와닿지 않아 몇 분간 앱을 탐색하다 결국 네이버 여행으로 전환한다.

트렌드 데이터 자체는 "어디가 요즘 많이 가는지" 참고는 되지만, 가족 여행 계획에 필요한 구체적 정보(주차, 유아 시설, 교육 프로그램)가 없어 앱 이탈이 발생한다.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 홈 레이더 화면 확인
- '자연/관광지' 카테고리 선택 시도
- 가족 여행 특화 필터 탐색 → 부재 확인

### 2단계: 행동
- 제주도 트렌드 데이터 확인 (이동통신 지수 등)
- 지표 의미 파악 시도 → 해설 정보 부족으로 혼란
- '주간 트렌드 TOP 10' 리스트 훑어보기
- 개별 장소 상세 페이지 확인 → 가족 편의시설 정보 없음
- 앱 이탈, 네이버 여행 앱으로 전환

### 3단계: 결과
- 제주도가 "요즘 뜨는 여행지"임은 확인
- 구체적 여행 계획 수립에는 활용하지 못함
- 앱 재방문 의향 낮음 (가족 여행 맥락에서)

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **4** | 거시적 트렌드 파악에는 참고가 되나, 가족 여행 계획에 필요한 안전·편의·교육 정보 부재로 실질적 가치 낮음 |
| 사용 편의 | **4** | 트렌드 지표 해석을 위한 가이드 부족, 가족 특화 필터 없음, 데이터 대시보드 복잡도가 40대 일반 사용자에게 진입장벽 |
| 구현 타당성 | **7** | 앱 자체 기능 구현은 타당하나, 타겟 페르소나 미스매치가 근본 문제. area-tourism-resource-demand API로 시설 정보 보완 가능 |

**종합 점수: 15/30**

## Pass/Fail 판정

> **❌ FAIL**

유용성 4점, 종합 15/30으로 기준(18점 이상 AND 유용성 6점 이상) 미달. 트렌드 데이터 중심 앱이 가족 여행 계획이라는 정보 집약적 니즈를 충족하지 못함. 이 페르소나는 앱의 주요 타겟이 아니나, 가족 여행객 포용을 위한 개선이 필요.

## 개선 제안

1. **가족 여행 전용 필터**: '가족/아이 동반' 카테고리 추가 및 area-tourism-resource-demand API 연계로 유아 시설·주차·교육 프로그램 정보 통합 제공
2. **지표 해설 레이어**: 각 데이터 지표(이동통신 지수, 카드 소비 지수) 옆에 물음표 버튼으로 쉬운 언어 설명 제공 ("이 지역에 요즘 사람이 47% 더 많이 방문하고 있어요")
