# SIM-194: TasteData Korea × Emma Chen (외국인 20대 백패커)

## 시나리오

에마는 한국 여행 콘텐츠 크리에이터로서 "현지인이 실제 가는 맛집"을 찾는 것이 콘텐츠의 핵심 차별점이다. 대만 여행 커뮤니티에서 TasteData Korea가 "신용카드 데이터 기반 현지인 맛집 앱"이라는 평을 받고 있다는 글을 보고 즉시 설치한다. 영어 UI로 전환해 부산을 검색하면 현지인 소비 밀집 지역 히트맵이 나타나고, 관광객 밀집 지역(남포동)과 현지인 소비 집중 지역(사직동·온천장)이 눈에 띄게 구분된다. 음식 포토 큐레이션 기능이 크리에이터로서 즉각적인 관심을 끈다.

---

## 사용자 여정

### 1단계: 진입
- 대만 여행 커뮤니티 추천 → 설치 → 영어 UI 전환 (번체 중국어 미지원 확인)
- 부산 검색 → 현지인/관광객 소비 분리 히트맵 → 크리에이터 시각에서 즉각적 가치 인식
- 음식 포토 큐레이션 갤러리 탭 → "이거 직접 찍기 전에 비주얼 참고용으로 쓸 수 있겠다"

### 2단계: 행동
- 사직동 현지인 소비 밀집 지역 탭: 국밥·돼지국밥·밀면 소비 강도 높음
- 음식 테마: 부산 길거리 음식 코스 선택 → 씨앗호떡→어묵→밀면 코스 자동 생성
- 음식 포토 큐레이션: 각 음식별 촬영 앵글 포토스팟 위치 연동 → 크리에이터에게 최고 기능
- 연관 관광지: 맛집 지역 주변 감천문화마을·영도 연결 → 포토 투어 동선 완성
- 번체 중국어 콘텐츠 없어 아쉽지만 영어로 충분히 활용 가능

### 3단계: 결과
- 사직동 돼지국밥 골목 방문 → "아무도 모르는 부산 현지 맛집" 릴스 제작
- 팔로워 반응: "이 앱 뭐야?" 댓글 폭주 → TasteData Korea 자발적 홍보 콘텐츠 생성
- 에마 평가: "음식 크리에이터에게 이 앱은 진짜 게임체인저"

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 현지인 소비 데이터 + 포토 큐레이션 조합이 음식 여행 크리에이터의 핵심 니즈(차별화 소재 + 비주얼)에 완벽 부합 |
| 사용 편의 | **8** | 영어 UI 완성도 양호. 포토 큐레이션과 코스 생성 흐름이 직관적. 번체 중국어 미지원만 아쉬움 |
| 구현 타당성 | **8** | 소비 데이터 + 포토 API + 연관 관광지 조합 실현 가능. 영어 지원으로 외국인 크리에이터 접근 가능 |

**종합 점수: 25/30**

## Pass/Fail 판정

> **✅ PASS**

TasteData Korea는 에마에게 콘텐츠 차별화 무기를 제공한다. 현지인 소비 데이터 기반 맛집 발굴이 "광고 없는 진짜 현지 맛집"이라는 크리에이터 서사와 완벽히 맞아떨어지며, 포토 큐레이션이 음식 촬영 크리에이터의 실무적 니즈를 직접 지원한다. 유용성 9점, 종합 25점으로 높은 점수를 기록했다.

## 개선 제안

1. 번체 중국어(繁體中文) UI 지원 — 대만·홍콩 음식 여행 크리에이터는 한국 음식 콘텐츠 생산의 핵심 채널이며, 이들의 자발적 홍보는 앱 성장의 최대 바이럴 소스
2. "크리에이터 모드" 추가 — 음식 촬영 최적 시간대(자연광 조건), 촬영 팁, 인스타 해시태그 추천을 코스에 함께 제공해 음식 크리에이터 특화 경험 강화
