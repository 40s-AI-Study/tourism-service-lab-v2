# SIM-03: KoreaPath AI × 최동훈 (한국 40대 가족)

## 시나리오

여름방학이 2주 후로 다가왔다. 최동훈(44세, 역사 교사)은 경주 2박 3일 가족 여행을 계획 중이다. 아이들(초5·초2)에게 교육적인 경험을 주고 싶고, 가끔 동행하는 70대 장인어른이 지팡이를 짚어 무장애 동선도 필요하다. 맘카페 정보는 광고가 섞여 있어 이번엔 KoreaPath AI를 써보기로 했다.

---

## 사용자 여정

### 1단계: 진입
- 조건 입력: **"경주, 2박 3일, 가족 4인 + 노부모 동반 가능, 교육적 체험, 초등생 포함, 예산 40만원"**
- 입력 소요 시간: 약 60초

### 2단계: 행동
- AI가 `central-attractions-by-municipality` + `related-attractions` + `visitor-concentration-forecast` 조합
- 생성된 코스:
  ```
  [1일차 — 역사 탐방]
  09:00 불국사 (오전 혼잡도 낮음, 아이 해설 체험 가능)
  12:00 황리단길 가족 식사
  14:00 첨성대 + 동궁원 (야외, 아이들 선호)
  17:00 호텔 체크인

  [2일차 — 체험 중심]
  09:00 경주 도예 공방 (초등생 체험 프로그램)
  12:00 교촌 한옥마을 로컬 식당
  14:00 경주 국립박물관 (AR 해설 체험)
  18:00 저녁 식사

  [3일차 — 자연]
  09:00 대릉원 (평지, 노부모 이동 편의)
  11:00 귀가
  ```
- ⚠️ 앱에 "노부모 무장애 필터"가 없어 동훈이 수동으로 접근성 정보를 구글에서 별도 검색

### 3단계: 결과
- 코스 구성 자체는 만족스러움 (아이들 체험 프로그램 연계 성공)
- 장인어른 동선에서 일부 계단 구간 발생 → 현장에서 당황
- 전반적으로 좋은 여행이었지만 접근성 gap이 아쉬움으로 남음

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **7** | 교육적 코스·혼잡도 타이밍은 훌륭. 단, 무장애 정보 부재로 핵심 Pain Point 일부 미해결 |
| 사용 편의 | **7** | 가족 조건 입력은 쉬우나 "노부모 동반" 세부 필터 없어 수동 보완 필요 |
| 구현 타당성 | **8** | 현재 API 조합으로 기본 가족 여행 코스 생성 충분히 가능 |

**종합 점수: 22/30**

---

## Pass/Fail 판정

> **⚠️ CONDITIONAL PASS**

가족 여행 코스 기획이라는 핵심 니즈는 충족하나, P3의 중요한 Pain Point인 "노부모 무장애 정보"가 해결되지 않는다. 개선 사항 반영 시 완전한 PASS 가능.

---

## 개선 제안

1. **"노부모 동반" 필터 추가**: 체크 시 `barrier-free-travel` API 연동 → 계단·경사 정보 포함 동선 자동 생성
2. **자녀 연령별 체험 매핑**: "8세 / 11세" 입력 시 각 나이에 맞는 체험 프로그램 필터링 (도예, AR 해설 등)
3. **가족 리뷰 태그**: "실제 가족 방문" 인증 리뷰만 필터링하여 광고성 정보 구분
