# SIM-162: KoreaFestival Live × 박민준 & 이수연 (한국 30대 커플)

## 시나리오

수연이 결혼 1주년 기념 여행을 계획하던 중, 우연히 SNS에서 "화천산천어축제 얼음 위 이벤트" 영상을 발견한다. 낭만적인 분위기에 당장 가고 싶어지지만, 민준은 "축제 때 사람 미어터지는 거 아니야?"라며 신중하게 반응한다. KoreaFestival Live 앱을 열어 화천산천어축제 페이지에서 방문자 집중률 예측을 확인하니, 평일 목·금이 주말 대비 혼잡도 40% 이하임을 알게 된다. 마침 민준이 연차를 쓸 수 있는 날이라 목요일 출발을 결정한다. 앱이 추천한 1박 2일 코스에는 산천어 낚시 체험 후 감성 펜션과 근처 웰빙 온천까지 포함되어 있어, 수연이 원하던 힐링 요소도 충족된다.

---

## 사용자 여정

### 1단계: 진입
- 수연이 인스타그램에서 축제 영상 발견 → 민준에게 공유 → 두 사람 함께 앱 검색
- KoreaFestival Live 앱 설치, "화천산천어축제" 검색 — 즉시 상세 페이지 진입
- 축제 기간 달력 뷰에서 날짜별 혼잡도 색상 표시(빨강/노랑/초록) 직관적으로 파악

### 2단계: 행동
- 목요일~금요일 혼잡도 "초록(낮음)" 확인 → 민준 반응 긍정으로 전환
- "1박 2일 코스 생성" 탭 → 산천어 낚시 → 감성 펜션 체크인 → 근처 온천 → 다음 날 빙판 체험 순서 자동 생성
- 지역 다양성 지수 ★★★★★ 확인 — 화천 주변 먹거리·볼거리 5종 연계 정보 제공
- 숙박 가격대(15~30만원) 필터 적용 → 독채 펜션 3곳 카드 형태로 추천
- 커플 여행 태그 적용 시 "로맨틱 코스 버전" 별도 제공

### 3단계: 결과
- 앱 내에서 코스 확정 후 숙박 예약 링크로 이동해 펜션 예약 완료
- 온천·낚시 체험 현장 당일 혼잡도도 실시간 확인하며 동선 조정
- 두 사람 모두 "사전 검색 스트레스 없이 코스가 완성됐다"는 만족감 → 앱 리뷰 4.8점 작성

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live는 이 커플의 핵심 페인포인트인 "코스 기획 부담"과 "현장 실망 리스크"를 혼잡도 예측 + 자동 코스 생성으로 동시에 해소한다. 다만 이 페르소나는 프리미엄 미식·숙박 예약 기능에 더 높은 기대를 가지고 있어, 축제 특화 앱으로서의 커버리지가 제한적일 수 있다는 점이 감점 요인이다.
