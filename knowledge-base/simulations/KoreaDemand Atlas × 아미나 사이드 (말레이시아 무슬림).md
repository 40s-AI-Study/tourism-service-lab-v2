# SIM-189: KoreaDemand Atlas × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

아미나는 말레이시아 무슬림 여행 카카오톡 그룹에서 누군가 "한국 관광 데이터 앱인데 영어 됨"이라고 공유한 KoreaDemand Atlas 링크를 받는다. 설치 후 영어 UI로 전환해 서울 히트맵을 보니 명동·홍대가 붉게 포화된 상태이고, 이태원 인근이 "Rising" 뱃지를 달고 있다. 이태원에는 무슬림 타운이 있다는 사실을 알고 있어 흥미를 느낀다. 그러나 앱을 탐색할수록 할랄 레스토랑 필터, 기도실 위치, 무슬림 친화 시설 정보가 전혀 없다는 것을 확인한다.

---

## 사용자 여정

### 1단계: 진입
- 무슬림 여행 그룹 공유 링크 → 설치 → 영어 UI 전환
- 히트맵 탐색: 이태원 지역 Rising 확인 → "이태원 무슬림 타운과 연결되는 데이터인가?" 기대
- 관광 잠재력 점수, 방문자 집중률 예측 확인 — 기능 파악은 빠름 (디지털 마케터 배경)

### 2단계: 행동
- 이태원 지역 탭: 수요 강도·다양성 지수 확인, 국제 다양성 지수 높음(=외국인 방문 多)
- 할랄 필터, 기도실 정보, 무슬림 친화 레스토랑 데이터 탐색 → 없음
- 지역 핵심 명소 영어 정보: 이슬람 성원(서울중앙성원) 언급 없음
- "이 앱은 관광 수요 분석 도구이지 무슬림 여행 가이드가 아니다"는 결론 도달
- 방문 타이밍 예측 기능만 실용적 — 금요일 주마 기도 시간 전후 혼잡도 참고 활용

### 3단계: 결과
- 방문 타이밍 데이터는 유용하게 활용 (인기 지역 한산한 시간대 방문)
- 할랄 식당·기도실 정보는 HalalTrip 앱을 병행 사용
- 인스타 팔로워에게 이 앱 추천하지 않기로 결정 — "무슬림 여행자에게 특별히 도움되는 기능 없음"

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **4** | 방문 타이밍 예측만 유용. 무슬림 여행자의 핵심 니즈(할랄·기도실·성분 확인)는 완전히 미지원 |
| 사용 편의 | **7** | 영어 UI 사용 가능, 디지털 마케터라 조작은 쉬움. 그러나 원하는 정보가 없어 사용 지속 동기 약함 |
| 구현 타당성 | **6** | 현재 API 범위에 할랄 데이터 없음 — 무슬림 친화 기능 추가는 별도 데이터 소스 필요 |

**종합 점수: 17/30**

## Pass/Fail 판정

> **⚠️ CONDITIONAL PASS**

종합 점수 17점으로 CONDITIONAL PASS 구간(15-17점)에 해당한다. 방문 타이밍 예측이라는 범용 기능이 최소한의 유용성을 제공하지만, 아미나의 여행에서 가장 중요한 할랄·기도실·무슬림 친화 정보가 전혀 없어 이 앱이 말레이시아 무슬림 여행자 커뮤니티에 전파될 가능성은 낮다. 영향력 있는 인플루언서가 추천하지 않기로 결정한 것이 실질적 손실이다.

## 개선 제안

1. 이태원 이슬람 성원, 할랄 인증 식당 밀집 지역을 "무슬림 친화 지역" 오버레이로 히트맵에 표시 — 무슬림 여행자의 첫 탐색 단계에서 즉각적 관련성 제공
2. 영어 관광 정보 페이지에 종교시설(이슬람 성원, 기도실) 카테고리 추가 — 현재 API 범위 내에서 시작 가능한 최소 무슬림 친화 대응
