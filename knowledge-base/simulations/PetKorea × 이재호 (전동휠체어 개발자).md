# SIM-37: PetKorea × 이재호 (전동휠체어 개발자)

## 시나리오

2026년 6월, 이재호(38세, IT개발자, 마포)는 파트너 박소연과 첫 번째 반려동물 동반 여행을 계획 중이다. 재호는 전동휠체어 사용자로 평소 자립 여행에 능숙하지만, 소연의 반려견 '하루'(포메라니안, 3kg)를 처음 데리고 가는 여행이라 반려견 정책과 휠체어 접근성을 동시에 확인해야 하는 복합 조건이 생겼다. 개발자 답게 앱을 설치하자마자 API 구조와 필터 옵션을 체계적으로 탐색한다.

재호는 PetKorea 앱에서 반려견 + 휠체어 접근성 복합 필터를 찾는다. 반려견 동반 가능 여부는 `pet-friendly-travel` API로 잘 필터링되지만, 전동휠체어 접근성(경사로, 문턱 높이, 엘리베이터 여부) 데이터는 앱 내에 제한적으로만 존재한다. 개발자로서 이 gap을 명확히 인식하고, 부족한 접근성 정보는 별도 베리어프리 앱과 병행 사용하기로 한다.

목적지는 전남 담양 대나무숲 인근 펜션, 1박2일. 대나무숲 산책로의 휠체어 접근 가능 여부와 반려견 동반 가능 여부를 앱에서 교차 확인한다.

## 사용자 여정

### 1단계: 진입
- 앱 설치 후 설정 메뉴부터 체계적으로 탐색 (개발자 성향)
- 하루 프로필 등록 (포메라니안, 3kg)
- 접근성 필터 검색 → 휠체어 전용 필터 없음 확인
- 반려견 + 접근성 우수 복합 필터로 검색 시작

### 2단계: 행동
- `pet-friendly-travel` API (카테고리: 숙박) → 소형견 허용 + 접근성 우수 필터, 담양 펜션 목록 수신
- `tourism-info-korean` API → 담양 죽녹원 반려견 동반 가능 여부 확인 (리드줄 필수, 일부 구간만 허용)
- 죽녹원 휠체어 접근 정보 확인 시도 → 데이터 미비, 외부 베리어프리 앱 병행 사용
- `visitor-concentration-forecast` API → 6월 주말 담양 혼잡도: "보통" → 평일 대비 혼잡 예상, 오전 이른 시간 방문 권장
- `related-attractions` API → 펜션 반경 내 평탄 산책로 + 반려견 동반 가능 카페 추천
- `pet-friendly-travel` API (카테고리: 레포츠) → 전동휠체어 접근 가능 + 반려견 동반 야외 활동 검색 (결과 1건으로 매우 제한적)
- 개발자 답게 앱 내 접근성 데이터 부재를 노트 앱에 피드백 메모로 기록

### 3단계: 결과
- 1박2일 담양 여행 완료. 하루 첫 장거리 여행 성공
- 죽녹원 휠체어 접근 가능 구간 현장에서 직접 확인 (사전 정보 불충분했으나 실제로는 일부 가능)
- PetKorea 앱은 반려견 정보 파트에서는 유용, 휠체어 접근성 파트는 베리어프리 앱이 훨씬 정확
- 개발자로서 두 앱을 API 수준에서 통합하면 강력한 서비스가 될 것이라는 개선 아이디어 도출
- 커뮤니티 게시판에 "휠체어 + 반려견 동반 담양 후기" 글 작성 → 앱 내 첫 커뮤니티 기여자

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **6** | 반려견 정보 파트는 유용. 휠체어 접근성 데이터 부재로 핵심 니즈 절반만 충족 |
| 사용 편의 | **7** | 개발자로서 복잡한 필터 탐색 능숙하게 처리. 일반 사용자라면 더 어려웠을 것 |
| 구현 타당성 | **7** | 반려견 API는 충실. 휠체어 접근성 데이터 통합은 추가 파트너십 필요 |

**종합 점수: 20/30**

## Pass/Fail 판정

> **✅ PASS**

반려견 동반 정보 파트에서 핵심 니즈를 충족하며 유용성 6점, 종합 20점으로 기준을 통과한다. 단, 휠체어 접근성 통합이 이루어지면 이 페르소나에게 킬러 앱이 될 잠재력이 있다.

## 개선 제안

1. **베리어프리 데이터 통합**: 한국관광공사 무장애 여행 정보 API 또는 베리어프리 앱과 제휴해 반려견 + 휠체어 접근성 복합 필터를 단일 앱에서 구현
2. **사용자 생성 접근성 리뷰**: 커뮤니티 기능에 휠체어·유모차 접근성 사용자 리뷰 태그를 추가해 실사용자 기반 접근성 데이터 누적
3. **레포츠 카테고리 접근성 필터**: 반려견 동반 야외 활동에서 이동 제한 사용자 접근 가능 여부를 별도 배지로 표시해 틈새 사용자층 지원
