# SIM-191: TasteData Korea × 김지원 (한국 20대 솔로)

## 시나리오

지원은 주말 당일치기 강릉 여행을 앞두고 "진짜 현지인 맛집"을 찾다가 네이버 블로그에 광고성 글만 가득한 것에 지쳐 TasteData Korea를 설치한다. 인스타그램에서 앱 광고를 봤는데 "신용카드 데이터 기반 현지인 실제 소비 맛집"이라는 카피가 광고 블로그에 질린 지원의 감성을 정확히 자극한다. 앱을 열면 강릉 소비 강도 히트맵이 뜨고, 관광객 집중 지역(경포대 인근)과 현지인 소비 집중 지역(포남동 일대)이 명확히 구분된다. "아, 이게 진짜 동네 맛집 위치구나"라고 직관적으로 이해한다.

---

## 사용자 여정

### 1단계: 진입
- 인스타그램 앱 광고 → 설치 (광고 카피가 pain point 정확히 자극)
- 강릉 검색 → 소비 강도 히트맵 + 현지인/관광객 소비 분리 지도 화면 즉시 납득
- "광고 없는 데이터 맛집"이라는 포지셔닝에 신뢰감 형성

### 2단계: 행동
- 포남동 현지인 소비 밀집 지역 탭 → 음식 카테고리별 소비 강도 확인 (해산물·순두부 강세)
- 지역 특산 음식 코스 생성: 아침 순두부→점심 회→카페 → 자동 생성된 동선 저장
- 음식 포토 큐레이션: 지역 음식 공식 사진 갤러리 → 인스타 감성에 맞는 비주얼 확인
- 연관 관광지: 맛집 지역 기반 주변 카페·시장 연결 → 하루 코스 자연스럽게 완성
- 앱 안에서 발견 → 코스 확정 → 포토스팟 확인까지 원스톱 완료

### 3단계: 결과
- 포남동 방문 → 관광지 주변 식당보다 가격 저렴하고 신선함 → 대만족
- 인스타에 "TasteData Korea가 찾아준 강릉 현지 맛집" 게시 → 저장 312개
- 앱 즐겨찾기 등록, 다음 여행 시 첫 번째로 열 앱으로 지정

## Pass/Fail 판정

> **✅ PASS**

TasteData Korea는 김지원에게 가장 이상적인 앱이다. "광고 없는 데이터 맛집"이라는 핵심 가치가 광고성 블로그에 지친 20대 솔로 여행자의 pain point를 정확히 해소하며, 코스 자동 생성과 포토 큐레이션이 당일치기 여행 계획 효율을 극대화한다. 유용성 9점, 종합 25점으로 높은 점수를 기록했다.

## 개선 제안

1. 인스타그램 직접 공유 기능 추가 — 앱 내 포토 큐레이션을 인스타 포맷으로 즉시 게시 가능하게 해 크리에이터 사용자의 바이럴 루프 강화
2. "예산별 코스" 필터 — 당일 2-4만원 예산 기준 맛집 조합 추천으로 가성비 여행자 니즈 직접 대응
