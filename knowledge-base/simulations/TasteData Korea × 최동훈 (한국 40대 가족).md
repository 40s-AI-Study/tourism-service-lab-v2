# SIM-193: TasteData Korea × 최동훈 (한국 40대 가족)

## 시나리오

경주 여름방학 여행을 계획 중인 동훈은 맘카페 광고성 글에 지쳐 다른 방법을 찾던 중 지인 추천으로 TasteData Korea를 설치한다. 역사 교사로서 지역 음식 문화도 교육적 소재가 된다는 점에서 관심을 갖는다. "경주 전통 음식"을 검색하면 황남빵·교동법주 등 지역 특산물 정보가 나오고, 현지인 소비 밀집 지역 히트맵이 표시된다. 8세·11세 자녀가 먹을 수 있는 메뉴인지, 가족 단체 입장 가능한 식당인지가 동훈의 주요 관심사다.

---

## 사용자 여정

### 1단계: 진입
- 지인 추천(신뢰 기반 진입) → 설치
- 경주 검색 → 현지인 소비 지역 히트맵 + 지역 특산 음식 목록 화면 탐색
- "광고 없는 데이터 맛집"이라는 설명에 맘카페 광고 경험을 떠올리며 공감

### 2단계: 행동
- 지역 특산 음식 탭: 황남빵·쌈밥·교동법주 등 경주 대표 음식 확인 — 아이들에게 설명하기 좋은 스토리 요소
- 소비 강도 지역 탭: 황리단길 현지인 소비 밀집 확인 → 관광지 주변보다 저렴한 가족 식당 기대
- 음식 코스 생성: 점심 쌈밥정식 → 간식 황남빵 → 저녁 한우 코스 자동 생성
- 아동 친화 필터(8세 가능 여부, 어린이 메뉴 여부) 탐색 → 없음
- 음식 포토 큐레이션: 지역 음식 사진 — 동훈보다 아이들이 좋아하는 비주얼 확인 (황남빵 인기)

### 3단계: 결과
- 황리단길 방문 → 가족 단체 입장 가능한 쌈밥집 발견 (현장에서 직접 확인)
- 황남빵 체험이 아이들 인기 폭발 → 여행 만족도 상승
- 동훈 평가: "음식 지역 발굴은 도움됨. 아이들 입맛에 맞는지는 앱에서 알 수 없어 결국 현장 판단"

---

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **7** | 지역 특산 음식 발굴과 현지인 소비 지역 안내가 가족 여행 음식 계획에 실질 도움. 아동 맞춤 필터 부재가 감점 요인 |
| 사용 편의 | **6** | 중간 기술 친숙도의 동훈에게 앱 탐색이 크게 어렵지 않음. 그러나 가족 여행 관점 필터가 없어 사용 효율 제한 |
| 구현 타당성 | **8** | 지역 특산 음식 + 소비 데이터 조합 구현 가능. 아동 친화 필터는 추가 데이터 소스 필요 |

**종합 점수: 21/30**

## Pass/Fail 판정

> **✅ PASS**

TasteData Korea는 최동훈의 "광고 없는 현지 음식 정보" 니즈를 충족하며 지역 특산 음식 발굴이라는 교육적 부가가치도 제공한다. 유용성 7점, 종합 21점으로 기준 충족. 아동 친화 필터와 가족 단체 입장 정보 부재가 이 페르소나에서의 완전한 만족을 막는 주요 한계다.

## 개선 제안

1. 음식 코스 생성 시 동반자 정보 입력 옵션 추가 — "초등생 자녀 2명 포함" 선택 시 아이들이 먹을 수 있는 메뉴 위주 코스 자동 조정
2. 식당 정보에 "단체 입장 가능", "어린이 의자 구비", "아동 메뉴 있음" 태그 추가 — 가족 여행자의 현장 불확실성 해소
