# SIM-102: EuroKorea Gateway × 박민준 & 이수연 (한국 30대 커플)

## 시나리오

민준은 PM답게 앱스토어 신규 여행 앱을 주기적으로 탐색한다. EuroKorea Gateway가 "8개 API 활용, 고화질 관광 사진 갤러리" 라는 설명을 보고 설치해본다. 하지만 앱을 실행하자마자 지원 언어가 러·스·프·독 4개뿐이고, 커플 여행 플래닝, 숙소 추천, 미식 코스 기능이 전혀 없음을 확인한다. 수연이 관심 갖는 감성 숙소·오마카세 정보도 없다. 결혼 1주년을 앞두고 강원도 여행을 기획 중인 두 사람에게는 서비스 전체가 타깃 밖이다.

---

## 사용자 여정

### 1단계: 진입
- 민준이 앱스토어에서 발견 후 설치, "비주얼 갤러리가 괜찮겠다" 기대
- 실행 후 언어 선택 화면에서 한국어 없음 인지 — 영어로 탐색 시도

### 2단계: 행동
- 영어 지원이 없어 독일어 또는 스페인어 UI로 간신히 탐색
- 사진 갤러리는 고화질로 인상적이나 숙소·맛집 정보 연계 없음
- 커플 코스, 기념일 플래닝, 예약 연동 기능 부재 확인
- 수연에게 보여주니 "이거 한국어도 안 되고 우리한텐 필요 없겠다" 반응

### 3단계: 결과
- 사진 갤러리 2-3장 구경 후 이탈
- 기존 네이버 블로그 + 트리플 조합으로 복귀

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 고화질 사진 갤러리 한 가지 기능만 소소하게 유용. 커플 여행 핵심 니즈 전혀 미충족 |
| 사용 편의 | **3** | 한국어 미지원으로 UI 탐색 불편. 비주얼 중심이라 사진은 직관적 |
| 구현 타당성 | **7** | 서비스 자체 구현은 타당하나 이 페르소나 대상이 아님 |

**종합 점수: 12/30**

---

## Pass/Fail 판정

> **❌ FAIL**

박민준·이수연 커플이 원하는 기념일 코스 기획, 프리미엄 숙소 추천, 미식 예약 연동 기능이 EuroKorea Gateway에는 존재하지 않는다. 한국어 미지원과 내국인 여행 콘텐츠 부재로 인해 실질적 가치를 제공하지 못한다. 종합 12점, FAIL.

---

## 개선 제안

1. 한국인 내국인 사용자를 위한 "한국의 유럽 감성 여행지" 콘텐츠를 추가하면 뜻밖의 내국인 사용층 확보 가능 (예: 가평 유럽 테마파크, 통영 등)
2. 비주얼 갤러리에 숙소·맛집 예약 링크를 연결하면 한국 커플 사용자에게도 부분적 가치 제공 가능
