# SIM-156: PhotoHunt Korea × 김영자 (한국 시니어)

## 시나리오

영자는 딸·사위와 경주 여행을 계획 중이다. 딸이 "엄마, 이 앱이 경주 예쁜 곳 알려준대, 사진도 잘 찍을 수 있어"라며 PhotoHunt Korea를 설치해줬다. 평소 스마트폰 카메라로 여행 사진을 찍지만 "왜 나는 항상 사람들이 다 나오게 찍히나"라는 불만이 있었다. "한산한 시간에 가면 된대"라는 딸의 설명에 관심을 보였다.

---

## 사용자 여정

### 1단계: 진입
- 딸이 설치 후 기본 사용법 설명
- 앱 실행: 딸이 경주 검색 후 포토스팟 지도 열어줌
- 영자: "사진이 예쁘네" — 공모전 수상작 비주얼에 긍정 반응

### 2단계: 행동
- 딸이 불국사 클릭 → 혼잡도 예측 확인: "월요일 오전 9시 방문자 18%" 정보 딸이 설명
- 영자: "그럼 월요일 아침 일찍 가자" — 혼잡도 정보를 이해하고 일정 결정에 반영
- 촬영 팁 페이지: 작은 글씨로 "다보탑 정면 기준 10m 거리에서 촬영" 등 설명 — 글씨 너무 작아 읽기 어려움
- 딸이 대신 읽어주며 "여기 서면 된대요, 어머니" 안내
- 앱 직접 조작은 딸이 대부분 담당, 영자는 결과 정보만 수신

### 3단계: 결과
- 월요일 오전 불국사 방문 → 한산한 타이밍에 가족 단체 사진 성공
- 영자: "이렇게 사진이 잘 나온 건 처음이야" 만족 표현
- 앱 직접 사용은 없고, 딸이 조작해주는 방식으로 간접 활용
- 여행 후 딸에게 "다음에도 이 앱 써서 가자" 요청

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **6** | 혼잡도 예측 정보가 실제 일정 결정에 기여, 단 직접 사용이 아닌 딸 의존 |
| 사용 편의 | **3** | 시니어 직접 사용 불가 수준의 글씨 크기·UI 복잡도, 가족 동반 시에만 활용 가능 |
| 구현 타당성 | **7** | 앱 자체 기능은 타당, 시니어 접근성 모드 부재가 직접 사용 장벽 |

**종합 점수: 16/30**

## Pass/Fail 판정

> **⚠️ CONDITIONAL PASS**

김영자는 PhotoHunt Korea의 직접 타깃이 아니지만, 딸을 매개로 한 "가족 동반 간접 사용" 유스케이스에서 부분적 가치를 창출했다. 혼잡도 예측 기능이 실제 여행 결정에 영향을 미쳤다는 점은 시니어 간접 사용자가 앱 가치를 체험할 수 있음을 보여준다. 시니어 UI 모드(큰 글씨, 단순 탐색)를 추가하면 직접 사용층으로 전환 가능하다.

---

## 개선 제안

1. 시니어 친화 UI 모드: 글씨 크기 확대, 버튼 단순화, 핵심 정보(혼잡도·추천 시간)를 큰 카드 형태로 표시하는 접근성 모드 옵션 추가
2. 가족 공유 기능: 딸이 생성한 포토 투어 코스를 가족 그룹 카카오톡으로 공유 시 어른도 이해하기 쉬운 요약 형태로 전달되는 공유 포맷 제공
