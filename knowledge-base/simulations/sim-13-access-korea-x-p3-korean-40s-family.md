# SIM-13: AccessKorea × 최동훈 (한국 40대 가족)

## 시나리오

최동훈(44세, 역사 교사)은 여름방학 경주 여행에 장인어른(72세, 지팡이 사용)이 합류하게 됐다. 아이들(초5·초2)도 있어 "모든 세대가 함께할 수 있는 동선"이 절실하다. 맘카페에서 "노부모 동반 경주 여행"을 검색하다 AccessKorea를 발견한다.

---

## 사용자 여정

### 1단계: 진입
- 앱 실행 → 온보딩: "지팡이 사용자 동행" + "아동 포함(8세·11세)" 선택
- "경주, 2박 3일, 노부모+아동 동반" 입력

### 2단계: 행동
- AI 무장애 코스 생성:
  ```
  [1일차 — 역사 탐방 (평지 중심)]
  09:00 불국사 (휠체어·지팡이 전용 우회로 안내, 경사도 표시)
  → 주차장에서 입구까지 경사로·벤치 위치 지도 표시
  12:00 무장애 인증 식당 (테이블 높이·장애인 화장실 안내)
  14:00 경주 국립박물관 (엘리베이터 완비, 아동 전시관 별도)
  17:00 숙소 (1층 방 요청 안내 포함)

  [2일차 — 체험 중심]
  09:00 대릉원 (완전 평지, 노부모 최적)
  11:00 도예 체험 공방 (좌식 체험, 휠체어 접근 가능)
  14:00 첨성대 (외부 관람, 접근로 평지)

  [3일차 — 이동]
  09:00 황리단길 카페 (계단 없는 매장 필터)
  11:00 귀가
  ```
- 오디오 가이드: 아이들용 한국어 아동 해설은 없으나 성인 해설 제공
- 각 장소마다 "지팡이 사용 가능 구간 100%", "계단 없음" 등 접근성 등급 표시
- 동훈이 미리 코스를 PDF로 인쇄해 장인어른께 드림

### 3단계: 결과
- 장인어른이 불국사 전망대까지 무리 없이 이동 성공
- "아버지가 불국사 처음 오셨는데 계단 안 오른다고 미안해하셨는데 전망대도 충분히 멋있었어요" → 앱 리뷰 5점
- 아이들 체험은 별도 검색 필요했으나 전반적으로 크게 만족
- 다음 여행에도 반드시 사용하겠다는 의사 표명

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **10** | 노부모 동반 가족 여행의 가장 큰 Pain Point(무장애 정보 부재)를 완벽 해소 |
| 사용 편의 | **8** | 동반자 조건 입력이 직관적. 접근성 등급 표시로 현장 당황 방지 |
| 구현 타당성 | **8** | `barrier-free-travel` + `audio-guide` + `central-attractions-by-municipality` 실제 API 조합 가능 |

**종합 점수: 26/30**

---

## Pass/Fail 판정

> **✅ PASS (핵심 유즈케이스)**

P3 최동훈은 AccessKorea의 가장 중요한 간접 타겟 사용자다. 장애인·시니어 본인이 아닌 "거동 불편한 가족을 동반한 일반 여행자" 세그먼트가 실제로는 가장 큰 시장이다. 이 유즈케이스에서 AccessKorea는 대체재가 없는 필수 서비스.

## 개선 제안

1. **아동 연령별 해설 연동**: 아이 나이 입력 시 눈높이 맞는 역사 해설 음성 제공 → P3 완전 충족 (현재 gap)
2. **세대별 동선 분기**: "노부모는 대릉원 쉬기, 아이들은 체험공방" 동시 진행 시 두 코스 병렬 표시 기능
3. **사용자 접근성 리뷰**: "불국사 경사로 현재 상태 양호" 크라우드소싱 업데이트 → 공공데이터 오류 실시간 보정
4. **인쇄 최적화**: 노부모·아이 동반 가족은 디지털 앱보다 인쇄물 선호 → PDF 내보내기 기능 추가
