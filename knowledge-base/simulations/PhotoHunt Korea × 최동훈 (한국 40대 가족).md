# SIM-153: PhotoHunt Korea × 최동훈 (한국 40대 가족)

## 시나리오

동훈은 여름방학 경주 가족 여행을 계획 중이다. 역사 교사답게 좋은 사진으로 여행의 기억을 남기고 싶고, 특히 아이들이 즐거워하는 순간을 찍고 싶다. 배우자가 네이버 블로그에서 PhotoHunt Korea를 소개한 글을 보고 "이 앱이 경주 포토스팟 알려준대"라며 공유했다. 동훈은 새 앱 설치에 다소 보수적이지만, "한국관광공사 공식 인증"이라는 문구에 신뢰감을 느끼고 설치를 결정했다.

---

## 사용자 여정

### 1단계: 진입
- 네이버 블로그 후기 → 배우자 공유 → 동훈 설치
- 온보딩: "가족 여행", "역사·문화", "자연" 선택 — "가족 여행" 카테고리 존재에 긍정적
- 경주 검색 → 불국사·첨성대·동궁과 월지 포토스팟 지도 표시

### 2단계: 행동
- 불국사 선택 → 공모전 수상작 사진 확인: 석가탑 앞 특정 각도 촬영 팁 — "아이들이랑 여기서 찍으면 되겠다" 메모
- `visitor-concentration-forecast`: 불국사 토요일 오전 10시 방문자 68%, 평일 오전 9시 22% — 방학 중 평일 일정 확인
- 포토 투어 코스 생성 → 아이 동반 필터 없음, 계단 많은 코스에 무장애 정보 미제공 — 실망
- 동궁과 월지 야경 스팟: 조명 최적 시간 "일몰 후 30분" 촬영 팁 — 구체적이고 유용
- 아이들 연령(8세, 11세)에 맞는 스팟 필터 기능 없음 확인

### 3단계: 결과
- 야경 촬영 팁과 혼잡도 정보는 실제 일정 수립에 활용
- 아이 맞춤 필터 없어 앱 활용도 제한적 — 50% 정도만 원하는 정보 획득
- 앱 유지하나 주요 정보 탐색은 여전히 네이버 블로그 병행

| 사용 편의 | **6** | 중간 기술 친숙도 사용자에게 무난한 UX, 가족 여행 맞춤 기능 부족 |
| 구현 타당성 | **8** | 핵심 API 구현 타당, 가족 여행 레이어 추가가 과제 |

## Pass/Fail 판정

> **⚠️ CONDITIONAL PASS**

최동훈은 PhotoHunt Korea의 부분적 타깃이다. 포토스팟 큐레이션과 혼잡도 예측 기능에서 실질적 가치를 얻었지만, 가족 여행 특화 기능(아이 동반 필터, 접근성 정보)이 없어 완전한 만족에 이르지 못했다. 가족 여행자는 한국 관광 시장에서 큰 비중을 차지하므로, 기본 기능 위에 "가족 모드" 레이어를 추가하면 PASS로 전환 가능하다.

## 개선 제안

1. 가족 모드 추가: 아이 연령 입력 시 연령 적합 포토스팟 필터링 및 "아이와 함께 찍기 좋은 각도" 촬영 팁 별도 제공
2. 접근성 레이어 통합: 포토스팟 상세 페이지에 경사로·계단 여부 간단 표시로 노부모·유모차 동반 가족의 사전 판단 지원
