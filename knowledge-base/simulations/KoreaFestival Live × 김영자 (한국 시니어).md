# SIM-166: KoreaFestival Live × 김영자 (한국 시니어)

## 시나리오

영자는 딸과 함께 봄 나들이를 계획 중이다. 작년에 유채꽃 축제를 갔다가 비포장 길에 무릎이 너무 아팠던 기억이 있어, 이번엔 "걸을 수 있는지"를 미리 확인하고 싶다. 딸이 KoreaFestival Live를 스마트폰에 설치해주고 "어머니, 여기서 축제 찾아봐요"라고 안내한다. 큰 글씨 모드로 전환하자 화면이 한결 편해진다. "봄꽃 축제"를 검색하니 함양산삼항노화엑스포와 고령대가야축제가 나타난다. 딸이 "무장애 접근 가능" 필터를 켜주자 경사로와 포장도로가 확보된 축제만 남아, 영자는 안심하고 방문 결정을 내린다.

---

## 사용자 여정

### 1단계: 진입
- 딸이 앱 설치 및 큰 글씨 모드 설정 → 영자 혼자서도 버튼 탭 가능한 UI 확인
- "봄꽃" 테마 필터 탭 → 지역별 봄 축제 목록 카드 형태로 크게 표시
- 딸이 "무장애 접근" 필터 추가 적용 → 결과가 경사로·포장도로 확보된 축제로 자동 좁혀짐

### 2단계: 행동
- 고령대가야축제 선택 → 상세 페이지에서 "휠체어·보행 보조기 접근 가능: ✅" 아이콘 확인
- 주차장에서 주요 전시장까지 경사로 경로 지도로 표시 — 영자가 직접 보며 "여기는 걸을 수 있겠네" 안도
- 혼잡도 예측: 토요일 오전 10~12시 혼잡도 낮음(38%) → 해당 시간대 방문 결정
- "반나절 가족 코스" 생성 → 가야 유물 전시 → 체험 공방 → 근처 식당 자동 연결
- 코스 내 이동 거리 총 1.2km(평지 기준) — 영자가 감당 가능한 수준 표시

### 3단계: 결과
- 방문 당일 앱 지도 따라 무장애 경로로 전시장 진입 성공
- 딸이 "어머니 혼자서도 앱 보면서 잘 다니셨어요"라고 후기 네이버 카페 게시
- 영자가 노인 여행 모임에 앱 소개 → 시니어 사용자 구전 확산

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **8** | 무장애 필터 + 경사로 경로 지도가 영자의 핵심 불안을 사전에 해소. 혼잡도 예측으로 여유 있는 방문 가능 |
| 사용 편의 | **6** | 큰 글씨 모드·단순 UI는 시니어 접근성을 높이나, 초기 설치 및 설정은 여전히 가족 도움이 필요. 독립 사용 진입장벽 존재 |
| 구현 타당성 | **7** | 무장애 경로 데이터는 별도 크라우드소싱 또는 지자체 협력 필요 — 전국 축제 완전 커버에는 시간이 걸림 |

**종합 점수: 21/30**

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live는 김영자의 가장 큰 여행 불안인 "현장 가서야 계단을 발견"하는 문제를 무장애 필터와 경로 지도로 사전에 해소한다. 시니어 친화 UI(큰 글씨, 단순 구조)가 갖춰진다면 딸의 도움으로 충분히 사용 가능하며, 노인 여행 모임 구전 효과로 시니어 사용자층 확대도 기대할 수 있다.

## 개선 제안

1. 시니어 전용 "쉬운 모드" 온보딩 추가: 가족이 한 번 설정해두면 이후 영자 혼자서도 "이번 주 접근 쉬운 축제"를 큰 버튼 하나로 확인할 수 있는 단순화된 홈 화면 제공
2. 무장애 경로의 이동 거리·예상 소요 시간을 노인 보행 속도 기준(시속 2~3km)으로 계산해 표시 — "총 1.2km, 어르신 기준 약 30분"처럼 구체적 안내가 불안 감소에 결정적으로 기여
