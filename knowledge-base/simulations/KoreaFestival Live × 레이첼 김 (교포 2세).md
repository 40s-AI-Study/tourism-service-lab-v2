# SIM-168: KoreaFestival Live × 레이첼 김 (교포 2세)

## 시나리오

래희는 10살 딸과 함께 처음으로 부모님 고향인 전주를 방문하기 위해 한국에 왔다. 딸에게 "진짜 한국 문화"를 보여주고 싶어 전통 축제를 찾고 있다. 교포 커뮤니티 KoreAm 뉴스레터에서 KoreaFestival Live 앱 소개 기사를 읽고 설치한다. 영어로 전주 지역 축제를 검색하니 전주한지문화축제와 전주대사습놀이 정보가 상세히 뜬다. 건축가로서 한옥 관련 전통 공예 체험이 함께 연계되는 코스가 자동 생성되는 것을 보고 "내가 찾던 게 이거야"라는 반응을 보인다. 딸(10세) 연령 맞춤 체험 프로그램까지 분리 표시되어, 모녀가 각자의 관심사에 맞는 시간을 보낼 수 있다는 점이 특히 마음에 든다.

## 사용자 여정

### 1단계: 진입
- KoreAm 뉴스레터 링크 → 앱스토어 설치 → 영어 UI 자동 설정
- "전주" 지역 + "전통문화" 테마 필터 → 전주한지문화축제 상단 노출
- tourism-info-english API 기반 영어 축제 배경 설명 — 한지 제조의 역사적 맥락 상세 기술

### 2단계: 행동
- 축제 상세 페이지에서 "어린이 체험 프로그램" 탭 분리 확인: 한지 만들기(6~12세 적합) 예약 가능
- 건축가 관심사 연계: "주변 관광" 탭에서 전주 한옥마을 건축 투어(영어 해설 포함) 코스 자동 연결
- 혼잡도 예측: 평일 오전 혼잡도 낮음(28%) → 모녀가 여유롭게 체험 가능한 시간대 확인
- "2일 코스" 생성 → 한지 체험(딸) + 한옥 건축 해설(래희) → 전통 비빔밥 식사 → 오목대 역사 탐방 자동 배치
- 지역 다양성 지수 ★★★★★ — 전주 일대 문화 다양성 최고 등급 확인

### 3단계: 결과
- 모녀 모두 각자의 관심사 충족 — 딸은 한지 체험, 래희는 한옥 건축 구조 탐구
- 래희가 부모님께 영상 통화로 현장 보여드리며 감동 공유
- 교포 커뮤니티 페이스북 그룹에 "이 앱 덕분에 진짜 한국 문화 체험했어요" 후기 게시

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live는 래희의 "표면적 관광 정보가 아닌 깊이 있는 한국 문화 탐구" 니즈를 전통 축제 + 영어 상세 해설 + 한옥 투어 연계로 충족한다. 딸 연령 맞춤 체험 분리 표시는 모녀 여행이라는 특수한 맥락에서 특히 높은 가치를 발휘한다. 교포 커뮤니티 내 신뢰 구전 효과도 기대할 수 있다.

## 개선 제안

1. "교포 특화 모드" 추가: 한국어 읽기 불편한 교포를 위해 영어 기본 + 한국어 병기 옵션을 제공하고, "부모님 고향 탐방" 테마 코스를 별도 큐레이션하면 래희 같은 정체성 여행자에게 결정적 가치 제공
2. 어린이 동반 시 체험 프로그램 사전 예약 기능을 앱 내 직접 연동 — 현장 대기 없이 원하는 프로그램에 참여할 수 있도록 함으로써 딸의 체험 만족도와 래희의 계획 실현도 동시에 향상
