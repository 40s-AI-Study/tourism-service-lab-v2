# SIM-97: K-Camp Finder × 이재호 (전동휠체어 개발자)

## 시나리오

이재호(38세, IT 개발자)는 전동휠체어를 사용하는 장애인 여행자다. 자연 속에서 시간을 보내고 싶은 욕구가 있고, 글램핑(침대·화장실 완비 숙박)은 현실적으로 가능한 선택지다. 그러나 가장 큰 문제는 접근성 정보의 부재다 — 캠핑장 입구부터 숙소까지 휠체어 경사로가 있는지, 화장실 문 너비가 충분한지, 전동휠체어 충전 콘센트가 있는지 등의 정보를 어떤 플랫폼에서도 찾기 어렵다.

K-Camp Finder에서 "글램핑 / 2인 / 장애인 접근성" 필터 검색을 시도한다.

---

## 사용자 여정

### 1단계: 진입

- 장애인 여행 커뮤니티(인스타그램 #휠체어여행) 에서 K-Camp Finder 언급 발견
- 앱 설치, 카카오 로그인
- 온보딩: 접근성 관련 설정 항목 탐색

### 2단계: 행동

- "장애인 접근성" 또는 "휠체어 접근 가능" 필터 검색 시도
- 필터 항목 없을 경우: 각 캠핑장 상세 페이지에서 접근성 정보 수동 탐색
- gocamping API 원본 데이터에 휠체어 접근성 항목 포함 여부 불확실
- 접근성 정보 미제공 캠핑장이 대다수일 경우 이탈
- 글램핑 후보 2-3곳 전화 문의로 전환 (앱 외부 행동)

### 3단계: 결과

- 앱 내에서 최종 예약 미완료
- 전화 확인 후 접근성 확인된 1곳 예약 (앱 외부)
- 앱에 "휠체어 접근 가능 여부를 표시해달라"는 피드백 남김

---

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **4** | 글램핑 탐색 자체는 가능하나 접근성 정보 부재로 핵심 의사결정을 앱 내에서 완결하지 못함 |
| 사용 편의 | **5** | 일반 UI 사용은 가능하나 접근성 특화 필터·정보 구조 없음 |
| 구현 타당성 | **6** | gocamping API 접근성 데이터 보강 + 캠핑장 자체 신고 시스템으로 구현 가능, 중장기 로드맵 과제 |

**종합 점수: 15/30**

---

## Pass/Fail 판정

> **❌ FAIL**

전동휠체어 사용자는 글램핑 이용 동기가 있으나 접근성 정보 공백이 결정적 장벽이다. 유용성 4점, 종합 15점으로 Pass 기준 미달. 단, 접근성 필터를 추가하면 MEDIUM 적합도로 상승 가능한 잠재력 있는 페르소나.

---

## 개선 제안

1. 장애인 접근성 필터 신설: 휠체어 경사로, 화장실 문 너비(80cm 이상), 전동휠체어 충전 콘센트, 장애인 전용 주차 여부를 체크리스트화하고 gocamping API 미제공 데이터는 캠핑장 직접 신고 및 방문자 확인으로 보완
2. 장애인 여행 커뮤니티 협업: 휠체어 여행 인플루언서·단체와 파트너십으로 접근성 검증 리뷰 콘텐츠 확보, 신뢰도 높은 UGC 생태계 구축
