# SIM-129: K-BeautyMed Tour × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

아미나는 말레이시아 K-뷰티 인플루언서 커뮤니티에서 K-BeautyMed Tour를 소개하는 릴스를 봤다. "강남 피부과 + 올리브영 투어 원스톱"이라는 내용이 자신의 팔로워들에게 공유할 콘텐츠로 딱 맞겠다고 생각해 설치했다. 영어 UI가 지원되어 언어 장벽은 없었다. 그러나 가장 중요한 두 가지가 없었다. 첫째, K-뷰티 제품의 할랄 성분 필터가 없다. 둘째, 피부과 클리닉 중 무슬림 여성을 위한 개인 공간(시술 중 히잡 관련) 정보가 없다. 피부과 시술에 관심이 있었지만 이 두 가지 정보 없이는 예약할 수 없었다.

---

## 사용자 여정

### 1단계: 진입
- 말레이시아 K-뷰티 인플루언서 릴스 추천 → 영어 UI 확인 후 즉시 설치
- K-뷰티 쇼핑 + 피부과 시술에 대한 높은 관심 — 타겟 페르소나 요건 부합

### 2단계: 행동
- 올리브영 투어 코스 탐색 → 성분 정보 없음. "이 제품 돼지 유래 성분 없어요?" 확인 불가
- 피부과 클리닉 목록 탐색 → 무슬림 친화 시술 환경(히잡 착용 상태 시술 공간, 여성 전담 의사) 정보 없음
- `visitor-concentration-forecast` 활용: 혼잡도 기반 방문 시간 확인 — 이 기능은 유용
- 포토스팟 큐레이션 탐색: 강남 뷰티 거리 인증샷 장소 발견 → 팔로워 콘텐츠 제작에 부분 활용

### 3단계: 결과
- 포토스팟·혼잡도 기능은 활용. 피부과 예약은 할랄·무슬림 환경 정보 부재로 미진행. 앱을 유지하되 시술 기능 핵심 전환 실패. 팔로워에게 "포토스팟 기능은 유용하지만 무슬림 전용 필터 없음" 솔직 리뷰 게시

## Pass/Fail 판정

> **✅ PASS**

아미나는 K-BeautyMed Tour의 핵심 타겟(동남아 외국인 여성, K-뷰티 쇼핑+시술 관심)에 해당한다. 포토스팟과 혼잡도 기능에서 실질적 가치를 찾았으나, 무슬림 친화 정보 부재로 시술 예약 전환이 막혔다. 말레이시아·인도네시아 무슬림 여성이 한국 의료관광의 주요 성장 세그먼트임을 감안하면 이 개선은 비즈니스적으로도 중요하다.

## 개선 제안

1. K-뷰티 제품 성분 정보에 할랄 인증 필터 추가 — `tourism-info-english` 기반 제품 데이터에 할랄 여부 태그 연동
2. 피부과 클리닉 상세 페이지에 "무슬림 친화 시술 환경" 정보 항목 추가 (여성 전담 의사 여부, 개인 시술실, 히잡 착용 가능 여부) — 동남아 무슬림 여성 사용자 시술 전환율 직결
