# SIM-R3-048: 치유여정 Korea × 이재호

## 시나리오

이재호는 38세, 서울 송파구에 사는 IT 개발자다. 척수 손상으로 전동휠체어를 사용한 지 11년째. 그는 코드로 세상을 바꾸는 일을 하면서도, 실제 세상에서 자신이 갈 수 있는 곳이 얼마나 제한적인지 매일 실감한다. 여행 앱들은 늘 그에게 실망을 안겨줬다 — "무장애 관광지"라고 표시된 곳에 가보면 입구 계단이 있거나, 화장실이 멀거나, 포장된 길이 50m를 넘지 않는 경우가 허다했다. 그래서 그는 여행 전 데이터를 직접 확인하는 습관이 생겼다. 경사도, 노면 재질, 휠체어 접근 가능 구간의 정확한 길이.

그가 치유여정 Korea를 발견한 건 접근성 관련 개발자 커뮤니티 슬랙에서였다. "두루누비 트레일 API + 무장애 경로 데이터 + 오디오 가이드를 결합한 앱인데, 경사도 데이터가 실제로 정확한 것 같다"는 개발자 동료의 한 줄 평. 그는 그 말을 믿지 않았다. 하지만 설치는 했다.

---

## 사용자 여정

### 1단계: 진입
- 개발자 슬랙 커뮤니티 추천 → 설치 (데이터 정확성 반신반의)
- 온보딩: 🌿쉬며 선택, 이동 수단 "전동휠체어" 설정
- 필터: 경사도 5% 이하, 포장 노면, 휠체어 접근 가능 화장실 500m 이내

### 2단계: 행동
- 추천 코스: "올림픽공원 몽촌토성 무장애 둘레길 오디오 트레일 (2.8km, 전 구간 포장, 경사 최대 3.2%)"
- 상세 페이지에서 구간별 경사도 그래프, 노면 재질, 화장실 위치를 확인 — "이 정도면 가볼 수 있겠다"
- 현장 도착: 앱 경로가 실제 경로와 일치함을 확인. 그 순간 작은 신뢰가 생겼다
- 오디오 시작: 백제 위례성의 이야기, 이 흙 언덕이 1,700년 전 도성이었다는 내레이션
- 전동휠체어 속도에 맞춰 내레이션이 자동 조정됨 (GPS 이동 속도 감지)
- 혼잡 예측: 오전 10시 현재 방문객 7명. "앞 구간 200m에 3명 분포" 실시간 안내 → 속도 조절로 자연스럽게 간격 유지
- 코스 완료 후 앱 내 경사도 데이터 피드백 제출: "3.2% 표시 구간 실측 체감 정확함. 화장실 위치 정확. 한 곳 노면 상태 사진 첨부."

### 3단계: 결과
- "데이터가 정확했다. 이건 쓸 수 있다." — 개발자로서의 기준을 통과
- 슬랙 커뮤니티에 상세 사용 후기 게시 (경사도·노면·화장실 데이터 정확도 검증 내용 포함)
- 다음 달 부산 방문 계획 시 앱에서 무장애 코스 우선 검색

## Pass/Fail 판정

> **✅ PASS**

접근성 데이터 정확성이라는 가장 높은 기준을 가진 페르소나에게 통과한 것은 의미 있는 결과다. 다만 두루누비 API 데이터 품질이 전국적으로 균일하지 않을 수 있어 지역별 데이터 신뢰도 편차 리스크가 남아 있다. 오디오 스토리가 이동 경험의 의미를 높인 것은 예상된 결과였지만, GPS 기반 내레이션 속도 조절이 휠체어 사용자에게 특히 유효했다.

## 🔄 리텐션 평가: 이 페르소나가 앱을 다시 열까?

**재사용 확률**: ⬆️높음

**감정적 이유**:
> "여행 앱이 나한테 거짓말을 너무 많이 했어. 이 앱은 첫 번째 코스에서 거짓말을 안 했어. 그게 전부야. 거짓말 안 하는 앱을 찾는 데 11년 걸렸어."

**재사용 트리거**:
- 다른 도시 방문 계획 시 무장애 코스 데이터 신뢰 기반으로 앱 우선 검색
- 개발자 커뮤니티에서 접근성 관련 논의 시 앱 경험 언급하며 재오픈
- "부산 무장애 코스 5개 추가" 알림 수신 시

**이탈 리스크**:
- 방문한 코스에서 데이터 오류 발견 시 신뢰 즉각 소멸 — 한 번의 실수가 모든 신뢰를 날림
- 전국 무장애 코스 커버리지가 좁을 경우 "원하는 지역에 코스 없음"으로 사용 빈도 제한

## 개선 제안

1. **"데이터 정확도 커뮤니티 검증" 시스템**: 이재호처럼 현장 피드백을 제출하는 사용자에게 "검증 배지" 부여. 해당 코스에 "현장 검증 완료 — 이재호(전동휠체어)" 표시로 다른 장애인 여행자에게 신뢰 신호 제공. 데이터 정확도가 앱의 가장 중요한 자산이므로 사용자 검증 체계를 게임화하여 지속 유지.
2. **구간별 접근성 세부 지도**: 코스 전체가 아닌 50m 단위 구간별 노면 상태·경사도·장애물을 시각화한 인터랙티브 지도 제공. 개발자 페르소나는 이 데이터를 보고 코스 선택 전 시뮬레이션이 가능하며, 현장 도착 전 정확한 기대 수립이 가능해짐.
