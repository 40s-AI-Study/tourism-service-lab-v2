# SIM-150: TourismCareer Korea × 타나카 하루카 (일본 직장인)

## 시나리오

하루카는 도쿄 IT 회사 마케터로 K-pop과 K-드라마 팬이다. 3박 4일 첫 한국 여행을 계획하던 중, 일본어 한국 여행 유튜브 채널에서 "한국에서 일하고 싶다면 이 앱" 이라는 영상을 보고 TourismCareer Korea를 알게 됐다. 막연히 한국 회사에서 일해보고 싶다는 꿈이 있었고, 마케터로서 관광 콘텐츠 분야 인턴십에 관심을 가졌다. 한국어는 기초 읽기 수준이라 앱 탐색에 큰 어려움을 예상하면서도 설치해봤다.

---

## 사용자 여정

### 1단계: 진입
- 일본어 유튜브 영상 → 앱스토어 설치
- 앱 실행: 일본어 UI 없음, 한국어 전용 — 한글 읽기 능력으로 간신히 메뉴 파악 시도
- "インターンシップ" 관련 메뉴를 찾다가 "인턴십" 탭 발견 (한글 인식 성공)

### 2단계: 행동
- 인턴십 탭 진입 → 관광 콘텐츠·SNS 운영 인턴십 공고 5건 노출
- 지원 자격 확인: 전부 "국내 대학 재학생 또는 졸업 예정자" — 외국인 불가
- 지역 관광 수요 데이터 탭: 차트 그래프는 언어 몰라도 수치로 어느 정도 파악
- 서울 관광 수요 데이터가 일본 대비 어떤지 비교하고 싶었으나 기능 없음
- 현장 탐방 코스 생성: 서울 홍대·이태원 코스 생성 성공 — 이 기능만 실용적으로 활용

### 3단계: 결과
- 인턴십 목적: 외국인 지원 불가로 완전 실패
- 탐방 코스 기능: 여행 일정 보조 수단으로 1회 활용
- 앱 삭제, 일본 커뮤니티에 "외국인은 쓸 수 없는 앱"이라고 언급

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 외국인 지원 불가 공고만 존재, 언어 장벽으로 탐방 기능도 제한적 활용 |
| 사용 편의 | **3** | 한글 기초 읽기로 메뉴 간신히 파악, 일본어 UI 전무, 실사용 심각히 제한 |
| 구현 타당성 | **6** | 앱 기능 구현 자체는 타당, 외국인 타깃 확장 시 즉각 가치 창출 가능 |

**종합 점수: 11/30**

## Pass/Fail 판정

> **❌ FAIL**

하루카는 TourismCareer Korea의 구조적 비타깃 사용자다. 외국인 취업 지원 불가와 일본어 UI 부재가 이중으로 작용해 앱의 어떤 핵심 기능도 실질적으로 활용하지 못했다. 일본에서의 부정적 구전("외국인은 쓸 수 없는 앱")은 브랜드에도 해가 된다. 글로벌 확장 전략이 없다면 일본·대만 등 외국 사용자 유입 차단이 오히려 평판 관리에 유리하다.

## 개선 제안

1. 외국인 K-취업 정보 허브 기능: 워킹홀리데이 비자 소지자 또는 한국 거주 외국인 대상 관광 분야 공고를 별도로 수집하여 일본·대만 등 K-culture 팬덤 국가 대상 마케팅 전개
2. 앱스토어 국가별 타깃 설정 및 설명 현지화: 현재 일본 앱스토어에서 노출 시 오인 다운로드 발생 — 한국 앱스토어 전용 배포 또는 외국인 지원 가능 여부를 첫 화면에 명시
