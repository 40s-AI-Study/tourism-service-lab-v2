# SIM-169: KoreaFestival Live × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

아미나는 친구 2명과 함께 5박 6일 한국 여행을 계획 중이다. K-drama에서 봤던 화려한 축제 장면을 직접 경험하고 싶지만, 축제 현장의 음식 부스가 할랄인지 확인할 방법이 없었다. 말레이시아 무슬림 한국 여행 인스타그램 계정에서 KoreaFestival Live 앱 소개 포스팅을 보고 설치한다. 영어 UI에서 "Muslim-friendly" 필터를 발견하는 순간 기대감이 높아진다. 서울 등불축제(서울빛초롱축제)를 검색하니, 축제 주변 할랄 인증 음식 부스 위치와 인근 무살라(기도실) 정보가 함께 표시된다. 이 정보 하나만으로도 "이 앱이면 믿고 갈 수 있겠다"는 신뢰가 형성된다.

---

## 사용자 여정

### 1단계: 진입
- 말레이시아 무슬림 여행 인스타 계정 추천 → 앱 설치 → 영어 UI 자동 전환
- 온보딩에서 "Muslim-friendly" 프리셋 선택 → 앱 전체에 할랄 필터 적용
- 서울 지역 겨울 축제 검색 → 서울빛초롱축제(청계천) 상단 노출

### 2단계: 행동
- 축제 상세 페이지 내 "Muslim-Friendly Info" 탭 확인:
  - 할랄 인증 음식 부스: 3곳 위치 지도 표시
  - 인근 무살라(기도실): 청계천 광장 서울YMCA 내 1층 (도보 5분)
  - 음식 성분 주의 정보: 돼지고기 비함유 확인 부스 별도 표시
- 혼잡도 예측: 토요일 오후 6~8시 혼잡도 85% → 일요일 오후 5시 출발로 조정
- 관련 관광 코스: 청계천 산책 → 인사동 전통 공예 → 할랄 식당 → 광화문 광장 자동 연결
- 마그립 기도 시간(일몰 기준) 자동 계산 및 코스 중간 기도 시간 알림 설정

### 3단계: 결과
- 기도 시간에 맞춰 코스 이동, 할랄 음식으로 식사, 사진 콘텐츠 제작까지 차질 없이 완료
- 인스타그램에 "무슬림도 한국 축제 완벽히 즐길 수 있어요" 릴스 게시 → 30만 뷰 달성
- 말레이시아 무슬림 여행 커뮤니티에 앱 링크 공유 → 동남아 무슬림 사용자 집단 유입

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live의 Muslim-friendly 기능은 아미나가 한국 축제 여행을 결정하는 데 결정적 역할을 한다. 할랄 부스 위치와 무살라 정보의 결합은 동남아 무슬림 관광객이라는 빠르게 성장하는 시장을 공략하는 강력한 차별화 포인트다. 다만 데이터 수집 체계가 일반 축제 정보보다 복잡해 구현 타당성이 낮은 것이 리스크다.

## 개선 제안

1. 할랄 축제 음식 부스 데이터를 한국이슬람교중앙회(KMF) 등 공인 기관과 파트너십으로 확보하고, 인증 마크를 시각적으로 명확히 표시 — 신뢰도 기반 전환 핵심
2. 기도 시간 알림을 코스 동선에 자동 삽입하는 기능("마그립 기도 30분 전 알림 → 근처 무살라 안내") 추가로 무슬림 여행자의 앱 이탈 없이 전 여행 동안 사용 지속성 확보
