# SIM-R3-026: KoreaWellness v2 × 박민준 & 이수연

## 시나리오

박민준(33세, IT 컨설턴트)과 이수연(31세, 마케터)은 만난 지 5년째인 커플이다. 이번 주말이 5주년 기념일이다. 민준은 한 달 전부터 뭔가 특별한 걸 준비하고 싶었는데 막상 찾아보면 늘 비슷한 것들 — 호텔 스파 패키지, 제주 펜션, 남이섬. 수연이 작년에 "온천 가보고 싶다"고 했던 말이 기억났다. 단순히 온천만이 아니라, 거기서 아무 생각 없이 하루를 보내는 것.

민준은 회사 동료가 "요즘 이 앱 진짜 좋다"며 추천한 KoreaWellness v2를 설치했다. 수요일 밤, 수연 몰래 기념일 코스를 짜려고 혼자 앱을 열었다.

첫 화면의 질문: "지금 가장 필요한 게 뭐예요?" 민준은 수연을 생각하며 🌿 **고요** 를 눌렀다가, "아니 둘 다 즐겨야지" 싶어 화면을 다시 보다가 🔥 **열기** 도 눌렀다. 고요 + 열기. 앱이 "회복하면서 몸도 깨우는 여정"으로 추천 카테고리를 재편했다.

---

## 사용자 여정

### 1단계: 진입
- 동료 추천 → 기념일 코스 사전 기획 목적으로 설치
- 🌿 고요 + 🔥 열기 복수 선택 → "온천·스파 + 자연 활동" 조합 추천 활성화
- 인원 설정: 커플(2인), 예산: 30만원 이내, 1박 2일

### 2단계: 행동
- 추천 1순위: 충북 수안보 온천 + 월악산 둘레길 1구간 조합 패키지 — "커플 힐링 코스 자동 생성" 기능으로 동선, 시간표, 식사 포인트 자동 구성
- 수안보 온천 숙소 3곳 비교 탭: 프리미엄 온천탕 여부, 커플 룸 유무, 조식 포함 여부 한눈에 비교
- 방문자 집중률 예측 API 기반 "기념일 주말 예상 혼잡도" 표시 — 토요일 오후가 최고점, 일요일 오전이 최저점 → 동선 역순으로 재구성 추천
- 월악산 둘레길 1구간 상세: 총 4.2km, 경사 완만, 중간 전망대 포토스팟 2곳, 소요 2시간
- 앱 내 숙소 예약 + 웰컴 메시지 옵션("5주년 기념 문구를 숙소에 전달") 선택
- 수연에게 "이번 주말 나만 따라와" 문자 한 통

### 3단계: 결과
- 토요일 저녁 온천탕에서 수연: "어떻게 이런 데 알았어?"
- 일요일 오전 둘레길, 전망대에서 사진 한 장
- 민준: "이 앱이 코스를 다 짜줬어" → 수연: "앱이 너보다 낫네"(웃음)
- 귀가 후 수연이 앱을 직접 설치
- 민준 앱 리뷰: "기념일 계획을 혼자 다 짜줬다. 진짜 고맙다."

## 리텐션 평가: 이 페르소나가 앱을 다시 열까?

**재사용 확률**: 🔥매우 높음

**감정적 이유**:
> "이 앱이 기념일을 만들어줬다. 다음 기념일도, 그다음도, 이걸로 짤 것 같아."

**재사용 트리거**:
- 매년 기념일 + 생일 + 명절 연휴 등 커플 여행 기획 시
- 수연이 앱을 설치한 후 "이번엔 내가 짤게"로 역할 전환
- 친구 커플에게 "우리 이 앱으로 갔어" 추천 → 바이럴

**이탈 리스크**:
- 코스 자동 생성이 실제 방문 후 정보와 다르면 (휴업, 혼잡도 오류 등) 신뢰 급락
- 숙소 예약 연동이 실패하거나 웰컴 메시지가 전달 안 되면 기념일 망칠 수 있어 치명적 이탈

## 개선 제안

1. **기념일 전용 "D-Day 플래너"**: 기념일 날짜 입력 시 1주 전부터 알림 + 추천 코스 자동 생성, 숙소 잔여 객실 실시간 모니터링
2. **커플 공유 여정 보드**: 두 사람이 각자 앱에서 찜한 장소를 실시간 공유 보드에서 합쳐보고 투표로 최종 코스를 정하는 협업 기능
