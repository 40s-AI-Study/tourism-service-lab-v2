# SIM-CFA-01: Crowd-Free Alternative × 박지은 (KR 30대 직장인)

## 시나리오

박지은(32세, IT 스타트업 마케터)은 남자친구와 이번 주말 서울 고궁 데이트를 계획 중이다. 지난 달 경복궁에서 인파에 치여 사진도 못 찍고 돌아온 기억이 트라우마로 남아 있다. 지인에게 Crowd-Free Alternative 앱을 추천받아 설치한다.

---

## 사용자 여정

### 1단계: 진입 (앱 설치 → 온보딩)
- 앱스토어에서 "혼잡도" 검색 → Crowd-Free Alternative 발견, 평점 4.6 확인 후 설치
- 온보딩: "어느 관광지가 혼잡한지 알고 싶으세요?" → "네" 선택
- 관심 카테고리: "궁궐/역사", "카페거리", "자연/공원" 선택
- 위치 권한 허용 → 홈 화면: 현재 위치(성동구) 주변 혼잡도 대시보드 표시

### 2단계: 혼잡도 확인
- 검색: "경복궁" → 혼잡도 예측 타임라인 즉시 로딩
- 이번 주 토요일 오전 11시: 🔴 매우 혼잡 (예측 방문자 2.1만명, 혼잡도 82%)
- 혼잡 원인 태그: "어린이날 연휴", "날씨 맑음 예보", "학교 단체 소풍 시즌"
- "지난 주 같은 시간: 혼잡도 74%, 이번 주 더 붐빌 것으로 예측" 알림 뱃지

### 3단계: 대안 탐색
- 화면 하단 "대안 명소 보기" 버튼 탭 → 자동 추천 리스트 표시
  1. **창덕궁 비원** — 🟢 혼잡도 31%, 0.8km, 최적 방문: 일 오전 9-10시
  2. **경희궁** — 🟢 혼잡도 11%, 1.3km, 지금 바로 가도 OK
  3. **덕수궁 돌담길** — 🟡 혼잡도 48%, 2.0km, 오후 3시 이후 권장
- 창덕궁 비원 카드 탭 → "경복궁과 비슷한 조선 왕궁 분위기, 비원 비밀 정원이 하이라이트" 설명
- 인스타그램 스타일 사진 미리보기 3장 (앱 내 기존 방문자 공유 사진)

### 4단계: 결정 & 계획
- "창덕궁 비원으로 가자!" 결정 → "이 날짜로 저장" 탭
- 최적 방문 시간 제안: "일요일 오전 9시~10시 (혼잡도 18%, 이번 주 최저)"
- 주변 맛집 추천: "창덕궁 근처 비혼잡 카페 3곳" (평일 오후 대비 비교)
- 남자친구에게 카카오톡 공유: "이번 일요일 창덕궁 비원 어때? 경복궁보다 훨씬 덜 붐빌 것 같아 🌿"

### 5단계: 현장 + 피드백
- 일요일 오전 9시 앱 알림: "창덕궁 비원 현재 혼잡도 22% — 예측(18%)과 유사, 쾌적합니다 ✅"
- 실제 방문: 여유로운 정원에서 사진 여러 장, 대기열 없이 입장
- 방문 후 피드백: 슬라이더로 "여유로움 쪽으로 당김" → +50 포인트 적립
- 인스타그램에 "사람 없는 창덕궁 비원 강추 🙌 이 앱 덕분에 발견" 업로드

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **10** | 핵심 Pain Point(혼잡 회피 + 대안 발견)를 완벽 해결. 이전 경복궁 실패 경험을 직접 해소 |
| 사용 편의 | **9** | 온보딩 3단계, 대안 추천 1탭, 공유 1탭. 직관적. 카테고리 필터가 더 있으면 10점 |
| 구현 타당성 | **9** | KTO visitor-concentration-forecast API 실존. related-attractions API 실존. 충분히 구현 가능 |

## Pass/Fail 판정

> **✅ PASS**

박지은은 이 서비스의 핵심 타겟 사용자다. 혼잡 회피 니즈가 명확하고, 디지털 리터러시가 높아 앱을 즉시 활용한다. 대안 명소 추천 → 계획 저장 → 현장 확인 → 피드백 전 사이클이 자연스럽게 완료된다. 인스타그램 공유로 바이럴 효과까지 기대할 수 있어 서비스 성장에도 기여한다.

## 개선 제안

1. **혼잡 원인 태그 확장**: 현재 일반적 태그 외에 "이번 주 유명 유튜버 방문 예고" 같은 실시간 이벤트 연동 시 예측 정확도 대폭 향상
2. **커플/그룹 모드**: "2인 데이트 코스 추천" 필터 추가 → 박지은 같은 타겟에게 더 개인화된 UX
3. **비혼잡 인증 뱃지**: "혼잡도 20% 이하에서 방문" 인증 스탬프 수집 기능 → 재방문 동기 부여
