# SIM-199: TasteData Korea × 아미나 사이드 (말레이시아 무슬림)

## 시나리오

아미나는 한국 여행 전 TasteData Korea가 "빅데이터 기반 로컬 맛집 큐레이터"이고 "알레르기·할랄 필터 적용"이 기능으로 나열되어 있다는 것을 앱스토어 설명에서 확인하고 설치한다. 무슬림 여행자로서 할랄 필터가 실제로 작동하는지 여부가 이 앱의 가치를 결정한다. 서울을 검색해 할랄 필터를 활성화하면 결과가 극도로 제한적이거나 "데이터 없음"으로 나타난다. 현재 한국 관광공사 API에 할랄 인증 데이터가 체계적으로 구축되어 있지 않다는 구조적 한계가 그대로 노출된다.

---

## 사용자 여정

### 1단계: 진입
- 앱스토어 설명의 "할랄 필터" 문구 → 설치 결정 (말레이시아 무슬림 여행자에게 핵심 키워드)
- 영어 UI 전환 → 서울 검색 → 소비 강도 히트맵 탐색
- 할랄 필터 버튼 찾음 → 활성화 시도

### 2단계: 행동
- 할랄 필터 활성화 → 결과: 이태원 무슬림 타운 일대 소수 지역만 표시, 대부분 지역 공백
- "할랄 인증 식당 데이터가 부족합니다" 안내 문구 확인 → 기대 크게 하회
- 비무슬림 음식 코스 화면: 삼겹살·돼지국밥 등 돼지고기 포함 음식이 소비 강도 상위 — 아미나에게 전혀 해당 없음
- 해산물·채소 기반 음식 필터 시도 → 할랄과 무관하게 돼지 제외 필터 자체가 없음
- 포토 큐레이션 기능은 확인 → 음식 비주얼은 아름답지만 먹을 수 있는지 알 수 없어 사용 불가

### 3단계: 결과
- TasteData Korea 사용 중단 → HalalTrip 앱으로 전환
- 인스타 팔로워 4만 5천 명에게 "이 앱 무슬림에게 도움 안 됨" 솔직한 후기 게시 → 부정적 바이럴
- 아미나 최종 평가: "할랄 필터 기능을 광고했는데 실제로는 데이터가 없어 사실상 무의미"

---

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **2** | 할랄 필터가 유명무실하고, 음식 코스 대부분이 돼지고기 포함. 무슬림 여행자에게 실질적 가치 거의 없음 |
| 사용 편의 | **6** | 영어 UI 자체는 사용 가능. 그러나 원하는 기능(할랄 필터)이 작동하지 않아 편의 점수가 의미 없음 |
| 구현 타당성 | **4** | 할랄 데이터 구축 없이 할랄 필터를 기능으로 나열한 것 자체가 문제. 앱스토어 설명과 실제 기능 간 괴리가 신뢰를 훼손 |

**종합 점수: 12/30**

---

## Pass/Fail 판정

> **❌ FAIL**

종합 점수 12점, 유용성 2점으로 FAIL이다. TasteData Korea가 앱스토어 설명에 "할랄 필터"를 명시했음에도 실제 데이터가 뒷받침되지 않아 말레이시아 무슬림 여행자의 핵심 기대를 완전히 충족시키지 못했다. 이는 단순한 기능 부재를 넘어 허위 기재에 가까운 신뢰 훼손이며, 인플루언서 아미나의 부정적 후기가 바이럴될 경우 말레이시아 무슬림 여행 커뮤니티 전체에서의 앱 평판에 타격을 준다.

## 개선 제안

1. 할랄 인증 데이터 구축 전까지 앱스토어 설명에서 "할랄 필터" 문구 즉시 제거 — 기능 미완성 상태에서의 마케팅 오용은 무슬림 여행자 커뮤니티에서 부정적 바이럴 위험이 크다
2. 단계적 할랄 지원 로드맵 — 이태원 이슬람 성원 주변 반경 500m 내 할랄 식당 데이터부터 정확하게 구축하고, 범위를 명확히 표시하는 "제한적 할랄 지원" 투명 공개 방식 채택
