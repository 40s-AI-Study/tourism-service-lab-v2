# SIM-31: PetKorea × 김지원 (한국 20대 솔로)

## 시나리오

2026년 4월 토요일 오전 10시, 김지원(25세, 프리랜서 디자이너)은 마포구 자취방에서 인스타그램 피드를 스크롤하다 친구가 공유한 "고양이랑 갈 수 있는 카페 리스트" 게시물을 본다. 반려묘 '먹구름'을 키운 지 6개월째인 지원은 최근 고양이 동반 외출에 관심이 생겼지만 어디서 정보를 찾아야 할지 몰랐다. 인스타 댓글에서 PetKorea 앱 링크를 발견하고 설치한다.

앱 온보딩에서 반려동물 프로필 등록을 권유받지만 일단 건너뛰고 탐색 모드로 진입한다. 목적은 단순하다: 오늘 당일치기로 먹구름과 함께 갈 수 있는 감성적인 공간, 사람이 너무 많지 않은 곳, 예산 3만원 이내. 앱 메인 화면의 "혼잡도 낮은 펫프렌들리 명소" 배너가 눈에 들어온다.

검색 결과를 보며 지원은 반신반의하다가 홍대 근처 고양이 동반 가능 루프탑 카페의 사진을 보고 마음을 굳힌다. 실시간 혼잡도가 '여유'로 표시된 것을 확인하고 오후 1시에 방문하기로 결정한다. 포토스팟 정보와 주차 필요 없음(도보 이동) 정보까지 확인하고 집을 나선다.

## 사용자 여정

### 1단계: 진입
- 인스타그램 지인 게시물에서 앱 링크를 통해 유입 (SNS 레퍼럴)
- 설치 후 온보딩에서 반려동물 프로필 등록 건너뜀 (귀찮음)
- 메인 화면에서 "오늘 당일치기 추천" 배너와 혼잡도 낮은 장소 피드 탐색
- 검색창에 "고양이 동반 카페 홍대" 입력

### 2단계: 행동
- `pet-friendly-travel` API (카테고리: 카페/음식점) 호출 → 고양이 동반 가능 카페 목록 수신
- 혼잡도 필터 '여유/보통' 선택 → `visitor-concentration-forecast` API로 실시간 혼잡도 확인
- 카페 상세 페이지에서 포토스팟 태그, 이용 규정(케이지 필수 여부), 메뉴 가격 확인
- `related-attractions` API → 해당 카페 반경 500m 내 산책 가능 공원·포토존 추가 추천 수신
- 예산 필터 (1인 3만원 이하) 적용 후 최종 2곳 비교
- 혼잡도 실시간 알림 설정 (방문 30분 전 푸시 알림 예약)

### 3단계: 결과
- 오후 1시 홍대 루프탑 카페 방문, 먹구름과 감성 사진 촬영 성공
- 앱이 추천한 인근 한강 공원 고양이 산책 코스도 들러 총 4시간 당일치기 완성
- 지출 27,000원 (카페 음료 2잔 + 고양이 전용 간식 1개)
- 불만족: 반려동물 프로필 없이 사용해 '고양이 맞춤' 추천이 아닌 전체 펫 결과가 섞여 나옴
- 만족: 혼잡도 알림이 정확해 줄 서지 않고 입장 성공. 사진 퀄리티 만족해 인스타 업로드 후 앱 태그

## 개선 제안

1. **소프트 온보딩**: 반려동물 종류를 프로필 등록 없이 세션 단위로 선택 가능하게 하여 (예: "오늘은 고양이와 함께해요" 배너 탭) 즉시 맞춤 결과 제공
2. **포토스팟 태그 강화**: 인스타그램 연동 또는 사용자 사진 업로드 기반 포토스팟 태그를 상세 페이지 상단에 배치해 20대 SNS 이용자 전환율 제고
3. **당일치기 코스 자동 생성**: 출발지·이동수단(도보/대중교통)·예산 입력 시 2-4시간 코스를 AI로 자동 구성하는 "즉흥 코스" 기능 추가
