# SIM-86: EcoTrail Korea × 김영자 (한국 시니어)

## 시나리오

김영자(67세, 은퇴 초등학교 교사)는 서울 노원구에 거주하며 은퇴 후 자연 탐방과 사진 찍기를 즐기기 시작했다. 무릎 관절염으로 인해 경사가 급하거나 긴 산행은 불가능하지만, 평지나 완만한 경사의 생태 탐방로는 충분히 걸을 수 있다. 스마트폰은 갤럭시를 사용하고 기본적인 앱 사용은 가능하나, 복잡한 UI는 어려워한다. 주민센터 노인 스마트폰 교육에서 EcoTrail Korea를 소개받았다. 딸이 앱 설치를 도와줬으며, "무릎 약한 분도 갈 수 있는 쉬운 코스가 있다"는 말에 관심을 가졌다.

---

## 사용자 여정

### 1단계: 진입

딸의 도움으로 앱을 설치하고 기본 설정을 마친다. 온보딩에서 '자연 산책', '쉬운 코스', '시니어' 프로필을 선택한다. 홈 화면에 큰 글씨 모드가 자동으로 적용되고, 추천 코스 첫 화면에 '난이도: 쉬움', '거리: 3km 이하', '포장 비율: 80% 이상' 필터가 기본 적용된 코스 목록이 뜬다.

### 2단계: 행동

'서울 둘레길 1구간 (수락산 방면 우회 평지 코스, 2.8km)' 코스를 확인한다. 코스 설명에 '무릎 부담 최소', '벤치 5개소', '화장실 위치' 정보가 표시되어 있어 신뢰감을 가진다. 생태관광 큐레이션 탭에서 '서울 노원 불암산 생태학습관' 프로그램을 발견하고 전화 예약 방법을 확인한다(앱 내 예약보다 전화 선호). 탄소 발자국 계산기는 복잡해 보여 패스한다.

### 3단계: 결과

주말에 서울 둘레길 평지 구간을 앱 지도를 보며 걷는다. 벤치 위치가 정확히 표시되어 있어 쉬엄쉬엄 걷기 좋았다는 후기를 남긴다. 불암산 생태학습관 프로그램은 전화로 예약해 참여한다. 앱을 주 1회 정도 새 코스 탐색 용도로 사용하며, 딸에게 "이 앱 괜찮다, 쉬운 코스 많이 있어"라고 소개한다. 다만 일부 메뉴 글씨가 작고, 뒤로 가기 버튼 위치가 헷갈린다는 불만을 표한다.

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **7** | 쉬운 코스 필터와 시설 정보(벤치·화장실)가 시니어 니즈와 부합. 생태관광 프로그램 연계도 유효. 탄소 계산기 등 부가 기능은 활용 낮음 |
| 사용 편의 | **5** | 큰 글씨 모드 자동 적용은 긍정적이나 전반적 UX가 시니어 최적화 수준에 미달. 뒤로 가기, 메뉴 구조 단순화 필요 |
| 구현 타당성 | **7** | 코스 내 편의시설(벤치·화장실·경사도) 데이터 추가는 두루누비 API 보완으로 구현 가능. 시니어 UI 모드는 별도 개발 필요 |

**종합 점수: 19/30**

## Pass/Fail 판정

> **✅ PASS**

종합 점수 19/30으로 기준(18/30) 초과(쉬운 코스 필터 전제 충족). 유용성 7점으로 기준(6점) 충족. 완만한 생태 탐방로 필터링 기능이 관절염 시니어에게 실질적 가치를 제공한다. 다만 사용 편의성이 5점으로 낮아, 시니어 UX 개선 없이는 지속 사용이 어렵다. 개선 여지가 명확한 PASS 판정.

## 개선 제안

1. **시니어 전용 UI 모드 강화**: 큰 글씨(18pt 이상), 고대비 색상, 단순화된 네비게이션(홈-코스-내정보 3탭), 음성 안내 기능을 포함한 '시니어 모드'를 제공한다. 코스 상세 페이지에 벤치·화장실·응급연락처·가장 가까운 지하철역 정보를 필수 항목으로 표시한다.
2. **신체 조건 기반 코스 필터 세분화**: 현재의 난이도(쉬움/보통/어려움) 외에 '무릎 부담 지수', '최대 경사도', '포장 비율', '전체 계단 수' 등 신체 조건별 세부 필터를 추가한다. 관절염·고혈압 등 건강 조건을 프로필에 입력하면 적합 코스만 추천하는 맞춤형 필터링을 구현하면 시니어 사용자 만족도를 크게 높일 수 있다.
