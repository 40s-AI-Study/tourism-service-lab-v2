# SIM-136: SilverTrail Korea × 김영자 (한국 시니어)

## 시나리오

딸(40세)이 "엄마, 이번 경주 여행 전에 이 앱 깔아드릴게요. 계단 있는지 없는지 다 나와요"라며 SilverTrail Korea를 설치해줬다. 영자는 처음에 "또 새 앱이야?" 하며 반신반의했지만, 큰 글씨와 단순한 버튼 구성이 자신도 혼자 조작할 수 있을 것 같다는 자신감을 줬다. 불국사 방문 전날 딸과 함께 앱으로 무장애 경로를 확인했고, 오디오 해설을 느린 속도로 미리 들어봤다. 35년 교직 생활 중 방문하지 못했던 불국사 전망대까지 계단 없이 올라간 경험이 감동적이었다.

---

## 사용자 여정

### 1단계: 진입
- 딸이 설치·설명 후 전달. 큰 글씨 UI에 즉각 안도감. "이건 나도 할 수 있겠다" 첫인상
- 경주 여행 전날 딸과 함께 앱 탐색

### 2단계: 행동
- `barrier-free-travel` + `central-attractions-by-municipality`: 불국사 무장애 경로 지도 확인 → 경사로 위치, 무장애 화장실 위치까지 상세 안내에 안심
- `audio-guide` 불국사 해설: 느린 속도 설정 적용 → "선생님이 설명해주는 것 같네" 호평. 어려운 용어 없이 이야기처럼 전개
- `durunubi-trails` 완만한 트레일 코스: 불국사 평지 산책로 → 무릎에 무리 없는 1.2km 코스 선택
- `wellness-tourism` 경주 온천(보문온천) 확인 → 다음날 온천 일정 추가
- `area-tourism-resource-demand`: 시니어 방문 적합 지역 데이터 기반 "경주 추천 이유" 확인 → 안심하고 선택

### 3단계: 결과
- 불국사 무장애 경로로 전망대 도달 성공. 귀가 후 네이버 실버 여행 카페에 "이 앱 정말 좋아요, 계단 정보 다 나와요" 후기 게시 → 카페 회원 30명 설치 유도

| 사용 편의 | **9** | 큰 글씨, 단순 UI, 딸 없이도 혼자 조작 가능한 수준. 완벽한 시니어 최적화 |
| 구현 타당성 | **9** | 고령화 사회 필수 서비스로서 구현 타당성 매우 높음. 실제 사용 데이터와 바이럴 후기까지 발생 |

## Pass/Fail 판정

> **✅ PASS**

김영자는 SilverTrail Korea의 완벽한 코어 타겟 페르소나다. 무장애 경로 안내, 느린 속도 오디오 해설, 완만한 트레일 큐레이션, 웰니스 온천 연계가 67세 관절염 시니어 여행자의 모든 고민을 해결했다. 단순 사용 전환을 넘어 커뮤니티 바이럴까지 이어진 최상의 시나리오이며, 이 앱이 존재해야 하는 이유를 가장 강력하게 증명한다.

## 개선 제안

1. 시니어 여행 커뮤니티(네이버 카페, 밴드)와의 공식 파트너십 — 영자 같은 사용자가 자발적 바이럴을 일으키는 구조를 공식 채널로 확장
2. 딸·사위가 대신 설치·설정 후 부모님께 전달하는 "가족 선물 여행 플랜" 공유 기능 — 영자 시나리오처럼 자녀 경유 설치가 핵심 유입 경로임을 감안한 공식화
