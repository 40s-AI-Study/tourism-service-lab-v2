# SIM-167: KoreaFestival Live × 이재호 (전동휠체어 개발자)

## 시나리오

재호는 파트너와 함께 가을 축제 여행을 계획 중이다. 매번 공공기관 PDF를 뒤지거나 장애인 카페에 물어봐야 했던 과정이 너무 비효율적이라 이번엔 앱으로 한 번에 해결하고 싶다. KoreaFestival Live 앱을 설치하고 "무장애 접근 가능" 필터를 켜보니, 전국 가을 축제 중 전동휠체어 진입 확인된 곳만 필터링된다. 안동국제탈춤페스티벌 상세 페이지에서 "전동휠체어 통과 가능 경사도 8° 이하 전 구간"이라는 데이터를 확인하는 순간, 개발자로서 "드디어 이 정보가 앱으로 나왔다"는 감탄이 나온다. 혼잡도 데이터를 활용해 휠체어 이동 공간이 확보되는 시간대까지 계산해 출발 시각을 정한다.

---

## 사용자 여정

### 1단계: 진입
- 장애인 여행 카페에서 KoreaFestival Live 링크 발견 → 앱 설치
- 프로필 설정에서 "전동휠체어 사용자" 체크 → 앱 전체 필터가 무장애 모드로 자동 전환
- 가을 축제 목록에서 전동휠체어 진입 가능 마크(♿) 표시된 축제만 표시

### 2단계: 행동
- 안동국제탈춤페스티벌 클릭 → "무장애 상세 정보" 탭 확인
  - 경사도: 최대 8° (전동휠체어 기준 통과 가능)
  - 바닥 재질: 아스팔트·블록 포장 전 구간
  - 전동휠체어 충전 포인트: 메인 무대 앞 2곳
  - 장애인 화장실: 입구·중간·출구 3곳
- 혼잡도 예측: 오전 10시 혼잡도 35% → 이동 공간 여유 확보 판단
- 최근 크라우드소싱 리뷰: "3주 전 방문, 메인 경로 전 구간 포장 상태 양호" 확인
- "1박 2일 코스" 생성 → 전동휠체어 접근 가능 숙소 자동 필터 적용

### 3단계: 결과
- 현장에서 예상치 못한 장벽 없이 전 코스 완주 — 10년 여행 중 처음 있는 일
- 앱 내 리뷰 남기기: "경사도·충전소까지 나오는 앱은 처음. 개발자 관점에서도 데이터 설계가 훌륭하다" 5점 평가
- 장애인 여행 카페와 개인 블로그에 상세 사용기 게시 → 장애인 여행자 커뮤니티 내 화제

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **9** | 경사도·충전소·바닥재질까지 제공하는 무장애 상세 데이터는 재호의 핵심 페인포인트를 완전히 해소. 크라우드소싱 최신 리뷰까지 결합 시 신뢰성 극대화 |
| 사용 편의 | **8** | 전동휠체어 사용자 프로필 설정 후 전체 자동 필터 전환은 매우 효율적. 개발자 배경으로 UI 품질에 민감하지만 기능이 충분하면 수용 |
| 구현 타당성 | **6** | 경사도·바닥재질 데이터를 전국 축제 현장별로 수집하려면 현장 조사 또는 지자체 협력 필수 — 초기 데이터 구축 비용이 높은 것이 가장 큰 실현 장벽 |

**종합 점수: 23/30**

## Pass/Fail 판정

> **✅ PASS**

KoreaFestival Live가 이재호에게 통하려면 무장애 데이터의 세밀함이 핵심이다. 단순 "접근 가능" 체크가 아닌 경사도·바닥재질·충전소까지 제공할 때 비로소 이 페르소나가 신뢰하고 전환한다. 구현 비용이 높으나 장애인 여행자 커뮤니티 내 신뢰 획득 시 구전 효과가 크고, 공모전 배점에서 사회적 가치 측면에서도 강점으로 작용한다.

## 개선 제안

1. 전동휠체어 사용자 크라우드소싱 리뷰 전용 섹션 신설: "N주 전 방문, 경사로 상태 [양호/공사중/불가]" 형식의 구조화된 리뷰를 유도하여 실시간 정보 갱신 체계 구축
2. 전동휠체어 충전소 위치를 지도 뷰에 별도 레이어로 표시하고, 축제장 입구·중간·출구 위치를 사전에 확인할 수 있는 상세 지도 제공 — 배터리 방전 불안 제거가 재호의 여행 결정에 가장 큰 영향
