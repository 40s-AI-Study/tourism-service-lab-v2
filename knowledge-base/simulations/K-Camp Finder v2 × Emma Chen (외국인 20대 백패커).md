# SIM-R3-012: K-Camp Finder v2 × Emma Chen

## 시나리오

Emma Chen(23세)은 대만 타이베이 출신 프리랜서 사진작가다. 인스타그램 팔로워 8만 명을 보유한 그녀의 콘텐츠 테마는 "아시아 감성 여행 × 자연광 포토그래피"다. 이번 한국 여행은 3주 일정으로, 서울을 베이스로 하되 주말마다 1박 2일 외곽 탐방을 계획했다. 가장 큰 고민은 '아직 아무도 올리지 않은 각도의 한국 캠핑 사진'을 찾는 것이다.

가로수길 카페에서 아이스 아메리카노를 마시며 인스타그램을 스크롤하던 중, 팔로우하는 한국 감성 여행 계정(@감성캠핑일기)의 스토리에서 K-Camp Finder v2 앱 링크를 본다. 繁體中文(번체 중국어) 지원 표시에 눈이 번쩍 뜨인다. 한국 앱 대부분이 영어 아니면 한국어뿐이라 답답했던 터였다. 즉시 다운로드.

앱을 열면 풀스크린으로 펼쳐지는 가을 단풍 캠핑 영상이 그녀의 사진가 본능을 자극한다. 황금빛 역광 속 텐트, 빨간 단풍잎이 카메라 렌즈처럼 보인다. 繁體中文 인터페이스로 전환하자 "你想在什麼樣的天空下休息？(어떤 하늘 아래에서 쉬고 싶어?)" 라는 문구가 뜬다. ⭐별멍 카드를 탭한다 — 밤하늘 은하수 사진이 목표다.

## 사용자 여정

### 1단계: 진입
- 인스타그램 스토리 → 앱스토어 → 번체 중국어 지원 확인 후 다운로드 (2분)
- 앱 언어를 繁體中文으로 설정. UI 번역 품질이 자연스러운 대만 중국어임을 확인 (대륙식 간체자 아님)
- ⭐별멍 선택 → 광해 지수 낮은 캠핑장 필터 자동 적용
- "포토스팟 태그" 필터 추가 탭 — 사진가 모드로 전환

### 2단계: 행동
- **광해 지수 + 포토스팟 복합 필터**: 강원도 인제 방향 캠핑장 2곳이 최상단 노출. 은하수 촬영 가능 예상 시간대까지 표시 (새벽 1-3시)
- **사용자 포토 갤러리 탐색**: 각 캠핑장의 실제 방문자 사진들을 슬라이드로 확인. "이 각도가 아직 덜 찍혔다" 분석. Emma는 가장 독창적인 각도가 많은 곳을 고름
- **숨은 명소 태그 발견**: 일반 예약 사이트에는 없는 "캠핑장 뒤편 600m 지점 포토스팟" 정보가 앱 내 커뮤니티에 등록되어 있음. 중국어 번역으로 내용 파악 가능
- **혼잡도 예측 활용**: 이번 주말 방문자 수 예측 그래프 확인. 토요일 오전보다 일요일 새벽이 혼잡도 최저임을 파악 → 일정 조정
- **연관 관광지 API**: 캠핑장 인근 폭포와 계곡 정보 추가 확인. 1박 2일 동선을 앱 내에서 완성

### 3단계: 결과
- 인제 캠핑장 1박. 새벽 1시 30분에 일어나 은하수 촬영 성공
- 앱이 알려준 "뒤편 600m 포토스팟"에서 촬영한 텐트 + 은하수 구도가 기존 인스타 게시물과 완전히 다른 각도임을 확인
- 귀국 후 해당 사진 인스타 게시 → 저장 수 12,000회. "어디서 찾았어요?" DM 폭주
- 앱 태그 및 스토리 언급 → 자연 바이럴 발생
- 평가: 번체 중국어 지원과 포토스팟 데이터의 깊이에 매우 만족. 단, 일부 커뮤니티 글은 한국어만 있어 번역이 안 됨

> "이 앱 덕분에 인스타에서 아무도 올리지 않은 각도의 사진을 찍었다. 콘텐츠 크리에이터한테 그게 얼마나 중요한 건지, 한번 맛보면 끊을 수가 없어."

**재사용 트리거**:
- 다음 주말 여행 계획 시 광해 지수 낮은 날씨 예보와 함께 앱 오픈
- 새로운 포토스팟 커뮤니티 게시글 알림 수신 시
- 계절 바뀔 때마다 새로운 감성(봄 벚꽃) 배경화면 보러 앱 오픈

**이탈 리스크**:
- 커뮤니티 포토스팟 정보가 업데이트되지 않아 낡아지면 이탈
- 한국어 커뮤니티 글 번역 미지원 지속 시 정보 격차로 이탈

## 개선 제안

1. **커뮤니티 게시글 AI 자동 번역**: 한국어로 작성된 포토스팟 팁, 후기 글을 번체 중국어·영어·일본어로 실시간 번역. 외국인 사용자가 현지인 커뮤니티에 완전히 접근할 수 있게 되어 앱의 핵심 가치인 "숨은 명소" 발굴 능력이 극대화됨
2. **사진작가 모드 전용 UI**: 골든아워 시간, 은하수 최적 촬영 시간대, 일출/일몰 방향각을 캠핑장별로 제공. Emma처럼 빛을 쫓는 사진가에게 앱이 없으면 안 되는 도구로 포지셔닝 가능
