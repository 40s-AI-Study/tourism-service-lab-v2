# SIM-183: KoreaDemand Atlas × 최동훈 (한국 40대 가족)

## 시나리오

여름방학 2주 전, 역사 교사 동훈은 경주 여행을 계획하며 네이버 블로그를 뒤지다가 지인에게 KoreaDemand Atlas를 소개받는다. "데이터로 여행지 뜨는 거 보여주는 앱"이라는 설명을 듣고 솔직히 자신이 쓸 앱인지 확신이 없지만, 경주 입력 시 방문자 집중률 예측 그래프가 뜨는 것을 보고 "아, 방학 때 얼마나 붐비는지 볼 수 있구나"라고 이해한다. 초등생 자녀 둘을 데리고 가는 여행이라 혼잡도 정보는 실용적 가치가 있다. 그러나 앱 전반이 지역 수요 분석에 특화되어 있어 "아이들이 좋아할 체험" 같은 가족 맞춤 필터를 기대했다가 없다는 것을 알게 된다.

---

## 사용자 여정

### 1단계: 진입
- 지인 추천으로 설치 (보수적 성향, 검증된 추천에 반응)
- 경주 검색 → 방문자 집중률 그래프 확인 → "방학 기간 집중 예측" 경고 확인
- 히트맵과 수요 지표 화면을 보며 "이게 무슨 의미인지" 읽는 데 5분 소요

### 2단계: 행동
- 경주 지역 탭: 방문자 집중률 방학 기간 78% → "붐빔 주의" 표시 확인
- 불국사·첨성대 포함 지역 핵심 명소 100위 목록 열람 — 교사로서 역사적 맥락 이해하기 좋음
- 방문 타이밍 알림 기능: 방학 첫째 주 화요일 오전이 상대적으로 한산 → 일정 조정
- 체험 활동, 아동 맞춤 필터 찾아봤지만 없음 → 코스 다양성 면에서 한계
- 영어·한국어 관광 정보 페이지 열람 — 텍스트 정보는 충실하나 아이들 흥미 유발 콘텐츠 없음

### 3단계: 결과
- 방문 타이밍 조정으로 불국사 혼잡 회피 성공 → 해당 기능은 만족
- 체험 프로그램, 도예공방 위치 등 가족 맞춤 정보는 결국 맘카페에서 별도 검색
- 전반적으로 "어른이 여행지 뽑을 때 쓸 앱"이라는 인식, 가족 여행 전용으로는 부족

## 평가

| 항목 | 점수 (1-10) | 근거 |
|---|---|---|
| 유용성 | **5** | 방문 타이밍 예측 기능은 유용하나, 가족 여행 핵심 니즈(아동 맞춤, 체험 활동, 무장애) 미지원으로 반쪽 유용성 |
| 사용 편의 | **5** | 중간 기술 친숙도인 동훈에게 수요 지표 용어가 어렵고, 가족 여행 맥락의 필터 부재로 탐색 효율 낮음 |
| 구현 타당성 | **8** | 플랫폼 자체 구현은 타당하나 가족 여행자 대상 기능 확장 시 가치 상승 |

**종합 점수: 18/30**

---

## Pass/Fail 판정

> **✅ PASS**

종합 점수 18점으로 최소 기준을 충족한다. 방문 타이밍 예측이라는 단일 기능이 가족 여행의 혼잡 회피 니즈에 부합해 최소 유용성을 확보했다. 그러나 KoreaDemand Atlas는 최동훈 같은 가족 여행자보다 데이터 지향적 파워 여행자에게 적합한 앱이며, 이 페르소나에서의 PASS는 아슬아슬한 수준이다.

## 개선 제안

1. 지역 탐색 결과에 "가족 여행 적합도" 서브 점수 추가 — 체험 시설 수, 아동 친화 명소 비율 등 파생 지표 활용
2. 방문자 집중률 예측 알림에 "아이 동반 추천 시간대" 레이블 붙이기 — 유모차·아동 통행 공간 여유 기준 명시
