# PM 아이디어 10개 사업성 점수 매트릭스

## 평가 기준 (각 20점, 총 100점)

| 기준 | 설명 | 고득점 조건 |
|---|---|---|
| **시장 규모** | 타겟 시장의 절대적 크기 + 성장성 | 대형 시장, 성장세 뚜렷 |
| **경쟁 강도** | 기존 경쟁 서비스 공백 정도 | 경쟁 서비스 없을수록 고점 |

| **구현 난이도** | 개발 복잡도 역산 (쉬울수록 고점) | MVP 빠른 구현 가능 |
| **공모전 적합성** | 심사 기준 부합도 (데이터 활용·기획력·사업성·사회적 가치) | 심사 기준 다수 충족 |

## 아이디어별 상세 점수

### 아이디어 1: KoreaPath AI — 개인화 코스 자동 생성 + 혼잡도 최적화

**한 줄 정의**: 사용자 조건(테마·지역·일정·동반자) 입력 시 혼잡도·연관 관광지 데이터로 최적 코스 자동 생성

**타겟**: 20-40대 국내 여행자 (코스 계획 귀찮은 여행자)

**활용 API**: `related-attractions` + `central-attractions-by-municipality` + `visitor-concentration-forecast` + `tourism-info-korean` + `area-tourism-demand-density`

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 18 | 국내 관광 연 3.2억회. 코스 추천 니즈 82%. 20-40대 전 연령 타겟 |
| 경쟁 강도 | 18 | AI 코스 자동 생성 국내 전무. 트리플·네이버 모두 수동 계획 방식 |
| API 활용도 | 18 | 5개 API 창의적 조합. 혼잡도+연관관광지 병행 국내 최초 |
| 구현 난이도 | 13 | 추천 엔진 백엔드 로직 복잡. AI 모델 연동 필요 |
| 공모전 적합성 | 18 | 혁신성·실용성·데이터 활용 배점 동시 충족 |
| **합계** | **85** | |

### 아이디어 2: K-Guide Global — 외국인 전용 다국어 통합 한국 관광 가이드

**한 줄 정의**: 9개 언어 완전 지원, Visit Korea 공식 데이터 + 오디오 가이드 + 실시간 혼잡도를 하나의 앱에서 제공

**타겟**: 방한 외국인 전체 (1,620만명) — 일/중/영어권 우선

**활용 API**: 9개 다국어 `tourism-info-*` + `audio-guide` + `visitor-concentration-forecast` + `tourism-big-data` + `related-attractions`

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 19 | 방한 외래객 2024 추정 1,620만명. 중국 24%, 일본 22%. 대형·성장 시장 |
| 경쟁 강도 | 14 | Visit Korea 앱 존재하나 UX 구식. Google·TripAdvisor 범용. 공식 데이터+현대 UX 틈새 |
| API 활용도 | 20 | 12개+ API 활용 (다국어 9개+오디오+혼잡도+빅데이터). 27개 중 최다 |
| 구현 난이도 | 11 | 9개 언어 UI/UX 구현 복잡. 콘텐츠 품질 편차·유지보수 부담 |
| 공모전 적합성 | 20 | 데이터 활용 배점 20점 만점 최유력. 심사위원 임팩트 최고 |
| **합계** | **84** | |

### 아이디어 3: PetTrip Korea — 반려동물 동반 여행 전문 플랫폼

**한 줄 정의**: 반려동물 동반 관광지·숙박·식당·캠핑장 7개 카테고리 통합 + 동선 최적화 제공

**타겟**: 반려동물 가구 600만 (30-40대 반려견/묘 보유자)

**활용 API**: `pet-friendly-travel` + `gocamping` + `related-attractions` + `visitor-concentration-forecast`

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 15 | 반려동물 가구 600만. 반려동물 여행 시장 연 25% 성장 추정 |
| 경쟁 강도 | 18 | 전용 통합 앱 전무. 야놀자 일부 펫 숙박, 블로그 정보 분산뿐 |
| API 활용도 | 15 | 4개 API. pet-friendly + gocamping 조합 독창적 |
| 구현 난이도 | 16 | 구현 비교적 단순. 데이터 최신성 유지 관건 |
| 공모전 적합성 | 15 | 니즈 명확하나 공모전 심사 차별화 보통 수준 |
| **합계** | **79** | |

### 아이디어 4: FreeTrip Korea — 무장애·접근성 특화 관광 플랫폼

**한 줄 정의**: 장애인·고령자·영유아 동반 가족을 위한 접근성 정보 통합 + 휠체어 코스 + 오디오 안내

**타겟**: 장애인 263만 + 65세 이상 970만 + 유모차 동반 부모

**활용 API**: `barrier-free-travel` + `tourism-info-korean` + `related-attractions` + `audio-guide` + `visitor-concentration-forecast`

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 14 | 장애인 263만+노인 970만. 고령화 가속 → 성장 시장 확실 |
| 경쟁 강도 | 19 | 무장애 통합 여행 앱 국내 전무. 공식 정보 분산만 존재 |
| API 활용도 | 15 | 5개 API. barrier-free 특화 활용 독창적 |
| 구현 난이도 | 16 | 접근성 데이터 검증 필요하나 구현 비교적 단순 |
| 공모전 적합성 | 18 | 사회적 가치 심사 항목 직결. RTO 특별상 연계 가능 |
| **합계** | **82** | |

### 아이디어 5: WellKorea — 웰니스 여행 큐레이터 앱

**한 줄 정의**: 번아웃 직장인 대상 온천·스파·산림욕·에코 힐링 코스 큐레이션 + 조용한 시간대 추천

**타겟**: 30대 번아웃 직장인, 회복 여행 원하는 커플

**활용 API**: `wellness-tourism` + `eco-tourism` + `visitor-concentration-forecast` + `area-tourism-diversity` + `related-attractions`

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 14 | 웰니스 시장 성장 중. 30대 집중으로 타겟 범위 제한 |
| 경쟁 강도 | 16 | 개별 스파 예약만 존재. 통합 웰니스 여행 앱 없음 |
| API 활용도 | 14 | 5개 API. wellness+eco 조합 적절하나 폭 제한적 |
| 구현 난이도 | 16 | 구현 비교적 단순. 큐레이션 콘텐츠 품질이 관건 |
| 공모전 적합성 | 13 | 차별화 약함. 웰니스 단독은 기획력 배점 확보 어려움 |
| **합계** | **73** | |

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 16 | 글로벌 K-팝 팬 수억명. 방한 일본(22%)·동남아(10%) 핵심 타겟 |
| 경쟁 강도 | 14 | 팬 커뮤니티 앱 일부 존재. 공식 데이터 기반 통합 없음 |
| API 활용도 | 16 | 5개 API. 큐레이션 데이터 별도 구축 필요 |

| 공모전 적합성 | 16 | K-컬처 트렌드 반영. 외국인 관광 활성화 공모전 취지 부합 |

### 아이디어 7: LocalSecret — 혼잡 회피 + 숨은 명소 발견 앱

**한 줄 정의**: 향후 30일 혼잡도 예측 캘린더 + 빅데이터 기반 급상승 로컬 명소 발견 결합

**타겟**: 혼잡 회피 선호 전 연령대 + 재방문 여행자 ('남들이 모르는 곳' 선호 20-30대)

**활용 API**: `visitor-concentration-forecast` + `tourism-big-data` + `area-tourism-diversity` + `area-tourism-demand-density` + `tourism-info-korean`

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 16 | 혼잡도 확인 니즈 76%. 재방문 여행자 틈새지만 충성도 높음 |
| 경쟁 강도 | 18 | 한국 특화 혼잡도 앱 + 빅데이터 숨은 명소 서비스 동시에 전무 |
| API 활용도 | 17 | 5개 API. 혼잡도+빅데이터 조합 독창적, 공모전 배점 유리 |
| 구현 난이도 | 14 | 빅데이터 분석 로직 + 지역 필터링 알고리즘 필요. 중간 복잡도 |
| 공모전 적합성 | 17 | 빅데이터 활용 강조 → 심사위원 인상적. 오버투어리즘 해소 정책 연계 |
| **합계** | **82** | |

**활용 API**: `barrier-free-travel` + `pet-friendly-travel` + `related-attractions` + `visitor-concentration-forecast` + `tourism-info-korean`

| 기준 | 점수 | 근거 |
|---|---|---|
| 시장 규모 | 16 | 40대 가족여행 큰 시장. 소비력 높고 '아이 데리고 갈 곳 찾기' Pain Point #1 |
| 경쟁 강도 | 17 | 가족여행 전용 플래너 앱 없음. 맘카페·블로그 분산 정보만 존재 |

| 공모전 적합성 | 16 | 40대 가족 니즈 명확. 사회적 가치(접근성 포함) + 기획 타당성 높음 |
