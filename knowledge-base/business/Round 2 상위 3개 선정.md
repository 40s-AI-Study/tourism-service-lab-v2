## 아이디어별 선정 근거

### 🥇 1위: K-Guide Global (87점)

**핵심 선정 이유**
- Round 2 10개 아이디어 중 **유일하게 API 가중치 보너스 +4점** 수혜 (8개 API)
- 방한 외래객 2024년 추정 1,620만 명 — 국내 관광 시장 중 가장 빠른 성장 세그먼트
- 공모전 "데이터 활용도(20점)" 항목에서 만점 최유력: 다국어 4종 + 오디오 + 혼잡도 + 연관관광지 + 사진 조합
- 관광공사 공식 데이터 재패키징 → MVP 구현 현실성 높음

**리스크 대응**
- 8개 언어 동시 구현 부담 → MVP는 영·일·중간체 3개 언어로 시작, 단계적 확장
- Visit Korea 앱과의 차별화 → AI 개인화 코스 + 실시간 혼잡도로 완전 차별화 (KoreaPath AI 모듈 연계)

**페르소나 커버**: P4 Emma Chen (High), P5 David Park (High) — 외국인 방한객 핵심 2명 완벽 충족

---

### 🥈 2위: KoreaPath AI (84점)

**핵심 선정 이유**
- BA·UX 양측 모두 최상위 추천: BA 공동 2위 / UX **단독 1위**
- 5개 페르소나 모두 High(4명) + Medium(1명) — Round 2 아이디어 중 가장 넓은 페르소나 커버리지
- 국내 AI 자동 코스 생성 서비스 전무 → 블루오션, 경쟁강도 18/20
- Round 1에서도 최고점(85점) — 양 라운드 모두 상위권의 검증된 강점

**리스크 대응**
- 추천 엔진 구현 복잡도 → MVP는 rule-based 로직, AI 고도화는 2단계로 분리

**페르소나 커버**: P1·P2·P3·P5 (High), P4 (Medium) — 전 세그먼트 대응

---

### 🥉 3위: AccessKorea (84점)

**핵심 선정 이유**
- **경쟁강도 만점(20/20)** — 무장애 통합 여행 앱 국내 완전 전무
- 공모전 사회적 가치 항목(10점) 직결 + RTO(지역관광조직) 특별상 연계 가능
- 문체부 무장애 관광 정책 직접 부합 → 심사위원 정책 공감도 확보
- 시니어 970만 + 장애인 263만 + 영유아 동반 가족 = 잠재 시장 대형

**리스크 대응**
- 기존 P1~P5 페르소나 매칭 약함 → P6(시니어) 추가 시 사업성 근거 대폭 강화 (TOU-17 P6~P10 작업 완료 후)
- 접근성 데이터 현장 괴리 → 크라우드소싱 리뷰 시스템으로 실시간 보완

**페르소나 커버**: P3 (Medium — 노부모 동반), P6 시니어 (High — 신규 페르소나 추가 후)

## KoreaPath AI와의 통합 가능성 검토

> CEO 검토 항목: "통합 서비스의 API 활용도가 15개 → 20개+로 확장되면 공모전 배점 확보"

### 통합 시나리오 A: K-Guide Global + KoreaPath AI

| 구성 | 기여 API | 역할 |
|---|---|---|
| K-Guide Global | 다국어(영·일·중·중번체) 4종, 오디오가이드, 혼잡도예보, 연관관광지, 사진공모전 수상작 | 외국인 다국어 UX, 콘텐츠 레이어 |
| KoreaPath AI | AI동선생성, 혼잡도예보(공유), 연관관광지(공유), 빅데이터, 수요밀도 | 코스 자동화, 혼잡도 최적화 |

**통합 후 순수 추가 API**: 약 10-12개 (중복 제거 시)
**API 가중치 보너스**: 10개 이상 → **+5점**
**공모전 배점 영향**: 데이터 20점 만점 + 기획력(30점) + 혁신성 동시 확보

### 통합 시나리오 B: 3개 통합 (K-Guide Global + KoreaPath AI + AccessKorea 모듈)

| 추가 모듈 | 기여 |
|---|---|
| AccessKorea 무장애 필터 | 무장애 동선 필터 옵션화 → 사회적 가치(10점) 가점 |

**전략적 판단**: 3개를 별도 앱으로 공모 vs 통합 1개로 공모 — 후속 세션에서 결정 필요
