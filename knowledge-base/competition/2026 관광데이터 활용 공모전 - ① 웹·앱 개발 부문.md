# 2026 관광데이터 활용 공모전 - ① 웹·앱 개발 부문

## 주최
- **주최**: 한국관광공사 (Korea Tourism Organization)
- **주관 사무국**: gongmo@stunning.kr
- **접수 사이트**: 한국관광 콘텐츠랩 (api.visitkorea.or.kr)
- **공고일**: 2026-03-30

## 공모전 슬로건
> **"데이터 가치를 넘어 서비스로 실현하다, 관광데이터 스케일 UP"**

## 전체 구조 (2026년 변경사항)

2026년부터 **통합 공모전 내 부문 단위 운영**으로 변경:
- ① **웹·앱 개발 부문** ← 우리가 참여할 부문
- ②-1 생성형 AI 활용 관광 프롬프톤
- ②-2 웹·앱 구현 (국내 대학 재학생 대상)
- ③ 웹·앱 고도화 (기업 대상)

## 우리 참가 부문: ① 웹·앱 개발

### 공모 개요
- **공모명**: 2026 관광데이터 활용 공모전 - ① 웹·앱 개발 부문
- **공모대상**: 한국관광공사 OpenAPI **필수 활용**하여 개발한 **웹·앱 등 신규 관광 서비스**
- **참가자격**: 대한민국 국민 누구나 (국내 거주자 한정, **팀당 최대 5명**)
  - ⚠️ 동일 서비스로 본 공모전 수상자 및 공사 주관 지원사업 수혜자 참가 불가

## 선발 절차 (3단계)

```
예비심사 (서류) → 1차 심사 (기능) → 최종심사 (PT 발표) → 시상식
```

1. **예비심사**: 제출서류 적격 여부 판단
2. **1차 심사 (서면 + 기능)**: 완성 서비스 기능 심사 (100점)
3. **최종심사 (PT)**: 상위 5팀 대상 발표 심사 (100점)
4. **시상식**: 통합 시상식 진행

## 심사 기준 ⭐

| 항목 | 배점 |

| **서비스 완성도** (기능성, 안정성, 편의성) | 30 |
| **데이터 활용 적절성** (공사 OpenAPI 활용 **필수**) | 20 |
| **서비스 발전성** (지속성, 확장성) | 20 |
| **합계** | **100** |

**가점 (각 2점)**
- ➊ 신용보증기금 'Start-up NEST' 선정기업 (선정서 및 수료증 제출 필수)
- ➋ 지역 특화 관광서비스 (1개 지역 특화, 서울 제외)

## 수상팀 후속지원

- 수상 서비스 대외 홍보 + 이벤트 연계 소비자 피드백 기회
- 신용보증기금 'Start-up NEST' 지원 시 서류 심사 가점 부여 (3년 이내 스타트업, 예비 창업자 포함)

## 제안서 양식 (제출물)

**파일 형식**: PDF (10MB 미만)
**분량**: 5페이지 이내 (이미지, 표 포함)
**글자 크기**: 12포인트 이상
**작성 항목 및 양식 수정 금지, 모든 항목 작성 필수**

### 제안서 섹션 (4개)

#### 1) 서비스 기획배경 및 필요성
- 서비스 기획 배경
- 서비스 필요성

#### 2) 서비스 개요
- 기획 서비스 소개 (한 줄) — 예: "인구 감소지역 관광정보 제공 서비스"
- 기획 서비스 주요 기능
- 서비스 차별성
- (지역 특화 서비스 시) 지역 특화 관련 사항

#### 3) 데이터 활용 방안 ⭐ **한국관광공사 OpenAPI 활용 필수**
- 활용 예정 한국관광공사 OpenAPI (예: 한국관광공사 국문관광정보 서비스)
- 데이터 활용 방식

#### 4) 서비스 발전 방향
- 개발 서비스 향후 발전방향 및 기대효과

## 주요 제약사항 / 유의사항

1. **OpenAPI 필수 활용** — 제출/심사 불가 조건
2. **동일 서비스 중복 지원 금지** — 다른 부문(②-1, ②-2, ③)에도 지원 불가
3. **서비스 완성 필수** — 1차/최종 심사는 "개발 완료된 완성 서비스"만 평가
4. **기획 단계 vs 완성 단계 구분**:
   - 공고/접수 단계 (3.30~5.6): 제안서 PDF만
   - 예비심사 후 (5월~9월): 실제 서비스 개발 진행
   - 1차 심사 (10월): 완성된 서비스 제출
5. **팀 최대 5명, 팀 대표는 한국관광콘텐츠랩 회원가입 필수**
6. **저작권/초상권 책임**: 지원자 본인
7. **서비스 2종 이상 개발 시 동일 내용**만 가능

## 우리 전략 시사점

### 공모전 이해 기반 방향성

1. **접수(5/6)까지는 제안서 PDF만 제출** — 완성 서비스 불필요
   - 현재 Phase 3 목표: **수준 높은 5페이지 PDF 제안서** 작성
2. **기획력(30점) + 차별성 + 완성도 + 데이터 활용** 4개 축
3. **지역 특화 서비스는 가점 2점 + RTO 특별상 가능성** — 고려할 옵션
4. **예비심사 통과 후**에는 실제 구현이 필요 (5~9월 5개월)
   - 그러나 이 프로젝트 범위는 **제안서까지**
5. **창의성·트렌드·독창성** 강조 (심사 기준 1순위)

### 우리 타겟 (사용자 요구사항 재확인)

- **20-40대 한국인 + 외국인** — 다양한 사람이 쓰는 범용 서비스
- **시뮬레이션 루프로 실제 니즈 검증** → "서비스 완성도" 및 "서비스 적정성" 심사 대응
- **Visit Korea OpenAPI 필수 활용** → 이미 공모전 요건과 일치

## 우리 목표 (수정 확정)

- **최우수상 (사장상, 300만원)** 이상 목표
- 지역특화 옵션도 부분적으로 고려 (가점 + RTO 특별상)
- **제안서 완성도 10/10** — 공모전 심사 기준 완벽 매핑
