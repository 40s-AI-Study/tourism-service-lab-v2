# [Round 4 Step 4] 카테고리 대표 8개 Hero Screen 스케치

> 작성: 🎨 디자인님 (claude-sonnet-4-6) | 2026-04-19  
> 입력: BA 카테고리 대표 8개 선정 결과 (`scoring-2026-04-19.md`)  
> 규칙: HTML 제작 X — 텍스트 묘사만 / UI 요소·색감·문구 구체적으로 / 트리플·야놀자·Klook 차별화 명시

---

## 작성 원칙

각 Hero Screen 스케치는 다음 구조로 작성합니다:

- **레이아웃 구조**: 화면 상단→하단 요소 배치
- **색상 팔레트**: 주색·보조색·배경·텍스트 컬러
- **주요 카피**: 헤드라인·서브카피·CTA 문구
- **핵심 UI 요소**: 첫 화면에서 사용자가 보고 만지는 것
- **감정 진입점**: 앱을 열었을 때 느끼는 감정
- **차별화 포인트**: 트리플/야놀자/Klook 첫 화면 대비

| 역할 | 색상 | 코드 (참고) |

#### 주요 카피

- **헤드라인**: "오늘 어디로 갈까요? AI가 최적 코스를 30초 만에 짜드릴게요."
- **서브카피**: "혼잡한 관광지는 피하고, 숨은 보석만 골라드립니다"
- **CTA**: "✨ AI 동선 생성하기"
- **알림 배너**: "경복궁 오늘 붐빔 예상 ⚠️ 대신 이 코스 어때요? →"
- **온보딩 첫 질문**: "지금 어떤 여행을 꿈꾸고 있나요?"

#### 감정 진입점

> "뭘 검색해야 할지 모르겠어" → "그냥 조건 몇 개 누르면 AI가 알아서 짜준다"는 **해방감·편안함**

#### 차별화 포인트 (트리플/야놀자/Klook 대비)

| 구분 | 트리플/야놀자/Klook | KoreaPath AI |
|------|-------------------|--------------|
| 첫 화면 | 검색창 + 베스트 목록 | 실시간 혼잡도 열지도 + 조건 입력 폼 |
| 사용자 행동 | 직접 검색·탐색 | 조건 입력 → AI 자동 생성 |
| 혼잡도 정보 | 없음 or 간접적 | Hero 화면에서 메인 시각화 |
| 동선 생성 | 수동으로 직접 구성 | AI 자동 최적화 (30초) |
| 감정 | "어디 가지?" 막막함 | "AI한테 맡기면 돼" 안도감 |

- **헤드라인**: "아직 아무도 모르는 한국"

- **선택지**: 🌿 고요한 발견 / ⚡ 반전 있는 발견 / 🎨 감성적인 발견 / 👣 아무도 모르는 발견

> "SNS에서 본 곳은 이미 다 붐빈다" → "진짜 아무도 모르는 곳을 나만 발견한다"는 **특별함·발견의 희열**

#### 차별화 포인트

| 구분 | 트리플/야놀자/Klook | LocalSecret v2 |
|------|-------------------|----------------|
| 첫 화면 | 인기 관광지 베스트 | 방문자 수 최하위 숨은 장소 |
| 첫 질문 | "어디 가고 싶어요?" | "지금 어떤 발견을 원하세요?" (감정 선택) |
| 추천 기준 | 별점·리뷰 수 | 혼잡도 최하 + 다양성지수 |
| 콘텐츠 톤 | 상업적·광고 포함 | 로컬 발견자 직접 기록 |
| 차별화 감정 | 정보 탐색 | 나만의 발견, 특별함 |
