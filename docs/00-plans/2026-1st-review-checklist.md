# TripCraft Korea — 1차 심사 준비 체크리스트

> **1차 심사**: 2026년 10월 중 (기능 심사)  
> **평가 기준**: 서비스 완성도 30점 + 데이터 활용 20점 + 서비스 기획력 30점 + 발전성 20점 = **100점**  
> 이 체크리스트는 **D-30(9월 말)** 기준으로 완료되어야 합니다.

---

## 🔴 Must — 심사 통과 필수

### 서비스 동작 (완성도 30점)

- [ ] F1 **관광 정보 검색** 정상 동작 — 지역·카테고리·키워드 검색 결과 반환
- [ ] F2 **코스 자동 생성** 정상 동작 — 5초 이내 1박2일 플랜 생성
- [ ] F3 **혼잡도 회피 + 대안 추천** — 인기 장소 포화 시 대안 3곳 표시
- [ ] F4 **오디오 가이드** — 위치 진입 50m 트리거 + 재생 동작 확인
- [ ] 모바일 iOS/Android 브라우저에서 전체 기능 정상 동작
- [ ] 오류율 5% 미만 (주요 기능 기준)
- [ ] 응답 속도 — 검색 3초 이내, 코스 생성 10초 이내

### 데이터 활용 증빙 (데이터 활용 20점)

- [ ] KTO OpenAPI **12종** 실제 호출 로그 준비 (스크린샷 또는 서버 로그)
- [ ] 카카오 API 4종 연동 확인 (지도·모빌리티·로그인·간편결제)
- [ ] API 호출 시퀀스 다이어그램 업데이트 (`04-api-utilization.md` 기준)
- [ ] 각 API가 어떤 기능에 연결되는지 매핑표 심사용 버전 준비

### 제출 파일

- [ ] **시연 URL** 확정 (공개 접속 가능한 도메인 또는 GitHub Pages)
- [ ] **소스코드** GitHub 레포 공개 설정 확인
- [ ] **시연 영상** 3~5분 (전체 기능 플로우 — 심사위원 오프라인 시청용)

---

## 🟡 Should — 고득점 차별화 요소

### 서비스 완성도 강화

- [ ] F5 **경비 자동 합산** 동작 (입장료 + 예상 이동비)
- [ ] F6 **팔도 스탬프** 적립 + UI 표시 (실제 리워드 지급은 Phase 3)
- [ ] 4개 언어 지원 (한·영·중·일) — 최소 UI 텍스트 번역 완료
- [ ] 소외 지역 가중치 1.5배 알고리즘 동작 확인 (`area-tourism-diversity` 기반)

### 기획력 보강 자료

- [ ] 시뮬레이션 검증 결과 업데이트 (Phase 2 실제 사용 테스트 기반)
- [ ] 페르소나 3종 × 전체 플로우 시연 스크립트 완성
- [ ] KTO 정책 정합성 근거 문서 (소외 지역 균형 지표 수치)

### 심사 PT 초안

- [ ] PT 자료 10~15장 초안 완성 (최종심사 준비)
- [ ] 데모 영상 편집 완료 (BGM + 자막)
- [ ] Q&A 예상 질문 30개 + 답변 준비

- [ ] Start-up NEST 선정 여부 확인 (+2점 가능)
