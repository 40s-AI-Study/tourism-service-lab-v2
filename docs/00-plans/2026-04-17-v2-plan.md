## AI 스터디용 히스토리 관리

### 기록 대상
1. 사용자 프롬프트 — `docs/02-prompts/YYYY-MM-DD-{summary}.md`
2. Claude 응답/계획 — `docs/00-plans/`, `docs/01-decisions/`
3. Paperclip 코멘트 — `docs/03-agent-outputs/{agent}/`
4. Heartbeat 로그 — `docs/04-session-logs/paperclip/`
5. Claude Code 세션 — `docs/04-session-logs/claude-code/`
6. 주요 의사결정 — `docs/01-decisions/{number}-{slug}.md` (ADR)

### 기록 빈도
- 실시간: 프롬프트, 주요 응답, 최종 산출물
- 배치 (5-10개): 일상 코멘트, 세부 API 스펙
- 일 1회: 세션 로그 합본

### 민감정보 필터링
- API 키, 토큰, OAuth secret 자동 스크럽
- 커밋 전 `scripts/scrub-secrets.sh` 실행
