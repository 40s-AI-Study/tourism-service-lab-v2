# TripCraft Korea — Phase 2 User Story Backlog

> **형식**: `US-{번호}: As a {사용자}, I want to {행동} so that {가치}.`  
> **우선순위**: 🔴 Must / 🟡 Should / 🟢 Could  
> **관련 기능**: F1~F6 (MVP 6대 기능 기준)

---

## 🔴 Must — Sprint 1~3 (F1~F3)

### F1. 관광 정보 검색

**US-001** 🔴  
As a 국내 여행자, I want to 지역과 테마 카테고리로 관광지를 검색할 수 있어서 원하는 장소를 빠르게 찾을 수 있다.  
*AC: 지역(시도) + 카테고리(7종) 선택 → 관광지 목록 3초 이내 반환*

**US-002** 🔴  
As a 외국인 여행자, I want to 영어·중국어·일본어로 관광지를 검색할 수 있어서 언어 장벽 없이 여행을 계획할 수 있다.  
*AC: 언어 선택 시 해당 언어 TourAPI 호출, 결과 동일 UI 표시*

**US-003** 🟡  
As a 여행자, I want to 키워드로 전국 관광지를 검색할 수 있어서 특정 장소를 바로 찾을 수 있다.  
*AC: 검색어 입력 → 연관 관광지 자동완성 + 결과 목록*

---

### F2. 맞춤형 여행 코스 자동 생성

**US-004** 🔴  
As a 당일치기 여행자, I want to 지역·인원·카테고리를 입력하면 당일 코스가 자동으로 만들어져서 2-3시간 계획 시간을 아낄 수 있다.  
*AC: 입력 5초 내 → 시간순 방문 코스(장소명·소요시간·이동방법) 생성*

**US-005** 🔴  
As a 1박2일 여행자, I want to 1박2일 일정 코스를 자동으로 받을 수 있어서 숙박지 기준 최적 동선을 알 수 있다.  
*AC: 숙박지 자동 포함, Day 1/Day 2 분리, 이동 거리 최소화*

**US-006** 🟡  
As a 가족 여행자, I want to 아이·노부모 동반 조건을 체크하면 접근성 코스가 생성되어서 모든 가족이 즐길 수 있다.  
*AC: 무장애 API 조회 → 계단·경사 제외 장소 우선 배치*

**US-007** 🟡  
As a 여행자, I want to 완성된 코스를 카카오맵으로 바로 볼 수 있어서 실제 이동 경로를 확인할 수 있다.  
*AC: 카카오맵 SDK → 코스 마커 + 경로 폴리라인 표시*

---

### F3. 혼잡도 회피 + 대안 추천

**US-008** 🔴  
As a 주말 여행자, I want to 인기 장소의 혼잡 예상 시간을 미리 알 수 있어서 대기 없이 방문할 수 있다.  
*AC: 방문자 추이 예측 API → 시간대별 혼잡도 배지(🟢🟡🔴) 표시*

**US-009** 🔴  
As a 여행자, I want to 인기 장소가 너무 혼잡할 때 비슷한 분위기의 한산한 대안을 3곳 받을 수 있어서 혼잡을 피할 수 있다.  
*AC: CLIP 유사도 상위 3개 대안 카드 표시 (이미지·이름·혼잡도·거리)*

**US-010** 🟡  
As a 소외 지역 탐방 여행자, I want to 덜 알려진 지역 관광지가 코스에 우선 포함되어서 새로운 여행지를 발견할 수 있다.  
*AC: 관광다양성 하위 30% 지역 가중치 1.5배 적용 확인*

---

## 🟡 Should — Sprint 4~5 (F4~F6)

### F4. 위치 기반 오디오 가이드 자동재생

**US-011** 🟡  
As a 문화유산 방문자, I want to 유산 50m 이내로 가면 오디오 가이드가 자동으로 재생되어서 스마트폰을 꺼내지 않아도 해설을 들을 수 있다.  
*AC: Geolocation API → 50m 감지 → 진동 1회 → 오디오 자동재생*

**US-012** 🟡  
As a 외국인 방문자, I want to 내 언어로 오디오 가이드를 들을 수 있어서 문화 맥락을 이해할 수 있다.  
*AC: 기기 언어 감지 또는 설정 기반 → 한·영·중·일 4개 언어 분기*

**US-013** 🟢  
As a 오프라인 여행자, I want to 와이파이 없이도 오디오를 들을 수 있어서 지하 문화재나 산간 지역에서도 사용할 수 있다.  
*AC: 코스 생성 시 오디오 파일 미리 다운로드 (캐시 저장)*

---

### F5. 여행 경비 자동 합산

**US-014** 🟡  
As a 예산 여행자, I want to 코스의 총 예상 경비(입장료·이동·식비)를 한 화면에서 볼 수 있어서 여행 예산을 미리 계획할 수 있다.  
*AC: TourAPI 입장료 + 카카오 모빌리티 이동비 + 평균 식비 합산 표시*

**US-015** 🟡  
As a 4인 가족 여행자, I want to 인원수별 총 경비를 볼 수 있어서 가족 여행 예산을 정확히 알 수 있다.  
*AC: 1인/2인/4인 경비 토글 전환*

---

### F6. 팔도 스탬프 적립

**US-016** 🟡  
As a 여행 수집가, I want to 16개 시도를 방문할 때마다 스탬프를 받을 수 있어서 전국 여행 기록을 남길 수 있다.  
*AC: 위치 인증(EXIF 또는 GPS) → 해당 시도 스탬프 적립*

**US-017** 🟡  
As a 지방 여행 장려 참여자, I want to 소외 지역 방문 시 스탬프 리워드가 2배가 되어서 지방 여행 동기가 생긴다.  
*AC: 관광다양성 하위 50% 지역 스탬프 2배 지급*

**US-018** 🟢  
As a 스탬프 수집자, I want to 전체 스탬프 완주 시 지역상품권을 받을 수 있어서 여행 비용을 일부 환급받을 수 있다.  
*AC: 16개 시도 완주 → 지역상품권 리워드 안내 화면 (MVP: 안내만, 실제 지급 Phase 3)*

---

## 🟢 Could — Sprint 6 이후

**US-019** 🟢  
As a SNS 사용자, I want to 내 코스를 카카오톡이나 인스타그램으로 공유할 수 있어서 친구와 여행 정보를 나눌 수 있다.

**US-020** 🟢  
As a 반복 사용자, I want to 내가 만든 코스를 저장하고 다시 볼 수 있어서 자주 가는 여행 패턴을 기록할 수 있다.

**US-021** 🟢  
As a 실시간 여행자, I want to 현재 위치 기반으로 주변 코스를 즉시 받을 수 있어서 즉흥 여행을 쉽게 할 수 있다.
