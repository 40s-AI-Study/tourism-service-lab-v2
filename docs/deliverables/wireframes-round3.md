# Round 3 — Hero Screen + 핵심 플로우 와이어프레임

> 6개 Round 3 업그레이드 아이디어별 Hero Screen ASCII 와이어프레임 + 핵심 사용자 플로우 + UI/UX 차별점. 모바일 우선 375px 기준. HTML 프로토타입(Step 7) 제작을 위한 상세 설계.

---

## 0. 디자인 원칙 (공통)

1. **감정이 첫 번째 입력값** — "어디 갈까?" 대신 "지금 어떤 기분이야?"로 시작
2. **혼잡도는 숨기지 않고 설계에 내재** — 혼잡도 하위 필터가 추천 알고리즘 핵심
3. **계절 × 감성 × 혼잡도 3축 교차** — 단순 목록이 아닌 교차 큐레이션
4. **영웅 배경은 풀스크린 감성** — 텍스트 최소화, 비주얼이 먼저 말한다
5. **한 손 조작 CTA** — 감정 카드·모드 선택 버튼은 하단 Thumb-zone 배치
6. **접근성 WCAG 2.1 AA** — 명도 대비 4.5:1 이상, 터치 영역 44×44pt

---

### 핵심 사용자 플로우

1. **[Screen: 권한 요청]** 앱 최초 실행 → 위치·마이크 권한 허용 팝업
2. **[Screen: Hero — 지도]** 산책 시작 → 지도 위 별(★) 아이콘 실시간 표시, GPS 자동 감지 시작
3. **[Screen: Hero — 근접 알림]** 문화유산 5m 이내 진입 → 하단 알림 카드 슬라이드업 ("5m 앞, 조선시대 우물터 — 탭해서 이야기 듣기")
4. **[Screen: 오디오 플레이어]** 카드 탭 → 계절(봄/여름/가을/겨울) × 성향(학구/감성/어린이) 분기 오디오 자동 재생, 전체화면 유산 사진 배경
5. **[Screen: 스토리 수집 지도]** 재생 완료 → 지도 위 해당 별 아이콘이 "수집됨(금색)" 으로 전환, 스토리 카드 갤러리에 저장
6. **[Screen: 오늘의 탐험 완료]** 스토리 3개 이상 수집 → "오늘 발견한 이야기 3개" 완료 배지 + 공유 카드 생성

---

### UI/UX 차별점

트리플·야놀자는 목적지를 검색한 뒤 이동하지만, 이 앱은 걷기만 해도 이야기가 위치 감지로 자동 따라와 탐험 자체가 콘텐츠가 된다.

1. **[Screen: Hero]** 앱 실행 → 계절 배경 자동 감지(기기 날짜 기준) + 4 감성 카드 표시

3. **[Screen: 필터 자동 적용]** 계절(봄) × 감성(불멍) × 혼잡도 조합 자동 계산 → "지금 추천 캠핑장 3개 찾는 중…" 로딩 카드
4. **[Screen: 캠핑장 추천 3개]** 카드 3장 (혼잡도 낮음 우선) — 캠핑장명 / 혼잡도 배지 / 날씨 예보 / 별 관측 확률 표시
5. **[Screen: 날씨·별 관측 알림]** 카드 탭 → 상세 화면: 날씨 예보 3일 + 별 관측 가능 시간대 + 일몰 시각
6. **[Screen: 예약]** 예약 버튼 탭 → 날짜 선택 → 결제
7. **[Screen: 출발 전 알림]** D-1 푸시 알림: "내일 🔥불멍 예보 최적! 장작 준비 체크리스트 확인하세요"

---

### UI/UX 차별점

캠핏·야놀자는 예약 가능 캠핑장 목록을 나열하지만, 이 앱은 감성 × 계절 × 혼잡도 3축 조합으로 "느낌"을 먼저 고른 뒤 캠핑장이 따라온다.

5. **[Screen: 방문 타이밍 알림]** 코스 저장 → "혼잡도 최저 시간대 알림 받기" 토글 → 최적 방문 시간 푸시 알림 설정
6. **[Screen: 코스 완료 — 회복 체크]** 코스 완료 후 팝업: "오늘 몸과 마음이 얼마나 회복됐나요?" 5단계 슬라이더 + 짧은 메모 입력

트리플·야놀자가 "어디 갈까?"를 묻는다면, 이 앱은 "지금 어떤 기분이야?"를 먼저 물어 감정 상태를 첫 번째 필터로 삼는다.

- 배경: 순백(#FFFFFF), 아무 장식 없는 클린 화면
- 질문: 28pt, 굵은 산세리프, 화면 상단 1/3 중앙
- 8 감정 카드: 4×2 그리드, 각각 고유 그라데이션 배경색
- 색상 코드: 감정별 심리 색채 배정 (위 hex 참고)

---

### 핵심 사용자 플로우

1. **[Screen: Hero]** 앱 실행 → 순백 화면 + "지금 기분이 어때?" 질문 + 8 감정 카드
2. **[Screen: 감정 선택]** 카드 1개 탭 (예: 😮‍💨지침) → 선택 카드 전체화면 확대 → 0.5초 인터랙션 후 전환
3. **[Screen: 교차 추천 로딩]** 감정(지침) × 혼잡도(하위) × 계절(봄) 3축 교차 알고리즘 → "지침을 달래줄 곳 찾는 중…"
4. **[Screen: 장소+코스+타이밍 카드]** 추천 결과: 장소명 / 이런 기분엔 이런 이유 / 혼잡도 배지 / 최적 방문 시각
5. **[Screen: 방문]** 길찾기 연동 → 이동 → 방문 체크인
6. **[Screen: 사후 체크 팝업]** 방문 2시간 후 푸시: "오늘 기분이 나아졌나요?" → 5단계 이모지 선택
7. **[Screen: 감정 여행 기록]** 기록 누적 → 월별 "내 감정 여행 지도" 리포트 (감정 색상별 방문 지도 시각화)

---

### UI/UX 차별점

기존 모든 관광 앱은 "어디 갈까?"에서 시작하지만, 이 앱은 감정 하나만이 유일한 입력값이며 목적지·코스·타이밍 모두 감정에서 역산된다.

### 핵심 사용자 플로우

1. **[Screen: Hero]** 앱 실행 → 새벽 사찰길 배경 + "지금 어떤 방식으로 회복하고 싶어?" + 3 모드 카드
2. **[Screen: 회복 모드 선택]** 3 카드 중 탭 (예: 🎧들으며) → 선택 카드 강조 + 알고리즘 3조건 교차 필터 시작
3. **[Screen: 교차 필터 로딩]** 숨은 명소(혼잡도 최하) + 오디오 스토리 보유 + 웰니스 코스 3조건 동시 충족 장소 탐색 → "당신만을 위한 치유 여정 만드는 중…"
4. **[Screen: 치유 여정 코스 추천]** 코스 카드: 장소명 / 거리 / 오디오 스토리 수 / 웰니스 체험 연계 가능 여부 / 혼잡도 배지
5. **[Screen: 이동 중 오디오 자동 재생]** 출발 후 위치 감지 → 해당 장소 50m 이내 진입 시 오디오 스토리 자동 재생 시작 (배경 재생, 화면 잠금 유지)
6. **[Screen: 웰니스 체험 연계 예약]** 코스 중간 웰니스 체험 (명상·산림욕·한방 체험) 예약 인라인 연결
7. **[Screen: 여정 완료 — 회복 일지]** 여정 완료 → "오늘 치유 여정 기록": 방문 장소 / 들은 오디오 스토리 / 회복 점수(5단계) + 짧은 메모 저장 → 누적 회복 일지

### UI/UX 차별점

트리플·야놀자가 "어디 갈까?"를 묻는다면, 이 앱은 "어떻게 회복할까?"를 먼저 물어 숨은 명소·오디오 스토리·웰니스 체험을 하나의 치유 여정으로 융합해 제공한다.
