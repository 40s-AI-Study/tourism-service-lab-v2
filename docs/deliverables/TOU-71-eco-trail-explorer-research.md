# [고도화] Eco Trail Explorer — 생태관광 코스 리서치

> 작성: 🔍 리서치님 | 참조 이슈: [TOU-71](/TOU/issues/TOU-71) | 부모: [TOU-65](/TOU/issues/TOU-65)

---

## 배경 요약

Round 6 통합 점수 **112/150** (탈락, 115점 컷 미달). 주요 약점: "두루누비 무료 앱 경쟁". 본 고도화 리서치는 이 약점을 정면 돌파하여 두루누비 대비 압도적 차별화 포인트를 구체화하고 공모전 제출 기준 7개 산출물을 완성한다.

## 산출물 1 — 페르소나 3종

### Persona A — 한국인 30-40대 에코 여행자

| 항목 | 내용 |
|---|---|
| **이름/나이** | 박지수 / 34세 |
| **직업** | IT 스타트업 디자이너, 서울 거주 |
| **가치관** | ESG·친환경 소비에 적극적. 인스타 여행 포스팅 즐김 |
| **여행 패턴** | 월 1-2회 당일 또는 1박 자연 트레킹. 대중교통 선호 |
| **주요 Goal** | "주말에 도심 탈출 — 자연 속에서 리프레시하면서 탄소발자국도 줄이고 싶다" |
| **Pain Point** | 두루누비 앱은 정보가 불편하고 낡음. 코스별 탄소발자국 비교가 불가능. 캠핑장·생태관광지 정보가 분산되어 통합 여정 설계 어려움 |
| **주요 행동** | 앱에서 코스 탐색 → 탄소 계산 → 인증샷 공유 → 에코 챌린지 배지 획득 |
| **관련 API** | 두루누비, 생태관광정보, 방문자집중률예측 |
| **수익 기여** | B2C 프리미엄 구독 (월 9,900원) |

## 산출물 2 — 핵심 사용자 시나리오 (5단계 플로우)

> Persona A (박지수, 34세) 기준 메인 시나리오

```
Step 1 [발견] 
  트리거: 금요일 저녁 "주말에 어디 갈까?" 고민
  행동:   앱 실행 → "이번 주말 혼자 / 수도권 / 10km 이하 / 생태관광" 설정
  API:    두루누비 courseList (지역·거리·난이도 필터)
             생태관광정보 areaBasedList (환경부 인증 생태관광지)
             방문자집중률예측 tatsCnctrRateList (향후 7일 혼잡도 조회)
  결과:   "이번 토요일 오전, 광주 무등산 생태관광 코스 — 혼잡도 28% (쾌적)"
             AI 추천 배너 노출

Step 2 [탐색]
  행동:   코스 상세 진입 → 탄소발자국 계산 실행
  API:    두루누비 crsTourInfo (코스 구간 상세, 포인트 정보)
             두루누비 routeList (경로 좌표 + 고도 정보)
  결과:   이동수단 3가지 비교 표 노출
             - 자가용: 3.6 kgCO₂e
             - 대중교통: 0.18 kgCO₂e ← AI 추천
             - 카셰어링: 1.2 kgCO₂e
          "대중교통 선택 시 나무 0.3그루 심은 효과"

Step 3 [계획]
  행동:   코스 확정 → 인근 친환경 캠핑장 검색
  API:    고캠핑 정보조회서비스 (코스 반경 20km 내 친환경 인증 캠핑장)
             방문자집중률예측 (캠핑장 주변 관광지 혼잡도)
  결과:   "가평 에코캠핑장 — 2.3km, 잔여 8사이트, 친환경 인증 ✓"
          원클릭 캠핑장 예약 연동

Step 4 [완주]
  행동:   트레킹 완주 → 앱에서 완주 인증
  API:    내부 에코 챌린지 서비스 (탄소 절감량 적립)
  결과:   "오늘 절감한 탄소: 3.42 kgCO₂"
          에코 뱃지 "숲 지킴이 Lv.3" 획득
          소셜 공유 카드 자동 생성 (인스타 최적화 비율)

Step 5 [공유·지속]
  행동:   이번 달 누적 탄소 절감량 리포트 확인
  결과:   "이번 달 12.4 kgCO₂ 절감 — 나무 1.2그루 효과"
          다음 추천 코스 3개 노출 (AI 개인화)
          기업 ESG 담당자라면: 팀 리포트 PDF 내보내기
```

## 산출물 3 — Visit Korea OpenAPI 매핑

### 사용 API 5종 명세

| # | API 명 | 핵심 오퍼레이션 | 주요 응답 필드 | 활용 화면 |
|---|---|---|---|---|
| 1 | **두루누비 정보 서비스** | `courseList`, `crsTourInfo`, `routeList` | 코스명, 거리, 난이도, 소요시간, 경로 좌표, 주요 포인트 | 코스 탐색, 코스 상세 |
| 2 | **생태관광 정보** | `areaBasedList`, `areaBasedList1` | 생태관광지명, 위치, 환경부 인증 여부, 탐방 프로그램 | 코스 탐색, 생태관광지 상세 |
| 3 | **방문자 추이 예측** | `tatsCnctrRateList`, `tatsCnctrRatedList` | 관광지별 향후 30일 집중률, 혼잡도 등급 | 코스 탐색 (혼잡도 배지), 추천 타이밍 |
| 4 | **무장애 여행 정보** | `areaBasedList2`, `locationBasedList2`, `detailInfo2` | 장애 유형별 접근 가능 여부, 휠체어 접근로, 장애인 화장실 | 배리어프리 필터, 접근성 상세 |
| 5 | **국문 관광정보 서비스** | `areaBasedList1`, `detailCommon1`, `locationBasedList1` | 캠핑장·숙박시설 상세, 주변 편의시설, 이미지 | 캠핑장 연결, 주변 정보 |

### API 호출 시퀀스

```
[코스 탐색 화면]
사용자 필터 설정
  ↓
① 두루누비 courseList (지역코드, 거리, 난이도)
  ↓ 코스 목록 반환
② 방문자집중률예측 tatsCnctrRateList (관광지ID, 날짜범위)
  ↓ 코스별 혼잡도 점수 매핑
③ 생태관광정보 areaBasedList (지역코드)
  ↓ 환경부 인증 생태관광지 오버레이
  → UI: 코스 카드 목록 + 혼잡도 배지 + 에코 인증 태그

[코스 상세 화면]
사용자 코스 선택
  ↓
④ 두루누비 crsTourInfo (코스ID)
  ↓ 구간 상세, 주요 포인트
⑤ 두루누비 routeList (코스ID)
  ↓ 경로 좌표 → 지도 렌더링
  → UI: 지도 + 탄소발자국 계산기 + 이동수단 비교

[캠핑장 연결 화면]
코스 확정 시
  ↓
⑥ 국문 관광정보서비스 locationBasedList1 (코스 중심좌표, 반경 20km, contentTypeId=28 캠핑장)
  ↓ 인근 캠핑장 목록
  → UI: 캠핑장 카드 + 잔여 사이트 + 친환경 인증 뱃지

[배리어프리 필터 적용 시]
  ↓
⑦ 무장애여행정보 areaBasedList2 (지역코드)
  ↓ 장애 유형별 접근 가능 관광지 필터링
⑧ 무장애여행정보 detailInfo2 (관광지ID)
  ↓ 세부 접근성 정보 (휠체어 경사로, 장애인 화장실, 주차)
  → UI: 배리어프리 코스 목록 + 접근성 상세 정보
```

### 차별화 포인트 3개 (핵심)

**차별화 1 — 탄소발자국 × 여정 통합 계산 (세계 최초 한국 에코트레킹 특화)**

두루누비는 코스 정보만 제공하며 탄소 계산 기능이 전무하다. Eco Trail Explorer는 이동수단(대중교통/카셰어링/자가용) × 숙박 유형 조합으로 코스별 탄소발자국을 즉시 비교하여 가장 친환경적인 여정을 자동 추천한다. "대중교통 선택 시 나무 X그루 심은 효과" UI로 행동 변화를 직접 유도한다.

**차별화 2 — AI 방문자 혼잡도 예측 × 에코 타이밍 추천 (국내 유일)**

방문자집중률예측 API(향후 30일)를 활용해 특정 코스·생태관광지의 최적 방문 시간대를 자동 추천한다. 두루누비·올댓캠핑 모두 혼잡도 정보 없음. 혼잡한 시간 회피 → 자연 보호 + 여행 품질 동시 향상. "이번 주 토요일 오전 8시, 혼잡도 28% (쾌적) — 지금 바로 코스 시작 추천" 알림 기능.

**차별화 3 — ESG B2B 에코투어 리포트 채널 (독자 수익 구조)**

두루누비는 공공앱으로 수익화 불가. Eco Trail Explorer는 기업 HR 담당자를 대상으로 팀 단체 에코투어 설계·참여 추적·탄소 절감량 자동 PDF 리포트를 제공한다. 국내 300대 기업 ESG 보고서 의무화(2025년~) 흐름과 직결된 B2B 수요를 독점한다. 계약 1건당 평균 3,000만원. 두루누비가 절대 제공할 수 없는 수익 채널.
