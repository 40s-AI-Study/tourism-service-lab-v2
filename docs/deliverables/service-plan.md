# TripCraft Korea — 서비스 기획서

> **한 줄 정의**: 카테고리와 지역을 선택하면 TripCraft Korea가 혼잡도·유사도·경비를 실시간으로 계산해 오디오 가이드·예약·팔도 스탬프가 붙은 원페이지 여행 플랜을 5초 만에 자동 생성한다.

> **슬로건**: 데이터 가치를 넘어, 서비스로 실현하다

> ⚠️ 이 문서는 KoreaPath AI 기획서를 대체합니다. 최종 제출 서비스는 **TripCraft Korea**입니다 (제출일: 2026-05-06).

---

## 1. 문제 정의 (Why)

### 1-1. 핵심 Pain Point

| # | 구조적 문제 | 핵심 지표 | 출처 |
|---|---|---|---|
| P1 | **오버투어리즘** | 북촌 거주 인구 △27.6%, 주말 일 1.5만 명 몰림 | 서울시 북촌 관광관리 실태조사 2023 |
| P2 | **수도권·제주 쏠림** | 관광 지출 점유 55%↑, 전남·전북·강원 내륙 다양성 하위 33% | 문체부 국내관광 통계조사 2024 |
| P3 | **여행 정보 파편화** | 평균 5개 앱 사용, 준비 시간 40% 차지 | 한국소비자원 여행 행태 조사 2024 |
| P4 | **외국인 개별 여행객 다국어 공백** | 방한 1,750만 명·개별 여행객 70%+, 통합 다국어 가이드 부재 | KTO 외래관광객 실태조사 2024 |

이 네 문제는 **인스타 바이럴 → 플랫폼 인기순 노출 → 정보 파편화 → 외국인 수도권 고립**으로 연결되는 단일 순환 고리를 형성한다. TripCraft Korea는 KTO 공공데이터 알고리즘으로 이 순환을 직접 끊는다.

### 1-2. 시장 공백

| 기존 서비스 | 한계 |
|---|---|
| 트리플 | 사용자가 일정 직접 배치, 혼잡도 예측 없음 |
| 야놀자 | 숙박·액티비티 예약 중심, 코스 최적화 없음 |
| Klook | 외국인 체험 예약 특화, 개인화 코스 없음 |
| Google Maps | 한국 혼잡도 데이터 취약, 로컬 큐레이션 없음 |
| Visit Korea 앱 | 다국어 OK, AI 기능 없음, UX 구식 |

→ **"카테고리 감성 보존 × 혼잡도 회피 × 오디오 가이드 자동재생 × 경비 합산 × 지역 균형"** 조합은 국내외 전무.

## 2. 타겟 사용자 (Who)

### 2-1. 핵심 페르소나 (3종)

| 페르소나 | 연령·특성 | 핵심 니즈 |
|---|---|---|
| **국내 MZ 솔로/커플** | 20-35, 스마트폰 여행자 | 5초 안에 실행 가능한 플랜, 줄 없는 포토스팟 |
| **4050 가족 여행자** | 35-50, 자녀·노부모 동반 | 접근성 필터, 경비 투명화, 안전한 코스 |
| **방한 외국인 FIT** | 20-40, 개별 여행객 | 다국어 오디오 가이드, 현지 맥락 이해 |

### 2-2. 시장 규모 (TAM/SAM/SOM)

- **TAM**: 국내 관광객 3.2억 회/년 + 방한 외래객 1,750만 명/년
- **SAM**: 20-50대 스마트폰 여행자 약 2,200만 명
- **SOM (3년)**: 초기 2.5% 침투 목표 → 55만 MAU

## 3. 핵심 기능 (What)

### 3-1. 공모전 MVP 6대 기능

| # | 기능 | 작동 방식 | 활용 KTO OpenAPI |
|---|---|---|---|
| F1 | **관광 정보 검색** | 지역·종류·키워드로 전국 26만 곳 빠른 검색 | 국문관광정보 서비스 |
| F2 | **맞춤형 여행 코스 자동 추천** | 지역·일정·인원 선택 → 5초 내 1박2일/2박3일 일정 자동 생성 | 연관관광지정보 / 기초지자체 중심관광지정보 |
| F3 | **혼잡도 회피 + 대안 추천** | 인기 장소 포화 시 유사 분위기 한산 대안 3곳 제시 | 관광지집중률·방문자추이예측 / 관광빅데이터 정보서비스 |
| F4 | **위치 기반 오디오 가이드 자동재생** | 문화유산 50m 진입 시 진동 + 4개 언어 자동 재생 | 관광지 오디오가이드 정보서비스 / 다국어 관광정보 서비스 |
| F5 | **여행 경비 자동 합산** | 입장료·숙박·이동·식비 자동 계산, 1인/4인 총경비 한 화면 표시 | 국문관광정보 서비스 / 카카오 모빌리티 |
| F6 | **팔도 스탬프 적립** | 16개 시도 방문 인증 → 지역상품권 리워드 (소외 지역 2배 보너스) | 지역별 관광다양성 / 카카오 간편 로그인 |

### 3-2. Phase 2 (공모전 이후 6개월)

- 실시간 예약 원스톱 연동 (숙박·식당·체험)
- 소셜 트렌드 분석 (네이버 블로그 바이럴 감지)
- UGC 코스 공유 플랫폼 (팔로우·평점)

### 3-3. Phase 3 (12개월+)

- 글로벌 확장 (일본·대만·동남아 역방향)
- B2B API 판매 (지자체·호텔·항공사)
- KTO 정책 데이터 공동 연구

## 5. 예상 사용 시나리오

### 시나리오 A — 국내 솔로 여행자

```
금요일 저녁 앱 실행
→ 카테고리: "자연/에코트레일" + 지역: "강원 내륙" + 당일
→ 5초 내 코스 자동 생성 (혼잡도 🟢 + 오디오 준비 완료)
→ 토요일: 가는 길마다 오디오 가이드 자동 재생
→ 팔도 스탬프 +1 적립 → 강원 지역상품권 500원 리워드
```

### 시나리오 B — 방한 외국인 FIT

```
인천공항 도착 → 앱 자동 언어 감지 (English)
→ 카테고리: "Cultural Heritage" + 지역: "전주"
→ 문화유산 50m 진입 시 영어 오디오 자동 재생
→ 트리플·야놀자 사용 없이 원페이지 플랜으로 실행
```
