# [고도화] Hidden Gem Scout — KTO 강소형 잠재관광지 발굴 탐험 플랫폼

> 참조 이슈: [TOU-70](/TOU/issues/TOU-70) | 부모: [TOU-65](/TOU/issues/TOU-65)
> BA 100/130 (전체 2위) · AI Branch: companion · 팀 합의 8.2/10

---

## 배경 요약

Round 6 통합 점수 **116/150** (통과, 7위). 핵심 차별화: KTO 강소형 잠재관광지 DB(2,400개소) 독점 활용 + 개척자 배지 게이미피케이션 + AI 혼잡도 기반 실시간 "진짜 한적한 곳" 추천. 지역분산 관광 BA 10점 만점. 본 고도화에서는 사업성(현재 6점) 보강을 위해 지자체 B2G 파트너십 채널을 구체화하고 외국인 페르소나 시나리오를 추가한다.

|---|---|
| **이름/나이** | 김태양 / 37세 |
| **직업** | 중학교 역사 교사, 경기도 수원 거주 |
| **가치관** | "관광 명소보다 아무도 모르는 진짜 한국을 발견하는 여행이 의미있다" |
| **여행 패턴** | 월 1-2회 자가용 당일 또는 1박. 경쟁적 탐험에 흥미 — 배지·랭킹에 적극 반응 |

| **주요 행동** | 앱으로 미개척 잠재관광지 탐색 → 혼잡도 낮은 곳 선택 → 방문 → 리뷰 등록 → 배지 획득 |
| **수익 기여** | B2C 프리미엄 구독 (월 9,900원) |

## 산출물 2 — 핵심 사용자 시나리오 (5단계 플로우)

> Persona A (김태양, 37세 역사교사) 기준 메인 시나리오

```
Step 1 [탐험 시작]
  트리거: 주말 아침, 앱 실행 → "탐험 모드" 진입
  행동:   지역 선택(충남), 난이도(보통), 배리어프리(불필요) 설정
  AI 처리: 기초지자체중심관광지 areaBasedList → 충남 지자체별 잠재관광지 목록 조회
             방문자집중률예측 → 현재 혼잡도 낮은 순서로 정렬
             지역별관광수요강도 → 방문자 증가 추세 있으나 아직 덜 알려진 곳 우선 필터
  결과:   "충남 청양 OO협곡 — 혼잡도 8% (극히 한적)"
             "미개척 점수: 94점 (최근 3개월 방문자 수 전국 하위 5%)"

Step 2 [검증]
  행동:   잠재관광지 상세 진입
  API:    관광지별연관관광지 areaBasedList1 → 함께 방문하기 좋은 인근 관광지 TOP 5
             국문관광정보서비스 detailCommon1 → 기본 정보(주소, 운영, 주차)
             방문자집중률예측 tatsCnctrRatedList → 오늘/이번 주 예측 혼잡도
  결과:   "오늘 방문자 예측: 34명 (평상시 수준)"
             "연관 관광지: OO온천 2.1km, OO사 4.3km"
             AI 코스 자동 생성: "협곡 → 온천 → 고즈넉한 사찰" 2박 3일 플랜

Step 3 [출발 준비]
  행동:   교통 정보 확인 → 대중교통 경로 탐색
  API:    위치기반관광정보서비스 locationBasedList1 (주변 버스터미널·주차장)
             기상청 API → 오늘 날씨 → 방문 적합도 판단
  결과:   "청양 직행버스 1일 4회 운행"
             AI 메시지: "오늘 맑고 기온 18도 — 협곡 트레킹 최적 조건 ✅"

Step 4 [현장 방문 & 발굴]
  행동:   도착 → 앱 체크인 → "첫 발굴자" 여부 확인
  앱 기능: GPS 기반 체크인 확인 → 해당 관광지 누적 체크인 순위 표시
             AR 카메라: 관광지 역사·생태 정보 오버레이
  결과:   "🎉 이번 달 충남 OO협곡 첫 번째 발굴자! 개척자 배지 획득"
             리뷰 작성 → 첫 리뷰어 포인트 지급

Step 5 [커뮤니티 공유 & 성장]
  행동:   리뷰 + 사진 등록 → 앱 내 발굴자 랭킹 확인
  결과:   발굴자 랭킹: 전국 38위 → 이번 발굴로 전국 31위 상승
             "이 관광지를 발굴한 사람: 김태양 (최초)" 영구 기록
             지자체 연계: 청양군 관광담당자 → 발굴 데이터 수령 → 정책 반영
```

## 산출물 3 — Visit Korea OpenAPI 매핑

### 사용 API 6종 명세

| # | API 명 | 핵심 오퍼레이션 | 주요 응답 필드 | 활용 목적 |
|---|---|---|---|---|
| 1 | **기초지자체 중심 관광지 정보** | `areaBasedList`, `areaBasedList1` | 지자체별 중심관광지 100위, 연결성 점수, 좌표 | 강소형 잠재관광지 핵심 DB (타 서비스 미활용) |
| 2 | **관광지별 연관 관광지** | `areaBasedList1`, `searchKeyword1` | 연관 관광지 최대 50위 (관광·음식·숙박 유형별) | AI 코스 자동 생성 (잠재관광지 → 주변 동선) |
| 3 | **방문자 추이 예측** | `tatsCnctrRateList`, `tatsCnctrRatedList` | 향후 30일 집중률·혼잡도 등급 | 진짜 한적한 곳 실시간 동적 추천 |
| 4 | **지역별 관광 수요 강도** | `areaTarExpDsList`, `areaTarSjrnDsList` | 관광 체류 강도, 소비 강도 | 방문자 증가 중이나 아직 덜 알려진 잠재 스팟 발굴 |
| 5 | **국문 관광정보 서비스** | `detailCommon1`, `locationBasedList1`, `detailImage1` | 관광지 상세, 주차, 운영시간, 이미지 | 잠재관광지 상세 정보 및 이미지 제공 |
| 6 | **무장애 여행 정보** | `areaBasedList2`, `detailInfo2` | 장애 유형별 접근 가능 여부, 편의시설 | 시니어·접근성 약자 필터 (Persona C 대응) |

### API 호출 시퀀스

```
[탐험 모드 — 잠재관광지 목록 화면]
사용자 지역·난이도·배리어프리 설정
  ↓
① 기초지자체중심관광지 areaBasedList (지역코드)
  ↓ 지자체별 중심관광지 100위 목록
② 지역별관광수요강도 areaTarSjrnDsList (지역코드, 기간)
  ↓ 체류 강도 낮은 (=덜 알려진) 순 필터
③ 방문자집중률예측 tatsCnctrRateList (관광지ID 배열, 날짜범위)
  ↓ 현재 혼잡도 낮은 순 정렬 → "미개척 점수" 계산
  → UI: 잠재관광지 카드 목록 + 미개척 점수 배지

[관광지 상세 화면]
사용자 관광지 선택
  ↓
④ 국문관광정보서비스 detailCommon1 (contentId)
  ↓ 상세 정보, 이미지, 주차, 운영시간
⑤ 관광지별연관관광지 areaBasedList1 (contentId)
  ↓ 함께 방문하기 좋은 연관 관광지 TOP 5
⑥ 방문자집중률예측 tatsCnctrRatedList (contentId, 날짜)
  ↓ 오늘/이번 주 예상 혼잡도 → 최적 방문 시간
  → UI: 상세 + AI 동선 추천 + 최적 방문 시간 배너

[배리어프리 필터 적용 시]
  ↓
⑦ 무장애여행정보 areaBasedList2 (지역코드)
  ↓ 접근 가능 잠재관광지 필터링
  → UI: 시니어·접근성 약자용 코스 목록

[현장 체크인 시]
  ↓
⑧ 위치기반관광정보서비스 locationBasedList1 (GPS 좌표, 반경 3km)
  ↓ 주변 음식점·주차장·편의시설
  → UI: 체크인 완료 + 발굴자 배지 + 주변 정보
```

---

### 차별화 포인트 3개

**차별화 1 — KTO 강소형 잠재관광지 DB 독점 활용 (진입장벽 최강)**

트리플·여기어때·네이버는 모두 이미 알려진 유명 관광지 중심. Hidden Gem Scout는 한국관광공사 기초지자체 중심관광지 DB(2,400개소)를 독점 활용해 "타 서비스에는 존재하지 않는 숨은 관광지"를 제공한다. 이 데이터를 서비스화한 경쟁사가 없어 진입장벽이 강하며, 지역별 관광수요강도 API와 조합해 "방문자 증가 추세지만 아직 덜 알려진 곳"을 자동 선별한다.

**차별화 2 — AI 혼잡도 기반 실시간 "미개척 점수" (국내 유일)**

방문자집중률예측 API(향후 30일)와 지역별관광수요강도 API를 조합해 관광지마다 "미개척 점수(0-100)"를 실시간 산출한다. 혼잡도 10% 미만 + 방문자 수 낮음 + 체류 강도 상승 = 미개척 점수 90점 이상. 기존 어떤 앱도 이런 동적 지표를 제공하지 않는다. 여행자 입장에서 "진짜 아무도 없는 곳"을 데이터로 확인하고 갈 수 있는 최초 서비스.

**차별화 3 — 발굴자 게이미피케이션 × 지자체 B2G 듀얼 효과**

개척자 배지·전국 발굴자 랭킹이라는 게이미피케이션이 사용자 재방문과 자발적 신규 관광지 리뷰 생성을 유도한다. 동시에 이 발굴 데이터가 지자체에 B2G 리포트로 제공되어 관광 정책 수립에 활용된다. 공모전 심사 기준인 "사회적 가치 + 지역분산 관광" 두 항목에 직접 기여하는 유일한 구조.
