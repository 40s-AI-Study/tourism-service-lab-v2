# KoreaPath AI — 사업 모델

> 단일 앱으로 국내·외국인 여행자 모두 공략 가능한 AI 여행 플래너. 공모전 수상 후 **18개월 내 손익분기(BEP)**, 3년 내 **연매출 30억원** 달성 목표.

---

## 1. Executive Summary

| 항목 | 내용 |
|---|---|
| 서비스 | KoreaPath AI — AI 여행 코스 자동 생성 + 혼잡도 최적화 |
| 타겟 | 20-40대 한국인 + 방한 외국인 FIT 여행자 |
| 수익 모델 | Freemium(B2C) + B2B API + 커머스 커미션 |
| 3년 ARR | 30억원 |
| BEP | Month 18 |
| 초기 자본 | 5억원 (공모전 상금 + 시드 투자) |

## 2. 시장 규모 (TAM / SAM / SOM)

### 2-1. TAM (전체 시장)

| 구분 | 규모 | 근거 |
|---|---|---|
| 국내 관광객 | 연 3.2억 회 이동 | 관광공사 2024 |
| 방한 외래객 | 연 1,620만 명 | 관광공사 2024 추정 |
| 국내 OTA 시장 | 연 22조원 | 한국온라인쇼핑협회 |
| **TAM (연)** | **약 25조원** | 국내 OTA + 외국인 여행 플랫폼 합산 |

### 2-2. SAM (진입 가능 시장)

- **20-40대 스마트폰 여행자**: 약 1,800만 명
- **방한 FIT 외국인 (영·중·일)**: 약 1,085만 명
- **코스 추천·혼잡도 정보 니즈**: 조사 인구 82%
- **SAM (연)**: 약 3.5조원 (플래닝·가이드 앱 세그먼트)

### 2-3. SOM (3년 내 확보 가능)

| 연차 | MAU 목표 | 점유율 | 연매출 |
|---|---|---|---|
| Y1 (2026) | 5만 | 0.2% | 2억 |
| Y2 (2027) | 25만 | 1.0% | 12억 |
| Y3 (2028) | 80만 | 3.0% | 30억 |

## 3. 수익 모델 (Revenue Streams)

### 3-1. 수익원 4가지

| # | 유형 | 설명 | 비중 (Y3) |
|---|---|---|---|
| R1 | **Freemium 구독** | 무제한 코스 생성·오디오 가이드·광고 제거 월 4,900원 | 35% |
| R2 | **커머스 커미션** | 숙박·식당·체험 예약 수수료 5-10% | 40% |
| R3 | **B2B API 판매** | 지자체·호텔·항공사에 AI 코스 API 제공 | 18% |
| R4 | **광고** | 지역 관광·먹거리 타겟 광고 | 7% |

### 3-2. 단가 가정

| 채널 | ARPU / 거래당 |
|---|---|
| Premium 구독 | 4,900원/월 (연 58,800원) |
| 예약 커미션 | 평균 숙박 단가 15만원 × 7% = 10,500원 |
| B2B API | 지자체 500만원/년, 대형 파트너 5,000만원/년 |

### 3-3. Unit Economics (Y2 기준)

- **CAC**: 3,500원 (주로 바이럴·SNS 기반)
- **LTV**: 32,000원 (평균 구독 11개월 + 예약 2회)
- **LTV/CAC**: **9.1x** (SaaS 기준 우수)
- **Payback**: 4.2개월

## 4. 경쟁 분석

### 4-1. 경쟁 매트릭스

| 서비스 | AI 코스 | 혼잡도 | 다국어 | 접근성 | 한국 특화 | 가격 |
|---|---|---|---|---|---|---|
| 트리플 | ❌ | ❌ | 부분 | ❌ | ✅ | Free + 광고 |
| 네이버 여행 | ❌ | ❌ | ❌ | ❌ | ✅ | Free |
| Google Maps | ❌ | 부분 | ✅ | ❌ | ❌ | Free |
| Visit Korea 앱 | ❌ | ❌ | ✅ | 부분 | ✅ | Free |
| Klook | ❌ | ❌ | ✅ | ❌ | ❌ | Free + 예약 |
| **KoreaPath AI** | **✅** | **✅** | **✅** | **✅** | **✅** | Freemium |

### 4-2. 해자(Moat)

1. **데이터 해자** — Visit Korea OpenAPI 15개+ 결합 (경쟁사 모방 시 시간 필요)
2. **사용자 해자** — 사용 이력 기반 개인화가 누적되며 리텐션 상승
3. **B2B 해자** — 지자체·호텔과의 계약 락인 (관광공사 협업으로 초기 우위)
4. **규제 해자 낮음** — 진입장벽이 낮으므로 빠른 확장 + UX 차별화 필수

## 7. Go-to-Market 전략

### Phase 1: 공모전 & 론칭 (2026 Q2-Q3)
- 공모전 수상 → 언론 노출 + 관광공사 공식 홍보
- 20대 여행 인플루언서 50명 시딩 (SNS 공유 카드 바이럴)
- 서울·제주·강릉 3개 도시 선행 론칭

### Phase 2: 국내 확산 (2026 Q4 - 2027 Q2)
- 카카오T/네이버지도 API 연계
- B2B 파트너십 — 경주·부산·전주 지자체 공동 캠페인
- 접근성 필터 사회적 가치 캠페인 (장애인의 날 등)

### Phase 3: 외국인 확장 (2027 Q3-Q4)
- 일본(X·LINE), 대만(Dcard), 영어권(Reddit/r/korea) 타겟 마케팅
- KTO 해외 지사 협업 (9개국)
- 공항 면세·호텔 체크인 파트너십

### Phase 4: B2B & 글로벌 (2028+)
- 호텔 체인(롯데·신라) API 판매
- 항공사(대한항공·아시아나) 제휴
- 일본·대만 역방향 서비스
