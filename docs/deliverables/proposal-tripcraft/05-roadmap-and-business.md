## 5-1. 단계별 발전 로드맵 (4단계)

### 1단계 — D-Day MVP (공모전 제출 버전)

| 항목 | 내용 |
|------|------|
| **핵심 기능** | ① 바이럴 장소 유사 분산 추천 (17시도 × 3후보 지도 UI) ② 원페이지 1박2일 플랜 자동 생성 ③ 팔도 스탬프 시스템 프로토타입 (인증 + 리워드 시뮬레이션) |
| **콘텐츠 커버리지** | TourAPI 26만건 임베딩 완료 / 17시도 전체 장소 수집 |
| **MAU 목표** | 심사위원·베타 테스터 — 실사용자 기반 없음 (시연 환경) |
| **기술 스택** | CLIP + sentence-transformers (로컬) / Chroma DB / 카카오 API 3종 / TourAPI |
| **여유로움 지수** | 빅데이터 통계 기반 (과거 6개월 평균 방문자 비교) — 실시간 아님 명시 |
| **인증 방식** | 위치 EXIF + 이미지 해시 / 영수증 OCR + 국세청 API |

### KPI 상호 연관 구조

```
MAU 증가
   ↓ 플랜 생성 증가
      ↓ 코스 완성률 향상 (실행 가치 입증)

            ↓ 소외 지역 스탬프 가중 → 분산률 달성
               ↓ KTO 정책 성과 지표 충족 → B2B 계약 근거
```

## 5-4. 지역 균형 관광 정책 연계

### KTO 정책 정합성

| 정책 사업 | TripCraft Korea 연계 방식 |
|----------|--------------------------|
| **KTO 지역관광 예산** (2025~2026 지역관광 육성 300억원 규모) | 팔도 스탬프 지역상품권 리워드 재원으로 KTO 예산 집행 채널화. 플랫폼이 디지털 쿠폰 배포·정산 인프라 역할 수행 |
| **디지털 관광주민증 사업** (KTO 시범 운영 중) | 팔도 스탬프 17시도 인증 데이터를 디지털 관광주민증 방문 이력과 API로 상호 연동. 동일 인증 행위가 두 시스템에 동시 기록되어 사용자 피로 없이 혜택 이중 적립 가능 |
| **지역 균형 관광 분산 목표** | `area-tourism-diversity` API 소외 시도 가중치 1.5~2배 적용으로 자연 분산 유도. 분산률 KPI(≥33%)를 KTO 보고 지표로 직접 제출 가능 |

### 17시도 팔도 스탬프 단계별 확장 시나리오

**1차 확장 — 광역 (D-Day ~ 1년)**

17개 광역시도 단위 스탬프 전체 개통. 대도시(서울·부산·대구·인천·광주·대전·울산·세종) 8곳은 대표 관광지 3~5곳 인증 필요. 도 단위(경기·강원·충북·충남·전북·전남·경북·경남·제주) 9곳은 대표 관광지 2~3곳 인증으로 완주 장벽을 낮춤. 소외 시도(관광 다양성 지수 하위 33%) 방문 시 리워드 1.5~2배 적용.

```
광역 스탬프 기준 예시
  서울  → 경복궁·북촌·한강공원 중 3곳 인증
  강원  → 강릉해변·속초·평창 중 2곳 인증
  전북  → 전주한옥마을·군산·순창 중 2곳 인증 (소외 지역 리워드 2배)
  세종  → 국립세종수목원·세종호수공원 중 1곳 인증 (신규 도시 특별 배지)
```

**2차 확장 — 시군구 (1년 ~ 3년)**

229개 시군구 단위 세부 스탬프 확장. 1차 파일럿: 관광 다양성 지수 최하위 30개 시군구 우선 개통 (예: 경북 봉화군, 전남 신안군, 강원 양구군). 지역 소상공인·지자체 관광재단과 공동 운영 협의체 구성. 시군구 스탬프 5개 이상 완주 시 해당 광역도 스탬프도 함께 달성 처리(업셀링 설계).

```
시군구 스탬프 예시 (전남 신안군 — 섬 지역 특화)
  퍼플섬(박지도·반월도) 인증 → "섬 여행자" 특별 배지
  천사대교 포토스팟 인증 → 신안군 스탬프 + 리워드 2배
```

## 5-7. 수상 후 확장 비전 — 글로벌 K-Tourism Hub

TripCraft Korea가 목표하는 최종 비전은 단순한 여행 앱을 넘어, 한국의 지역 균형 관광 데이터 인프라 위에 구축된 글로벌 K-Tourism Hub다. 방한 외국인이 출국 전 본국에서 앱을 설치하고, 17시도 팔도 스탬프 완주를 여행 버킷리스트로 삼아 다회 방한을 계획하며, 달성한 스탬프와 오디오 스토리 기록이 하나의 디지털 여행 여권이 되는 세계를 설계한다. 이 플랫폼 위에 지자체·관광공사·항공사·카드사가 API로 연결되어 각자의 리워드를 실어 보내고, TripCraft는 분산 알고리즘이라는 중립 엔진으로 모든 이해관계자의 이익이 소외 지역 활성화로 수렴하도록 설계된 인프라 레이어로 진화한다. K-컬처가 한국을 세계의 관심사로 만든 지금, TripCraft Korea는 그 관심을 서울 한 곳이 아닌 17시도 전체로 연결하는 디지털 다리가 될 것이다.
