## 1) 서비스 기획배경 및 필요성

### 1-1 서비스 기획 배경

국내 관광 시장은 네 가지 구조적 문제가 하나의 악순환을 이루고 있습니다.

| 문제 | 현황 | 출처 |
|------|------|------|
| **오버투어리즘** | 북촌한옥마을 거주 인구 5년간 **27.6% 감소**, 주말 방문자 하루 1.5만 명 초과 | 서울시 북촌 관광관리 실태조사 2023 |
| **지역 관광 불균형** | 서울·경기·제주가 전체 관광 지출의 **55% 이상** 독점 | 문체부 국내관광 통계조사 2024 |
| **여행 정보 파편화** | 여행자 1인당 평균 **5개 앱** 사용, 준비 시간의 40% 소모 | 한국소비자원 여행 행태 조사 2024 |
| **외국인 FIT 공백** | 방한 **1,750만 명** 중 개별여행 70%+, 통합 다국어 가이드 전무 | KTO 외래관광객 실태조사 2024 |

> 상기 통계는 각 기관의 2023~2024년 공식 보고서 기준이며, 시뮬레이션 수치는 TourAPI 과거 데이터 기반 추정입니다.

이 네 문제는 "인스타 바이럴 → 플랫폼 인기순 노출 → 정보 파편화 → 외국인 수도권 고립"으로 이어지는 하나의 순환 고리입니다. 현재 어떤 서비스도 이 순환 자체를 끊어내지 못하고 있습니다.

### 1-2 서비스 필요성

**TripCraft Korea**는 한국관광공사 공공데이터(혼잡도·다양성·수요강도)와 카카오 서비스 4종을 알고리즘으로 결합해 위 순환을 직접 끊어냅니다.

- 한국관광공사 2025~2027 핵심 정책 **'지역 균형 관광'** 과 직접 정합
- 소외 지역 가중치 1.5배 + 팔도 스탬프 리워드 2배로 **정책 목표를 사용자 동기로 전환**
- 분산 추천 적용 시 상위 10% 과부하 관광지 집중도 **20~30% 완화** 목표
- 소외 지역 선택 비율 현행 15% → **33% 이상** 달성 목표

핵심 사용자: P1 이지은(27세, 포토스팟 크리에이터) / P2 박성호(39세, 가족 여행) / P3 Marc Dupont(33세, 프랑스 FIT) / P4 윤·김 커플(40대, 슬로우 트래블)

## 2) 서비스 개요

### 2-1 기획 서비스 소개명

> **TripCraft Korea — 카테고리 테마 기반 개인화 여행 플래너**

여행 카테고리(문화유산·미식·자연·축제·힐링·포토스팟·K-컬처)와 지역을 선택하면, 한국관광공사 혼잡도·다양성 데이터를 실시간 결합해 동선·경비·예약·오디오·스탬프가 통합된 **원페이지 실행 플랜**을 5초 만에 자동 생성합니다.

### 2-2 기획 서비스 주요 기능

| # | 기능명 | 이렇게 동작합니다 | 활용 공공데이터 |
|---|--------|-------------------|----------------|
| ① | **관광 정보 검색 기능** | 지역·카테고리·키워드로 전국 26만 곳에서 원하는 장소를 빠르게 찾습니다 | 국문관광정보 서비스 |
| ② | **맞춤형 코스 자동 추천 기능** | 지역·일정·인원을 고르면 5초 안에 Day1/Day2 일정표가 자동 생성됩니다 | 연관관광지정보 서비스 / 카카오 모빌리티 |
| ③ | **혼잡도 회피 + 대안 추천** | 인기 폭발 장소 대신 분위기 91%+ 유사하면서 한산한 다른 시도의 후보 3곳을 보여줍니다 | 관광지집중률·방문자추이예측 서비스 |

| ⑤ | **여행 경비 자동 합산** | 입장료·숙박·이동·식비를 자동 계산해 총경비를 한 화면에 보여줍니다 | 국문관광정보(usefee·roomrate) / 카카오 모빌리티 |
| ⑥ | **팔도 스탬프 적립** | 16개 시도 방문 시 인증사진·영수증으로 스탬프를 모으고 지역상품권을 받습니다 | 지역별 관광다양성 서비스 / 카카오 로그인 |

> 프로토타입 6화면(카테고리 선택 → 17시도 지도 → 타임라인 → 오디오 → 경비 → 스탬프)은 별첨 `prototypes/tripcraft-korea/index.html`에서 시연 가능합니다.

### 2-3 서비스 차별점

기존 서비스는 단순 검색·예약에 머물지만, TripCraft Korea는 공공데이터를 결합한 알고리즘으로 **5가지를 동시 제공**하는 유일한 서비스입니다.

| 비교 항목 | 트리플 | 야놀자 | 마이리얼트립 | ChatGPT | **TripCraft Korea** |
|:---|:-:|:-:|:-:|:-:|:-:|
| AI 코스 자동 생성 | △ | △ | △ | ✗ | **✓** |
| 위치 기반 오디오 자동 재생 | ✗ | ✗ | ✗ | ✗ | **✓** |
| 혼잡도 자동 회피 + 분산 | ✗ | ✗ | ✗ | ✗ | **✓** |
| 경비 자동 합산 (4항목) | ✗ | △ | △ | ✗ | **✓** |
| 소외 지역 스탬프 가중 리워드 | ✗ | ✗ | ✗ | ✗ | **✓** |

**3대 핵심 차별점:**

1. **분위기 보존 분산** — 바이럴 장소를 포기하지 않고 AI 유사도 91%+ 대안을 같은 카테고리 맥락에서 제시
2. **원페이지 실행 플랜** — 지도 길찾기·예약·경비·오디오가 단일 화면에 통합 (텍스트 코스가 아닌 즉시 실행)
3. **걸으면 따라오는 이야기** — 문화유산 50m 진입 시 5개국어 오디오 자동 재생, QR 없이 외국인까지 포괄

### 2-4 서비스 내 지역 특화 관련 사항 (지역 특화 서비스 — 작성 필수)

TripCraft Korea는 **서울 제외 16개 시도 전체를 커버하는 지역특화 엔진**입니다. 사용자가 지역을 선택하는 순간 해당 시도의 향토자원·소외 지수·RTO 콘텐츠를 1:1 결합한 코스가 자동 생성됩니다.

| 지역특화 구현 | 설명 | 적용 시도 | 활용 공공데이터 |
|---|---|---|---|
| 소외 지역 노출 강화 | 관광 다양성 지수 하위 33% 시도 노출·리워드 **1.5~2배** 자동 가중 | 강원·충남·전북·전남·경북 등 | 지역별 관광다양성 서비스 |
| RTO 협업 콘텐츠 우선 | 8개 RTO 추천 코스·축제 카드 우선 노출 | 부산·대전·광주·세종·충남·경북·강원·제주 | 기초지자체 중심관광지정보 서비스 |
| 지역 경제 활성화 | 팔도 스탬프 + 지역상품권 리워드 | 서울 제외 16개 시도 | 지역별 관광자원수요 서비스 |
| KTO 정책 연계 | 디지털 관광주민증 인증 데이터 양방향 연동 | 전 시도 | KTO 협약 채널 |

> **동작 사례**: '강릉(강원)' 선택 → 다양성 지수 하위 가중 1.5배 적용, 안목해변 바이럴 장소는 강문해변·정동진으로 분산 추천, 강원 RTO 추천 축제 우선 노출, 강원도 팔도 스탬프 적립

본 항목은 공모전 지역 특화 가점 대상이며, 8개 RTO 특별상 후보 지위를 확보합니다.

### 3-2 데이터 활용 방식 (용도)

단순히 공공데이터를 화면에 나열하는 것이 아닌, **데이터를 결합하는 알고리즘**으로 설계했습니다. 한국관광공사 빅데이터 3종을 AI 분석과 교차 연산해 기존에 없는 **새로운 기준 3가지**를 만들어냅니다.

| 새로운 기준 | 만드는 방법 | 사용자에게 보이는 모습 |
|---|---|---|
| **여유로움 지수** | 관광빅데이터(인기 장소 방문자) ÷ 관광빅데이터(후보지 방문자) | "이 장소는 인기 장소 대비 3배 여유로워요" |
| **소외 지역 가산점** | 관광다양성 지수 하위 33% 판정 → 노출·리워드 ×1.5 적용 | "이 지역 방문 시 스탬프·상품권 1.5배!" |
| **분위기 유사도** | AI 이미지 분석 40% + 카테고리 일치 40% + 텍스트 분석 20% | "안목해변과 분위기 91% 유사한 곳: 오이도" |

**데이터 결합 시나리오 — 문화유산 카테고리 (경주)**

```
① 관광자원수요 → 경주 문화자원 수요 지수 확인
② 관광다양성   → 소외 지역 가산점 결정
③ 국문관광정보 → 문화유산 후보 50곳 수집
④ 집중률예측   → 혼잡 시간대 회피 배치
⑤ 관광사진     → 카드 비주얼 표시
⑥ 공모전수상작 → 촬영 각도 가이드
⑦ 국문관광정보 → 입장료·예약·좌표 획득
⑧ 오디오가이드 → 50m 반경 오디오 자동 연결
⑨ 연관관광지   → 다음 코스 추천
⑩ 오디오가이드 → 언어·계절·성향별 재생
⑪ 카카오맵     → 길찾기 즉시 실행
```

| 구분 | 설명 |
|---|---|
| 단순 나열 방식 | API 결과를 화면에 그대로 보여주기만 함 |
| **TripCraft 결합 방식** | 빅데이터 3종 + AI 벡터 → 신규 기준 3가지 생성 → 원페이지 플랜에 통합 |

## 4) 서비스 발전 방향

### 4-1 개발 서비스 향후 발전방향

**4단계 로드맵**

| 단계 | 핵심 기능 | 이용자 목표 | 파트너십 |
|---|---|---|---|
| **MVP (공모전)** | 분산 추천·원페이지 플랜·팔도 스탬프 프로토타입 | 시연 환경 | KTO 공모전 |
| **6개월 후** | 코스 저장·실시간 재조정·다국어 플랜 | 월 **5,000명** | 야놀자·여기어때 제휴 |
| **1년 후** | 통신사 실시간 혼잡도·AR 포토스팟·지자체 화이트라벨 | 월 **5만 명** (외국인 20%) | KTO 디지털 관광주민증 양방향 연동·지자체 3곳 SaaS 계약 |
| **3년 후** | 일본·동남아 스탬프, K-Tourism Pass | 월 **30만 명** | 항공사·카드사·JTB 글로벌 협업 |

**수익 모델 (3년차 합계 약 50억 원)**

| 수익원 | 구조 | 비중 |
|---|---|---|
| 제휴 커미션 | 숙박·식당·체험 예약 전환 수수료 5~10% | 45% |
| B2B 라이선스 | 지자체·관광재단 화이트라벨 SaaS | 30% |
| 광고·인센티브 | 스폰서드 카드·지역상품권·스탬프 스폰서십 | 25% |

> **산출 근거 (3년차 MAU 30만 기준)**
> - 제휴 커미션: 30만 × 플랜 생성율 50% × 예약 전환율 25% × 건당 5,000원 = 월 1.87억 → 연 22.5억
> - B2B 라이선스: 지자체·관광재단 25곳 × 평균 연 6,000만원 = 15억
> - 광고·인센티브: 스폰서드 카드 CPM + 스탬프 스폰서십 월 500~2,000만원 × 12개월 = 12.5억

**핵심 운영 목표 (KPI)**

| 지표 | 정의 | 1년 목표 |
|---|---|---|
| 분산률 | 소외 시도(다양성 하위 33%) 선택 비율 | **35% 이상** |
| 재방문율 | 스탬프 적립자의 30일 내 재이용 비율 | **40% 이상** |
| 외국인 비율 | 비한국어 설정 월간 이용자 비율 | **20% 이상** |

> **목표치 산출 근거**
> - 분산률 35%: 17시도 균등 분산 시 하위 33%(6개 시도) 합산 = 약 35%. 가중치 1.5배 적용으로 균등 수준 달성 가능
> - 재방문율 40%: 게이미피케이션 적용 서비스 업계 평균 30~50% 범위의 중간값 (Duolingo 사례 참고)
> - 외국인 20%: 방한 FIT 1,750만 중 앱 설치 전환율 0.1% = 연 1.75만 명 ≈ MAU 5만의 35%
