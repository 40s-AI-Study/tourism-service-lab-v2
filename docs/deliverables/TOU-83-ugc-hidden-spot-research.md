# [기획고도화] UGC Hidden Spot — SNS 바이럴 숨은 명소 발굴 플랫폼

> 참조 이슈: [TOU-83](/TOU/issues/TOU-83) | 부모: [TOU-65](/TOU/issues/TOU-65)
> BA 98/130 · AI Branch: ai-only · 팀 합의 8.0/10

---

## 배경 요약

Round 6 통합 점수 **113/150** (탈락, 115점 컷 미달 1점 차). 핵심 아이디어: Instagram·YouTube·TikTok 공개 데이터에서 "48시간 이내 폭발적으로 업로드 증가 중인 관광지"를 자동 감지하고, 협찬·광고 콘텐츠를 자동 분류해 진성 바이럴 스팟만 제공. ai-only 브랜치 — AI 없이 서비스 자체가 성립 불가. 본 고도화에서는 SNS 파이프라인 구체화 + B2B 여행사 대시보드로 수익성 강화.

| 항목 | 내용 |
|---|---|
| **이름/나이** | 이지아 / 27세 |
| **직업** | 인스타그램 여행 크리에이터, 팔로워 5.2만 |
| **가치관** | "아무도 찍지 않은 곳을 먼저 찍어야 내 계정 가치가 유지된다" — 독점 콘텐츠가 곧 수익 |
| **여행 패턴** | 주 1-2회 당일치기 서울 근교. 이미 유명한 장소는 무가치하다고 인식 |

| **주요 행동** | 앱 알림 수신 → 바이럴 급증 스팟 확인 → 협찬 여부 확인 → 이동 → 독점 촬영 → 포스팅 |
| **수익 기여** | B2C 프리미엄 구독 (월 12,900원) |

## 산출물 2 — 핵심 사용자 시나리오 (5단계 플로우)

> Persona A (이지아, 인스타 크리에이터) 기준 메인 시나리오

```
Step 1 [알림 수신]
  트리거: 월요일 오전 7시, 앱 푸시 알림
           "🔥 강원도 홍천 어느 카페 — 지난 48시간 인스타 게시물 +340%"
  행동:   알림 탭 → 바이럴 스팟 카드 확인
  AI 처리: Instagram 공개 해시태그 크롤링 → 게시물 급증 패턴 감지
             NLP 감성 분석 → "협찬/광고" 표시 없는 진성 포스팅 비율 산출
             → 진성 바이럴 점수 87점 (신뢰도 HIGH)
  API:    위치기반관광정보서비스 (해당 스팟 좌표·기본 정보 조회)
             방문자집중률예측 tatsCnctrRateList (현재 혼잡도)

Step 2 [검증]
  행동:   스팟 상세 진입 → "왜 지금 뜨는가?" 분석 탭 확인
  AI 처리: YouTube 공개 API → 최근 72시간 관련 영상 조회수 추이
             협찬 마커 감지 (해시태그 #광고, #sponsored, 캡션 패턴 분석)
             진짜 바이럴 vs 마케팅 집행 자동 분류
  결과:   "협찬 콘텐츠 비율: 12% (낮음 — 자연 발생 바이럴)"
             "예측 피크 시간: 이번 주 토-일요일 오후 2-5시"
  API:    방문자집중률예측 → 이번 주말 예상 혼잡도 72% (보통)

Step 3 [사전 예약]
  행동:   방문 계획 → 최적 방문 시간 확인 → 인근 주차·교통 정보 확인
  API:    국문관광정보서비스 detailCommon1 (상세 정보, 주차, 운영시간)
             위치기반관광정보서비스 locationBasedList1 (반경 3km 편의시설)
  결과:   최적 방문 시간 추천: "수요일 오전 9-11시 혼잡도 23% (최저)"
             "금일 방문자 예측: 340명 (평상시 대비 +280%)"

Step 4 [현장 촬영]
  행동:   방문 → 앱으로 체크인 → 포토스팟 AI 가이드 활성화
  앱 기능: 다른 크리에이터가 찍은 각도 표시 → "아직 찍히지 않은 앵글" 추천
             실시간 조도·날씨 기반 최적 촬영 타임 알림
  결과:   체크인 완료 → "발굴자 배지" 획득 (최초 방문자 Top 10)

Step 5 [공유 & 리포트]
  행동:   포스팅 후 앱 연동 → 내 포스팅 성과 추적
  앱 기능: 내 콘텐츠 도달률 + 해당 스팟 바이럴 점수 상관 분석
             "이 스팟을 먼저 찾은 발굴자" 프로필 뱃지 → 팔로워 신뢰도 강화
  수익 연동: 여행사/관광청 B2B 대시보드에 크리에이터 매칭 광고 제안
```

## 산출물 3 — Visit Korea OpenAPI 매핑

### 사용 API 명세

| # | API 명 | 핵심 오퍼레이션 | 주요 응답 필드 | 활용 목적 |
|---|---|---|---|---|
| 1 | **국문 관광정보 서비스** | `detailCommon1`, `locationBasedList1`, `searchKeyword1` | 관광지명, 주소, 좌표, 운영시간, 이미지, 주차 | 바이럴 스팟 상세 정보 조회, 주변 편의시설 |
| 2 | **위치기반 관광정보 서비스** | `locationBasedList1` | 좌표 기반 반경 내 관광지·음식·숙박 목록 | 현장 도착 시 주변 정보, 지도 핀 표시 |
| 3 | **방문자 추이 예측** | `tatsCnctrRateList`, `tatsCnctrRatedList` | 관광지별 향후 30일 집중률·혼잡도 등급 | SNS 버즈량 교차 검증, 방문 타이밍 추천 |
| 4 | **관광 빅데이터 정보** | (빅데이터 시계열 조회) | 지역별 관광 트렌드, 방문자 통계 | 바이럴 급증 스팟의 실제 방문 데이터 검증 |
| 5 | **관광지별 연관 관광지** | `areaBasedList1`, `searchKeyword1` | 연관 관광지 최대 50위 (음식·숙박·관광 유형별) | 바이럴 스팟 방문 시 동선 연계 추천 |

### 외부 API (공개 범위 내 합법 수집)

| # | API 명 | 활용 목적 | 법적 근거 |
|---|---|---|---|
| E1 | **Instagram Basic Display API** | 공개 해시태그 게시물 수 추이, 좋아요·댓글 수 집계 | Meta 공개 API 정책 내 |
| E2 | **YouTube Data API v3** | 최근 72시간 관련 영상 조회수, 공개 메타데이터 | Google API 이용약관 내 |
| E3 | **기상청 단기예보 API** | 방문 당일 날씨 → 촬영 최적 조건 예측 | 공공 API 무료 |

### API 호출 시퀀스

```
[바이럴 탐지 파이프라인 — 백엔드 스케줄러, 6시간 주기]
① Instagram 공개 해시태그 크롤링 (위치 태그 포함 게시물)
  ↓ 게시물 수 시계열 데이터
② YouTube Data API → 관련 영상 조회수 추이
  ↓
③ NLP 협찬 감지 모델 → 진성 바이럴 점수 산출
  ↓
④ KTO 방문자추이예측 tatsCnctrRateList → 실제 혼잡도 교차검증
  ↓
⑤ 바이럴 스팟 DB 업데이트 → 푸시 알림 발송 (점수 80점 이상)

[사용자 스팟 상세 조회]
⑥ 국문관광정보서비스 detailCommon1 (contentId)
  ↓ 관광지 기본 정보, 이미지
⑦ 위치기반관광정보서비스 locationBasedList1 (좌표, 반경 3km)
  ↓ 주변 음식점·카페·주차장
⑧ 방문자추이예측 tatsCnctrRatedList (contentId, 날짜)
  ↓ 이번 주 예상 혼잡도 → 최적 방문 시간 추천
  → UI: 스팟 카드 + 바이럴 그래프 + 최적 방문 시간 배너
```

## 산출물 5 — 차별화 포인트 (경쟁 서비스 대비)

### 경쟁 비교 매트릭스

| 기능 | 인스타그램 탐색탭 | 네이버 블로그 | 트리플 | **UGC Hidden Spot** |
|---|---|---|---|---|
| 숨은 명소 발굴 | ❌ (이미 유명한 곳만) | ❌ (SEO 상위만) | ❌ | ✅ 급등 48h 내 스팟 |
| 협찬 vs 진성 구분 | ❌ | ❌ | ❌ | ✅ NLP 자동 분류 |
| 바이럴 예측 | ❌ | ❌ | ❌ | ✅ 48시간 예측 알고리즘 |
| 방문 최적 타이밍 | ❌ | ❌ | ❌ | ✅ KTO 혼잡도 연동 |
| 발굴자 게이미피케이션 | ❌ | ❌ | ❌ | ✅ 배지·랭킹 |
| B2B 여행사 대시보드 | ❌ | ❌ | ❌ | ✅ 트렌드 API 제공 |
| B2G 지자체 리포트 | ❌ | ❌ | ❌ | ✅ 주간 SNS 모니터링 |

---

### 차별화 포인트 3개

**차별화 1 — "48시간 바이럴 예측" (세계 유일 한국 로컬 관광지 특화)**

기존 SNS는 이미 유명해진 곳만 탐색탭에 노출. UGC Hidden Spot은 Instagram·YouTube 공개 데이터에서 게시물 급증 패턴을 실시간 감지해 "지금 막 뜨기 시작하는 장소"를 크리에이터에게 48시간 먼저 알려준다. KTO 방문자집중률 예측 API로 교차 검증해 정확도를 높인다. 경쟁 서비스 어디에도 이 기능 없음.

**차별화 2 — "협찬 vs 진성 바이럴 자동 구분" (신뢰도 핵심 차별화)**

마케팅 예산으로 인위적으로 만들어진 바이럴인지, 자연 발생인지 구분하는 것이 핵심 가치. 해시태그 패턴·캡션 언어 패턴·반복 계정 행동·후원 표시 의무 규정 위반 탐지 등 NLP 모델로 자동 분류. 크리에이터 관점: "진짜 좋아서 가는 곳"만 추천받음으로써 팔로워 신뢰 유지.

**차별화 3 — B2B/B2G 듀얼 수익 채널 (두 개의 독립 수익원)**

B2C 크리에이터 구독 외에 여행사(주간 트렌드 리포트 API 제공)와 지자체(SNS 모니터링 대시보드)라는 독립된 두 B2B 채널 보유. 두루누비·인스타그램 등 단일 채널 서비스 대비 수익 구조 안정성 압도적으로 높음.
