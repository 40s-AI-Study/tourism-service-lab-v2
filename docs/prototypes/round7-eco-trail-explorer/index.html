Section 01 — 서비스 개요

What: 생태관광 통합 플랫폼

두루누비 트레킹 코스 + 생태관광지 + 친환경 캠핑장을 AI가 통합 설계하는 에코 여행 앱. 탄소발자국 실시간 계산과 ESG 기업 복지 B2B 파트너십으로 수익화.

🎯

타겟 페르소나 A

한국인 30-40대 에코 여행자. 주말 트레킹과 자연 체험을 즐기며 탄소발자국에 관심 있는 MZ 세대 전문직.

🌏

타겟 페르소나 B

외국인 에코투어리스트. 한국 자연을 깊이 경험하고 싶은 친환경 가치 지향 여행자. 영어·일어·중국어 지원.

🏢

타겟 페르소나 C

ESG 담당 HR 매니저. 300대 기업 직원 복지 여행 프로그램을 친환경으로 전환하려는 B2B 고객사 담당자.

Section 12 — MVP 기능 우선순위

Must / Should / Could 우선순위

4주 MVP 개발 범위. Must는 공모전 제출 필수, Should는 UX 완성도, Could는 V2 로드맵.

우선순위기능근거난이도
Must두루누비 코스 탐색 + AI 탄소발자국 계산핵심 차별화. 공모전 심사위원 1순위 확인 포인트중간
Must생태관광지 + 친환경 캠핑장 연결 예약3-way 통합 핵심. 두루누비 단독과의 차별점낮음
Must에코 챌린지 (뱃지·레벨·누적 절감량)재방문 동기 구조. 공모전 창의성 항목 핵심낮음
Should영어 UI + 외국인 페르소나 지원Sarah 시뮬 PASS 조건. 글로벌 관광 공모전 가산점낮음
Should국립공원 실시간 혼잡도 + 대체 코스 추천안전·편의성 가산점. 실시간 데이터 차별화중간
ShouldB2B ESG 리포트 자동 생성 (PDF)박성호 시뮬 계약 전환 조건. B2B 수익 핵심중간
Could탄소 크레딧 거래 연동환경부 정책 연계. V2 추가 수익 채널높음
Could오프라인 코스 캐싱 (산악 음영지역)등산 안전성 강화. 통신 불안 구간 해결높음
CouldSNS 에코 인증 공유 (인증 카드 생성)바이럴 성장. UGC 에코 데이터 수집 연계낮음
