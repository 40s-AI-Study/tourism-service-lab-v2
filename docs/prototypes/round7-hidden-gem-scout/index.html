Section 03
리서치 근거 (Why)
01
국내 관광지 상위 50곳 방문 집중도 78%
전국 관광지 방문의 78%가 상위 50개 인기 관광지에 몰림. 나머지 수천 개 관광지는 방문자 부족으로 지역 경제 활성화 실패.
출처: 한국관광공사 관광지집중률 예측 데이터 2024
02
강소형 잠재관광지 2,400개소 미발굴
한국관광공사가 발굴·등재한 강소형 잠재관광지 2,400개소 이상이 있으나, 방문자 정보·접근로·콘텐츠 부족으로 사실상 방치 상태.
출처: 한국관광공사 강소형관광지 사업보고서 2024
03
비수도권 방문율 60% 달성 — 정부 목표 (2026)
문화체육관광부가 2026년까지 지방 방문율 60% 이상을 목표로 수도권 집중 관광 분산 정책 추진 중. 플랫폼과 정책 방향 일치.
출처: 문화체육관광부 관광진흥기본계획 2024
04
20-30대 "새로운 곳 발굴" 여행 동기 1위 56%
국내 여행을 계획하는 20-30대의 56%가 "아직 가보지 못한 새로운 곳 발굴"을 1순위 동기로 응답. 유명 관광지 재방문 욕구 낮음.
출처: 한국관광공사 여행트렌드 리포트 2025
05
SNS 인증샷 관광지 효과: 방문자 300% 증가
SNS에 처음 알려진 숨은 명소가 바이럴 후 3개월 이내 방문자가 평균 300% 이상 증가한 사례 다수 확인. 게임화 + SNS 인증은 바이럴 핵심 메커니즘.
출처: 한국관광공사 SNS 관광 효과 분석 2024
06
장애인 여행 접근성 정보 부족 — 88% 불만족
장애인 여행자의 88%가 여행 전 접근성 정보 불충분을 최대 장벽으로 응답. 무장애 잠재관광지 정보를 통합 제공하는 서비스 전무.
출처: 한국장애인개발원 여행 실태조사 2024
Section 04
핵심 사용자 시나리오 5단계 플로우
SCENARIO
박지수(34세, 탐험형 내국인)가 Hidden Gem Scout를 처음 열어 강원도 영월군 숨은 명소를 발굴하고, 첫 방문자 배지를 획득하기까지의 전 과정
1
🔍 선호 조건 입력
앱 실행 → 현재 위치(대전) 자동 감지 → 선호 자연 유형(산·계곡), 이동 시간(차량 2시간 이내), 방문자 수(월 10명 이하), 접근성 필요 여부 선택. AI가 조건을 벡터화해 강소형 DB와 매칭 시작.
📡 강소형잠재관광지정보 API 📡 위치기반관광정보서비스
2
🤖 AI 발굴 리스트 생성
AI가 실시간 방문자 추이 예측 데이터와 강소형 DB를 교차 분석 → "지금 이 순간 방문자가 가장 적은 곳" 상위 10개 추출. 각 장소마다 발굴 난이도(희귀도 점수), 예상 방문 시간, "첫 방문자 가능성" 퍼센트 표시.
📡 관광지집중률방문자추이예측 📡 관광지별연관관광지정보
3
📍 장소 상세 확인 + 경로 탐색
"영월 요선암 돌개구멍" 선택 → 상세 정보(한국어/영어), 오디오 가이드 자동 재생, 접근 경로(차량·대중교통), 주차 정보, 무장애 접근 가능 여부, 연관 관광지(식당·숙소) 표시. KTO 관광 사진으로 현장 분위기 미리보기.
📡 국문관광정보서비스 📡 관광지오디오가이드 📡 무장애여행정보
4
🤳 방문 인증 + GPS 검증
현장 도착 → GPS 위치 자동 인증(반경 100m 이내) → 인증샷 업로드 → AI가 사진과 장소 매칭 검증 → 방문 기록 DB 저장. 방문자 카운터 실시간 업데이트. 타임스탬프 기반 "첫 방문자" 여부 즉시 판정.
📡 위치기반관광정보서비스 📡 관광사진정보
5
🏆 배지 획득 + 커뮤니티 공유
"첫 방문자" 배지 즉시 발급 → 개인 탐험 랭킹 갱신 → SNS 공유 카드 자동 생성(장소명·배지·지도) → 지역 소상공인 할인 쿠폰 발송 → 연관 숨은 명소 3곳 추가 추천. 지자체 파트너에게 방문 통계 데이터 제공(B2G 수익화).
📡 관광지방문자통계 📡 지역별관광수요강도

Visit Korea OpenAPI 매핑 + 호출 시퀀스
강소형잠재관광지정보 관광지집중률방문자추이예측 국문관광정보서비스 위치기반관광정보서비스 무장애여행정보 관광지오디오가이드 관광지별연관관광지정보
📡 API 호출 시퀀스 — 장소 발굴 플로우
1
위치기반관광정보서비스 · locationBasedList
사용자 GPS 좌표 → 반경 내 전체 관광지 목록 조회 (강소형 포함). 첫 필터링 레이어.
mapX={lng} mapY={lat} radius=50000 contentTypeId=12
2
강소형잠재관광지정보 · areaBasedList
Step 1 결과와 강소형 DB 교차 → 강소형 등재 여부 필터링. 핵심 독점 데이터 소스.
areaCode={sido} sigunguCode={sigungu} numOfRows=20
3
관광지집중률방문자추이예측 · tatsCnctrRateList
강소형 후보지별 향후 30일 방문자 추이 예측 조회 → "지금 가장 한적한 곳" 역산 정렬.
areaCode={sido} signguCode={sigungu}
4
국문관광정보서비스 · detailCommon2 (+ 무장애여행정보)
선택 장소의 상세 정보(설명·운영시간·연락처) + 무장애 접근 가능 여부(경사로·주차·화장실) 병렬 조회.
