HOW IT WORKS

빅데이터 4종으로
잠재력 지역 발굴

수요 강도, 자원 공급, 다양성, 실제 행동 데이터 — 4가지 렌즈로 한국 여행 지도를 새로 그립니다.

01
🔥
수요 히트맵
탐색
체류 강도·소비 강도를 지역별 히트맵으로 시각화. 관광 자원 수요 데이터를 중첩해 "수요는 높지만 공급이 부족한" 잠재력 지역 자동 식별.
02
⭐
관광 잠재력
종합 점수 산출
다양성 지수 + 빅데이터 실제 행동 데이터(이동통신·카드·내비)로 지역별 "관광 잠재력 종합 점수" 계산. 데이터 근거 의사결정.
03
🔔
뜨기 전 알림 +
코스 생성
30일 방문자 예측으로 "아직 붐비지 않은 최적 방문 시점" 알림 구독. 지역 중심 관광지 코스 자동 생성.
