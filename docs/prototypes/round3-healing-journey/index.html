서비스 개요 — What

치유여정 Korea란 무엇인가

KoreaWellness + Audio Story Korea + LocalSecret 3개 서비스를 통합한 감성 치유 여행 플랫폼. 소리와 침묵으로 회복하는 감성 여행을 핵심 가치로, 번아웃된 20–40대가 '어디 갈까?'가 아닌 '어떻게 회복할까?'라는 감정 JTBD에서 출발해 치유 모드(🎧 들으며 / 🌿 쉬며 / ✨ 발견하며)를 선택하면 맞춤 웰니스 코스, 오디오 스토리, 혼잡 회피 타이밍을 자동으로 큐레이션합니다.

MVP 기능 우선순위 — MoSCoW

4주 MVP 범위 — Must / Should / Could

공모전 데모 기준 Must 완성 → Should 2주 내 추가 → Could 이후 로드맵으로 배치합니다.

Must — 데모 필수
감정 JTBD 진입 — 치유 모드 3종 선택
홈 화면에서 들으며 / 쉬며 / 발견하며 선택 → 모드별 코스 큐레이션 분기
wellness-tourism · visitor-concentration-forecast
코스 추천 + 혼잡도 뱃지 실시간 표시
추천 코스 리스트에 🟢/🟡/🔴 혼잡도 상태 표시 · 필터 (한산한 곳 먼저)
visitor-concentration-forecast · area-tourism-diversity
코스 상세 + 오디오 미리듣기 (30초)
장소별 오디오 스토리 30초 샘플 재생 · 혼잡도 시간대별 예보 캘린더
audio-guide · wellness-tourism
방문 일정 선택 + 출발 확정
혼잡도 데이터 기반 최적 시간대 자동 추천 · 캘린더 저장
visitor-concentration-forecast
Should — 2주 내 추가
감정 체크인 — 번아웃 진단 5문항
감정 상태 선택 → 맞춤 치유 모드 자동 분기 (S7)
wellness-tourism
코스 진행 중 오디오 전체 재생 + 위치 연동
GPS 포인트 도착 시 오디오 자동 트리거 · 파형 애니메이션
audio-guide · durunubi-trails
치유 완료 기록 + 회복 점수 시각화
완료 시 회복 점수(+12) 부여 · 숨은 명소 발견 배지 지급
photo-contest-winners · area-tourism-diversity
Could — 이후 로드맵
외국인 영문 K-웰니스 모드 (S8)
영문 오디오 가이드 + 다국어 UI (EN/JP/ZH/ES)
tourism-info-english · audio-guide
Premium 구독 결제 + B2B 파트너십 포털 (S9)
$15/월 구독 모델 · 웰니스센터 B2B 계약 관리 대시보드
모든 API 풀 접근
4계절 치유 코스 큐레이션 + 소셜 공유
계절별 특화 코스 자동 추천 · 인스타그램 공유 카드 생성

§11
4주 개발 일정 — Roadmap

공모전 제출까지 주차별 마일스톤

Must 기능 3주 완성 → 4주차 QA·최적화·제출 준비. 각 주차 KTO OpenAPI 연동과 UI를 병행합니다.

Week 1
기반 설계 + API 연동
05.05 – 05.11
  • 프로젝트 구조 설계 (React Native / Flutter)
  • KTO OpenAPI 8종 키 발급 + 연동 테스트
  • 감정 JTBD 데이터 모델 설계
  • S1 홈/모드선택 화면 구현
🎯 마일스톤: API 연동 완료 + S1 동작
Week 2
핵심 플로우 구현
05.12 – 05.18
  • S2 코스추천 + 혼잡도 뱃지 실시간 반영
  • S3 코스상세 + 오디오 미리듣기 30초
  • S4 일정선택 — 최적 시간대 자동 추천
  • visitor-concentration-forecast API 캐싱 전략 구현
🎯 마일스톤: Must 코어 플로우 완성
Week 3
Should 기능 + UI 완성
05.19 – 05.25
  • S5 진행중 — GPS + 오디오 자동 트리거
  • S6 완료 — 회복 점수 + 숨은 명소 배지
  • S7 감정체크인 플로우 연결
  • 다크 포레스트 UI 완성 + 애니메이션 polish
🎯 마일스톤: 9개 화면 인터랙션 완성
Week 4
QA · 최적화 · 제출 준비
05.26 – 06.01
  • API 응답 캐싱 최적화 (Redis / 로컬 캐시)
  • 접근성 검수 (시니어 페르소나 기준)
  • S8 외국인 영문 모드 + S9 구독플랜 UI
  • 공모전 제출 자료 (기획서 PDF + 데모 영상)
🎯 마일스톤: 공모전 제출 완료
