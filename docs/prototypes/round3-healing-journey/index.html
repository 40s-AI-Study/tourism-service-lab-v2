서비스 개요 — What

치유여정 Korea란 무엇인가

KoreaWellness + Audio Story Korea + LocalSecret 3개 서비스를 통합한 감성 치유 여행 플랫폼. 소리와 침묵으로 회복하는 감성 여행을 핵심 가치로, 번아웃된 20–40대가 '어디 갈까?'가 아닌 '어떻게 회복할까?'라는 감정 JTBD에서 출발해 치유 모드(🎧 들으며 / 🌿 쉬며 / ✨ 발견하며)를 선택하면 맞춤 웰니스 코스, 오디오 스토리, 혼잡 회피 타이밍을 자동으로 큐레이션합니다.

심사 강점
  • 감정 JTBD("어떻게 회복할까?") 기반 서비스 기획 — 기존 관광 플랫폼과 명확히 차별화된 진입점 확보
  • KTO OpenAPI 8종 풀 활용 — 오디오 가이드·웰니스·혼잡도·두루누비 등 데이터 다각화로 데이터 활용 평가 최고점 목표
  • 3개 서비스 융합으로 단일 서비스 대비 기능 완성도·커버리지 압도적 우위
  • 5가지 성향 스펙트럼 + 3가지 치유 모드로 MZ·외국인·시니어 전 세대 포괄 가능
  • 웰니스 투어리즘 글로벌 성장($8,940억) 트렌드와 정확히 일치하는 사업 발전성
  • 다크 포레스트 감성 UI — 치유 콘셉트를 시각적으로 구현, 심사위원 첫인상 차별화
