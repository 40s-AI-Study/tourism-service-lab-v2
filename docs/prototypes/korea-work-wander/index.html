HOW IT WORKS

3단계로 완성되는
K-워케이션 플랜

언어 선택부터 글램핑 매칭, 업무 후 관광까지 — 데이터로 증명되는 최적 워케이션 지역.

01
🌍
언어 선택 +
지역 탐색
FR / DE / ES / EN 중 선택 → 관광 자원 수요·다양성 데이터 기반 "한 달 살기 가능한 워케이션 지역" 추천. 제주·강릉·전주·담양 큐레이션.
02
🏕️
글램핑 + 작업 환경
매칭
goCamping API로 와이파이·전기 제공 글램핑장 필터링. 파워콘센트·고속 인터넷·조용한 환경 태그로 "일 할 수 있는 숙소" 즉시 확인.
03
🌅
업무 후 관광
자동 연동
숙소 기준 연관 관광지 자동 매핑. 오전 작업 → 오후 관광 → 저녁 로컬 식당 시간대별 추천으로 완벽한 워케이션 루틴.
