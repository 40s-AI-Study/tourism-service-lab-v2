HOW IT WORKS

데이터로 찾는
진짜 로컬 맛집

신용카드 실소비 데이터가 증명하는 "현지인 맛집 지역"을 3단계로 발굴합니다.

01
📍
지역 선택 +
음식 테마 설정
전주·부산·전남 등 지역 선택 후 한정식·길거리·파인다이닝·특산물 등 음식 테마 설정. 계절별 제철 음식 자동 추천.
02
💳
빅데이터 소비 지역
히트맵 확인
신용카드 실소비 강도 + 이동통신 체류 데이터로 "현지인이 진짜 가는 골목"을 히트맵으로 시각화.
03
🗺️
미식 코스 +
포토 스팟 생성
연관 관광지·카페·시장을 자동 연결해 "먹고 + 찍고 + 즐기는" 미식 여행 코스 완성.
