🤝 Companion AI

1
실시간 다변수 대안 도출: 혼잡도 + 날씨 + 대중교통 거리 + 주차 가능 여부 + 유사도 점수 5개 변수 동시 처리 → 즉시 최적 대안 3곳 랭킹. 단순 지도 앱 필터링으로 불가능.
2
방문 이력 개인화 학습: 과거 방문 패턴 · 만족도 피드백 지속 학습 → "지난번 한적하다고 좋아했던 곳과 비슷한 곳" 점점 정확해지는 개인화 추천. 정적 알고리즘 불가.
3
