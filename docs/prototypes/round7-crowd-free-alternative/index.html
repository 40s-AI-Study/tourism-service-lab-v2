Section 02 — Visit Korea OpenAPI 매핑

사용 API 명세 + 호출 시퀀스

5개 KTO OpenAPI를 조합합니다. tatsCnctrRateList가 핵심 혼잡도 엔진, areaBasedList1이 대안 후보 생성 역할.

API명오퍼레이션역할호출 시점

국문 관광정보 서비스detailCommon2
locationBasedList2		SUPPORT 기본 정보·위치Step 1 검색·Step 4
관광빅데이터 정보서비스locgoRegnVisitrDDListDATA 지역 방문자 통계Step 2 필터링 가중치

호출 시퀀스 (경복궁 입력 → 대안 3곳)
① GET /searchKeyword2?keyword=경복궁  → contentId: "126508"
② GET /tatsCnctrRateList?contentId=126508  → cnctrRate: 0.87 (혼잡 판정)
③ GET /areaBasedList1?contentId=126508  → 후보 50곳
④ PARALLEL /tatsCnctrRateList × 50  → 창덕궁:0.31 · 덕수궁:0.18 · 경희궁:0.11
⑤ GET /locgoRegnVisitrDDList  → 비수도권 +0.1 가중치
⑥ score = (1-cnctrRate)×0.5 + similarity×0.3 + (1/dist)×0.1 + bonus×0.1
   → Top 3: [창덕궁, 덕수궁, 경희궁]
