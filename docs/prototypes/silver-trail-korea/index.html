HOW IT WORKS

3단계로 완성되는
시니어 맞춤 여행

건강 상태 입력부터 트레일 매칭, 핸즈프리 오디오 가이드까지 — 시니어를 위한 모든 것.

01
🏃
건강 상태 +
체력 난이도 선택
가볍게 산책 / 보통 걷기 / 활발한 트레킹 3단계 선택. 무릎·관절 문제, 동반자(자녀·배우자) 정보 입력 → 맞춤 코스 자동 필터링.
02
🗺️
트레일 + 웰니스
코스 매칭
걷기 코스 중 시니어 적합 난이도 필터링 + 온천·산림욕·명상 등 건강 관광지 연동. 무장애 구간·편의시설 자동 확인.
03
🎧
AI 오디오 역사
해설 활성화
관광지 도착 시 자동으로 오디오 가이드 재생. 큰 음량·느린 속도 옵션. 터치 없이 귀로 듣는 "핸즈프리 투어" 경험.

무릎 관절 약해 긴 등산 불가. SilverTrail에서 경주 "실버 2" 코스 선택 → 3km 평지 중심 역사 탐방 코스 생성. 불국사 도착 후 오디오 가이드 자동 재생 — 해설사 없이도 깊은 역사 이해.
