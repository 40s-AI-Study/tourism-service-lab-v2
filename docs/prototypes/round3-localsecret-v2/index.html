§2 · What

서비스 개요

LocalSecret v2가 해결하는 문제와 핵심 가치

혼잡도 예측을 역활용해 아무도 없는 숨은 명소만 추천하는 오버투어리즘 해소형 여행 플랫폼. 발견의 설렘을 감정 JTBD로 재정의하여 기존 인기 순위 기반 추천의 한계를 넘어서고, 조용한 탐험가부터 미식모험가까지 다양한 여행 성향에 맞는 숨은 명소를 연결합니다. KTO 비수도권 관광 분산 정책과 자연스럽게 연계되는 사회적 가치 창출형 서비스입니다.

성향 스펙트럼

조용한 탐험가 사진작가 전통감성 미식모험가 혼자여행러

경쟁 서비스 비교

서비스 추천 방식 핵심 차별점 오버투어리즘 대응
LocalSecret v2 🔮 혼잡도 최하위 추천 감정 JTBD × 발견 경험 설계 핵심 가치
야놀자 인기 순위 기반 가격·예약 편의성 미대응

주요 기능

기능 01
🗝️
감정별 숨은 명소 추천
비밀·고요·정취·계절 4가지 감정 JTBD를 선택하면, 해당 감정에 맞는 숨은 명소 3곳을 즉시 추천합니다. 발견의 설렘을 감정 언어로 설계한 핵심 UX.
기능 02
📊
혼잡도 반전 알고리즘
일반 서비스가 "인기 많은 순위"를 보여줄 때, LocalSecret은 혼잡도 최하위 장소를 최상위로 정렬합니다. KTO 관광지 집중률 데이터 역활용.
기능 03
🔔
계절별 숨은 명소 타이밍 알림
"지금 이 순간 방문하면 가장 아름다운 곳"을 푸시 알림으로 전달. 사계절 데이터 기반으로 방문 최적 시기를 자동 감지합니다.

사용자 여정

📱
앱 열기
다크 미스터리 스타 맵 Hero Screen 진입. "오늘, 어떤 발견을 원해?" 질문으로 시작.
🎭
발견 감정 선택
비밀 / 고요 / 정취 / 계절 중 오늘의 기분에 맞는 감정 카드 선택.
🗝️
숨은 명소 3곳 추천
혼잡도 최하위 + 감정 매칭 알고리즘으로 선별된 아무도 모르는 3곳 카드로 표시.
⏰
혼잡도 최저 시간 알림
"오늘 오후 2-4시 방문 시 혼잡도 최저" 실시간 타이밍 가이드 제공.
🚶
방문
나만의 숨은 명소에서 발견의 설렘 경험. 진정한 로컬 감성 여행.
📷
기록 & 공유
방문 기록 저장 후 선택적 공유. 다른 탐험가와 숨은 명소 발견 커뮤니티 형성.
