## 🎯 우리 목표

**최우수상 (공사 사장상, 300만원) 이상** 목표

### 사용자 요구사항 반영
- **타겟**: 20-40대 한국인 + 외국인 (범용)
- **방법론**: 시뮬레이션 루프 기반 실제 니즈 검증
- **필수**: Visit Korea Open API 활용

## 📄 제안서 양식

- **파일**: PDF 10MB 미만
- **분량**: 5페이지 이내 (이미지, 표 포함)
- **글자**: 12포인트 이상
- **섹션 4개** (수정 금지, 전체 필수):
  1. 서비스 기획배경 및 필요성
  2. 서비스 개요 (차별성 강조)
  3. **데이터 활용 방안** (OpenAPI 필수)
  4. 서비스 발전 방향

## ✅ 성공 기준 (회사 내부)

1. **시뮬레이션 통과**: 5개 페르소나 중 4개 이상 "유용" 평가
2. **사업성 점수**: 7/10 이상
3. **API 활용도**: 최소 3개 Visit Korea API 조합
4. **제안서 완성도**: 심사 기준 4개 항목 각각 우수 수준

## 🚫 주요 제약

1. OpenAPI 필수 활용 (없으면 실격)
2. 동일 서비스 중복 지원 금지 (다른 부문 X)
3. 팀 최대 5명 (국내 거주 한국인)
4. 1차/최종 심사는 "완성된 서비스" 기준 (이번엔 제안서만 필요)
5. 저작권/초상권 책임은 지원자
