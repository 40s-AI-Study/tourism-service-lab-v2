# 09. Deliverables

> **Phase 3 Stage 5에서 최종 산출물 완성**

## 최종 제출물 목록

### 1. 서비스 기획서
- **위치**: `deliverables/service-plan.md`
- **작성자**: ProductManager
- **내용**:
  - 서비스명, 슬로건
  - 핵심 가치 제안
  - 타겟 사용자 (페르소나 요약)
  - 주요 기능 5-7개
  - 사용자 여정
  - 기대 효과

### 2. 와이어프레임
- **위치**: `deliverables/wireframes/`
- **작성자**: UXDesigner
- **내용**:
  - 메인 화면 (텍스트 기반 설명 + ASCII 다이어그램)
  - 핵심 플로우 3-5개
  - 컴포넌트 레이아웃
  - 인터랙션 설명

### 3. 사업 모델
- **위치**: `deliverables/business-model.md`
- **작성자**: BusinessAnalyst
- **내용**:
  - Executive Summary
  - SWOT 분석
  - 수익 모델
  - 시장 규모 / 목표 시장
  - 3년 로드맵
  - KPI / 지표

### 4. 기술 아키텍처
- **위치**: `deliverables/tech-architecture.md`
- **작성자**: APISpecialist
- **내용**:
  - 시스템 구조도 (ASCII)
  - 기술 스택 (FE/BE/DB/Infra)
  - API 활용 방안 (어떤 Visit Korea API를 언제 호출하는지)
  - 데이터 흐름
  - 배포 전략

### 5. API 활용 설계
- **위치**: `deliverables/api-usage.md`
- **작성자**: APISpecialist
- **내용**:
  - 사용 API 목록 (Visit Korea)
  - 각 API별 용도
  - 샘플 요청/응답
  - 조합 시나리오

### 6. 공모전 제출 양식 맞춤
- **위치**: `deliverables/submission/`
- **작성자**: CEO + ProductManager
- **내용**:
  - 공모전 요구 양식에 맞게 재구성
  - 제출 전 최종 리뷰

## 제출 전 체크리스트

- [ ] 모든 문서 작성 완료
- [ ] Visit Korea API 활용 3개 이상
- [ ] 시뮬레이션 5개 중 4개 이상 Pass
- [ ] 사업성 점수 7/10 이상
- [ ] 사용자 피드백 반영 완료
- [ ] 공모전 요구 양식 준수
- [ ] 제출 기한 확인
