# 08. Business Analysis

> **Phase 3 Stage 3-5에서 채워질 예정** — BusinessAnalyst 주도

## 분석 프레임워크

### 각 아이디어별 평가 항목
1. **시장 규모** (Market Size) — 1-10
   - 국내 관광 시장 + 외국인 관광객 시장
   - TAM / SAM / SOM 구분
2. **차별화** (Differentiation) — 1-10
   - 기존 서비스 대비 독특한 가치
3. **구현 난이도** (Feasibility) — 1-10 (역산)
   - API 조합 복잡도, 개발 공수
4. **수익 모델** (Revenue Model) — 1-10
   - 광고, 수수료, 구독, B2B 등
5. **경쟁 우위 지속성** (Moat) — 1-10
   - 데이터, 네트워크 효과, 전환 비용

**종합 점수 = 가중 평균**

### SWOT 분석 (최종 1개 선정 후)
- Strengths: 서비스만의 강점
- Weaknesses: 약점/리스크
- Opportunities: 시장 기회
- Threats: 외부 위협

### 수익 모델 옵션
- **B2C 광고 모델**: 관광지/음식점 노출 광고
- **B2C 프리미엄**: 고급 기능 구독
- **B2B 수수료**: 예약/구매 중개 수수료
- **B2B 데이터**: 관광 트렌드 인사이트 판매
- **정부 용역**: 지자체 관광 활성화 사업

### 시장 규모 추정
- **국내 관광 시장**: 연간 약 X조 원
- **외국인 관광 시장**: 연간 약 Y조 원
- **디지털 관광 서비스 점유율**: Z%

*(Phase 3 Stage 3에서 실제 숫자로 채움)*

## 최종 기획 시 포함

- 사업 계획서 (Executive Summary)
- 수익 모델 상세
- 3년 로드맵 (MVP → 베타 → 런칭)
- 주요 지표 (KPI): MAU, 세션, 전환율 등
- 공모전 심사 기준 매핑
