# 05. Market Research

> **Phase 1-4 및 Phase 3 Stage 1에서 채워질 예정**

## 수집 예정 데이터

### 한국 관광 트렌드 (최근 1-2년)
- 국내 관광 총 인구/횟수
- 인바운드 관광객 (외국인 방문 통계)
- 연령대별 관광 패턴 (20-40대 집중)
- 관광 형태별 (개별, 단체, 테마 등)

### 20-40대 관광 선호도
- 20대 vs 30대 vs 40대 차이
- 솔로 vs 커플 vs 가족
- 국내 vs 해외
- 선호 테마 (자연, 도시, 문화, 체험, 미식 등)

### 외국인 관광객 특성
- 국가별 점유율
- 체류 기간
- 주요 방문지
- 정보 탐색 채널

### 기존 관광 서비스 분석
- VisitKorea 공식 앱
- 국내 앱: 트리플, 야놀자, 여기어때, 마이리얼트립
- 해외 앱: Google Maps, Tripadvisor, Klook
- 각 서비스의 기능 매트릭스
- 사용자 리뷰에서 드러난 pain point

### 경쟁 gap 분석
- 기존 서비스가 커버하지 못하는 니즈
- 한국 특화 가능한 기능
- 외국인 관광객의 불편함

## 리서치 방법

- 웹 검색 (웹 search MCP)
- 공공 통계 (한국관광공사 통계)
- 앱스토어 리뷰 분석
- 블로그/커뮤니티 (네이버, 레딧 r/korea 등)
